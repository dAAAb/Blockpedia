# Bitcoin Ordinals
> **最後更新：2026 年 1 月**


## Ordinals 協議讓比特幣可以承載 NFT 和其他數位資產，為「數位黃金」帶來全新的應用場景，同時引發社群激烈爭論。

2023 年初，比特幣社群發生了一件大事。

有人在比特幣上「銘刻」了 NFT。

比特幣——那個「只能轉帳」的區塊鏈——突然可以放圖片、影片、甚至程式碼了。

這就是 **Ordinals**。

### Ordinals 是什麼？

Ordinals 是一個協議，讓你可以在比特幣上創建「銘文」（Inscriptions）。

**技術原理**

比特幣的最小單位是「聰」（Satoshi），1 BTC = 1 億聰。

Ordinals 做了兩件事：

1. **序數理論（Ordinal Theory）**：給每一個聰一個唯一的序號
2. **銘文（Inscriptions）**：把資料「刻」在特定的聰上

這樣，每一個聰都變得「獨特」，可以承載不同的內容。

### 銘文是什麼？

銘文是「刻」在聰上的資料。可以是：
- 圖片
- 文字
- 影片
- 音樂
- HTML
- JSON

**資料儲存方式**

銘文資料直接儲存在比特幣區塊鏈上（不是像很多 NFT 那樣存在 IPFS）。

這意味著：
- 永久性：只要比特幣存在，銘文就存在
- 不可變：一旦「刻」上去，無法修改
- 成本：佔用區塊空間，需要支付礦工費

### 創辦人

Ordinals 協議由 **Casey Rodarmor** 創建。

Casey 是一位比特幣開發者，曾在 Bitcoin Core 團隊工作。

他的理念是：比特幣不只是「數位黃金」，也可以是「數位文物」的載體。

### 發展歷程

**2023 年 1 月**：Ordinals 協議發布

**2023 年 2 月**：第一波銘文熱潮，比特幣網路擁堵

**2023 年 3 月**：[BRC-20](brc20.md) 代幣標準出現，進一步推高熱度

**2023 年底**：銘文數量突破數百萬

**2024 年**：[Runes](runes.md) 協議推出，作為 BRC-20 的改進

### 爭議

Ordinals 在比特幣社群引發激烈爭論：

**支持者觀點**

- 增加比特幣的用途
- 帶來新用戶和開發者
- 增加礦工收入
- 展示比特幣的可能性

**反對者觀點**

- 「污染」比特幣區塊空間
- 比特幣應該專注於貨幣功能
- 推高交易手續費
- 「垃圾資料」佔用寶貴空間

**技術層面**

反對者指出：
- 這些「非金融」交易佔用空間
- 普通用戶的轉帳變得更貴
- 全節點的儲存負擔增加

支持者回應：
- 用戶支付了手續費，有權使用區塊空間
- 這是自由市場
- 增加礦工收入有利於網路安全

### 與以太坊 NFT 的比較

| 面向 | Bitcoin Ordinals | Ethereum NFT |
|------|------------------|--------------|
| 資料儲存 | 完全上鏈 | 通常在 IPFS |
| 智能合約 | 無 | 有 |
| 交易成本 | 較高 | 變化大 |
| 生態系統 | 較新 | 成熟 |
| 版稅機制 | 無原生支持 | 有（但可繞過）|
| 安全性 | 比特幣級別 | 以太坊級別 |

**Ordinals 的優勢**
- 資料真正永久（完全上鏈）
- 比特幣的安全性和去中心化

**Ordinals 的劣勢**
- 沒有智能合約，功能有限
- 交易所和錢包支援不如以太坊成熟

### 技術細節

**SegWit 和 Taproot**

Ordinals 能夠運作，得益於比特幣的兩次升級：

- **SegWit**（2017）：增加區塊空間
- **Taproot**（2021）：允許更複雜的腳本

這些升級讓銘文「塞進」比特幣區塊成為可能。

**銘文大小**

- 理論上最大約 400KB
- 實際上大多數銘文較小
- 大型銘文需要高額手續費

### 對比特幣的影響

Ordinals 對比特幣生態的影響：

**1. 礦工收入增加**

銘文帶來的手續費，在某些時期佔礦工收入很大比例。

這對比特幣安全有正面影響（特別是在區塊獎勵減半後）。

**2. 區塊空間競爭**

銘文和普通交易競爭區塊空間。繁忙時期，手續費飆升。

**3. 新用戶進入**

很多原本對比特幣不感興趣的 NFT 玩家，因為 Ordinals 進入比特幣生態。

**4. 生態系統擴展**

圍繞 Ordinals 出現了新的錢包、市場、開發工具。

### 衍生協議

Ordinals 催生了其他協議：

**BRC-20**
- 在 Ordinals 基礎上的代幣標準
- 允許在比特幣上發行代幣
- 詳見 [BRC-20](brc20.md)

**Runes**
- Casey Rodarmor 推出的新代幣標準
- 比 BRC-20 更高效
- 詳見 [Runes](runes.md)

### 未來展望

Ordinals 的未來可能包括：

**1. 技術改進**

- 更高效的銘文方式
- 更好的索引和查詢

**2. 應用場景**

- 數位藝術
- 收藏品
- 身分憑證
- 遊戲資產

**3. 與 Layer 2 整合**

- 在 Lightning Network 上交易銘文
- 降低成本和延遲

**4. 監管關注**

- 如果 Ordinals 大規模使用，可能引起監管關注

{% hint style="info" %}
Ordinals 是比特幣的「意外創新」。它不是比特幣設計的初衷，但展示了區塊鏈作為資料層的可能性。
{% endhint %}

### 如何參與？

如果你想嘗試 Ordinals：

1. 下載支援 Ordinals 的錢包（如 Xverse、Unisat）
2. 存入一些比特幣
3. 在市場（如 Magic Eden）瀏覽和購買銘文
4. 或者，使用銘文工具自己創建銘文

注意：交易手續費可能很高，特別是在網路繁忙時。
