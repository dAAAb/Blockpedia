# x402 Protocol

## x402 是 Coinbase 於 2025 年推出的 HTTP 原生支付協議，讓網站和 API 可以直接向 AI 收費。

名字很奇怪對吧？x402。其實這個名字藏著一個網路歷史的小彩蛋。

### 402：沉睡三十年的狀態碼

如果你有一點網頁開發經驗，應該知道 HTTP 狀態碼：

- **200** — OK，一切正常
- **404** — Not Found，找不到頁面
- **500** — Server Error，伺服器炸了

但你可能沒聽過 **402 Payment Required**。

這個狀態碼在 1990 年代 HTTP 規範制定時就被保留了，原意是「需要付款才能存取」。但因為當時沒有好用的網路支付方案，這個狀態碼就這樣閒置了將近 30 年，一直等待著屬於它的時代。

2025 年，Coinbase 決定喚醒它。

### 讓網站直接收費

想像你經營一個提供 AI 服務的網站，比如說圖片生成、文字翻譯、資料分析。

傳統做法是：用戶先註冊帳號、綁定信用卡、購買點數或訂閱，然後才能使用服務。流程繁瑣，而且每次都要經過支付公司抽成。

x402 的做法完全不同：

```
用戶（或 AI）發送請求 →
伺服器回傳 402 狀態碼和價格 →
用戶付款（穩定幣）→
伺服器確認收到 →
回傳服務結果
```

整個過程可以在**一秒內完成**，不需要註冊、不需要信用卡、不需要中間商。

### 為什麼這對 AI 很重要？

回到我們在「[AI x Web3 總覽](README.md)」說的場景：AI 助理需要呼叫各種外部服務。

有了 x402，AI 可以：

1. 發現一個有用的 API
2. 收到 402 回應，知道價格是 0.001 USDC
3. 自動從錢包付款
4. 取得服務結果

整個過程完全自動化，不需要人類介入。這就是所謂的「AI Agent 自主支付」。

### x402 的技術架構

x402 協議包含幾個關鍵元件：

| 元件 | 功能 |
|------|------|
| HTTP Header | 在標準 HTTP 請求中加入支付資訊 |
| 價格發現 | 伺服器告訴客戶端服務價格 |
| 支付驗證 | 確認鏈上交易已完成 |
| 支援幣種 | 主要是 USDC，運行在 Base 鏈上 |

{% hint style="info" %}
為什麼選 Base 鏈？因為 Base 是 Coinbase 推出的 Layer 2，交易費用極低（通常不到 0.01 美元），適合頻繁的小額支付。
{% endhint %}

### 重量級合作夥伴

x402 不只是 Coinbase 自己在玩。2025 年 9-10 月，一系列重量級公司宣布支援：

- **Cloudflare** — 全球最大的 CDN 之一，讓任何網站都能輕鬆整合 x402
- **Google** — 推出 AP2 協議，與 x402 互通
- **Visa** — 推出 TAP（Token Allowance Protocol）
- **AWS** — 在 Amazon Bedrock AI 服務中整合 x402
- **Anthropic** — Claude AI 的開發商，支援 AI Agent 支付

這些合作讓 x402 的交易量在短短兩個月內**暴增 1,000%**。

### 實際應用場景

x402 正在被用於：

**1. AI API 計費**
- 圖片生成 AI：每張圖 0.01 USDC
- 翻譯 API：每 1000 字 0.005 USDC
- 資料分析：按運算複雜度收費

**2. 內容付費**
- 新聞網站：單篇文章 0.10 USDC
- 研究報告：一份 1.00 USDC
- 不需要訂閱，看多少付多少

**3. 機器對機器交易**
- AI Agent A 呼叫 AI Agent B 的服務
- 完全自動化，無人值守

### 對網路的影響

x402 可能改變整個網路的商業模式。

現在的網路主要靠兩種方式賺錢：
1. **廣告** — 免費使用，但要看廣告
2. **訂閱** — 付月費，無限使用

x402 提供第三種選擇：
3. **按次付費** — 用多少付多少，即時結算

這對內容創作者特別有意義。不需要百萬訂閱才能賺錢，每個人都可以為自己的內容標價，讓市場決定價值。

### 風險與挑戰

當然，x402 也面臨一些挑戰：

1. **用戶習慣** — 大家習慣免費或訂閱，按次付費需要適應
2. **錢包門檻** — 用戶需要有加密貨幣錢包和 USDC
3. **監管不確定性** — 各國對加密支付的態度不同
4. **隱私問題** — 鏈上交易是公開的，支付記錄可被追蹤

{% hint style="info" %}
x402 的「x」代表「experimental」（實驗性），表示這個標準還在發展中，可能會有變動。
{% endhint %}
