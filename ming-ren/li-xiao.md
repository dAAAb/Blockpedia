# 李笑來

### 李笑來是中國幣圈的「傳奇」——不管是好的還是壞的那種。他是最早期的比特幣投資者之一，聲稱持有六位數的比特幣。但他也因為「割韭菜」錄音洩露而成為幣圈最具爭議的人物。他的故事，某種程度上就是中國幣圈早期野蠻生長的縮影。

---

## 李笑來是誰？

| 基本資料 | |
|----------|--|
| 出生 | 1972 年，中國吉林 |
| 前職業 | 新東方英語老師 |
| 著作 | 《把時間當作朋友》《通往財富自由之路》|
| 幣圈身份 | 早期比特幣投資者、ICO 項目方 |

## 李笑來的故事

### 英語老師到比特幣

```
李笑來的轉型：

新東方英語老師（成名）
    ↓
寫作《把時間當作朋友》（暢銷書）
    ↓
2011 年發現比特幣
    ↓
大量買入，聲稱持有 10 萬+ BTC
    ↓
成為中國「比特幣首富」
```

| 時期 | 身份 |
|------|------|
| 2000s | 新東方英語老師、暢銷書作家 |
| 2011-2013 | 比特幣早期投資者 |
| 2017-2018 | ICO 項目方、投資機構 |
| 2018 後 | 低調、偶爾發聲 |

{% hint style="info" %}
**「首富」身份存疑**

李笑來自稱持有六位數比特幣，但從未提供證明。幣圈對此說法一直存疑。
{% endhint %}


### 錄音洩露醜聞

2018 年 7 月，一段李笑來的私下談話錄音洩露，震驚幣圈：

| 錄音內容 | 原話（節選）|
|----------|-----------|
| 對韭菜的看法 | 「傻X 的共識也是共識」|
| 對項目的看法 | 「說實話做得最牛的一件事是帥初」|
| 對投資者 | 暗示散戶就是被收割的對象 |

{% hint style="danger" %}
**錄音的影響**

這段錄音讓「韭菜」一詞成為幣圈流行語，也讓很多人第一次聽到項目方對散戶的真實看法。
{% endhint %}

## 李笑來說的一些話

他的一些言論值得反思：

```
「傻X 的共識也是共識」
— 意思是，即使人們基於錯誤的理由達成共識，這個共識仍然有市場價值

「價值投資者是傻X」
— 在幣圈，短期炒作往往比長期持有更賺錢

「做網紅比做項目重要」
— 個人品牌和行銷比技術更重要
```

{% hint style="warning" %}
**這些話的意義**

李笑來的話雖然刺耳，但反映了幣圈的某些現實。投資者應該保持清醒，了解市場的真實運作方式。
{% endhint %}
