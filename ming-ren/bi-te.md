# 比特幣耶穌

### Roger Ver 是「比特幣耶穌」——這不是恭維，而是早年他對比特幣傳教般狂熱的寫照。他是最早看到比特幣潛力的人之一，曾傾盡全力推廣。但後來，他因為支持 Bitcoin Cash 而與比特幣社群決裂，最終在 2024 年因稅務問題被美國起訴。一個傳奇的起落。

---

| 國籍 | 放棄美國籍，成為聖基茨公民 |
| 身份 | 早期比特幣投資者、Bitcoin Cash 支持者 |

## Roger Ver 的故事

### 早期：比特幣傳教士

```
2011 年：
- 發現比特幣，認為這是「有史以來最重要的發明」
- 投資 100 萬美元購買比特幣
- 投資 BitInstant、Blockchain.info、Kraken 等公司
- 到處演講推廣比特幣
```

他被稱為「比特幣耶穌」是因為：

| 行為 | 說明 |
|------|------|
| 狂熱推廣 | 到處演講、贈送比特幣 |
| 早期投資 | 投資了許多重要的比特幣公司 |
| 傳教風格 | 宣傳方式像在傳播宗教 |

{% hint style="info" %}
**早期貢獻不可否認**

無論後來發生了什麼，Roger Ver 在比特幣早期的推廣功不可沒。他是第一批真正相信比特幣會改變世界的人。
{% endhint %}

---

### 分裂：Bitcoin Cash 之爭

2017 年，比特幣社群因「擴容」問題分裂：

```
比特幣擴容之爭：

比特幣核心派（BTC）：
- 維持 1MB 區塊大小
- 用 Layer 2（閃電網絡）擴容
- 強調「數位黃金」定位

大區塊派（BCH）：
- 增加區塊大小
- 鏈上直接擴容
- 強調「點對點現金」願景

Roger Ver → 支持大區塊派 → Bitcoin Cash
```

| 事件 | 時間 |
|------|------|
| 比特幣擴容爭論 | 2015-2017 |
| Bitcoin Cash 分叉 | 2017 年 8 月 |
| Roger Ver 全力支持 BCH | 2017 年起 |
| Bitcoin Cash 再分叉 | 2018 年（BSV 分裂）|

{% hint style="warning" %}
**「真正的比特幣」之爭**

Roger Ver 曾多次聲稱 Bitcoin Cash 才是「真正的比特幣」，引發比特幣社群強烈反彈。這場爭論至今仍是幣圈最大的分歧之一。
{% endhint %}

---

### 落幕：稅務起訴

2024 年，Roger Ver 的故事有了戲劇性轉折：

| 時間 | 事件 |
|------|------|
| 2024 年 4 月 | 在西班牙被捕 |
| 起訴內容 | 逃稅超過 4800 萬美元 |
| 指控 | 放棄美國籍時隱瞞比特幣資產 |
| 現況 | 等待引渡回美國受審 |

```
起訴背景：

2014 年：Roger Ver 放棄美國國籍
美國法律：放棄國籍時需繳納「離境稅」
問題所在：Ver 據稱低報了比特幣資產價值
結果：被指控逃稅超過 4800 萬美元
```

## 如何評價 Roger Ver？

### 支持者的觀點

| 觀點 | 說明 |
|------|------|
| 早期功臣 | 沒有他，比特幣可能不會這麼快普及 |
| 堅持願景 | 堅持比特幣「點對點現金」的初心 |
| 敢於挑戰 | 不怕與主流意見對抗 |

| 觀點 | 說明 |
|------|------|
| 分裂社群 | BCH 分叉削弱了比特幣 |
| 誤導新人 | bitcoin.com 讓新手混淆 BTC 和 BCH |

## 對投資者的啟示

{% hint style="danger" %}
**教訓**

1. **早期功臣不代表永遠正確**：過去的貢獻不保證未來的判斷
2. **社群分裂的代價**：BCH 和 BTC 的分裂讓雙方都付出代價
3. **稅務問題很嚴重**：加密貨幣不代表可以逃稅
{% endhint %}
