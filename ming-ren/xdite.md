# xDite

### xDite（鄭伊廷）是台灣科技圈的傳奇人物——在幣圈更是如此。她從知名的 Ruby 工程師、程式教學創業家，跨足到加密貨幣領域創辦 OTCBTC。她的故事是台灣幣圈 2017-2018 年熱潮的縮影，也反映了創業者在牛熊轉換中的起落。

### 幣圈時期

2017 年，xDite 進入加密貨幣領域：

| 時間 | 事件 |
|------|------|
| 2017 年 | 開始關注加密貨幣 |
| 2017 年 | 創辦 OTCBTC（場外交易平台）|
| 2017-2018 | 發行 OTB 代幣 |
| 2018 年 | 熊市來臨，爭議開始 |

### OTCBTC

```
OTCBTC 是什麼：

OTC = Over-The-Counter（場外交易）
BTC = Bitcoin

功能：讓用戶直接買賣加密貨幣，類似拍賣平台
定位：解決中國用戶在交易所關閉後的交易需求
```

| 特色 | 說明 |
|------|------|
| P2P 交易 | 用戶對用戶直接交易 |
| 支援法幣 | 人民幣、台幣等 |
| 多幣種 | BTC、ETH 等主流幣 |
| 評價系統 | 類似淘寶的信用評價 |

## 爭議

### OTB 代幣爭議

| 爭議點 | 說明 |
|----------|------|
| 代幣用途 | OTB 代幣的實際用途被質疑 |
| 價格波動 | 代幣價格大幅下跌 |
| 承諾兌現 | 部分功能未如期實現 |

### 熊市中的選擇

2018 年加密寒冬中：

```
市場環境：
- 加密貨幣整體暴跌 90%
- 用戶流失
- 營收下降

爭議焦點：
- 項目方的責任範圍
- 投資者的期望管理
- 創業失敗 vs 惡意行為的界線
```

{% hint style="warning" %}
**公平看待**

xDite 的案例反映了 2017-2018 年很多幣圈項目的處境。在牛市中募資、在熊市中掙扎，這是那個時代的普遍現象。評價時需要考慮整體市場環境。
{% endhint %}
