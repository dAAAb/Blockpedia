# 小查

### Charlie Lee（小查）創造了萊特幣——比特幣的「白銀」。他是 MIT 畢業的工程師，曾在 Google 工作，後來創辦了萊特幣。但他最著名的事蹟可能是在 2017 年幣價高點時賣掉了所有持有的 LTC。這讓他成為幣圈最有爭議的創辦人之一。

## Charlie Lee 的故事

### 創造萊特幣

```
2011 年：萊特幣誕生

Charlie Lee 在 Google 工作
    ↓
發現比特幣，被技術吸引
    ↓
認為比特幣「太慢、太貴」
    ↓
分叉比特幣代碼，創造萊特幣
    ↓
「比特金，萊特銀」概念誕生
```

| 萊特幣 vs 比特幣 | 萊特幣 | 比特幣 |
|-----------------|--------|--------|
| 出塊時間 | 2.5 分鐘 | 10 分鐘 |
| 總供應量 | 8,400 萬 | 2,100 萬 |
| 挖礦算法 | Scrypt | SHA-256 |
| 定位 | 日常支付 | 價值儲存 |

{% hint style="info" %}
**「比特金，萊特銀」**

Charlie Lee 將萊特幣定位為比特幣的補充而非競爭者。比特幣是數位黃金，萊特幣是數位白銀。這個定位讓萊特幣避免了直接和比特幣競爭。
{% endhint %}

### 爭議：高點清倉

2017 年 12 月，Charlie Lee 做了一件震驚幣圈的事：

```
時間線：

2017 年 12 月：LTC 價格接近 $350 歷史新高
2017 年 12 月 20 日：Charlie Lee 宣布賣出所有 LTC
理由：「避免利益衝突」
結果：LTC 隨後暴跌超過 90%
```

| 反應 | 觀點 |
|------|------|
| 支持者 | 他有權處置自己的資產 |
| 批評者 | 創辦人在高點跑路，不負責任 |
| 質疑者 | 他是否知道內部資訊？ |

{% hint style="danger" %}
**創辦人清倉的影響**

Charlie Lee 的清倉對萊特幣的信任造成了永久傷害。很多人認為，如果創辦人自己都不持有，為什麼別人要持有？
{% endhint %}

---

## Charlie Lee 的辯護

他在賣出時給出的理由：

| 理由 | 說明 |
|------|------|
| 利益衝突 | 作為創辦人，發推文會影響幣價 |
| 專注開發 | 不想讓個人財富干擾工作 |
| 透明度 | 公開宣布而非偷偷賣 |
| 繼續貢獻 | 賣出後仍持續參與萊特幣開發 |
