# 老查

### Charles Hoskinson（老查）是幣圈的「學術派」。他參與創辦了以太坊，又因理念不合離開創立 Cardano。他以每天直播幾小時聞名，討論從哲學到歷史再到數學的各種話題。有人視他為先知，有人認為他只是會吹牛。無論如何，他確實影響了加密貨幣的發展。

## Charles Hoskinson 的故事

### 以太坊時期

```
2013-2014 年：以太坊創辦

Vitalik Buterin 發表以太坊白皮書
    ↓
召集團隊，包括 Charles Hoskinson
    ↓
Charles 擔任 CEO（約 6 個月）
    ↓
因營利 vs 非營利路線爭議
    ↓
Charles 離開以太坊
```

| 爭議點 | Charles 立場 | Vitalik 立場 |
|--------|-------------|-------------|
| 組織形式 | 營利公司 | 非營利基金會 |
| 融資方式 | 接受 VC 投資 | 社群眾籌 |
| 發展方向 | 商業化 | 開源去中心化 |

{% hint style="info" %}
**永遠的遺憾？**

Charles 離開以太坊是幣圈歷史上最重要的分道揚鑣之一。如果他留下來，以太坊和 Cardano 的歷史都會完全不同。
{% endhint %}

### Cardano 時期

2015 年，Charles 創立 IOHK（Input Output Hong Kong），並開始開發 Cardano：

| 時間 | 事件 |
|------|------|
| 2015 年 | 創立 IOHK |
| 2017 年 | Cardano 主網上線 |
| 2020 年 | Shelley 升級（去中心化）|
| 2021 年 | Alonzo 升級（智能合約）|
| 2023 年 | Voltaire 時代（治理）|

```
Cardano 的特色：

學術研究 → 同行評審 → 形式化驗證 → 實現

與其他項目的區別：
- 其他項目：先做再說
- Cardano：先研究、證明、再實現
```

## Cardano 的評價

### 支持者的觀點

| 觀點 | 說明 |
|------|------|
| 學術嚴謹 | 每個功能都有論文支持 |
| 長期思維 | 不追求短期炒作 |
| 可持續 | 設計考慮長期發展 |

### 批評者的觀點

| 觀點 | 說明 |
|------|------|
| 發展太慢 | 智能合約花了 4 年才上線 |
| 生態不足 | DApp 和 DeFi 遠少於競爭對手 |

## 對投資者的啟示

```
投資 Cardano 的考量：

優點：
- 長期主義思維
- 學術背景和嚴謹開發
- 活躍的創辦人

缺點：
- 發展速度慢
- 生態系統較小
- 創辦人爭議性高
```

{% hint style="info" %}
**DYOR**

Charles 是個複雜的人物。他的支持者和批評者都很極端。投資前，建議親自看他的直播和讀 Cardano 的論文，形成自己的判斷。
{% endhint %}
