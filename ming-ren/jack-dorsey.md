# Jack Dorsey

## Jack Dorsey 是 Twitter 共同創辦人，也是支付公司 Block（原 Square）的 CEO。他是科技圈最狂熱的比特幣信徒，相信比特幣將成為「網際網路的原生貨幣」。

如果你在矽谷問：「哪個科技大佬最愛比特幣？」

答案幾乎一定是 Jack Dorsey。

不是投資組合裡放一點的那種愛，是把整間公司的未來都押上去的那種愛。

### 從推特到支付

Jack Dorsey 有兩個身分：

1. **Twitter 共同創辦人**（2006 年創立，2022 年更名為 X）
2. **Block CEO**（原名 Square，2009 年創立）

Twitter 讓他成名，但 Block 才是他真正的舞台。

Block 旗下有：
- **Square**：商家支付服務
- **Cash App**：個人支付 App（美國超流行）
- **Spiral**：專注比特幣開發
- **TIDAL**：音樂串流平台
- **TBD**：去中心化網路開發

看出來了嗎？這些業務有一個共同的方向：**支付**。

### 為什麼這麼愛比特幣？

2018 年，Jack 在 Twitter 上說：

> 「比特幣將在十年內成為世界的單一貨幣。」

這不是隨便說說。他把這個信念變成了行動：

**2020 年 10 月**：Square 買入 5,000 萬美元比特幣
**2021 年 2 月**：再買 1.7 億美元
**2021 年**：Cash App 讓用戶能直接買賣比特幣
**2021 年**：宣布建造比特幣硬體錢包
**2022 年**：公司改名為「Block」，暗示區塊鏈

為什麼這麼狂熱？

Jack 的邏輯很簡單：

> 「網際網路需要原生的貨幣。Email 是網際網路的原生通訊，HTTP 是原生協議。但錢呢？我們還在用 1970 年代的信用卡系統。比特幣是第一個真正屬於網際網路的貨幣。」

### 比特幣極簡主義者

在加密貨幣圈，有一個詞叫「Bitcoin Maximalist」（比特幣極簡主義者）。

這些人認為：
- 比特幣是唯一有價值的加密貨幣
- 其他都是「垃圾幣」（shitcoins）
- 以太坊？騙局。NFT？泡沫。DeFi？賭場。

Jack Dorsey 就是 Bitcoin Maximalist 的代表人物。

他不投資以太坊，不玩 NFT，不參與 DeFi。

「我只相信比特幣。」

這讓他在加密貨幣圈既受愛戴又受爭議。比特幣信徒視他為英雄，其他陣營則覺得他太偏執。

### Block 的比特幣計畫

Jack 不只是嘴上說說，Block 有完整的比特幣佈局：

| 計畫 | 內容 |
|------|------|
| **Cash App** | 美國人買賣比特幣最簡單的方式 |
| **Spiral** | 開源比特幣開發團隊 |
| **硬體錢包** | 自己做冷錢包，讓比特幣更安全 |
| **比特幣挖礦** | 2022 年宣布開發挖礦晶片 |
| **[Lightning Network](../bitcoin/lightning.md)** | 大力支持閃電網路支付 |

特別是 [Lightning Network](../bitcoin/lightning.md)——這是比特幣的「第二層」，讓小額支付變得更快更便宜。

Jack 認為這是比特幣成為「日常支付工具」的關鍵。

### 與 Elon Musk 的友誼

科技圈兩大比特幣信徒，Jack Dorsey 和 Elon Musk，關係很好。

2021 年，他們一起參加了一場名為「The B Word」的比特幣線上論壇，互相吹捧比特幣的未來。

但兩人風格很不同：

| | Jack Dorsey | Elon Musk |
|--|-------------|-----------|
| 立場 | 只愛比特幣 | 比特幣、狗狗幣都行 |
| 風格 | 低調、技術派 | 高調、迷因派 |
| 行動 | 建設基礎設施 | 發推文影響市場 |

Jack 是工程師思維，想要「建造」東西。
Elon 是行銷天才，想要「推動」敘事。

### 離開 Twitter

2021 年 11 月，Jack 宣布辭去 Twitter CEO。

「我想專注在 Block 和比特幣。」

這個決定讓很多人驚訝，但對了解他的人來說，這是意料之中。Twitter 對他來說已經變成「分心的事物」，比特幣才是他的使命。

後來 Elon Musk 買下 Twitter，把它改名為 X。Jack 似乎對此很滿意。

### Jack 的願景

Jack 心中的未來是這樣的：

1. 比特幣成為全球儲備貨幣
2. 閃電網路讓日常支付變得可行
3. 每個人都有自己的比特幣錢包
4. 不需要銀行、不需要政府批准

這聽起來像烏托邦嗎？也許吧。

但 Jack 正在用 Block 的資源，一步一步把這個願景變成現實。

### Jack Dorsey 語錄

> 「比特幣改變了一切，這是我一生中見過最重要的事。」

> 「我希望比特幣能為世界帶來和平。」

> 「網際網路需要原生的貨幣，比特幣就是那個貨幣。」

{% hint style="info" %}
Jack Dorsey 代表了一種純粹的比特幣信仰。在一個充滿炒作和投機的市場裡，他選擇專注於建設基礎設施，讓比特幣真正成為可用的支付工具。不管你是否認同他的極端立場，他的執著和行動力都令人印象深刻。
