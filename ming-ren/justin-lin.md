# Justin Lin

### Justin Lin（林之晨）是台灣創投圈的代表人物。他創辦的 AppWorks 是東南亞最大的創業加速器之一，投資了許多區塊鏈和 Web3 新創。他本身也是區塊鏈技術的倡導者，經常在公開場合分享對 Web3 未來的看法。

## Justin Lin 的 Web3 觀點

他經常分享的觀點包括：

### 對 Web3 的看法

```
Web3 的本質：

Web1 = 只讀（Read）
Web2 = 讀寫（Read-Write）
Web3 = 讀寫擁有（Read-Write-Own）

關鍵變化：
- 用戶擁有自己的數據
- 去中心化的身份和資產
- 創作者經濟的崛起
```

### 對台灣的建議

| 建議 | 說明 |
|------|------|
| 監管明確化 | 需要清楚的法規框架 |
| 人才培育 | 支持區塊鏈技術教育 |
| 國際連結 | 與全球 Web3 生態接軌 |
