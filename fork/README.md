# 分岔
> **最後更新：2026 年 1 月**


**分岔（Fork）是區塊鏈程式碼被修改後，導致新舊版本平行共處的狀態。**依分岔後新舊版本的相容與否分為硬分岔和軟分岔。

_A "fork," in programming terms, is an open-source code modification._

分岔的起因是更改開源程式碼，通常更改後會在開發者間造成歧異，導致新舊兩個版本平行共處。
