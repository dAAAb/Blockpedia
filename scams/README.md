# 常見詐騙手法
> **最後更新：2026 年 1 月**


## 加密貨幣世界充滿機會，但也充滿陷阱。了解常見詐騙手法是保護自己的第一步。

在區塊鏈的世界裡，有一句話：

> **「Not your keys, not your coins.」**

這句話的另一面是：**一旦被騙，無法追回。**

區塊鏈交易不可逆轉。沒有客服、沒有退款、沒有後悔藥。

這讓它成為詐騙犯的天堂。

### 為什麼加密貨幣詐騙這麼多？

**1. 匿名性**

詐騙犯可以隱藏身份。

**2. 不可逆轉**

一旦轉出，無法追回。

**3. 跨境執法困難**

詐騙犯可能在任何國家。

**4. 新手不熟悉**

很多人不了解區塊鏈運作方式。

**5. FOMO 心理**

怕錯過暴富機會的心理被利用。

### 常見詐騙類型

| 類型 | 手法 | 危險程度 |
|------|------|----------|
| [假錢包 APP](fake-wallet.md) | 竊取私鑰/助記詞 | 極高 |
| [釣魚網站](phishing.md) | 偽造官網騙取授權 | 極高 |
| [空投詐騙](airdrop-scam.md) | 假空投誘導授權 | 高 |
| [龐氏騙局](ponzi.md) | 高報酬吸引投資 | 高 |

### 詐騙的共同特徵

**1. 太好的報酬**

- 「保證日報酬 1%」
- 「年化 1000%」
- 「穩賺不賠」

**正常投資沒有保證報酬。**

**2. 製造緊迫感**

- 「限時優惠」
- 「名額有限」
- 「錯過不再」

**讓你沒時間思考。**

**3. 要求私鑰/助記詞**

任何要求你提供私鑰或助記詞的，100% 是詐騙。

**永遠不要給任何人你的助記詞。**

**4. 模仿官方**

- 假的官網（網址差一個字母）
- 假的客服
- 假的空投公告

**永遠從官方管道確認。**

### 如何保護自己

**1. 永遠不分享助記詞**

沒有任何正當理由需要你的助記詞。

**2. 檢查網址**

- 確認是官方網址
- 注意相似字母（如 0 和 O、l 和 I）
- 使用書籤訪問常用網站

**3. 小額測試**

大額操作前，先用小額測試。

**4. 獨立驗證**

- 不要點擊陌生連結
- 從官方管道確認資訊
- 詢問社群其他人

**5. 使用硬體錢包**

硬體錢包可以防止很多攻擊。

**6. 保持懷疑**

如果聽起來太好，那可能就是假的。

### 被騙了怎麼辦

**1. 立即行動**

- 如果還沒確認，嘗試取消交易
- 撤銷相關授權
- 轉移剩餘資產到新錢包

**2. 記錄證據**

- 截圖對話記錄
- 保存交易哈希
- 記錄詐騙方式

**3. 通報**

- 向社群警告他人
- 向相關平台舉報
- 必要時報警

**4. 接受教訓**

很遺憾，大部分情況下資產無法追回。但這是寶貴的經驗。

{% hint style="warning" %}
**記住**：在區塊鏈世界，你是自己的銀行。這意味著你也要承擔銀行級別的安全責任。沒有人會來拯救你——除了你自己。
{% endhint %}
