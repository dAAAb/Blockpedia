# 日本加密法規
> **最後更新：2026 年 1 月**


## 日本是全球最早對加密貨幣進行全面監管的國家之一，從 Mt. Gox 事件的教訓中建立了嚴格的交易所監管制度。

2014 年，全球最大的比特幣交易所 Mt. Gox 在東京宣布破產。

85 萬枚比特幣消失。

這場災難發生在日本，也促使日本成為全球最早建立加密法規的國家。

### 日本加密監管的起源

Mt. Gox 事件震驚了日本政府。

一個在日本運營的交易所，讓全球投資者損失慘重，卻沒有任何法規監管它。

日本的反應不是禁止加密貨幣，而是**建立監管框架**。

2016 年，日本修訂《資金結算法》，正式將加密貨幣納入監管。

日本成為全球第一個賦予比特幣「法律地位」的主要經濟體。

### 監管框架

日本的加密監管主要基於兩部法律：

**資金結算法（Payment Services Act）**

- 定義「加密資產」
- 規範加密資產交換業者
- 反洗錢要求

**金融商品交易法（FIEA）**

- 規範加密資產衍生品
- 證券型代幣（STO）
- 投資者保護

**監管機構**

- **FSA（金融廳）**：主要監管機構
- **JFSA（日本金融服務局）**：執行機構

### 加密資產交換業者

日本要求加密交易所必須註冊為「加密資產交換業者」。

**註冊要求**

1. **資本要求**
   - 最低 1000 萬日圓（約 250 萬台幣）

2. **客戶資產分離**
   - 必須將客戶資產與公司資產分開
   - 使用冷錢包存儲大部分資產

3. **網路安全**
   - 必須有適當的安全措施
   - 定期安全審計

4. **反洗錢**
   - KYC 要求
   - 可疑交易報告

5. **信息披露**
   - 向客戶披露風險
   - 費用透明

**註冊交易所**

日本有約 30 家註冊的加密資產交換業者，包括：
- bitFlyer（最大）
- Coincheck
- GMO Coin
- DMM Bitcoin
- Rakuten Wallet
- LINE BITMAX

### JVCEA 自律組織

2018 年，日本成立了 **JVCEA**（日本虛擬通貨交換業協會）。

**功能**

- 行業自律組織
- 制定行業規範
- 審核新代幣上架
- 處理投訴

**代幣上架**

在日本，交易所上架新代幣需要經過 JVCEA 審核。這導致：
- 上架速度比其他國家慢
- 可交易的幣種較少
- 但安全性較高

很多人批評這太保守，但也減少了詐騙幣。

### 穩定幣規定

2023 年，日本修訂法規，允許銀行和信託公司發行穩定幣。

**發行者限制**

- 只有銀行、信託公司、資金移動業者可以發行
- 非金融機構不能發行

**儲備要求**

- 足額儲備
- 資產分離

**日圓穩定幣**

多個機構計劃發行日圓穩定幣：
- MUFG（三菱 UFJ）
- SMBC（三井住友）
- Binance Japan（與合作夥伴）

日本希望有本國的穩定幣，而不是只依賴美元穩定幣。

### 稅務

日本的加密稅務相對複雜：

**稅率**

- 加密收益被視為「雜項所得」
- 累進稅率：15% 到 55%
- 這比股票的資本利得稅（約 20%）高很多

**申報**

- 年收入超過 20 萬日圓需要申報
- 每筆交易都需要記錄

**問題**

高稅率被認為是日本加密發展的障礙。很多人呼籲改革：
- 調降稅率
- 改為分離課稅（如股票）
- 但改革進展緩慢

### NFT 與 Web3

日本政府積極推動 NFT 和 Web3：

**2022 年 Web3 戰略**

日本政府提出 Web3 發展戰略：
- 稅制改革（部分）
- DAO 法律地位
- NFT 促進

**NFT 應用**

日本在 NFT 方面有獨特優勢：
- 動漫、遊戲 IP
- 偶像文化
- 收藏品市場

很多日本 IP 進入 NFT 領域。

**挑戰**

但日本 Web3 發展仍有挑戰：
- 稅制不友善
- 監管保守
- 人才外流

很多日本加密企業選擇在新加坡、杜拜設立總部。

### 重大事件

日本加密歷史上的重大事件：

**2014：Mt. Gox 破產**

- 損失 85 萬 BTC
- 促成監管改革

**2018：Coincheck 被駭**

- 損失約 5 億美元的 NEM
- 加強安全要求

**2021：Liquid 被駭**

- 損失約 9000 萬美元
- 進一步加強安全監管

這些事件讓日本監管越來越嚴格。

### 與其他國家的比較

| 面向 | 日本 | 美國 | 新加坡 |
|------|------|------|--------|
| 法律地位 | 明確 | 不明確 | 明確 |
| 交易所監管 | 嚴格 | 分散 | 嚴格 |
| 代幣上架 | 需審核 | 相對自由 | 較自由 |
| 穩定幣 | 限金融機構 | 立法中 | 框架已建立 |
| 稅務 | 高（55%）| 中（~37%）| 低（0%）|

### 日本的優勢與劣勢

**優勢**

1. **法律確定性**
   - 早早建立框架
   - 企業知道規則

2. **投資者保護**
   - 嚴格的安全要求
   - 客戶資產保護

3. **大型市場**
   - 1.25 億人口
   - 高科技接受度

4. **文化資產**
   - 動漫、遊戲 IP
   - NFT 潛力大

**劣勢**

1. **稅制不利**
   - 最高 55% 的稅率
   - 複雜的申報

2. **保守監管**
   - 代幣上架慢
   - 創新受限

3. **人才外流**
   - 企業和人才往新加坡、杜拜移動

4. **反應慢**
   - DeFi、NFT 監管落後

### 未來展望

日本加密監管的可能發展：

**1. 稅制改革**

最被期待的改革。可能：
- 調降稅率
- 分離課稅

但進展緩慢。

**2. Web3 國家戰略**

政府繼續推動 Web3：
- DAO 法人地位
- 代幣發行便利化
- 國際合作

**3. CBDC**

日本央行在研究數位日圓，但態度謹慎。

**4. 國際協調**

與其他 G7 國家協調監管標準。

**5. 穩定幣發展**

日圓穩定幣可能在 2025-2026 年正式推出。

{% hint style="info" %}
日本是加密監管的先驅，但近年來被認為太保守。能否在「安全」與「創新」之間找到新平衡，是日本面臨的挑戰。
{% endhint %}
