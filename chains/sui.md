# Sui

## Sui 是使用 Move 語言的新一代公鏈，以物件導向設計和並行處理著稱，由前 Meta 團隊打造。

當 Meta（Facebook）的區塊鏈項目 Diem 被監管扼殺後，

那些頂尖工程師去了哪裡？

他們創立了 **Sui** 和 **Aptos**。

Sui 代表了區塊鏈技術的新方向。

### Sui 是什麼？

Sui 是新一代 Layer 1 公鏈。

- **開發者**：Mysten Labs
- **創辦人**：Evan Cheng（前 Meta）
- **創立**：2023 年主網上線
- **代幣**：SUI
- **語言**：Move

### Move 語言

Sui 使用 **Move** 編程語言。

**什麼是 Move？**

Meta（當時的 Facebook）為 Libra/Diem 開發的語言。

**特點**
- 資源導向（Resources）
- 線性類型系統
- 更安全的智能合約
- 防止常見漏洞

**與 Solidity 比較**

| 面向 | Move | Solidity |
|------|------|----------|
| 資產模型 | 資源（不可複製）| 數值 |
| 安全性 | 更高 | 需要小心 |
| 學習曲線 | 較陡 | 較平緩 |
| 生態成熟度 | 發展中 | 成熟 |

### 物件導向設計

Sui 的核心創新：**以物件為中心**。

**傳統區塊鏈（帳戶模型）**

- 所有狀態存在帳戶中
- 交易修改帳戶狀態

**Sui（物件模型）**

- 資產是獨立的「物件」
- 每個物件有唯一 ID
- 物件可以獨立處理

**好處**
- 並行處理（不衝突的物件可以同時處理）
- 更直觀的資產概念
- 更靈活的權限控制

### 並行處理

Sui 可以**並行處理**交易：

**原理**

- 如果兩筆交易涉及不同物件
- 它們可以同時執行
- 不需要排隊等待

**結果**
- 理論上極高的 TPS
- 低延遲
- 隨節點增加而擴展

### zkLogin

Sui 的創新功能：

**什麼是 zkLogin？**

用 Google、Facebook 等社交帳號登入區塊鏈：
- 不需要記助記詞
- 用熟悉的方式登入
- 零知識證明保護隱私

**意義**

大幅降低使用門檻，讓普通人更容易進入。

### 與 Aptos 比較

Sui 和 [Aptos](aptos.md) 都來自 Meta，都用 Move：

| 面向 | Sui | Aptos |
|------|-----|-------|
| 物件模型 | 物件導向 | 帳戶導向 |
| Move 版本 | Sui Move | Aptos Move |
| 並行處理 | 物件級 | Block-STM |
| 共識 | Narwhal+Bullshark | AptosBFT |

兩者各有特點，形成競爭。
