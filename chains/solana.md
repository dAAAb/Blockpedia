# Solana
> **最後更新：2026 年 1 月**


## Solana 是以高速低費著稱的公鏈，經歷過 FTX 崩盤的低谷後強勢復甦，成為以太坊最有力的挑戰者。

2021 年，Solana 是明星。
2022 年，FTX 崩盤，Solana 被判「死刑」。
2023-2024 年，Solana 復活了。

這是區塊鏈世界最戲劇性的故事之一。

### Solana 是什麼？

Solana 是高性能的 Layer 1 公鏈。

- **創辦人**：Anatoly Yakovenko
- **創立**：2020 年主網上線
- **代幣**：SOL
- **特色**：極高 TPS、極低費用

### 技術特點

**1. 歷史證明（Proof of History）**

Solana 的核心創新：

傳統區塊鏈：節點需要互相通信確認時間順序。
Solana：用密碼學證明事件的時間順序。

**好處**
- 減少節點間通信
- 大幅提升速度
- 可預測的區塊時間

**2. 並行處理（Sealevel）**

同時處理多筆交易：
- 不衝突的交易可以並行
- 類似多核處理器
- 進一步提升吞吐量

**3. 高性能**

| 指標 | Solana | Ethereum |
|------|--------|----------|
| TPS | 理論 65,000+ | 15-30 |
| 出塊時間 | 400ms | 12 秒 |
| 費用 | 幾美分 | 幾美元 |

### FTX 危機

2022 年 11 月，FTX 崩盤對 Solana 造成重創：

**關聯**
- FTX/Alameda 是 Solana 生態的主要投資者
- 很多 Solana 項目與 FTX 相關
- SBF 是 Solana 的支持者

**影響**
- SOL 價格暴跌 90%+
- 很多人宣布 Solana「已死」
- 生態活躍度驟降

### 復甦

2023-2024 年，Solana 強勢復甦：

**原因**
- 技術持續改進
- 穩定性提升
- 新應用出現
- 社區堅持

**表現**
- SOL 價格大幅回升
- DeFi TVL 成長
- NFT 和 Meme 幣活躍
- 開發者回歸

### 生態系統

**DeFi**

| 協議 | 類型 |
|------|------|
| Jupiter | DEX 聚合器 |
| Raydium | DEX/AMM |
| Marinade | 流動性質押 |
| Kamino | 借貸 |

**NFT**

- Magic Eden：主要 NFT 市場
- Tensor：專業 NFT 交易
- 各種 NFT 項目

**支付和消費**

- Solana Pay
- Saga 手機
- 實體商家整合

**Meme 幣**

Solana 成為 Meme 幣熱門平台：
- Bonk
- WIF
- 還有很多...

### Saga 手機

Solana 推出的智慧手機：

**特點**
- 內建加密錢包
- dApp Store
- 與 Solana 生態整合

**意義**
- 探索 Web3 硬體
- 降低使用門檻
- 獨特的市場策略

### 開發環境

**語言**

主要使用 **Rust** 和 **Anchor** 框架。

**與 Solidity 不同**

開發者需要學習新的範式：
- 帳戶模型不同
- 程式結構不同
- 但 Anchor 簡化了很多

**工具**

- Anchor：Solana 開發框架
- Solana Playground：線上 IDE
- Phantom：主流錢包

### 穩定性問題

Solana 曾有穩定性問題：

**歷史事件**
- 多次網路中斷
- 交易處理延遲
- 驗證節點問題

**改進**
- 持續升級
- 2023-2024 年穩定性明顯改善
- 但仍有批評

### 去中心化質疑

**批評**
- 驗證節點數量較少
- 硬體要求高
- Superteam 影響力大

**回應**
- 驗證節點在增加
- 地理分布在改善
- 中心化程度在降低

### 與以太坊比較

| 面向 | Solana | Ethereum |
|------|--------|----------|
| 速度 | 極快 | 較慢 |
| 費用 | 極低 | 較高 |
| 去中心化 | 較低 | 較高 |
| 生態 | 成長中 | 最豐富 |
| 開發者 | Rust | Solidity |
| 穩定性 | 改善中 | 穩定 |

### 未來發展

**Firedancer**

Jump Crypto 開發的新驗證節點客戶端：
- 用 C 語言從頭寫
- 可能大幅提升性能
- 增加客戶端多樣性

**持續改進**

- 壓縮 NFT（降低成本）
- 狀態壓縮
- 更多 L2/擴展方案

### 使用 Solana

**錢包**
- Phantom（最流行）
- Solflare
- Backpack

**開始**
1. 安裝 Phantom 錢包
2. 購買 SOL（交易所或法幣入金）
3. 探索 DApp

{% hint style="info" %}
Solana 證明了在困境中可以復甦。它的高速低費是真實的優勢，但去中心化和穩定性仍需持續改進。如果你重視速度和低成本，Solana 是值得關注的選擇。
{% endhint %}
