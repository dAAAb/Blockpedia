# BitoEX

### BitoEX（幣託）是台灣老牌的加密貨幣服務商，成立於 2014 年。它最大的特色是讓用戶可以在便利商店購買比特幣，也設置了台灣第一批比特幣 ATM。對於不想使用銀行轉帳或不熟悉交易所的用戶來說，BitoEX 提供了更簡單的入門方式。

---

## BitoEX 簡史

| 時間 | 事件 |
|------|------|
| 2014 年 | BitoEX 成立 |
| 2015 年 | 推出超商購買比特幣服務 |
| 2017 年 | 推出比特幣 ATM |
| 2018 年 | 推出 BitoPro 交易所 |
| 現在 | 雙品牌運營（BitoEX + BitoPro）|

{% hint style="info" %}
**BitoEX vs BitoPro**

BitoEX 是經銷商服務，適合小額購買；BitoPro 是交易所，適合進階交易。兩者屬於同一集團，但定位不同。
{% endhint %}

---

## BitoEX 的服務

### 1. 超商購買

```
1. 在 BitoEX App 或網站下單
2. 選擇購買金額（如 1000 元）
3. 獲得超商繳費條碼
4. 到全家或萊爾富繳費
5. 比特幣自動入帳
```

| 優點 | 缺點 |
|------|------|
| 不需銀行帳戶 | 有上限限制 |
| 現金購買 | 手續費較高 |
| 方便快速 | 價差較大 |

### 2. 比特幣 ATM

台灣設有多台 BitoEX ATM，可以：

- 用現金購買比特幣
- 將比特幣換成現金

{% hint style="warning" %}
**ATM 費用較高**

ATM 的便利性是有代價的——費用通常比線上服務高。適合小額、緊急的情況。
{% endhint %}

### 3. 錢包服務

BitoEX 提供基本的加密貨幣錢包功能。

## 支援的幣種

BitoEX 作為經銷商，支援的幣種較少：

| 幣種 | 說明 |
|------|------|
| BTC | 比特幣 |
| ETH | 以太幣 |
| USDT | 穩定幣 |
| 部分其他幣 | 較少選擇 |

如果需要更多幣種，可以使用 BitoPro 交易所。

---

## BitoEX 的優勢

| 優勢 | 說明 |
|------|------|
| 台灣本地 | 中文介面、台幣計價 |
| 入門簡單 | 不需要學習交易 |
| 多元購買 | 超商、ATM、銀行轉帳 |
| 合規運營 | 台灣合法業者 |

---

## BitoEX 的限制

| 限制 | 說明 |
|------|------|
| 價差較大 | 買賣價格差距明顯 |
| 幣種少 | 選擇有限 |
| 不適合大額 | 費用會累積 |
| 不適合交易 | 沒有訂單簿功能 |

---

## 適合誰使用？

| 適合 | 不適合 |
|------|--------|
| 第一次買幣的新手 | 要做頻繁交易的人 |
| 想用現金購買的人 | 追求最佳價格的人 |
| 購買小額的人 | 大額投資者 |
| 不想用銀行轉帳的人 | 要買冷門幣的人 |
