# Coinbase

### Coinbase 是 2012 年成立的美國加密貨幣公司，2021 年在納斯達克上市，成為第一家公開上市的加密貨幣交易所。它以簡單的介面和合規的運營聞名，是許多美國人進入加密貨幣世界的第一站。Coinbase 既是經銷商（簡單買賣），也是交易所（Coinbase Pro / Advanced Trade）。

---

## Coinbase 簡史

| 時間 | 事件 |
|------|------|
| 2012 年 | Brian Armstrong 和 Fred Ehrsam 創立 |
| 2014 年 | 推出 Coinbase Exchange（後改名 Coinbase Pro）|
| 2018 年 | 估值達 80 億美元 |
| 2021 年 4 月 | 在納斯達克直接上市（股票代號 COIN）|
| 2022 年 | 加密寒冬，股價大跌、大量裁員 |
| 2024 年 | 獲得多個比特幣 ETF 託管合約 |

{% hint style="success" %}
**歷史性的上市**

Coinbase 的上市是加密貨幣產業的里程碑。它證明了加密貨幣公司可以通過傳統金融市場的審查，進入主流資本市場。
{% endhint %}

## Coinbase 的服務

### 1. Coinbase（簡單版）

面向新手的經銷商介面：

| 特色 | 說明 |
|------|------|
| 簡單買賣 | 一鍵購買 |
| 定期定額 | 設定自動購買 |
| 新手友善 | 不需要學習交易 |
| 費用較高 | 方便的代價 |

### 2. Coinbase Advanced Trade

面向進階用戶的交易所介面：

| 特色 | 說明 |
|------|------|
| 訂單簿交易 | 可以掛限價單 |
| 費用較低 | 專業用戶優惠 |
| 更多工具 | 圖表、技術分析 |

### 3. 其他服務

| 服務 | 說明 |
|------|------|
| Coinbase Wallet | 非託管錢包 |
| Coinbase Prime | 機構服務 |
| Coinbase Custody | 託管服務 |
| Coinbase Card | Visa 簽帳卡 |
| Base | 自建的 L2 區塊鏈 |

## 為什麼 Coinbase 重要？

### 合規的代表

| 方面 | 說明 |
|------|------|
| 上市公司 | 受 SEC 監管 |
| 州牌照 | 美國多州持牌 |
| 托管合規 | 獲得 ETF 託管合約 |
| 透明度 | 定期財報披露 |

{% hint style="info" %}
**比特幣 ETF 的託管商**

2024 年比特幣現貨 ETF 獲批後，多家 ETF 發行商選擇 Coinbase 作為託管商。這顯示了機構對 Coinbase 安全性的信任。
{% endhint %}

---

## Coinbase 的爭議

| 爭議 | 說明 |
|------|------|
| 費用高 | 簡單版費用較貴 |
| SEC 訴訟 | 被控上架證券 |
| 客服問題 | 用戶投訴客服差 |
| 帳戶凍結 | 部分用戶帳戶被鎖 |

{% hint style="warning" %}
**SEC 訴訟**

2023 年，SEC 起訴 Coinbase，指控其運營未註冊的證券交易所。這場官司仍在進行中，結果將影響整個美國加密貨幣產業。
{% endhint %}
