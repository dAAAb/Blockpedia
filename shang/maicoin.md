# MaiCoin

### MaiCoin 是台灣老牌的加密貨幣服務商，成立於 2014 年，主打簡單的加密貨幣買賣服務。它的關係企業 MAX 則是台灣最大的加密貨幣交易所之一。如果你是新手想要簡單買幣，用 MaiCoin；如果你想進階交易，用 MAX。

---

## MaiCoin 簡史

| 時間 | 事件 |
|------|------|
| 2014 年 | MaiCoin 成立 |
| 2018 年 | 推出 MAX 交易所 |
| 2021 年 | 取得台灣虛擬通貨公會會員資格 |
| 持續運營 | 台灣主要加密貨幣服務商之一 |

{% hint style="info" %}
**MaiCoin vs MAX**

MaiCoin 是經銷商，適合想要簡單買賣的新手；MAX 是交易所，適合想要進階交易的用戶。兩者帳號互通。
{% endhint %}

---

## MaiCoin 的服務

### 1. 簡單買賣

直接向 MaiCoin 購買或出售加密貨幣：

```
選擇幣種 → 輸入金額 → 確認價格 → 付款 → 收到幣
```

| 支付方式 | 說明 |
|----------|------|
| 銀行轉帳 | 最常用，費用低 |
| 信用卡 | 快速但手續費高 |
| 超商繳費 | 小額適用 |

### 2. 定期定額

設定每週或每月自動購買：

| 優點 | 說明 |
|------|------|
| 分散風險 | 平均買入成本 |
| 省時間 | 不用盯盤 |
| 長期投資 | 適合 HODL 策略 |

### 3. 錢包服務

MaiCoin 提供基本的加密貨幣錢包功能。

## 支援的幣種

MaiCoin 作為經銷商，支援主流幣種：

| 幣種 | 說明 |
|------|------|
| BTC | 比特幣 |
| ETH | 以太幣 |
| USDT | 穩定幣 |
| LTC | 萊特幣 |
| 其他 | 視平台公告 |

如需更多幣種，建議使用 MAX 交易所。

---

## MaiCoin 的限制

| 限制 | 說明 |
|------|------|
| 價差較大 | 買賣價格有差距 |
| 幣種有限 | 不如交易所多 |
| 費用較高 | 方便的代價 |
| 不適合大額 | 大額建議用交易所 |

{% hint style="warning" %}
**注意價差**

經銷商的「買入價」和「賣出價」通常有 2-5% 的差距。如果你頻繁買賣，這個成本會累積。
{% endhint %}

## 適合誰使用？

| 適合 | 不適合 |
|------|--------|
| 第一次買幣的新手 | 頻繁交易的人 |
| 想要定期定額的人 | 追求最佳價格的人 |
| 想用信用卡的人 | 要買冷門幣的人 |
| 長期持有的投資者 | 短線交易者 |

## MaiCoin vs MAX 怎麼選？

| 比較 | MaiCoin | MAX |
|------|---------|-----|
| 類型 | 經銷商 | 交易所 |
| 介面 | 簡單 | 專業 |
| 價格 | 有價差 | 市場價格 |
| 幣種 | 較少 | 較多 |
