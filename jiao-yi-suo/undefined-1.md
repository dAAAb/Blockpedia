# 資產

### 資產（Assets）在加密貨幣語境中，泛指你擁有的所有數位價值。這可以是比特幣、以太幣這樣的加密貨幣，也可以是 NFT、DeFi 協議中的 LP Token，甚至是遊戲中的虛擬道具。在交易所中，「資產」通常指你帳戶裡的所有幣種和餘額。

---

## 資產的類型

### 1. 加密貨幣

最基本的數位資產。

| 類型 | 例子 | 特性 |
|------|------|------|
| 原生幣 | BTC、ETH、SOL | 各自區塊鏈的原生代幣 |
| 穩定幣 | USDT、USDC、DAI | 價值錨定法幣 |
| 平台幣 | BNB、FTT、CRO | 交易所發行的代幣 |

### 2. 代幣（Token）

建立在其他區塊鏈上的資產。

| 標準 | 區塊鏈 | 例子 |
|------|--------|------|
| ERC-20 | Ethereum | UNI、LINK、AAVE |
| BEP-20 | BNB Chain | CAKE、XVS |
| SPL | Solana | RAY、SRM |

### 3. NFT（非同質化代幣）

獨一無二的數位資產。

- 藝術品
- 收藏品
- 遊戲道具
- 會員資格
- 域名

### 4. DeFi 資產

參與去中心化金融獲得的資產。

| 類型 | 說明 |
|------|------|
| LP Token | 流動性提供者憑證 |
| 質押憑證 | stETH、cbETH 等 |
| 借貸憑證 | aToken、cToken 等 |

## 交易所中的資產管理

在中心化交易所，你的資產通常分布在不同帳戶：

```
總資產
├── 現貨帳戶 — 買賣幣用
├── 合約帳戶 — 做合約交易
├── 槓桿帳戶 — 槓桿交易
├── 資金帳戶 — 入金/提幣
└── 理財帳戶 — 存幣生息
```

{% hint style="info" %}
**帳戶間劃轉**

不同帳戶的資產通常不能直接使用，需要先「劃轉」到對應帳戶。比如要做合約交易，需先把幣從現貨帳戶劃轉到合約帳戶。
{% endhint %}

## 資產估值

### 以何計價？

| 計價方式 | 說明 |
|----------|------|
| 法幣計價 | 以 USD、TWD 等顯示總值 |
| BTC 計價 | 以比特幣為單位（幣本位）|
| ETH 計價 | 常用於 NFT 和 DeFi |

### 市值計算

```
你的資產價值 = 持有數量 × 當前市價

例如：
持有 0.5 BTC，BTC 價格 $60,000
資產價值 = 0.5 × $60,000 = $30,000
```

{% hint style="warning" %}
**紙上富貴**

資產價值是浮動的。今天價值 100 萬，明天可能變 50 萬。只有賣出變現，才是「真正」賺到的錢。
{% endhint %}

## 資產安全

### 在哪裡最安全？

| 存放位置 | 安全性 | 便利性 |
|----------|--------|--------|
| 冷錢包 | ★★★★★ | ★★ |
| 硬體錢包 | ★★★★ | ★★★ |
| 軟體錢包 | ★★★ | ★★★★ |
| 交易所 | ★★ | ★★★★★ |

### 資產保護建議

| 建議 | 說明 |
|------|------|
| 分散存放 | 不要把蛋放同一個籃子 |
| 定期備份 | 助記詞、私鑰要備份 |
| 硬體錢包 | 大額資產用冷儲存 |
| 監控異常 | 設置地址監控警報 |

{% hint style="danger" %}
**FTX 的教訓**

FTX 破產時，用戶帳戶顯示的「資產」一夜歸零。交易所顯示的餘額，只是交易所「欠你」的數字。只有在自己錢包裡、由自己私鑰控制的，才是真正的資產。
{% endhint %}
