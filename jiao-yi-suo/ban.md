# 搬磚

### 搬磚（Arbitrage）是一種在不同市場間賺取價差的交易方式。想像同一塊磚頭，在 A 市場賣 10 元，在 B 市場賣 12 元——你在 A 買入、搬到 B 賣出，這個「搬運」的過程就叫搬磚。在加密貨幣世界，搬磚曾經是獲利穩定的策略，但隨著市場效率提高，機會已經越來越少。

---

## 搬磚的原理

```
交易所 A：BTC 價格 $59,800
交易所 B：BTC 價格 $60,200

價差：$400（約 0.67%）

操作：
1. 在 A 買入 1 BTC（花費 $59,800）
2. 轉帳到 B
3. 在 B 賣出 1 BTC（獲得 $60,200）

毛利：$400
扣除：轉帳費 + 交易手續費
淨利：約 $200-300
```

{% hint style="info" %}
**為什麼會有價差？**

不同交易所的用戶組成、流動性、地區法規都不同，導致同一幣種在不同地方可能有不同價格。這種「市場無效率」就是套利的機會。
{% endhint %}

---

## 搬磚的類型

### 1. 交易所間搬磚

最傳統的方式，在不同交易所之間搬運。

| 優點 | 缺點 |
|------|------|
| 概念簡單 | 轉帳需要時間 |
| 風險相對低 | 價差可能消失 |
| | 需要多處資金 |

### 2. 三角套利

在同一交易所，利用三種幣的匯率差異套利。

```
BTC → ETH → USDT → BTC

如果兜一圈後 BTC 變多了，就是套利成功
```

### 3. 期現套利

利用現貨和期貨的價差。

```
期貨價格 > 現貨價格（正價差）
→ 買現貨、賣期貨
→ 到期時價格收斂
→ 賺取價差
```

### 4. 跨鏈搬磚

利用同一幣種在不同區塊鏈的價差。

```
Solana 上的 USDC 價格
vs
Ethereum 上的 USDC 價格
```

---

## 搬磚的風險

| 風險 | 說明 |
|------|------|
| 價格波動 | 搬運過程中價差可能消失甚至反轉 |
| 轉帳延遲 | 區塊鏈擁堵導致轉帳變慢 |
| 手續費侵蝕 | 交易費 + 轉帳費可能吃掉利潤 |
| 資金效率 | 需要在多個交易所預置資金 |
| 交易所風險 | 出金可能被延遲或凍結 |

{% hint style="warning" %}
**時間就是金錢**

搬磚最大的敵人是時間。BTC 轉帳可能需要 30-60 分鐘，這段時間內價格可能劇烈變化。很多人搬磚搬到一半，價差已經沒了。
{% endhint %}

---

## 現代搬磚的挑戰

### 為什麼搬磚越來越難？

| 因素 | 說明 |
|------|------|
| 市場效率化 | 價差更快被抹平 |
| 機器人競爭 | 高頻交易程式比人快 |
| 手續費提高 | 利潤空間被壓縮 |
| 監管加強 | 跨境套利更困難 |

### 專業套利者的做法

```
1. 在多個交易所預置資金（避免轉帳）
2. 使用 API 自動化交易
3. 毫秒級執行
4. 統計套利而非簡單價差
```

{% hint style="danger" %}
**別輕信「搬磚教學」**

很多「搬磚教學」、「套利機器人」其實是詐騙。他們可能：
- 騙你把幣轉到假地址
- 賣你無用的軟體
- 讓你授權惡意合約

真正能獲利的套利策略，沒人會公開教你。
{% endhint %}
