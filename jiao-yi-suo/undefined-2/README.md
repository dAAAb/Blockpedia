# 幣價

### 幣價就是一個加密貨幣在交易市場上的價格。聽起來很簡單，但「這個幣現在多少錢？」這個問題，答案其實比你想像的複雜——因為不同交易所的價格可能不一樣，而且價格每秒都在變動。

---

## 幣價是怎麼決定的？

### 供需法則

```
很多人想買 + 很少人想賣 = 價格上漲
很多人想賣 + 很少人想買 = 價格下跌
```

和菜市場的青菜一樣，加密貨幣的價格也是由買賣雙方的供需決定的。

### 為什麼不同交易所價格不同？

| 原因 | 說明 |
|------|------|
| 用戶組成 | 不同地區、不同類型的交易者 |
| 流動性 | 交易量大的所價格較準確 |
| 法規差異 | 某些地區可能有溢價或折價 |
| 入金難度 | 法幣通道影響套利效率 |

{% hint style="info" %}
**韓國溢價（Kimchi Premium）**

韓國因為法幣出入金限制，加密貨幣價格常常比國際價格高 5-20%。這就是著名的「泡菜溢價」。
{% endhint %}

## 如何查看幣價？

### 常用工具

| 工具 | 特點 |
|------|------|
| CoinMarketCap | 最老牌，交易所覆蓋全 |
| CoinGecko | 數據詳細，免費 API |
| TradingView | 專業圖表工具 |
| 交易所 App | 即時價格，可直接交易 |

### 價格指標

| 指標 | 說明 |
|------|------|
| 現價 | 最新成交價 |
| 24h 高/低 | 過去 24 小時的最高和最低價 |
| 成交量 | 交易活躍度指標 |
| 市值 | 幣價 × 流通量 |

## 幣價的常見陷阱

### 1. 只看幣價不看市值

```
A 幣：$0.001/顆，總量 1 兆顆
B 幣：$50,000/顆，總量 2100 萬顆

A 幣「便宜」？市值其實是 10 億美元
B 幣「貴」？市值是 1 兆美元

「便宜」不等於「有上漲空間」
```

### 2. 被「價格低」誤導

{% hint style="warning" %}
**單價陷阱**

「這個幣才 0.0001 美元，漲到 1 美元就是 10000 倍！」

這種想法忽略了市值。如果流通量是 1 兆顆，漲到 1 美元意味著市值要達到 1 兆美元——比比特幣還大。
{% endhint %}

### 3. 忽略流動性

有些小幣看起來價格很「穩定」，其實是因為幾乎沒有交易。一旦你想賣出，可能根本找不到買家，或者價格會大幅滑落。

---

## 影響幣價的因素

| 因素 | 影響方式 |
|------|----------|
| 比特幣走勢 | 大多數幣跟著 BTC 漲跌 |
| 宏觀經濟 | 利率、通膨、經濟危機 |
| 監管消息 | 政策利好或打壓 |
| 技術進展 | 升級、合作、採用 |
| 市場情緒 | FOMO、恐慌、炒作 |
| 鯨魚動向 | 大戶買賣影響價格 |

---

## 幣價 vs 價值

| 概念 | 說明 |
|------|------|
| 幣價 | 市場此刻願意付的價格 |
| 價值 | 這個項目「真正值多少」|

{% hint style="danger" %}
**價格不等於價值**

短期幣價可能因為炒作而偏離價值。很多空氣幣曾經價格很高，最終歸零；有些有價值的項目曾經被低估。

「市場短期是投票機，長期是秤重機。」——班傑明·格雷厄姆
{% endhint %}
