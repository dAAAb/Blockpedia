# 法幣交易

### 法幣交易（Fiat-to-Crypto Trading）是用傳統貨幣購買加密貨幣的方式。想要第一次擁有比特幣？你需要用台幣、美元這些「法幣」去買。這是每個加密貨幣投資者的第一步，也是整個加密貨幣生態的入口。

---

## 法幣 vs 加密貨幣

| 比較 | 法幣（Fiat） | 加密貨幣（Crypto） |
|------|-------------|-------------------|
| 發行者 | 政府/央行 | 區塊鏈協議 |
| 例子 | TWD、USD、EUR | BTC、ETH、USDT |
| 監管 | 嚴格監管 | 監管程度不一 |
| 形式 | 實體+數位 | 純數位 |

{% hint style="info" %}
**「法幣」的由來**

「法幣」是「法定貨幣」的簡稱（Fiat Money），指由政府發行、沒有實物支撐、靠法律強制流通的貨幣。加密圈用「法幣」來區分傳統貨幣和加密貨幣。
{% endhint %}

---

## 法幣交易的方式

### 1. 中心化交易所

在交易所用法幣直接購買加密貨幣。

| 方式 | 說明 |
|------|------|
| 銀行轉帳 | 最常見，費用低 |
| 信用卡 | 快速但手續費高 |
| 電匯 | 國際轉帳 |
| 第三方支付 | Apple Pay、Google Pay 等 |

### 2. P2P 交易

和其他用戶直接交易，平台提供擔保。

```
你想買 BTC
    ↓
找到願意賣 BTC 的賣家
    ↓
平台鎖定賣家的 BTC
    ↓
你直接轉帳給賣家（銀行轉帳、支付寶等）
    ↓
賣家確認收款
    ↓
平台釋放 BTC 給你
```

### 3. 經銷商/Broker

像買外幣一樣，直接向經銷商購買加密貨幣。

{% hint style="warning" %}
**台灣用戶的限制**

國際大型交易所（如 Coinbase）通常不支援台幣直接入金。台灣用戶通常先在本地交易所用台幣買 USDT，再轉到國際交易所使用。
{% endhint %}

## 法幣交易的 KYC 要求

法幣交易通常需要完成身分驗證（KYC）：

| 等級 | 要求 | 限額 |
|------|------|------|
| 基礎 | 電子郵件、手機 | 很低或無法法幣交易 |
| 標準 | 身分證、自拍 | 每日數十萬台幣 |
| 進階 | 地址證明、收入證明 | 更高限額 |

{% hint style="info" %}
**為什麼要 KYC？**

法幣涉及銀行系統，受到反洗錢（AML）法規監管。交易所必須驗證用戶身分，否則可能失去銀行合作關係。
{% endhint %}

## 法幣交易的費用

| 費用類型 | 說明 |
|----------|------|
| 入金費用 | 銀行轉帳通常免費 |
| 交易手續費 | 0.1%-1% 不等 |
| 價差 | 買價和賣價的差距 |
| 信用卡費 | 3-5% 額外費用 |

---

## 法幣交易的風險

| 風險 | 說明 |
|------|------|
| 交易所風險 | 平台可能倒閉 |
| 監管風險 | 政策可能變化 |
| 銀行凍結 | 涉及可疑資金可能被凍卡 |
| 詐騙風險 | P2P 交易可能遇到騙子 |

## 選擇法幣交易平台

| 考量 | 建議 |
|------|------|
| 合規性 | 選擇有牌照的平台 |
| 流動性 | 交易量大、價差小 |
| 安全性 | 有保險、冷儲存 |
| 用戶體驗 | 介面友善、客服好 |
| 費用 | 比較各平台費率 |
