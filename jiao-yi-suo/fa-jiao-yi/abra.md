# Abra

### Abra 是 2014 年成立的加密貨幣金融服務公司，最初是一個比特幣錢包和匯款應用，後來轉型為加密貨幣理財平台。它讓用戶可以用法幣購買加密貨幣，並提供存幣生息等服務。但在 2023 年，Abra 因財務問題和監管壓力而大幅縮減業務。

---

## Abra 簡史

| 時間 | 事件 |
|------|------|
| 2014 年 | 成立，主打比特幣匯款 |
| 2017 年 | 轉型為加密貨幣投資平台 |
| 2020 年 | 推出高息存幣產品 |
| 2022 年 | 加密寒冬影響業務 |
| 2023 年 | 被德州監管機構起訴 |
| 2023 年 | 大幅縮減零售業務 |

{% hint style="danger" %}
**2023 年的危機**

2023 年，Abra 被美國德州監管機構指控「資不抵債」，並被迫停止部分業務。這再次提醒我們：把資產存在第三方平台有風險。
{% endhint %}

## Abra 的服務

### 曾經提供的服務

| 服務 | 說明 |
|------|------|
| 購買加密貨幣 | 用銀行帳戶或信用卡買幣 |
| 存幣生息 | 存入加密貨幣賺取利息 |
| 加密借貸 | 用加密貨幣作為抵押借款 |
| 交易服務 | 買賣多種加密貨幣 |

### 存幣生息的運作方式

```
你存入 1 BTC
    ↓
Abra 把你的 BTC 借給機構用戶
    ↓
機構用戶支付利息
    ↓
Abra 抽成後，剩餘給你
    ↓
你賺取約 5-10% 年化收益
```

{% hint style="warning" %}
**存幣生息的風險**

這類服務的高收益來自於借貸。如果借款方違約，或市場劇烈波動，你的資產可能無法取回。Celsius、BlockFi、Voyager 的倒閉都是前車之鑑。
{% endhint %}

## 為什麼 Abra 遇到麻煩？

### 加密寒冬的衝擊

| 因素 | 影響 |
|------|------|
| 幣價暴跌 | 用戶資產縮水、贖回壓力 |
| 借貸違約 | 機構用戶無法還款 |
| 監管收緊 | SEC 開始打擊類似服務 |
| 信任危機 | Celsius 等倒閉影響整個行業 |

### 和其他公司的比較

| 公司 | 狀態 |
|------|------|
| Celsius | 破產 |
| BlockFi | 破產 |
| Voyager | 破產 |
| Abra | 縮減業務 |
| Nexo | 仍在運營 |

## 給投資者的教訓

| 教訓 | 說明 |
|------|------|
| 高收益 = 高風險 | 5-10% 年化收益不是免費的 |
| 不是銀行 | 這些平台沒有存款保險 |
| 分散風險 | 不要把所有資產放在一處 |
| 自我託管 | 考慮使用自己的錢包 |

{% hint style="danger" %}
**Not your keys, not your coins**

存在這類平台的「餘額」只是一個數字。當平台出問題，你可能什麼都拿不回來。2022-2023 年的加密寒冬讓這個道理變得格外深刻。
{% endhint %}
