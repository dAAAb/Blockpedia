# COBINHOOD

### COBINHOOD 是 2017 年在台灣成立的加密貨幣交易所，由陳泰元（Popo Chen）創辦。它以「全球首家零手續費交易所」為號召，ICO 募資超過 1300 萬美元，一度是台灣區塊鏈產業的明星公司。但好景不長，COBINHOOD 在 2019 年爆發經營危機，成為台灣幣圈最大的失敗案例之一。

---

## COBINHOOD 簡史

| 時間 | 事件 |
|------|------|
| 2017 年 9 月 | ICO 募資 1300 萬美元 |
| 2017 年 12 月 | 交易所正式上線 |
| 2018 年 | 快速擴張，員工超過 100 人 |
| 2019 年初 | 傳出欠薪、資金問題 |
| 2019 年 2 月 | 大規模裁員 |
| 2019 年 | 實質停止運營 |
| 2020 年後 | 創辦人另起爐灶，但爭議不斷 |

{% hint style="danger" %}
**台灣幣圈最大教訓**

COBINHOOD 的失敗影響了數千名投資者和員工，也讓台灣區塊鏈產業的聲譽受損。
{% endhint %}

---

## 為什麼 COBINHOOD 失敗了？

### 商業模式問題

```
零手續費
    ↓
收入來源不足
    ↓
依賴 ICO 資金燒錢
    ↓
資金燒完
    ↓
無法維持營運
```

### 經營管理問題

| 問題 | 說明 |
|------|------|
| 過度擴張 | 員工過多、成本過高 |
| 產品發散 | 同時做交易所、ICO 平台、公鏈 |
| 市場時機 | 2018 年熊市來襲 |
| 透明度低 | 財務狀況不透明 |

{% hint style="warning" %}
**ICO 的詛咒**

很多 ICO 項目在牛市募資後，遇到熊市就撐不下去。COBINHOOD 不是唯一，但絕對是最知名的案例。
{% endhint %}

---

## 爭議事件

### 欠薪風波

2019 年初，多名員工在社群爆料公司欠薪，引發媒體關注。

### ICO 退款問題

部分投資者要求退款，但公司表示無力償還。

### 後續發展

創辦人陳泰元後來參與了其他區塊鏈項目，但持續引發社群爭議。

{% hint style="danger" %}
**投資前做功課**

COBINHOOD 的案例告訴我們：光看團隊背景和宣傳不夠，要深入了解商業模式是否可持續。
{% endhint %}

---

## 從 COBINHOOD 學到什麼？

| 教訓 | 說明 |
|------|------|
| 商業模式要可行 | 零手續費難以維持 |
| 不要被行銷迷惑 | 高調宣傳不等於成功 |
| 分散投資 | 不要把資產放在單一平台 |
| 研究團隊過往 | 創辦人的歷史記錄很重要 |
| 熊市考驗真章 | 牛市募資容易，熊市生存困難 |

---

## 對台灣區塊鏈產業的影響

COBINHOOD 的失敗：

- 降低了大眾對台灣區塊鏈公司的信任
- 讓投資者對 ICO 更加謹慎
- 促使監管機構關注加密貨幣產業
- 成為創業者的警惕案例
