# 交易所
> **最後更新：2026 年 1 月**


## 交易所是買賣加密貨幣的地方——就像股票市場讓你買賣股票一樣。差別是，加密貨幣交易所 24 小時運作，沒有週末休息，沒有漲跌停板。

你買了人生第一枚比特幣。

去哪買的？

大概率是在交易所。

交易所是大多數人進入加密貨幣世界的第一站。它可能是幣安、Coinbase、MAX 或其他平台。不管是哪一個，你都是把法幣（台幣、美元）換成加密貨幣。

### 交易所是什麼？

**簡單說**：媒合買家和賣家的平台。

- 有人想用 $30,000 買一枚比特幣
- 有人想用 $30,000 賣一枚比特幣
- 交易所幫他們配對，完成交易

**交易所賺什麼？**

| 收入來源 | 說明 |
|----------|------|
| 交易手續費 | 每筆交易收 0.1%-0.5% |
| 提領費用 | 提領加密貨幣或法幣的費用 |
| 上幣費 | 新項目想上架要付費 |
| 槓桿利息 | 借貸交易的利息 |
| 其他服務 | 質押、理財產品等 |

### 中心化 vs 去中心化

交易所分為兩大類：

| 類型 | 英文 | 代表 | 特點 |
|------|------|------|------|
| 中心化交易所 | CEX | 幣安、Coinbase | 有公司運營、需要 KYC |
| 去中心化交易所 | DEX | Uniswap、PancakeSwap | 智能合約運作、不需 KYC |

{% hint style="info" %}
大約 95% 的加密貨幣交易量發生在中心化交易所。去中心化交易所雖然更符合加密精神，但對一般用戶來說還是比較難用。
{% endhint %}

### 中心化交易所（CEX）

#### 運作方式

1. 你在交易所開戶、完成身分驗證（KYC）
2. 把錢存進交易所
3. 在交易所內買賣
4. 你的資產「顯示」在帳戶裡
5. 想用時再提領到自己的錢包

**重點**：你的幣其實是存在交易所的錢包裡，你只是有一個「欠條」。

{% hint style="warning" %}
「Not your keys, not your coins.」——如果你沒有私鑰，那些幣嚴格來說不是你的。交易所可能破產、被駭、跑路。FTX 的崩盤就是血淋淋的例子。
{% endhint %}

#### 主要的中心化交易所

| 交易所 | 特點 | 狀態 |
|--------|------|------|
| [幣安（Binance）](binance.md) | 全球最大、幣種最多 | 運營中 |
| [Coinbase](coinbase.md) | 美國上市公司、最合規 | 運營中 |
| [Kraken](kraken.md) | 歷史悠久、安全性高 | 運營中 |
| [OKX](okx.md) | 衍生品交易強 | 運營中 |
| [Bybit](bybit.md) | 衍生品專精 | 運營中 |
| [MAX](max.md) | 台灣交易所 | 運營中 |
| FTX | 曾是第二大 | **已倒閉** |

#### CEX 的優點

- 操作簡單，像使用銀行 App
- 流動性好，大額交易不會滑價太多
- 客服支援，有問題可以找人
- 法幣出入金，可以直接用台幣/美元
- 多元產品：現貨、期貨、槓桿、理財

#### CEX 的風險

| 風險 | 說明 | 案例 |
|------|------|------|
| 破產 | 交易所可能經營不善倒閉 | FTX（2022）|
| 駭客攻擊 | 被駭導致用戶資產損失 | Mt. Gox（2014）|
| 凍結資產 | 因監管或其他原因凍結帳戶 | 常見於各國 |
| 跑路 | 詐騙交易所捲款潛逃 | 無數小交易所 |
| 監管風險 | 各國法規變動 | 幣安退出多國 |

### 去中心化交易所（DEX）

#### 運作方式

不同於 CEX 的「訂單簿」模式，大多數 DEX 使用「自動做市商（AMM）」：

```
傳統訂單簿：
買家出價 ←→ 賣家出價 ←→ 配對成交

AMM 模式：
用戶 ←→ 流動性池（智能合約）←→ 自動計算價格
```

**你只需要**：

1. 連接你的錢包（如 MetaMask）
2. 選擇要交換的代幣
3. 確認交易
4. 代幣直接到你的錢包

**沒有註冊、沒有 KYC、沒有中間人。**

#### 主要的去中心化交易所

| DEX | 區塊鏈 | 特點 |
|-----|--------|------|
| [Uniswap](../defi/dex/uniswap.md) | Ethereum | DEX 龍頭 |
| [PancakeSwap](pancakeswap.md) | BNB Chain | BSC 上最大 |
| [Curve](../defi/dex/curve.md) | 多鏈 | 穩定幣交換專家 |
| SushiSwap | 多鏈 | Uniswap 分叉 |
| dYdX | 自有鏈 | 衍生品 DEX |

#### DEX 的優點

- 無需信任第三方
- 不需 KYC，保護隱私
- 資產始終在自己錢包
- 不會被凍結帳戶
- 上幣不需許可，新幣通常先上 DEX

#### DEX 的風險

| 風險 | 說明 |
|------|------|
| 智能合約漏洞 | 合約被駭可能損失資金 |
| 滑價 | 流動性不足時，大額交易價格差很多 |
| 無常損失 | 提供流動性可能虧錢 |
| 假幣 | 沒有審核，可能買到假代幣 |
| Gas 費 | 每筆交易都要付鏈上手續費 |
| 操作複雜 | 對新手不友善 |

### CEX vs DEX 比較

| 項目 | CEX | DEX |
|------|-----|-----|
| 註冊 KYC | 需要 | 不需要 |
| 資產控制 | 交易所保管 | 自己掌控 |
| 交易速度 | 快 | 較慢（等區塊確認）|
| 法幣出入金 | 支援 | 不支援 |
| 流動性 | 高 | 較低 |
| 上幣審核 | 嚴格 | 無 |
| 客服 | 有 | 無 |
| 被駭風險 | 中心化風險 | 智能合約風險 |

### OTC 交易

除了交易所，還有一種交易方式叫 OTC（Over-the-Counter，場外交易）。

**適合**：大額交易（通常 10 萬美元以上）

**為什麼用 OTC？**

- 在交易所買 100 BTC 會嚴重影響價格
- OTC 可以談好價格，一次成交
- 不會造成市場波動

{% hint style="info" %}
OTC 交易通常由專門的 OTC 櫃檯或經紀人撮合。大型交易所如幣安、Coinbase 都有 OTC 服務。
{% endhint %}

### 如何選擇交易所？

#### 新手建議

| 考量 | 建議 |
|------|------|
| 台灣用戶 | MAX、幣託、ACE（有台幣出入金）|
| 國際用戶 | 幣安、Coinbase |
| 想嘗試 DeFi | 從 Uniswap 開始 |

#### 選擇標準

1. **安全性**：有沒有被駭過？保險？冷錢包儲存比例？
2. **流動性**：交易量大不大？
3. **手續費**：費率多少？有沒有折扣？
4. **幣種**：有沒有你想交易的幣？
5. **出入金**：支援什麼幣種？手續費？
6. **監管**：在你的國家合法嗎？
7. **客服**：有問題找得到人嗎？

### 交易所安全建議

#### 在 CEX 上

1. **開啟 2FA**：一定要用 Google Authenticator
2. **設定提款白名單**：只能提到指定地址
3. **不要放太多**：只放你準備交易的金額
4. **分散風險**：不要全部放在同一家交易所
5. **注意釣魚**：確認網址正確

#### 使用 DEX 時

1. **確認合約地址**：從官方來源取得
2. **注意滑價設定**：太高可能被 MEV 攻擊
3. **小額測試**：新幣先小額嘗試
4. **檢查授權**：定期撤銷不用的合約授權

### 交易所的歷史教訓

| 事件 | 年份 | 損失 | 教訓 |
|------|------|------|------|
| Mt. Gox 破產 | 2014 | 85 萬 BTC | 交易所可能被駭 |
| QuadrigaCX | 2019 | 1.9 億美元 | 創辦人持有所有私鑰 |
| FTX 倒閉 | 2022 | 數十億美元 | 挪用客戶資金 |

{% hint style="danger" %}
FTX 的崩盤證明了一件事：不管交易所看起來多可靠、多大、多有背景，都不應該把所有資產放在上面。永遠記住：交易所是交易的地方，不是存錢的地方。
{% endhint %}

### 交易所的未來

| 趨勢 | 說明 |
|------|------|
| 監管趨嚴 | 全球各國加強監管 |
| 儲備證明 | 更多交易所公開資產證明 |
| 混合模式 | CEX 整合 DEX 功能 |
| 自託管 | 更多用戶選擇自己保管資產 |

{% hint style="success" %}
交易所是進入加密貨幣世界的必經之路。選擇一個安全、合規的交易所很重要，但更重要的是了解：交易所只是工具，你的資產安全最終還是自己的責任。學會使用去中心化交易所和自託管錢包，是每個加密貨幣投資者應該掌握的技能。
{% endhint %}
