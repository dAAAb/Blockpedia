# 餘額

### 餘額（Balance）是你在交易所帳戶中持有的資產數量。它顯示你有多少幣可以交易、提領或進行其他操作。交易所的餘額分為「可用餘額」和「凍結餘額」兩種狀態。

---

## 餘額的類型

### 1. 可用餘額（Available Balance）

你可以自由使用的資產，可以：
- 下單交易
- 提幣到外部錢包
- 轉帳到其他帳戶

### 2. 凍結餘額（Frozen/Locked Balance）

暫時無法使用的資產，可能因為：
- 有未成交的掛單
- 正在進行提幣
- 參與某些活動（如 Launchpad）
- 帳戶被安全凍結

```
總餘額 = 可用餘額 + 凍結餘額

例如：
總餘額：10 BTC
可用餘額：7 BTC
凍結餘額：3 BTC（因為你掛了 3 BTC 的賣單）
```

{% hint style="info" %}
**為什麼會凍結？**

當你下一筆限價單時，交易所會「凍結」相應的資產，確保訂單可以成交。這樣可以防止你重複使用同一筆錢下多筆訂單。
{% endhint %}

## 交易所的帳戶類型

大型交易所通常有多種帳戶，各有獨立餘額：

| 帳戶類型 | 用途 |
|----------|------|
| 現貨帳戶 | 現貨交易 |
| 合約帳戶 | 期貨/合約交易 |
| 資金帳戶 | 入金、提幣、轉帳 |
| 理財帳戶 | 存幣生息 |
| 槓桿帳戶 | 槓桿交易 |

{% hint style="warning" %}
**帳戶間劃轉**

要在不同帳戶間使用資金，通常需要先「劃轉」。比如你要做合約交易，需要先把幣從現貨帳戶劃轉到合約帳戶。
{% endhint %}

## 餘額與區塊鏈的關係

| 比較 | 交易所餘額 | 區塊鏈餘額 |
|------|-----------|-----------|
| 在哪裡 | 交易所的資料庫 | 區塊鏈上 |
| 誰控制 | 交易所 | 私鑰持有者 |
| 更新方式 | 即時 | 需等區塊確認 |
| 查詢方式 | 交易所 App | 區塊鏈瀏覽器 |

{% hint style="danger" %}
**交易所餘額 ≠ 真正擁有**

交易所顯示的餘額只是交易所「欠你」的數字。你真正擁有的幣，是你自己錢包裡、由你的私鑰控制的幣。FTX 破產時，用戶的「餘額」一夜歸零就是最好的教訓。
{% endhint %}

## 餘額的常見問題

### 餘額不對？

可能原因：
- 有未確認的入金
- 有凍結的資產
- 交易費用被扣除
- 系統延遲更新

### 提幣顯示餘額不足？

可能原因：
- 部分餘額被凍結
- 需要保留手續費
- 餘額在其他帳戶
- 低於最小提幣限額

## 保護你的餘額

| 建議 | 說明 |
|------|------|
| 啟用 2FA | 增加帳戶安全性 |
| 設置提幣白名單 | 只能提到指定地址 |
| 定期提幣 | 不要在交易所存太多 |
| 檢查交易紀錄 | 發現異常及時處理 |
