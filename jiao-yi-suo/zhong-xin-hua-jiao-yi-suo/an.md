# 幣安

### 幣安（Binance）是全球交易量最大的加密貨幣交易所，由趙長鵬（CZ）於 2017 年創立。它以低手續費、豐富的幣種、和快速上新幣聞名，被稱為「幣圈的亞馬遜」——什麼都有，什麼都賣。

---

## 幣安簡史

| 時間 | 事件 |
|------|------|
| 2017 年 7 月 | 幣安成立，ICO 募資 1500 萬美元 |
| 2017 年 12 月 | 成為全球最大交易所 |
| 2019 年 | 推出 Binance Smart Chain |
| 2021 年 | 面臨多國監管壓力 |
| 2023 年 | CZ 因違反美國法律被罰款並辭職 |

{% hint style="warning" %}
**監管風波**

2023 年，CZ 因幣安違反美國洗錢法規被罰款 43 億美元，並辭去 CEO 職位。這提醒我們：即使是最大的交易所，也存在監管風險。
{% endhint %}

## BNB 代幣

幣安發行的平台幣 BNB（BNB Chain，原 Binance Coin）：

| 用途 | 說明 |
|------|------|
| 手續費折扣 | 用 BNB 付手續費打折 |
| Launchpad | 參與新幣認購需要持有 BNB |
| BNB Chain Gas | BNB 鏈的交易費用 |
| 銷毀機制 | 幣安定期回購銷毀 BNB |

{% hint style="info" %}
**從 ERC-20 到自己的鏈**

BNB 最初是以太坊上的 ERC-20 代幣。2019 年，幣安推出自己的區塊鏈 Binance Chain，後來又推出兼容以太坊的 BNB Smart Chain（BSC）。
{% endhint %}

## 幣安的優勢

| 優勢 | 說明 |
|------|------|
| 幣種最多 | 上架數百種加密貨幣 |
| 流動性最好 | 全球最大交易量 |
| 費用低 | 現貨交易 0.1%，用 BNB 更低 |
| 功能全面 | 從現貨到衍生品一站式服務 |
| 上幣快 | 新幣經常首發於幣安 |

---

## 幣安的爭議

| 爭議 | 說明 |
|------|------|
| 監管問題 | 多國禁止或限制幣安運營 |
| 洗錢指控 | 被美國司法部起訴 |
| 準備金透明度 | 曾被質疑儲備不足 |
| 市場操縱 | 被指控「洗交易量」 |

{% hint style="danger" %}
**風險提醒**

幣安雖大，但不代表絕對安全。2023 年的監管事件顯示，即使是行業巨頭也可能面臨嚴重的法律風險。建議不要在任何單一交易所存放過多資產。
{% endhint %}
