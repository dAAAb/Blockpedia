# 中心化交易所

### 中心化交易所（Centralized Exchange, CEX）是由一家公司運營的加密貨幣交易平台。它就像銀行或股票券商——你把錢存進去，透過他們的系統買賣，方便快捷，但你必須信任這家公司不會跑路或被駭。

---

## CEX 的運作方式

```
用戶 → 入金到交易所 → 交易所託管你的資產
                      ↓
              交易所內部撮合訂單
                      ↓
             買賣完成，更新帳戶餘額
```

{% hint style="info" %}
**你的幣不是你的幣**

當你把加密貨幣存入 CEX，你實際上是把私鑰交給了交易所。在交易所的帳本上，那些幣是你的；但在區塊鏈上，那些幣屬於交易所的錢包。
{% endhint %}

---

## CEX 的優勢

| 優勢 | 說明 |
|------|------|
| 速度快 | 訂單在內部撮合，不用等區塊確認 |
| 流動性好 | 大型 CEX 有大量交易者，容易成交 |
| 功能多 | 提供槓桿、期貨、借貸等進階功能 |
| 法幣入金 | 可以用信用卡或銀行轉帳買幣 |
| 用戶體驗 | 介面友善，適合新手 |

---

## CEX 的風險

| 風險 | 案例 |
|------|------|
| 交易所跑路 | FTX 破產（2022） |
| 駭客攻擊 | Mt. Gox 被盜（2014） |
| 資產凍結 | 交易所配合政府調查 |
| 系統故障 | 劇烈行情時當機 |
| 監管風險 | 被迫關閉或限制服務 |

{% hint style="danger" %}
**Not your keys, not your coins**

這是幣圈最重要的警語之一。只要你的幣放在 CEX，你就有「歸零風險」——交易所可能破產、被駭、或被凍結。
{% endhint %}

---

## CEX vs DEX

| 比較 | CEX | DEX |
|------|-----|-----|
| 託管 | 交易所託管 | 自己保管私鑰 |
| 速度 | 快 | 較慢（需等區塊確認）|
| 流動性 | 高 | 較低 |
| 隱私 | 需 KYC | 匿名 |
| 風險 | 交易所風險 | 智能合約風險 |
| 幣種 | 有限（需上架）| 無限（任何代幣）|

---

## 使用 CEX 的建議

| 建議 | 說明 |
|------|------|
| 選擇大所 | 優先選擇有信譽的大型交易所 |
| 啟用 2FA | 強化帳戶安全 |
| 不要存太多 | 只留交易需要的量 |
| 分散風險 | 不要把所有資產放在同一個交易所 |
| 定期提幣 | 長期持有的幣放到自己的錢包 |

{% hint style="success" %}
**最佳實踐**

把 CEX 當作「交易場所」，而不是「存款銀行」。買完幣就提到自己的錢包，需要交易時再存進去。
{% endhint %}
