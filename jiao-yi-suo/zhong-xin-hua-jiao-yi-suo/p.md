# P 網

### P 網是 Poloniex 的簡稱，是加密貨幣歷史上最具影響力的早期交易所之一。在 2017 年之前，P 網是山寨幣交易的聖地，以太坊、瑞波幣等許多知名幣種的早期流動性都在這裡。雖然現在已經式微，但 P 網在幣圈發展史上佔有重要地位。

---

## Poloniex 簡史

| 時間 | 事件 |
|------|------|
| 2014 年 | 在美國成立 |
| 2015 年 | 成為山寨幣交易主要平台 |
| 2017 年 | 被 Circle 收購 |
| 2018 年 | Circle 進行合規整改 |
| 2019 年 | 被孫宇晨收購 |
| 2023 年 | 遭駭客攻擊損失上億美元 |

{% hint style="info" %}
**幣圈的「老黃埔」**

很多幣圈老人的第一個交易所就是 P 網。在沒有幣安的年代，P 網就是山寨幣愛好者的天堂。
{% endhint %}

---

## P 網的黃金時代

### 為什麼 P 網曾經這麼重要？

| 因素 | 說明 |
|------|------|
| 先發優勢 | 最早支援多種山寨幣 |
| 幣種豐富 | 上架了大量新幣 |
| 槓桿交易 | 早期少數提供槓桿的平台 |
| 流動性好 | 曾是山寨幣流動性最佳的地方 |

```
2016 年的 P 網：
- 支援超過 100 種代幣
- ETH/BTC 交易量全球最大
- 是新幣首發的首選平台
```

---

## 為什麼叫「P 網」？

和 B 網一樣，是幣圈對交易所的簡稱習慣：

```
Poloniex → 取首字母 → P 網
Bittrex  → 取首字母 → B 網
```

{% hint style="warning" %}
**注意混淆**

「P 網」這種簡稱很容易被用於釣魚詐騙。不要點擊不明連結，務必確認是官方網址。
{% endhint %}

## Poloniex 的衰落

### 從龍頭到邊緣

```
2017：幣安上線
    ↓ 用戶快速流失
2018：Circle 收購後大量下架代幣
    ↓ 失去「山寨幣天堂」優勢
2019：孫宇晨收購
    ↓ 爭議不斷
2023：重大安全事件
    ↓ 信任崩潰
```

### 衰落的原因

| 因素 | 說明 |
|------|------|
| 競爭對手 | 幣安提供更好的體驗和更低費用 |
| 監管壓力 | 下架大量代幣失去特色 |
| 安全問題 | 多次發生安全事件 |
| 管理混亂 | 多次易主，方向不明 |

{% hint style="danger" %}
**2023 年駭客事件**

2023 年 11 月，Poloniex 遭受駭客攻擊，損失超過 1 億美元。這對本已式微的 P 網是沉重打擊。
{% endhint %}

---

## 現在還能用 P 網嗎？

雖然 Poloniex 仍在運營，但需要注意：

| 考量 | 建議 |
|------|------|
| 安全風險 | 近年多次被駭 |
| 流動性 | 遠不如主流交易所 |
| 監管不明 | 法律地位存疑 |
| 建議 | 僅用於特定需求，不宜存放大額資金 |
