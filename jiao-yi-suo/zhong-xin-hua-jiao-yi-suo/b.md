# B 網

### B 網是 Bittrex 的簡稱，曾經是美國合規交易所的代表，以嚴格的安全措施和豐富的山寨幣種類聞名。在幣圈早期，B 網和 P 網（Poloniex）是兩大主流交易所。但在監管壓力下，Bittrex 於 2023 年宣布關閉美國業務，成為加密貨幣交易所興衰的縮影。

---

## Bittrex 簡史

| 時間 | 事件 |
|------|------|
| 2014 年 | 在美國西雅圖成立 |
| 2017 年 | 成為全球前五大交易所 |
| 2018 年 | 首次被 SEC 調查 |
| 2022 年 | 被 SEC 正式起訴 |
| 2023 年 4 月 | 宣布關閉美國業務 |
| 2023 年 5 月 | 申請破產保護 |

{% hint style="warning" %}
**監管浪潮的犧牲者**

Bittrex 被 SEC 指控未註冊經營證券交易所，同時上架了多種被認定為「證券」的代幣。這個案例顯示了美國監管對加密貨幣的強硬態度。
{% endhint %}

---

## 為什麼叫「B 網」？

幣圈早期，為了方便稱呼，交易所常以首字母命名：

| 簡稱 | 交易所 |
|------|--------|
| B 網 | Bittrex |
| P 網 | Poloniex |
| A 網 | 幣安（Binance）或其他 |
| O 網 | OKEx |

{% hint style="info" %}
**小心釣魚網站**

「B 網」這種簡稱容易被詐騙者利用，創建假網站騙取用戶資產。務必直接輸入官方網址，不要點擊不明連結。
{% endhint %}

## Bittrex 的特色

### 優點（全盛時期）

| 特色 | 說明 |
|------|------|
| 安全性高 | 從未發生重大駭客事件 |
| 幣種豐富 | 上架數百種山寨幣 |
| 美國合規 | 持有多州牌照 |
| 介面專業 | 適合進階交易者 |

### 缺點

| 問題 | 說明 |
|------|------|
| KYC 嚴格 | 驗證過程繁瑣 |
| 客服差 | 回覆慢、問題難解決 |
| 手續費高 | 0.25% 交易費 |
| 流動性下降 | 後期交易量萎縮 |

## Bittrex 的衰落

### 為什麼從巔峰走向關閉？

```
2017：巔峰時期
    ↓ 幣安崛起，用戶流失
2018：監管開始收緊
    ↓ 下架大量代幣
2019-2021：市占率持續下滑
    ↓ 無法與頭部交易所競爭
2022：SEC 起訴
    ↓ 法律成本飆升
2023：宣布關閉、申請破產
```

{% hint style="danger" %}
**前車之鑑**

Bittrex 的結局提醒我們：即使是老牌交易所，也可能因為監管、競爭等因素突然關閉。永遠不要在任何單一交易所存放所有資產。
{% endhint %}

---

## 對用戶的影響

Bittrex 關閉時，用戶需要：

1. 在截止日期前提領所有資產
2. 完成 KYC 才能提款
3. 部分用戶因各種原因無法提領

這再次印證了「Not your keys, not your coins」的重要性。
