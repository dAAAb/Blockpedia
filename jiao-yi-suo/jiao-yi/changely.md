# Changelly

### Changelly 是 2015 年成立的加密貨幣兌換服務，不是傳統交易所，而是「聚合器」——它從多個交易所找到最佳匯率，讓你一鍵完成幣種兌換。不需要複雜的交易介面，也不需要註冊帳戶，輸入要換的幣、貼上錢包地址、付款，幾分鐘就完成。

---

## Changelly 是什麼？

### 聚合器 vs 交易所

```
傳統交易所：
註冊 → KYC → 入金 → 下單 → 等待成交 → 提幣

Changelly：
選擇幣種 → 輸入地址 → 付款 → 收幣
```

Changelly 在背後連接多個交易所，幫你找到最佳價格並自動執行交易。

{% hint style="info" %}
**適合誰？**

Changelly 適合想要「快速換幣」的用戶，不想處理複雜的交易所操作。但通常匯率會比直接在交易所交易差一點。
{% endhint %}

---

## Changelly 的運作方式

```
1. 你選擇：把 1 BTC 換成 ETH
2. Changelly 顯示預估匯率
3. 你輸入 ETH 收款地址
4. Changelly 給你一個 BTC 存款地址
5. 你發送 BTC
6. Changelly 在後台交易所換成 ETH
7. ETH 發送到你的地址
```

整個過程通常 5-30 分鐘。

## 優缺點分析

### 優點

| 優點 | 說明 |
|------|------|
| 方便快速 | 幾分鐘完成兌換 |
| 隱私性 | 小額不需要 KYC |
| 簡單 | 不用學習交易介面 |
| 支援多幣種 | 可以換到小眾幣 |

### 缺點

| 缺點 | 說明 |
|------|------|
| 匯率較差 | 比直接交易所貴 |
| 隱藏費用 | 費用包含在匯率中 |
| 客服問題 | 出問題時解決較慢 |
| 不適合大額 | 大額交易成本高 |

{% hint style="warning" %}
**注意匯率差距**

Changelly 的匯率通常比直接在交易所交易差 1-5%。金額越大，損失越多。建議先比較一下價格。
{% endhint %}
