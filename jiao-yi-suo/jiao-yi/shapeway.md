# ShapeShift

### ShapeShift 是 2014 年成立的加密貨幣兌換服務，由 Erik Voorhees 創辦。它曾經是「無需註冊即可換幣」的代名詞，讓用戶可以在不同加密貨幣之間快速兌換。2021 年，ShapeShift 宣布完全去中心化，將公司解散並將治理權交給 FOX 代幣持有者。

---

## ShapeShift 簡史

| 時間 | 事件 |
|------|------|
| 2014 年 | Erik Voorhees 創立 ShapeShift |
| 2014-2018 年 | 無需 KYC 的換幣服務爆紅 |
| 2018 年 | 因監管壓力被迫實施 KYC |
| 2021 年 | 宣布去中心化，解散公司 |
| 2021 年 | 發行 FOX 治理代幣 |

{% hint style="info" %}
**加密朋克精神**

ShapeShift 的創辦人 Erik Voorhees 是知名的加密貨幣倡導者，堅信隱私和金融自由。當他被迫實施 KYC 時，選擇了一條激進的道路：直接把公司去中心化。
{% endhint %}

## 傳統的 ShapeShift（2014-2021）

### 運作方式

```
1. 選擇要兌換的幣種對（如 BTC → ETH）
2. 輸入接收地址
3. ShapeShift 生成一次性存款地址
4. 發送 BTC 到該地址
5. ShapeShift 自動將 ETH 發送到你的地址
```

### 特色

| 特色 | 說明 |
|------|------|
| 即時兌換 | 無需等待訂單撮合 |
| 無需帳戶 | 不用註冊（2018 年前）|
| 多幣種 | 支援數十種加密貨幣 |
| 簡單介面 | 新手也能輕鬆使用 |

## 去中心化的 ShapeShift（2021 後）

### FOX 代幣

| 項目 | 說明 |
|------|------|
| 名稱 | FOX Token |
| 類型 | 治理代幣 |
| 用途 | 投票決定平台發展方向 |
| 空投 | 對早期用戶進行大規模空投 |

### 現在的 ShapeShift

現在的 ShapeShift 是一個開源的多鏈聚合器：

- 聚合多個 DEX 的流動性
- 支援多條區塊鏈
- 完全去中心化運營
- 由 DAO 治理

{% hint style="success" %}
**真正的去中心化**

ShapeShift 是少數真正「解散公司、交出控制權」的案例。不是假裝去中心化，而是真的把代碼開源、把治理權交給社群。
{% endhint %}

## 為什麼要去中心化？

### 監管壓力

2018 年，ShapeShift 被迫實施 KYC，違背了創辦人的理念。

```
監管要求 KYC
    ↓
用戶流失（大家去用不要 KYC 的服務）
    ↓
商業模式受損
    ↓
決定去中心化
    ↓
沒有公司 = 沒有監管對象
```

### 對產業的意義

ShapeShift 的去中心化實驗證明：

- 公司可以轉型為 DAO
- 代幣可以用來分配治理權
- 去中心化是一種「監管套利」
