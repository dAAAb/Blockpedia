# OTCBTC

### OTCBTC 是 2017 年成立的加密貨幣交易平台，主打場外交易（OTC）和幣幣交易。它的名字就說明了一切：OTC + BTC。在中國禁止加密貨幣交易所後，OTCBTC 成為華語用戶繞過禁令的重要管道之一，但後來也難逃監管壓力。

---

## OTCBTC 簡史

| 時間 | 事件 |
|------|------|
| 2017 年 9 月 | 中國禁止 ICO 和交易所 |
| 2017 年 10 月 | OTCBTC 上線 |
| 2017-2018 年 | 快速成長，成為華語 OTC 主要平台 |
| 2019 年後 | 競爭加劇，監管壓力增大 |
| 近年 | 逐漸式微 |

{% hint style="info" %}
**時代產物**

OTCBTC 的崛起是中國監管政策的直接結果。當交易所被禁止，OTC 平台成為用戶的替代選擇。
{% endhint %}

---

## OTCBTC 的特色

### 1. 場外交易（OTC）

買賣雙方直接交易，平台提供擔保：

```
賣家掛單：「1 BTC，價格 45,000 USDT」
買家下單
賣家的 BTC 被鎖定（託管）
買家付款給賣家
賣家確認收款
平台釋放 BTC 給買家
```

### 2. 幣幣交易

支援多種加密貨幣之間的交易。

### 3. 華語介面

主要服務中國大陸、台灣、香港等華語地區用戶。

---

## OTC 平台的擔保機制

| 機制 | 說明 |
|------|------|
| 資金託管 | 賣家的幣被平台暫時鎖定 |
| 身分驗證 | 減少詐騙風險 |
| 評價系統 | 用戶可以看到對方的交易記錄 |
| 客服介入 | 發生糾紛時平台仲裁 |

{% hint style="warning" %}
**OTC 交易的風險**

即使有平台擔保，OTC 交易仍有風險：
- 對方可能使用贓款付款（你的銀行卡可能被凍結）
- 假轉帳截圖詐騙
- 平台本身的信用風險
{% endhint %}

---

## 為什麼 OTCBTC 式微了？

| 因素 | 說明 |
|------|------|
| 競爭激烈 | 幣安 P2P、火幣 OTC 等巨頭進場 |
| 監管壓力 | 中國持續打擊加密貨幣交易 |
| 銀行管控 | 涉及加密貨幣的帳戶被凍結 |
| 信任問題 | 小平台難以和大所競爭信任度 |

---

## OTC 交易的現況

現在華語用戶更常使用大型交易所的 P2P 功能：

| 平台 | 說明 |
|------|------|
| 幣安 P2P | 流動性最好 |
| OKX P2P | 另一個主流選擇 |
| 本地交易所 | 如台灣的 MAX、BitoPro |

---

## 對用戶的啟示

| 教訓 | 說明 |
|------|------|
| 平台可能消失 | 不要在任何平台存太多資金 |
| 監管風險 | 政策變化可能影響平台運營 |
| 選擇大平台 | 流動性和信任度更有保障 |
| 了解風險 | OTC 交易有其特有的風險 |
