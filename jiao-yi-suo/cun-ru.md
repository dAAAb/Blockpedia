# 入金

### 入金（Deposit）是將資金存入交易所帳戶的動作。就像把錢存進銀行才能用 ATM，你要先把錢「入」進交易所，才能開始買賣加密貨幣。入金分為「法幣入金」和「幣幣入金」兩種方式。

---

## 入金的類型

### 1. 法幣入金

用傳統貨幣（如台幣、美元）直接存入交易所。

| 方式 | 說明 | 手續費 | 到帳時間 |
|------|------|--------|----------|
| 銀行轉帳 | 最常見的方式 | 通常免費 | 1-3 工作天 |
| 信用卡 | 快速但費用高 | 3-5% | 即時 |
| 電匯 | 國際轉帳 | $15-50 | 1-5 工作天 |
| 第三方支付 | 如 ePay、街口 | 視平台而定 | 即時~1天 |

### 2. 幣幣入金（充幣）

將已有的加密貨幣從外部錢包轉入交易所。

```
你的錢包 → 複製交易所充幣地址 → 發送交易 → 等待確認 → 入帳
```

{% hint style="warning" %}
**選對網路！**

轉幣時一定要選對網路。比如 USDT 有 ERC-20（以太坊）、TRC-20（波場）、BEP-20（幣安鏈）等版本。選錯網路，幣可能永久丟失！
{% endhint %}

## 法幣入金的常見管道

### 台灣地區

| 交易所 | 入金方式 |
|--------|----------|
| MAX | 台幣銀行轉帳、超商繳費 |
| BitoPro | 台幣銀行轉帳 |
| ACE | 台幣銀行轉帳 |

### 國際交易所

| 交易所 | 入金方式 |
|--------|----------|
| 幣安 | P2P、信用卡、第三方 |
| Coinbase | 銀行轉帳、信用卡 |
| Kraken | 銀行轉帳、電匯 |

{% hint style="info" %}
**台灣用戶的選擇**

國際大所通常不支援台幣直接入金。常見做法是：先在台灣交易所買好 USDT，再轉到國際交易所使用。
{% endhint %}

## 入金流程（以銀行轉帳為例）

```
1. 登入交易所，進入「入金」頁面
2. 選擇「台幣入金」或「銀行轉帳」
3. 獲取專屬虛擬帳號（每人不同）
4. 使用網銀或 ATM 轉帳至該帳號
5. 等待交易所確認入帳（通常 1-24 小時）
6. 帳戶餘額更新，可以開始交易
```

{% hint style="danger" %}
**務必使用本人帳戶**

大多數交易所要求入金帳戶與 KYC 身分一致。用別人的帳戶轉帳可能導致資金被凍結或退回。
{% endhint %}

---

## 充幣流程（幣幣入金）

```
1. 登入交易所，進入「充幣」頁面
2. 選擇要充值的幣種（如 BTC、ETH）
3. 選擇正確的網路（如 ERC-20）
4. 複製充幣地址或掃描 QR Code
5. 從你的錢包發送交易
6. 等待區塊確認（BTC 約 1 小時，ETH 約 5 分鐘）
7. 確認數達標後自動入帳
```

---

## 入金的注意事項

| 項目 | 說明 |
|------|------|
| 最低入金 | 部分交易所有最低金額限制 |
| 入金限額 | KYC 等級可能影響每日限額 |
| 手續費 | 部分方式會收取費用 |
| 到帳時間 | 依方式和網路擁堵程度而定 |

## 常見問題

### 入金一直沒到帳？

可能原因：
- 銀行處理需要時間（非營業時間更久）
- 區塊鏈網路擁堵
- 充幣地址或網路選錯
- 低於最低入金金額

### 充幣選錯網路怎麼辦？

如果選錯網路：
- 同一交易所支援的網路：可能可以找回（收取高額手續費）
- 完全不支援的網路：資產可能永久丟失

{% hint style="success" %}
**小額測試**

第一次入金或充幣時，建議先用小額測試，確認流程正確後再轉大筆金額。這個習慣可以避免很多悲劇。
{% endhint %}
