# 出金

### 出金（Withdrawal）是將資金從交易所帳戶轉出的動作。賣掉幣之後，你需要「出金」才能把錢拿回銀行帳戶；或者「提幣」把加密貨幣轉到自己的錢包。出金是交易獲利的最後一哩路。

---

## 出金的類型

### 1. 法幣出金

將交易所帳戶中的法幣（如台幣、美元）提領到銀行帳戶。

```
交易所餘額 → 申請提領 → 審核 → 銀行轉帳 → 到帳
```

### 2. 加密貨幣提幣

將加密貨幣從交易所轉到外部錢包。

```
交易所餘額 → 輸入錢包地址 → 選擇網路 → 確認 → 區塊確認 → 到帳
```

{% hint style="info" %}
**為什麼要提幣？**

「Not your keys, not your coins」——放在交易所的幣，其實是交易所幫你保管。只有提到自己的錢包，才真正擁有這些幣。
{% endhint %}

---

## 法幣出金流程

| 步驟 | 說明 |
|------|------|
| 1. 綁定銀行帳戶 | 通常需要與 KYC 身分一致 |
| 2. 申請提領 | 輸入提領金額 |
| 3. 安全驗證 | 2FA、郵件確認等 |
| 4. 交易所審核 | 可能需要 1-24 小時 |
| 5. 銀行處理 | 1-3 個工作天 |
| 6. 資金到帳 | 查看銀行餘額 |

---

## 提幣流程

```
1. 進入「提幣」頁面
2. 選擇幣種（如 BTC、ETH、USDT）
3. 輸入目標錢包地址
4. 選擇正確的網路
5. 輸入提幣數量
6. 完成安全驗證（2FA、郵件）
7. 等待區塊確認
```

{% hint style="danger" %}
**地址和網路都要對！**

提幣地址輸錯 = 錢永遠拿不回來
網路選錯 = 幣可能永遠丟失

務必三重確認！
{% endhint %}

## 出金手續費

### 法幣出金

| 交易所 | 手續費 |
|--------|--------|
| 台灣交易所 | 通常免費或 $15-30 |
| 國際交易所 | 電匯 $25-50 |

### 加密貨幣提幣

| 幣種 | 網路 | 約略手續費 |
|------|------|------------|
| BTC | Bitcoin | 0.0001-0.0005 BTC |
| ETH | Ethereum | 0.001-0.01 ETH |
| USDT | TRC-20 | 1 USDT |
| USDT | ERC-20 | 5-20 USDT |

{% hint style="success" %}
**省手續費小技巧**

USDT 使用 TRC-20 網路通常最便宜，只要 1 USDT。ERC-20 在網路擁堵時可能高達 20 USDT 以上。
{% endhint %}

## 出金限制

交易所通常會設定各種限制：

| 限制類型 | 說明 |
|----------|------|
| 每日限額 | 依 KYC 等級而定 |
| 最低提幣 | 低於門檻無法提領 |
| 提幣白名單 | 只能提到預設地址 |
| 冷卻期 | 新地址可能需等待 24 小時 |

## 常見問題

### 出金要多久到帳？

| 類型 | 時間 |
|------|------|
| 法幣（台灣） | 1-3 工作天 |
| 法幣（國際電匯） | 3-5 工作天 |
| BTC 提幣 | 30-60 分鐘 |
| ETH 提幣 | 5-15 分鐘 |
| USDT (TRC-20) | 1-5 分鐘 |

### 為什麼出金被拒絕？

可能原因：
- KYC 未完成
- 銀行帳戶與身分不符
- 帳戶有異常活動
- 餘額不足（包含手續費）
- 超過每日限額

### 出金被凍結怎麼辦？

- 聯繫客服了解原因
- 提供要求的身分文件
- 配合風控調查
- 等待審核解凍

{% hint style="warning" %}
**大額出金注意**

大額法幣出金可能觸發銀行的洗錢防制（AML）審查。保留好交易紀錄，必要時能解釋資金來源。
{% endhint %}

## 安全提幣習慣

| 建議 | 說明 |
|------|------|
| 使用白名單 | 預設安全的提幣地址 |
| 小額測試 | 大額提幣前先測試 |
| 三重確認 | 地址、網路、金額都要對 |
| 開啟 2FA | 增加一層安全保護 |
| 定期提幣 | 不要在交易所存太多 |
