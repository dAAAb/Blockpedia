# Discord

### Discord 是幣圈的「會員俱樂部」。如果 Twitter 是公開廣場，Discord 就是私人會所。在這裡，項目方和社群直接互動，重要消息第一時間公布，真正的 alpha 資訊流動。不會用 Discord，在幣圈就像不會用 Email 一樣跛腳。

---

## Discord 是什麼？

Discord 原本是為遊戲玩家設計的語音和文字聊天平台，創立於 2015 年。但在 2020 年 NFT 熱潮後，它成為了加密貨幣社群的標準配備。

```
Discord 的結構：
伺服器（Server） → 頻道（Channel） → 訊息（Message）

就像：
一棟大樓 → 不同房間 → 房間裡的對話
```

| 功能 | 說明 |
|------|------|
| 文字頻道 | 即時文字聊天 |
| 語音頻道 | 即時語音通話 |
| 舞台頻道 | 類似 Clubhouse 的演講模式 |
| 角色系統 | 不同身份有不同權限 |

## 為什麼幣圈愛用 Discord？

### 1. 高度客製化

```
典型 NFT 項目 Discord 結構：

📢 公告區
├── announcements（重要公告）
├── roadmap（發展路線）
└── partnerships（合作消息）

💬 討論區
├── general（一般聊天）
├── alpha-chat（投資討論）
└── price-talk（價格討論）

🎨 NFT 相關
├── gallery（展示 NFT）
├── rarity-check（稀有度查詢）
└── secondary-market（二級市場）

🔒 持有者專區
├── holder-chat（限持有者）
└── airdrops（空投資訊）
```

### 2. 機器人整合

| 機器人 | 功能 |
|--------|------|
| Collab.Land | 驗證 NFT 持有者 |
| MEE6 | 等級系統、自動管理 |
| Carl-bot | 角色管理、反垃圾 |
| Premint | 白名單抽獎 |

### 3. 即時互動

{% hint style="info" %}
**AMA（Ask Me Anything）**

很多項目會在 Discord 舉辦 AMA，創辦人直接回答社群問題。這種透明度在傳統金融市場很少見。
{% endhint %}

## 如何使用 Discord？

### 基本操作

1. **加入伺服器**：通過邀請連結
2. **驗證身份**：完成驗證流程（防機器人）
3. **選擇角色**：獲取想要追蹤的頻道通知
4. **開始互動**：閱讀規則後參與討論

### 進階技巧

| 技巧 | 說明 |
|------|------|
| 設定通知 | 只開啟重要頻道的通知 |
| 使用搜尋 | 用 `from:` `in:` 等過濾器 |
| 建立資料夾 | 整理不同類型的伺服器 |
| 設定暱稱 | 每個伺服器可用不同名稱 |

## Discord 在幣圈的重要性

### 獲取第一手資訊

```
項目有重大更新
    ↓
Discord 公告（最快）
    ↓
Twitter 發布
    ↓
媒體報導
    ↓
散戶知道
```

時間差可能是幾分鐘到幾小時，在幣圈這足以影響交易決策。

### 白名單和空投

很多 NFT 項目會透過 Discord 活動發放白名單：

| 方式 | 說明 |
|------|------|
| 活躍度 | 聊天等級達標 |
| 抽獎 | 參與 Premint 等工具 |
| 競賽 | 創作、推廣競賽 |
| 邀請 | 邀請朋友加入 |

## Discord 的風險

{% hint style="danger" %}
**Discord 是詐騙重災區**

1. **假冒管理員**：詐騙者會用相似名稱私訊你
2. **釣魚連結**：假的 mint 網站、假空投
3. **伺服器被駭**：官方伺服器被入侵發布假消息
4. **DM 詐騙**：任何私訊要求你連接錢包都是詐騙
{% endhint %}

### 保護自己

| 做法 | 說明 |
|------|------|
| 關閉 DM | 設定 → 隱私 → 關閉伺服器成員私訊 |
| 驗證連結 | 只用官方 Twitter 上的連結 |
| 不急著點 | FOMO 是詐騙者最好的朋友 |
| 用小錢包 | 測試用的錢包別放大錢 |
