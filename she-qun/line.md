# LINE

### LINE 是台灣人的「國民通訊軟體」。雖然幣圈國際社群主要用 Telegram，但在台灣，很多本地的幣圈社群和交易群還是會用 LINE。LINE 本身也積極進軍區塊鏈領域，推出了 LINE Blockchain 和 LINK 代幣。

---

## LINE 是什麼？

LINE 由韓國 Naver 公司於 2011 年推出，在日本、台灣、泰國等地非常流行。

| 特色 | 說明 |
|------|------|
| 貼圖文化 | 豐富的表情貼圖 |
| 官方帳號 | 企業和品牌使用 |
| LINE Pay | 行動支付 |
| LINE Points | 點數回饋系統 |

---

## LINE 在台灣幣圈的角色

### 本地社群的選擇

```
台灣幣圈的平台選擇：

國際項目討論 → Telegram
本地社群交流 → LINE
新聞追蹤 → Twitter
深度研究 → Discord
```

| 用途 | 說明 |
|------|------|
| 交易所官方群 | MAX、ACE 等有 LINE 官方帳號 |
| 投資社群 | 私人投資群多用 LINE |
| 教學課程 | 很多課程用 LINE 作為聯繫工具 |
| 在地活動 | 線下活動通知和聯繫 |

### 為什麼台灣人用 LINE？

| 原因 | 說明 |
|------|------|
| 習慣 | 台灣人日常就用 LINE |
| 年齡層 | 較年長的投資者更習慣 LINE |
| 本地化 | 很多台灣服務都整合 LINE |

{% hint style="info" %}
**年齡和平台選擇**

觀察顯示，台灣幣圈中較年長的投資者傾向用 LINE，較年輕的和比較國際化的用戶則偏好 Telegram。
{% endhint %}

## LINE 的區塊鏈佈局

### LINE Blockchain

LINE 在 2018 年開始發展自己的區塊鏈生態：

| 產品 | 說明 |
|------|------|
| LINE Blockchain | 自有區塊鏈平台 |
| LINK (LN) | LINE 的加密貨幣 |
| LINE BITMAX | 日本的加密貨幣交易所 |
| LINE NFT | NFT 市場 |

```
LINE 區塊鏈生態：

LINE Blockchain（基礎設施）
    ↓
LINK 代幣（價值交換）
    ↓
LINE BITMAX（交易）+ LINE NFT（收藏品）
    ↓
與 LINE 服務整合（支付、點數）
```

### LINK 代幣

| 資訊 | 說明 |
|------|------|
| 代號 | LN |
| 用途 | LINE 生態系統內的獎勵和支付 |
| 上線交易所 | LINE BITMAX（日本）、部分國際交易所 |

{% hint style="warning" %}
**台灣不能直接使用**

LINE BITMAX 目前只在日本營運，台灣用戶無法直接使用。LINK 代幣在台灣的流通和使用也受限。
{% endhint %}

## LINE 社群的特色

### 優點

| 優點 | 說明 |
|------|------|
| 使用習慣 | 不需要另外下載 App |
| 貼圖表達 | 溝通更生動 |
| 已讀功能 | 知道訊息是否被看到 |
| 記事本/相簿 | 群組資料管理 |

### 缺點

| 缺點 | 說明 |
|------|------|
| 群組上限 | 最多 500 人 |
| 搜尋功能 | 比 Telegram 弱 |
| 機器人 | 功能比 Telegram Bot 少 |
| 隱私性 | 不如 Telegram 的加密 |

---

## LINE vs Telegram

| 比較 | LINE | Telegram |
|------|------|----------|
| 群組上限 | 500 人 | 20 萬人 |
| 機器人 | 有限 | 非常強大 |
| 頻道功能 | 官方帳號 | 頻道 |
| 加密 | 部分 | 端到端可選 |
| 幣圈生態 | 台灣本地 | 國際主流 |

## 如何安全使用 LINE 幣圈社群？

### 注意事項

| 風險 | 建議 |
|------|------|
| 詐騙群 | 只加入可信來源的群組 |
| 假客服 | 交易所不會用 LINE 私訊要求轉帳 |
| 投資騙局 | 保證獲利的都是騙局 |
| 個資外洩 | 不要在群組公開個人資料 |

{% hint style="danger" %}
**LINE 詐騙常見手法**

1. **假交易所客服**：「您的帳戶有問題，請轉帳驗證」
2. **假投資群**：「跟著老師操作，穩賺不賠」
3. **假朋友**：盜用朋友帳號借錢
4. **假活動**：「點此連結領取空投」

記住：天下沒有白吃的午餐。
{% endhint %}
