# WeChat

### 微信是中國互聯網的「超級 App」，也是中國幣圈人交流的主要工具。但由於中國嚴格的加密貨幣監管，微信上的幣圈討論必須極其小心。公開討論交易可能導致封號，甚至更嚴重的後果。這是一個在夾縫中生存的社群生態。

---

## WeChat 是什麼？

微信由騰訊於 2011 年推出，是中國使用人數最多的社交應用，月活躍用戶超過 12 億。

```
微信的功能：

通訊：聊天、語音、視訊
社交：朋友圈、公眾號
支付：微信支付
服務：小程序、生活繳費
```

| 功能 | 說明 |
|------|------|
| 私聊 | 一對一通訊 |
| 群聊 | 最多 500 人群組 |
| 朋友圈 | 類似 Facebook 動態 |
| 公眾號 | 類似訂閱的內容發布 |

## 中國的加密貨幣監管

### 為什麼微信幣圈很敏感？

```
中國監管時間線：

2017 年 9 月：禁止 ICO 和國內交易所
2021 年 5 月：禁止金融機構提供加密服務
2021 年 9 月：全面禁止加密貨幣交易和挖礦
```

| 禁止項目 | 說明 |
|----------|------|
| 交易所 | 國內交易所全部關閉 |
| 挖礦 | 礦場遷移海外 |
| OTC | 銀行凍結涉幣帳戶 |
| 宣傳 | 不得推廣加密貨幣 |

{% hint style="danger" %}
**嚴重警告**

在微信上公開討論加密貨幣交易、發布交易廣告、或組織投資群，可能導致：
- 微信帳號永久封禁
- 支付寶/銀行帳戶凍結
- 嚴重者可能涉及法律問題

這不是開玩笑的事情。
{% endhint %}

## 微信幣圈的現況

### 如何在夾縫中生存？

| 方式 | 說明 |
|------|------|
| 暗語 | 用「U」代表 USDT，「大餅」代表 BTC |
| 私聊 | 避免在群組公開討論 |
| 切換平台 | 敏感討論轉移到 Telegram |
| 行業群 | 以「區塊鏈技術」名義運營 |

### 常見暗語

```
幣圈微信暗語：

大餅 = 比特幣 (BTC)
二餅 = 以太坊 (ETH)
U = USDT
搬磚 = 套利交易
上車 = 買入
下車 = 賣出
韭菜 = 散戶
割韭菜 = 莊家收割散戶
```

## 微信公眾號的角色

### 幣圈內容發布

即使監管嚴格，仍有一些公眾號在發布區塊鏈相關內容：

| 類型 | 說明 |
|------|------|
| 技術類 | 區塊鏈技術、智能合約教學 |
| 行業類 | 產業新聞、監管動態 |
| 避開 | 投資建議、交易策略 |

{% hint style="warning" %}
**公眾號的風險**

涉及加密貨幣投資內容的公眾號經常被封禁。很多知名幣圈公眾號已經消失或轉型。
{% endhint %}

## 微信 vs 其他平台

| 比較 | 微信 | Telegram |
|------|------|----------|
| 監管 | 嚴格，內容審查 | 寬鬆，相對自由 |

## 中國幣圈人的選擇

### 平台遷移

| 平台 | 用途 |
|------|------|
| Telegram | 主要討論、交易群 |
| Twitter/X | 追蹤國際動態 |
| 微信 | 日常聯繫、行業社交 |

```
典型中國幣圈人的工具組合：

日常生活：微信
幣圈討論：Telegram
新聞追蹤：Twitter
交易操作：VPN + 海外交易所
```
