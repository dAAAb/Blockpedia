# Portal Network

### 0x7a3b...4f2c 太難記了吧？Portal Network 是區塊鏈世界的電話簿，讓你用簡單易記的名稱取代那串又臭又長的錢包地址。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。

---

## 什麼是 Portal Network

| 項目 | 說明 |
|------|------|
| 類型 | 去中心化域名服務 |
| 功能 | 區塊鏈地址命名 |
| 類比 | 傳統 DNS 的區塊鏈版本 |
| 目標 | 改善 Web3 使用體驗 |

---

## 解決的問題

### 為什麼需要區塊鏈域名

```
傳統地址的問題：

難記憶：
- 錢包地址：0x7a3b5f...
- 42 個字元的亂碼
- 容易打錯

難驗證：
- 無法一眼確認是否正確
- 釣魚網站容易偽造
- 轉帳錯誤不可逆

解決方案：
- alice.eth → 0x7a3b5f...
- 人類可讀
- 容易記憶和驗證
```

### 與 ENS 的關係

| 項目 | ENS | Portal Network |
|------|-----|----------------|
| 全名 | Ethereum Name Service | Portal Network |
| 鏈支援 | 以太坊為主 | 多鏈支援 |
| 知名度 | 最大的區塊鏈域名 | 早期探索者 |

## 應用場景

### 使用方式

```
區塊鏈域名的應用：

轉帳：
- 不用複製貼上長地址
- 直接輸入名稱
- 減少錯誤風險

身份：
- 作為 Web3 身份標識
- 社群媒體顯示
- 個人品牌

網站：
- 去中心化網站域名
- 搭配 IPFS 使用
- 抗審查網站
```

### 實際體驗

| 功能 | 說明 |
|------|------|
| 註冊域名 | 選擇並購買喜歡的名稱 |
| 設定指向 | 將域名指向錢包地址 |
| 跨平台使用 | 在支援的錢包和 DApp 使用 |

{% hint style="info" %}
**Web3 使用體驗**

區塊鏈域名是改善 Web3 使用者體驗的重要一步。當你可以對朋友說「轉帳到 alice.eth」而不是「轉帳到 0x7a3b...」時，區塊鏈就離普及更近了一步。
{% endhint %}
