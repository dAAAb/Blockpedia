# ENS（以太坊域名服務）

## ENS（Ethereum Name Service）讓你可以用 「vitalik.eth」 這樣好記的名字，代替 「0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045」 這樣的錢包地址。它是 Web3 世界的 DNS——去中心化的身份系統。

你的錢包地址是什麼？

```
0x742d35Cc6634C0532925a3b844Bc9e7595f8fAbc
```

你記得住嗎？

**當然記不住。**

這就是為什麼我們需要 ENS。

### 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | Ethereum Name Service |
| 簡稱 | ENS |
| 代幣 | ENS |
| 上線 | 2017 年 |
| 創辦人 | Nick Johnson |
| 類型 | 去中心化域名系統 |

### ENS 是什麼？

**簡單說**：把難記的地址變成好記的名字。

| 沒有 ENS | 有 ENS |
|----------|--------|
| 0xd8dA6BF26964aF9D7eEd9e03E53415D37aA96045 | vitalik.eth |
| 0x742d35Cc6634C0532925a3b844Bc9e7595f8fAbc | yourname.eth |

**就像網際網路的 DNS**：

| 系統 | 轉換 |
|------|------|
| DNS | google.com → 142.250.185.46 |
| ENS | vitalik.eth → 0xd8dA... |

### 為什麼 ENS 很重要？

#### 1. 使用體驗

轉帳時：

```
沒有 ENS：
「請把 ETH 轉到 0x742d35Cc6634C0532925a3b844Bc9e7595f8fAbc」
「等等，讓我再確認一次...」

有 ENS：
「請把 ETH 轉到 alice.eth」
「好的！」
```

#### 2. Web3 身份

你的 `.eth` 域名可以是你的：

- **錢包地址**：接收任何加密貨幣
- **網站地址**：託管去中心化網站
- **身份標識**：社交媒體、頭像、個人資訊
- **數位名片**：所有資訊的集合點

#### 3. 去中心化

| DNS | ENS |
|-----|-----|
| 由 ICANN 管理 | 由智能合約管理 |
| 可以被審查 | 難以審查 |
| 中心化 | 去中心化 |

{% hint style="info" %}
很多人把 ENS 域名當作 Web3 身份。你的 Twitter 顯示名稱是 `alice.eth`，大家就知道你是加密貨幣圈的人。
{% endhint %}

### 如何使用 ENS？

#### 註冊域名

1. 前往 [app.ens.domains](https://app.ens.domains)
2. 搜尋你想要的名字
3. 連接錢包
4. 支付註冊費 + Gas 費
5. 完成！

#### 費用

| 域名長度 | 年費（約）|
|----------|----------|
| 5+ 字元 | $5/年 |
| 4 字元 | $160/年 |
| 3 字元 | $640/年 |

**另外**：每次操作都需要支付以太坊 Gas 費。

#### 設定

註冊後，你可以設定：

| 記錄 | 用途 |
|------|------|
| ETH 地址 | 接收 ETH |
| BTC 地址 | 接收 BTC |
| 頭像 | 可連結 NFT |
| 文字記錄 | 社交媒體、Email 等 |
| 內容 | IPFS 網站 |

### ENS 代幣

2021 年 11 月，ENS 發行了治理代幣並空投給早期用戶。

#### 空投

| 條件 | 說明 |
|------|------|
| 資格 | 2021/10/31 前註冊過 ENS 域名 |
| 數量 | 根據持有時間、域名數量計算 |
| 最大 | 有人獲得價值數萬美元的空投 |

{% hint style="success" %}
這是加密貨幣史上最慷慨的空投之一。很多只花了 $5-10 註冊域名的人，收到了價值數千美元的 ENS 代幣。
{% endhint %}

#### 代幣用途

| 用途 | 說明 |
|------|------|
| **治理** | 投票決定 ENS 的發展方向 |
| **委託** | 把投票權委託給別人 |

**注意**：ENS 代幣不是用來支付域名費用的。

### ENS 的技術

#### 智能合約架構

```
用戶查詢 alice.eth
        ↓
    Registry（註冊表）
    「alice.eth 的 Resolver 是什麼？」
        ↓
    Resolver（解析器）
    「alice.eth 的 ETH 地址是...」
        ↓
    返回地址
```

#### 子域名

你可以創建子域名：

```
alice.eth        （你的主域名）
├── pay.alice.eth    （支付用）
├── nft.alice.eth    （NFT 用）
└── blog.alice.eth   （部落格）
```

#### 跨鏈支援

ENS 可以解析多種區塊鏈地址：

- ETH
- BTC
- LTC
- DOGE
- 更多...

### ENS vs 其他域名服務

| 服務 | 區塊鏈 | 特點 |
|------|--------|------|
| **ENS** | Ethereum | 最大、最多整合 |
| Unstoppable Domains | 多鏈 | 一次性付費 |
| Bonfida (.sol) | Solana | Solana 生態 |
| Handshake (HNS) | 自有鏈 | 頂級域名 |

### 投資 ENS 的考量

#### 域名投資

| 優勢 | 風險 |
|------|------|
| 稀缺性 | 需求不確定 |
| Web3 成長 | 年費成本 |
| 品牌價值 | 流動性問題 |

#### ENS 代幣

| 優勢 | 風險 |
|------|------|
| Web3 基礎設施 | 僅治理功能 |
| 協議收入成長 | 競爭對手 |
| 社群強大 | 代幣經濟有限 |

{% hint style="warning" %}
ENS 代幣主要是治理代幣，沒有直接的收益分配。它的價值來自對 ENS 協議未來的預期，而不是直接的現金流。
{% endhint %}

### ENS 的未來

| 方向 | 說明 |
|------|------|
| Layer 2 | 在 L2 上降低 Gas 費 |
| 更多整合 | 錢包、DApp、社交平台 |
| 跨鏈 | 支援更多區塊鏈 |
| 身份標準 | 成為 Web3 身份標準 |

{% hint style="success" %}
ENS 是 Web3 最成功的基礎設施之一。它解決了一個真實的問題（難記的地址），並創造了新的可能性（去中心化身份）。如果 Web3 繼續發展，ENS 很可能會是其中不可或缺的一部分。
{% endhint %}
