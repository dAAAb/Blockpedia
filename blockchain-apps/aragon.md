# Aragon

### 想成立一個沒有老闆的公司嗎？Aragon 讓你幾分鐘內就能創建一個 DAO（去中心化自治組織），有投票系統、金庫管理、成員權限——全部都在鏈上運行。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是 Aragon？

| 項目 | 說明 |
|------|------|
| 類型 | DAO 創建平台 |
| 推出 | 2017 年 |
| 代幣 | ANT |
| 功能 | DAO 創建、治理、金庫管理 |

---

## Aragon 的功能

### 核心模組

```
Aragon 提供的功能：

組織創建：
- 幾分鐘建立 DAO
- 預設模板可選
- 自定義組織結構

投票系統：
- 代幣加權投票
- 提案管理
- 多種投票機制

金庫管理：
- 多簽錢包
- 資金管理
- 支出提案

成員管理：
- 權限分配
- 角色設定
- 加入/退出機制
```

### 使用流程

| 步驟 | 說明 |
|------|------|
| 1. 連接錢包 | MetaMask 等 |
| 2. 選擇模板 | 公司型、會員型等 |
| 3. 設定參數 | 投票規則、代幣等 |
| 4. 部署 DAO | 支付 Gas 完成 |
