# \(JF\)IPFS
> **最後更新：2026 年 1 月**


## 放不上智能合約就放這

星際檔案系統\(InterPlanetary File System或IPFS\)提供一個分散式檔案存儲系統，將放上的檔案拆碎，然後分散到網絡中的一部分節點，每當檔案被尋找時才進行重組。

## 檔名就是內容

每個獨立檔案都會有一個[哈希](../wa/ha-xi-zhi.md)地址，利用該地址可以尋得檔案並取出。而哈希值的生成就是原檔案內容的哈希，也就是說一個檔案內容如果被些微改動了，其地址會截然不同。
