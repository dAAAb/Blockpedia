# 北科大

### 北科大（國立臺北科技大學）是台灣區塊鏈教育的先行者之一。它不只開設區塊鏈課程，還成立了研究中心，與業界合作推動區塊鏈應用。對於想要在台灣學習區塊鏈的人來說，北科大是重要的選擇。

## 區塊鏈相關資源

### 研究中心

北科大設有區塊鏈相關研究單位：

| 資源 | 說明 |
|------|------|
| 區塊鏈研究中心 | 學術研究和產學合作 |
| 金融科技研究 | 區塊鏈金融應用 |
| 產學合作計畫 | 與企業合作實務專案 |

### 課程內容

```
北科大區塊鏈課程涵蓋：

基礎理論：
- 區塊鏈概論
- 密碼學基礎
- 分散式系統

實務應用：
- 智能合約開發
- DApp 開發實作
- 區塊鏈專案管理
```

{% hint style="info" %}
**技職 vs 學術**

北科大的優勢在於技職體系的實作傳統。如果你想要「做得出東西」而非「寫論文」，北科大可能更適合你。
{% endhint %}
