# MIT

### MIT 是區塊鏈學術研究的聖地。Bitcoin Core 開發者、Lightning Network 發明者、多位頂尖密碼學家都與 MIT 有關。它的 Digital Currency Initiative（數位貨幣計畫）是全球最重要的區塊鏈學術研究機構之一。

## Digital Currency Initiative（DCI）

DCI 是 MIT Media Lab 下的研究機構：

| 特色 | 說明 |
|------|------|
| 成立 | 2015 年 |
| 定位 | 數位貨幣學術研究 |
| 研究領域 | 比特幣安全、隱私、擴容 |
| 知名人物 | Neha Narula（主任）|

### 研究方向

```
DCI 的研究領域：

核心技術：
- Bitcoin Core 開發支援
- Lightning Network 研究
- 密碼學和隱私技術

政策研究：
- 央行數位貨幣（CBDC）
- 監管框架研究
- 數位貨幣經濟學
```

{% hint style="info" %}
**學術獨立性**

DCI 強調學術獨立性，不接受加密貨幣公司的資金。這讓它的研究更具公信力。
{% endhint %}
