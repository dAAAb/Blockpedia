# 代幣化房地產
> **最後更新：2026 年 1 月**


## 代幣化房地產讓普通人也能投資商業地產，只需要幾十美元就能擁有一棟大樓的一小部分。

在「[RWA 總覽](README.md)」中，我們說 RWA 可以降低投資門檻。房地產是最好的例子。

### 房地產的問題

房地產是人類歷史上最重要的資產類別之一。但它有幾個致命缺點：

**1. 門檻太高**

一棟公寓動輒上千萬，普通人根本買不起。

**2. 流動性太差**

房子想賣，要找仲介、刊廣告、談價格、過戶，可能要半年。

**3. 管理麻煩**

當房東要處理租客、維修、稅務...很多人沒這個精力。

**4. 地域限制**

你住在台灣，很難投資紐約的房產。

代幣化可以解決這些問題嗎？

### 代幣化房地產的運作

典型的房地產代幣化流程：

```
1. 平台選定一棟房產（如紐約某公寓）
2. 成立 SPV 購買該房產
3. SPV 在區塊鏈上發行代幣
4. 每個代幣代表房產的一小部分所有權
5. 投資者購買代幣
6. 租金收入按比例分配給代幣持有者
7. 房產出售時，收益按比例分配
```

你持有代幣 = 你擁有這棟房子的一部分。

### 主要平台

目前主要的房地產代幣化平台：

**RealT**

- 總部：美國
- 專注：美國住宅房產
- 特點：最低投資約 50 美元
- 收益：租金每日發放（用穩定幣）
- 代幣標準：ERC-20

RealT 是目前最成熟的房地產代幣化平台，已經代幣化超過 500 處房產。

**Lofty**

- 總部：美國
- 專注：美國住宅房產
- 特點：可以用信用卡購買
- 收益：租金每日發放

**Propy**

- 專注：房產交易（不只是投資）
- 特點：用區塊鏈處理房產過戶

**Landshare**

- 專注：美國房產
- 特點：有自己的 DeFi 生態系

### RealT 深入介紹

讓我們以 RealT 為例，看看實際運作：

**選擇房產**

RealT 團隊在美國各地尋找房產，通常是：
- 已經有租客的住宅
- 價格在 10-50 萬美元
- 租金回報率穩定

**代幣發行**

假設一棟房子值 20 萬美元，RealT 可能發行 20,000 個代幣，每個代幣代表 1/20,000 的所有權，價值約 10 美元。

**購買代幣**

投資者用穩定幣（USDC、xDAI）購買代幣。最低投資額可能只有 50 美元。

**收取租金**

租客付租金給物業管理公司。物業管理公司扣除費用後，把淨租金轉給 SPV。SPV 按代幣持有比例分配給投資者。

RealT 特別之處是**每日發放租金**，直接發到你的錢包。

**出售代幣**

如果你想退出，可以在 RealT 的二級市場或 Uniswap 等 DEX 上出售代幣。

{% hint style="info" %}
RealT 代幣可以在 DeFi 中使用，比如作為抵押品借款。這是傳統房地產投資做不到的。
{% endhint %}

### 收益率

代幣化房地產的收益通常包括：

**1. 租金收益**

年化通常在 5-12% 之間，取決於房產位置和類型。

**2. 房價增值**

如果房產升值，代幣價格也會上升。但這部分不確定。

整體而言，代幣化房地產的收益率跟 REITs（房地產投資信託）類似，但有更高的靈活性。

### 與 REITs 的比較

傳統上，普通人投資房地產主要靠 REITs。代幣化房地產跟 REITs 有什麼不同？

| 特性 | REITs | 代幣化房地產 |
|------|-------|-------------|
| 最低投資 | 一股（數十美元）| 可能更低 |
| 投資標的 | 多棟房產組合 | 可選擇單一房產 |
| 交易時間 | 股市開市時間 | 24/7 |
| 透明度 | 季報/年報 | 鏈上即時 |
| DeFi 整合 | 無 | 可以 |
| 監管程度 | 嚴格 | 較寬鬆（也較不確定）|

代幣化房地產的獨特優勢是**細粒度選擇**——你可以選擇投資哪一棟特定的房子，而不是被動接受 REITs 經理的選擇。

### 法律結構

代幣化房地產的法律結構很重要。

**股權型（Equity）**

代幣代表房產所有權的一部分。你是這棟房子的部分擁有者。

- 優點：真正的所有權
- 缺點：法律複雜，尤其是跨境

**債權型（Debt）**

代幣代表對房產的借款。你借錢給房產持有者，收取利息。

- 優點：法律較簡單
- 缺點：沒有所有權，無法享受升值

**收益權型**

代幣代表收取租金收益的權利，但不擁有房產。

- 優點：最簡單
- 缺點：權利最有限

不同平台採用不同結構，投資前要看清楚。

### 風險

代幣化房地產的風險不能忽視：

**1. 房產風險**

- 租客不付租
- 房產需要維修
- 房價下跌
- 所在區域衰落

這些是所有房地產投資都有的風險。

**2. 平台風險**

- 平台倒閉
- 平台管理不善
- 平台被駭

你的代幣權益依賴平台運作。

**3. 流動性風險**

雖然代幣可以交易，但二級市場可能不活躍。想賣可能賣不掉，或要大幅折價。

**4. 監管風險**

各國對代幣化證券的監管不同。政策改變可能影響你的投資。

**5. 智能合約風險**

代幣合約可能有漏洞。

### 地理限制

代幣化房地產有地理限制：

- **美國房產**：大部分代幣化房產在美國
- **美國投資者限制**：有些平台不接受美國投資者（因為監管）
- **非美國投資者**：通常可以投資，但要注意稅務

投資前要確認你所在地區是否可以參與。

### 稅務考量

代幣化房地產的稅務可能很複雜：

- 租金收入可能要繳稅
- 房產出售的資本利得可能要繳稅
- 跨境投資的稅務更複雜

建議諮詢專業稅務顧問。

### 實際體驗

如果你想試試代幣化房地產，可以這樣開始：

1. **研究平台**：了解 RealT、Lofty 等平台的運作
2. **小額嘗試**：先投資 50-100 美元，了解流程
3. **選擇房產**：看房產位置、租金歷史、物業狀態
4. **分散投資**：不要把所有錢放在同一棟房子
5. **長期持有**：房地產是長期投資，不要期待短期暴利

### 未來展望

代幣化房地產還在早期，但潛力巨大：

1. **更多資產類型** — 商業地產、酒店、物流中心
2. **更多地區** — 不只美國，歐洲、亞洲的房產
3. **更好的流動性** — 二級市場成熟
4. **更深的 DeFi 整合** — 房產代幣作為抵押品借款、參與流動性挖礦

有一天，每個人都能像買股票一樣買房地產——這是代幣化房地產的願景。
