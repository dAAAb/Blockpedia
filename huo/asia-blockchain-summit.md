# Asia Blockchain Summit

### 每年夏天，全世界的區塊鏈人都會飛到台北。Asia Blockchain Summit 是亞洲最大的區塊鏈峰會，講者陣容星光熠熠，Side Event 更是社交的好機會。

> ⚠️ **免責聲明**：參加活動請注意安全，謹防詐騙。

---

## 關於 ABS

| 項目 | 說明 |
|------|------|
| 全名 | Asia Blockchain Summit |
| 地點 | 台北 |
| 頻率 | 每年一次 |
| 規模 | 數千人參與 |

## 活動特色

### 主要內容

```
ABS 的精彩之處：

主會場：
- 頂尖講者演講
- 產業趨勢分享
- Panel 討論

展區：
- 項目方展示
- 新創 Demo
- 合作洽談

Side Events：
- 各種主題派對
- 深度交流活動
- 投資人媒合
```
