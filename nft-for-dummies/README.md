## WhaleShark 的 NFT 觀點

### 核心理念

```
WhaleShark 的投資哲學：

1. 長期持有
   - 不頻繁交易
   - 相信藝術價值
   - 時間複利

2. 支持藝術家
   - 直接購買支持
   - 建立關係
   - 參與創作生態

3. 社群價值
   - NFT 不只是圖片
   - 是社群的入場券
   - 是文化認同

4. 分散配置
   - 不 all-in 單一專案
   - 跨類別收藏
   - 風險管理
```

### 投資策略

| 策略 | 說明 |
|------|------|

## NFT 財富管理

### 創作者策略

| 建議 | 說明 |
|------|------|
| 保留版稅 | 設定合理的二次銷售版稅 |
| 不要賤賣 | 建立作品價值底線 |
| 長期規劃 | 不要一次賣光 |
| 多元收入 | 不只依賴 NFT 銷售 |

```
創作者的財富管理建議：

收入分配：
- 留一部分作為儲蓄
- 投資其他資產
- 不要全部換成法幣

版稅策略：
- 5-10% 是合理範圍
- 太高會影響二手市場
- 太低失去長期收益

作品策略：
- 保留一些自己的作品
- 逐步釋出而非一次賣光
- 建立系列作品
```

## NFT 新手建議

### 入門須知

```
WhaleShark 給新手的建議：

心態：
- NFT 是高風險資產
- 只投你願意損失的
- 以收藏心態非投機

研究：
- 了解藝術家背景
- 研究項目路線圖
- 查看社群活躍度

實踐：
- 從小額開始
- 學習使用錢包
- 體驗不同平台

避免：
- FOMO 追高
- 相信「穩賺」
- 忽視 Gas 費成本
```

### 常見錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 追高 FOMO | 冷靜研究再買 |
| 忽視 Gas | 計入總成本 |
| 只看價格 | 關注藝術價值 |
| 短期思維 | 長期視角 |

{% hint style="info" %}
**先學習，再投資**

在投入資金前，花時間學習 NFT 的基本概念、如何評估項目、如何保護資產安全。知識是最好的投資。
{% endhint %}
