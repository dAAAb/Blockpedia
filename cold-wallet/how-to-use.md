# 硬體錢包使用指南

### 硬體錢包是保護加密資產最安全的方式之一。它把你的私鑰存在一個專用的小設備裡，即使你的電腦被駭客入侵，資產依然安全。

---

## 為什麼需要硬體錢包？

想像一下：你的電腦就像一間四面透明的房子，駭客隨時可能闖入偷窺。而硬體錢包就像是一個獨立的保險箱——即使小偷進了你家，也打不開它。

### 軟體錢包 vs 硬體錢包

| 特性 | 軟體錢包（熱錢包） | 硬體錢包（冷錢包） |
|------|-------------------|-------------------|
| 私鑰儲存位置 | 電腦/手機（連網） | 專用設備（離線） |
| 駭客風險 | 較高 | 極低 |
| 使用便利性 | 高 | 中等 |
| 價格 | 免費 | 約 $60-200 美元 |
| 適合場景 | 小額日常交易 | 大額長期持有 |

{% hint style="info" %}
**簡單的判斷標準**

如果你持有的加密資產價值超過一支手機的價格，就值得買一個硬體錢包。
{% endhint %}

## 開箱與初始設定

以 Ledger Nano S Plus 為例：

### 步驟 1：確認包裝完整

- 檢查外盒有無拆封痕跡
- 確認防偽標籤完整
- **重要**：如果收到時已經設定好助記詞，那是詐騙！正品一定是你自己產生助記詞

### 步驟 2：安裝官方軟體

1. 到官網 [ledger.com](https://www.ledger.com/start) 下載 Ledger Live
2. **千萬不要**從 Google 搜尋結果的廣告連結下載
3. 確認網址正確無誤

### 步驟 3：設定 PIN 碼

- 設定 4-8 位數的 PIN 碼
- 這是每次使用時的第一道防線
- 連續輸入錯誤 3 次，設備會重置

### 步驟 4：生成並備份助記詞

{% hint style="danger" %}
**最重要的步驟！**

1. 設備會顯示 24 個英文單字
2. 依序抄寫在**紙上**（包裝內附的卡片）
3. 再次確認順序正確
4. **永遠不要**拍照、截圖、存在電腦
{% endhint %}

### 步驟 5：驗證助記詞

設備會要求你確認幾個單字的順序，確保你已正確備份。

### 步驟 6：安裝應用程式

在 Ledger Live 中安裝你需要的幣種 App：
- Bitcoin App（比特幣）
- Ethereum App（以太坊及 ERC-20）
- 其他你需要的鏈

## 日常使用教學

### 接收加密貨幣

1. 連接硬體錢包到電腦
2. 開啟 Ledger Live，選擇「接收」
3. 選擇幣種
4. **在硬體錢包螢幕上確認地址**
5. 複製地址給對方

{% hint style="info" %}
**為什麼要在設備上確認？**

即使電腦被駭客控制，他們也無法竄改硬體錢包螢幕上顯示的地址。這是防止「地址替換攻擊」的關鍵。
{% endhint %}

### 發送加密貨幣

1. 在 Ledger Live 選擇「發送」
2. 輸入目標地址和金額
3. **在硬體錢包螢幕上確認**：
   - 目標地址（至少核對開頭和結尾）
   - 金額
   - 手續費
4. 按下設備上的確認按鈕

### 與 DeFi 互動

硬體錢包可以搭配 MetaMask 使用：

1. 在 MetaMask 選擇「連接硬體錢包」
2. 選擇 Ledger/Trezor
3. 之後所有交易都需要在硬體錢包上確認

## 安全最佳實踐

### 該做的事 ✓

- **定期更新韌體** — 但只從官方軟體更新
- **測試備份** — 買第二個設備，用助記詞恢復，確認可用
- **分散存放** — 大額資產考慮多個錢包
- **設定強 PIN** — 不要用生日或連號

### 不該做的事 ✗

- **不要買二手** — 可能被動過手腳
- **不要從非官方渠道購買** — 可能是假貨
- **不要在公共電腦使用** — 可能有鍵盤記錄器
- **不要把助記詞存在設備裡** — 紙本最安全

## 常見問題

### Q：硬體錢包壞了怎麼辦？

只要你有助記詞，買一個新的設備，用助記詞恢復即可。資產存在區塊鏈上，不在設備裡。

### Q：忘記 PIN 碼怎麼辦？

連續輸入錯誤 3 次後設備會重置。之後用助記詞恢復即可。

### Q：可以同時用在多台電腦嗎？

可以。你可以在任何電腦上安裝 Ledger Live，連接同一個設備使用。

### Q：Ledger 和 Trezor 選哪個？

兩者都很安全。Ledger 使用安全晶片（SE），但軟體不開源；Trezor 完全開源，但使用一般晶片。選擇你信任的品牌即可。

### Q：需要一直連著電腦嗎？

不需要。只有在你要發送交易或查看餘額時才需要連接。平時可以放在抽屜裡。

## 進階功能

### 隱藏錢包（Passphrase）

在助記詞之外，可以設定一個額外的密碼短語。這會產生一個完全不同的錢包。

用途：
- 創建「誘餌錢包」（放少量資產）
- 增加一層保護

{% hint style="info" %}
**注意**

使用 Passphrase 時，如果忘記這個密碼，即使有助記詞也無法恢復該錢包。
{% endhint %}

### 多重簽名

結合多個硬體錢包，設定需要多個設備同時授權才能轉帳。適合團隊或高淨值用戶。
