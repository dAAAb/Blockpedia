# 輕量節點
> **最後更新：2026 年 1 月**


**輕量節點（英文：Lightweight-node）**就是一個區塊鏈帳本儲存設備，但不一定存有該帳本的完整資料，只存有最近一定量的交易記錄，或是某大類別或時間的交易記錄。但只要有心，一個輕量節點也可以透過其所擁有部份或全部分帳本資訊位置資訊，擴充升級為全節點。
