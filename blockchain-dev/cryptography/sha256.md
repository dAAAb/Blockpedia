# SHA256
> **最後更新：2026 年 1 月**


SHA256是Secure Hash Algorithm 256的簡寫，屬於SHA2家族，其中256指的是其產生的哈希值共有256位元\(bit\)。目前被發現和MD5、SHA1一樣具有被[長度擴展攻擊](https://zh.wikipedia.org/wiki/长度扩展攻击)\(Length extension attack\)的風險。
