# SHA3
> **最後更新：2026 年 1 月**


SHA3是Secure Hash Algorithm 3的簡寫，相較於先前的SHA0、SHA1和SHA2，SHA3是Secure Hash Algorithm家族的最新標準，因[SHA2證實出有被攻擊的風險](sha256.md)而誕生。SHA3發明了一種名為海綿函數（sponge function）的算法來防止上述攻擊。
