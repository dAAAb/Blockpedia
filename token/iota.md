# IOTA

### 如果說區塊鏈是一條鏈，IOTA 就是一張網。它不用礦工、不收手續費，專為物聯網時代設計——讓機器與機器之間可以直接交易，哪怕只是 0.0001 元。

---

## 什麼是 IOTA？

| 項目 | 說明 |
|------|------|
| 代號 | MIOTA |
| 創立 | 2015 年 |
| 技術 | Tangle（有向無環圖）|
| 特色 | 零手續費、可擴展 |
| 定位 | 物聯網支付 |

## 核心技術：Tangle

### 不是區塊鏈

```
傳統區塊鏈 vs IOTA Tangle：

區塊鏈：
區塊 1 → 區塊 2 → 區塊 3 → ...
（線性結構，需要礦工打包）

Tangle：
     交易A ←─ 交易D
       ↖     ↗
        交易B ←─ 交易E
       ↗     ↖
     交易C ←─ 交易F
（網狀結構，每筆交易驗證前兩筆）

關鍵差異：
- 沒有區塊，只有交易
- 不需要礦工
- 越多人用，越快
```

### 運作原理

| 概念 | 說明 |
|------|------|
| DAG | 有向無環圖結構 |
| 驗證機制 | 發交易時需驗證 2 筆舊交易 |
| 共識 | Coordinator（正在移除）|
| 手續費 | 完全免費 |

{% hint style="info" %}
**為什麼零手續費？**

因為你在發交易時，同時幫網路驗證了別人的交易。這是一種「以工代費」的設計——你付出算力，換取免費交易。
{% endhint %}

## 物聯網應用場景

### 為什麼需要 IOTA？

```
物聯網的支付問題：

場景：智慧城市
- 自動駕駛車付停車費：$0.001
- 感測器賣數據：$0.0001
- 智慧電表微支付：$0.00001

傳統支付：
- 信用卡手續費 > 交易金額（不可行）
- 比特幣手續費 > 交易金額（不可行）

IOTA 解決方案：
- 零手續費
- 即時結算
- 機器對機器（M2M）支付
```

### 應用案例

| 領域 | 應用 |
|------|------|
| 汽車 | 與 VW、Jaguar 合作 |
| 能源 | 電力微支付 |
| 供應鏈 | 數據追蹤 |
| 智慧城市 | 感測器數據交易 |

## 發展歷程

### 重要事件

| 時間 | 事件 |
|------|------|
| 2015 | 創立，ICO 籌得約 50 萬美元 |
| 2017 | 價格飆漲，市值前 10 |
| 2018 | 發現安全漏洞 |
| 2019 | Trinity 錢包被駭 |
| 2020 | 內部紛爭，創辦人離開 |
| 2021+ | Chrysalis 升級，移除 Coordinator |

### 爭議與挑戰

| 問題 | 說明 |
|------|------|
| Coordinator | 中心化協調器（正在移除）|
| 安全疑慮 | 曾發現密碼學漏洞 |
| 團隊紛爭 | 創辦人與基金會衝突 |
| 採用率 | 物聯網普及比預期慢 |

{% hint style="warning" %}
**中心化爭議**

早期 IOTA 依賴 Coordinator（協調器）防止攻擊，這使其本質上是中心化的。IOTA 2.0（Coordicide）承諾移除它，但進度多次延遲。
{% endhint %}

## 投資考量

### 優勢與風險

| 優勢 | 風險 |
|------|------|
| 零手續費 | 技術複雜，開發緩慢 |
| 物聯網敘事 | 競爭對手眾多 |
| 可擴展性 | 過去的安全事件 |
| 企業合作 | 實際採用有限 |

{% hint style="danger" %}
**投資警告**

IOTA 從 2017 年高點已下跌超過 95%。物聯網支付是長期願景，但競爭激烈，技術路線圖多次延遲。請謹慎評估。
{% endhint %}
