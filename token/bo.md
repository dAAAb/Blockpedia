# 波場幣

### 波場是區塊鏈世界的「爭議話題」——創辦人孫宇晨靠營銷出名，但波場確實成為了 USDT 轉帳的主要網路之一。拋開爭議，它的技術和生態值得了解。

---

## 什麼是波場幣？

| 項目 | 說明 |
|------|------|
| 英文名 | TRON |
| 代號 | TRX |
| 創立 | 2017 年 |
| 創辦人 | 孫宇晨（Justin Sun）|
| 共識 | DPoS（委託權益證明）|

---

## 波場網路

### 技術特色

```
波場的定位：

宣傳口號：
「去中心化的娛樂內容網路」

實際用途：
1. USDT 轉帳（最大用途）
2. DeFi 應用
3. 博弈類 DApp
4. 代幣發行

技術參數：
- TPS：約 2,000+
- 確認時間：約 3 秒
- 手續費：極低
```

### 與以太坊比較

| 項目 | 波場 | 以太坊 |
|------|------|--------|
| TPS | ~2,000 | ~15-30 |
| 手續費 | 極低 | 較高 |
| 共識機制 | DPoS | PoS |
| 去中心化程度 | 較低 | 較高 |
| 智能合約 | 兼容 EVM | 原生 |

{% hint style="info" %}
**USDT 的家**

波場上的 USDT（TRC-20）是目前流通量最大的 USDT 版本，超越以太坊版本。因為手續費低、速度快，很多交易所和用戶選擇使用波場轉帳。
{% endhint %}

## 發展歷程

### 重要事件

| 時間 | 事件 |
|------|------|
| 2017 | ICO 籌得約 7,000 萬美元 |
| 2018/6 | 主網上線，脫離以太坊 |
| 2018/7 | 收購 BitTorrent |
| 2019 | 孫宇晨天價午餐事件 |
| 2021 | 成為 USDT 最大網路 |
| 2022 | 孫宇晨任格瑞那達常駐 WTO 代表 |

### 爭議事件

| 事件 | 說明 |
|------|------|
| 白皮書抄襲 | 被指控抄襲其他項目 |
| 空氣營銷 | 大量炒作但缺乏實質 |
| 天價午餐 | 與巴菲特午餐的鬧劇 |
| 中心化 | DPoS 被批評過度中心化 |

{% hint style="warning" %}
**爭議人物**

孫宇晨是區塊鏈圈最具爭議的人物之一。他的營銷手法常被批評，但不可否認波場網路確實有一定的使用量和生態。
{% endhint %}

### USDD 穩定幣

```
波場的算法穩定幣：USDD

機制：
- 與 TRX 的銷毀/鑄造機制
- 目標錨定 1 美元
- 波場 DAO 儲備支持

風險：
- 類似 UST 的機制
```

## 技術架構

### DPoS 機制

| 概念 | 說明 |
|------|------|
| 超級代表（SR）| 27 個出塊節點 |
| 投票權 | 持有 TRX 可投票 |
| 出塊獎勵 | SR 獲得區塊獎勵 |
| 能量機制 | 資源模型 |

### 資源模型

| 資源 | 用途 |
|------|------|
| 能量（Energy）| 執行智能合約 |
| 頻寬（Bandwidth）| 交易傳輸 |
| 獲取方式 | 凍結 TRX 或付費 |

## 投資考量

### 優勢與風險

| 優勢 | 風險 |
|------|------|
| USDT 最大網路 | 中心化程度高 |
| 手續費低 | 創辦人爭議多 |
| 交易速度快 | 生態品質參差 |
| 開發活躍 | 監管風險 |

{% hint style="danger" %}
**投資警告**

波場幣從 2018 年高點已大幅下跌。雖然網路使用量不錯，但代幣價值不一定反映網路價值。請謹慎評估。
{% endhint %}
