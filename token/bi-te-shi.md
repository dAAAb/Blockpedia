# 比特鑽石

### 2017 年是比特幣分叉潮的高峰期，比特鑽石是其中之一。它承諾更快、更便宜、更隱私的交易。但在眾多分叉幣中，它能否脫穎而出？

---

## 什麼是比特鑽石？

| 項目 | 說明 |
|------|------|
| 英文名 | Bitcoin Diamond |
| 代號 | BCD |
| 分叉時間 | 2017 年 11 月 24 日 |
| 分叉高度 | 區塊 495,866 |
| 總量 | 2.1 億（比特幣的 10 倍）|

---

## 分叉特色

### 與比特幣的差異

```
比特鑽石的「改進」：

1. 區塊更大
   比特幣：1 MB
   比特鑽石：8 MB
   → 每秒處理更多交易

2. 供應量 10 倍
   比特幣：2,100 萬
   比特鑽石：2.1 億
   → 單價更低，降低心理門檻

3. 挖礦算法
   使用 X13 算法
   → 抗 ASIC（早期）

4. 加密功能
   交易金額可選擇加密
   → 隱私保護
```

### 技術參數

| 參數 | 比特幣 | 比特鑽石 |
|------|--------|----------|
| 總量 | 2,100 萬 | 2.1 億 |
| 區塊大小 | 1 MB | 8 MB |
| 區塊時間 | 10 分鐘 | 10 分鐘 |
| 挖礦算法 | SHA-256 | X13 → BCD |

{% hint style="info" %}
**10 倍供應量的意義**

分叉時，每持有 1 BTC 可獲得 10 BCD。這個設計讓持幣者覺得「賺到了」，也降低了單價，讓新手覺得「買得起」。但實際上，數量多寡不影響總價值。
{% endhint %}

## 2017 分叉潮

### 時代背景

```
2017 年比特幣分叉潮：

為什麼那麼多分叉？

1. 比特幣價格飆漲
   - 一顆比特幣值 2 萬美元
   - 分叉幣可以「蹭」這個品牌

2. 免費獲得新幣
   - 持有 BTC 的人自動獲得分叉幣
   - 「空投」效應吸引關注

3. 技術門檻低
   - 分叉比特幣不難
   - 改幾個參數就能創造「新幣」

4. 投機氛圍
   - 任何新幣都可能漲
   - FOMO 心態推動交易

結果：
- 數十種比特幣分叉幣出現
- 大多數現在已經歸零
```

## 爭議與問題

### 常見質疑

| 質疑 | 說明 |
|------|------|
| 團隊匿名 | 創辦人身份不明 |
| 缺乏創新 | 只是參數調整 |
| 預挖爭議 | 團隊預先持有大量幣 |
| 價值質疑 | 為什麼需要另一個比特幣？|

### 預挖問題

```
比特鑽石的預挖爭議：

官方說法：
- 分叉時「預挖」了部分幣
- 用於團隊獎勵和發展基金

社群質疑：
- 預挖比例過高
- 團隊可能拋售套現
- 分配不透明

結果：
- 團隊持有大量籌碼
- 有隨時砸盤的能力
- 散戶處於劣勢
```

{% hint style="danger" %}
**匿名團隊風險**

比特鑽石的創辦團隊身份不明。這意味著如果出問題，沒有人需要負責。這是很嚴重的風險訊號。
{% endhint %}

## 投資警告

### 分叉幣的共同風險

| 風險 | 說明 |
|------|------|
| 缺乏價值主張 | 為什麼不直接用比特幣？|
| 流動性低 | 難以大量買賣 |
| 開發停滯 | 沒有持續改進 |
| 歸零風險 | 很可能變得毫無價值 |

{% hint style="danger" %}
**極高風險警告**

比特鑽石從高點下跌超過 99%。匿名團隊、缺乏開發、幾乎沒有使用場景。這是典型的「已死」分叉幣。除非你願意失去全部投資，否則不建議購買。
{% endhint %}
