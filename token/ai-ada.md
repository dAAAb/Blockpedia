# 艾達幣（Cardano / ADA）
> **最後更新：2026 年 1 月**


## Cardano 是一個以「學術研究為基礎」的區塊鏈平台，由以太坊共同創辦人 Charles Hoskinson 創立。它強調同行評審和形式化驗證，被稱為「第三代區塊鏈」——但也因為開發速度緩慢而備受批評。

大多數區塊鏈項目的開發模式是：

**先做，再改，出問題再修。**

Cardano 說：

**我們要先研究清楚，寫成論文，經過同行評審，然後才寫程式。**

這種「學術派」的做法，讓 Cardano 成為加密貨幣圈最獨特的項目之一。

也是最有爭議的。

### 創辦人：Charles Hoskinson

Charles Hoskinson 是以太坊的八位共同創辦人之一。

**但他在 2014 年離開了以太坊。**

原因？理念不合。

| 議題 | Vitalik | Hoskinson |
|------|---------|-----------|
| 組織形式 | 非營利基金會 | 營利公司 |
| 開發風格 | 快速迭代 | 學術研究 |
| 募資方式 | 一次 ICO | 更多風投 |

離開後，Hoskinson 創立了 Cardano，要用「正確的方式」建造區塊鏈。

{% hint style="info" %}
Hoskinson 是加密貨幣圈最活躍的 KOL 之一。他經常直播數小時，談論從技術到政治的各種話題。支持者認為他是遠見者，批評者說他只會說不會做。
{% endhint %}

### Cardano 的特色

#### 1. 同行評審

Cardano 的每個重要技術決定，都先寫成學術論文，送到頂級會議審查。

**已發表論文超過 100 篇**，涵蓋：

- 共識機制
- 智能合約
- 治理系統
- 密碼學

**好處**：理論基礎紮實，經過專家驗證。

**壞處**：開發速度慢，市場機會可能錯過。

#### 2. 分層架構

Cardano 把區塊鏈分成兩層：

| 層級 | 名稱 | 功能 |
|------|------|------|
| 結算層 | CSL | 處理 ADA 轉帳 |
| 計算層 | CCL | 執行智能合約 |

**這種設計的好處**：

- 可以分別升級
- 更容易維護
- 更靈活

#### 3. Ouroboros 共識機制

Cardano 自己設計的權益證明（PoS）協議。

**特點**：

- 第一個經過同行評審的 PoS
- 數學上可證明的安全性
- 能源消耗極低

```
Ouroboros 運作方式：

時間分成「紀元」(Epoch)
紀元分成「時段」(Slot)
每個時段隨機選一個驗證者出塊
選中機率與質押量成正比
```

#### 4. Plutus 智能合約

Cardano 的智能合約語言，基於 Haskell。

| 項目 | Solidity（以太坊）| Plutus（Cardano）|
|------|-------------------|------------------|
| 語言類型 | 物件導向 | 函數式 |
| 學習曲線 | 中等 | 陡峭 |
| 安全性 | 一般 | 較高 |
| 開發者數量 | 很多 | 很少 |

{% hint style="warning" %}
Haskell 是一種學術界喜愛但業界很少用的語言。這讓 Cardano 的智能合約開發門檻很高，開發者很難找。
{% endhint %}

### Cardano 的開發路線圖

Cardano 的開發分為五個階段，每個階段以詩人命名：

| 階段 | 名稱 | 內容 | 狀態 |
|------|------|------|------|
| 1 | Byron | 基礎功能 | 完成 |
| 2 | Shelley | 去中心化質押 | 完成 |
| 3 | Goguen | 智能合約 | 完成 |
| 4 | Basho | 擴容 | 進行中 |
| 5 | Voltaire | 治理 | 進行中 |

**從 2017 到現在，開發已經超過 7 年。**

### Cardano 的生態系統

#### DeFi

相比以太坊，Cardano 的 DeFi 生態系統很小：

| DEX | TVL |
|-----|-----|
| Minswap | 最大 |
| SundaeSwap | 第二 |
| WingRiders | 第三 |

**總 TVL**：約 2-3 億美元（以太坊約 500 億+）

#### NFT

Cardano 上也有 NFT，但市場規模遠小於以太坊和 Solana。

### Cardano 的爭議

#### 1. 開發太慢

「永遠在開發中」是最常見的批評。

| 功能 | 以太坊 | Cardano |
|------|--------|---------|
| 智能合約 | 2015 | 2021 |
| PoS | 2022 | 2020（但不完整）|
| 擴容方案 | 多種 L2 運作中 | 仍在開發 |

#### 2. 生態系統弱

- 開發者數量少
- DApp 數量少
- TVL 低
- 用戶少

#### 3. Charles Hoskinson

創辦人本身也是爭議：

- 支持者：他是遠見者、技術天才
- 批評者：他過度行銷、承諾太多

#### 4. 學術 vs 實用

同行評審確實保證了理論正確，但：

- 區塊鏈是實驗性技術
- 市場變化很快
- 完美的理論不如能用的產品

### Cardano vs 其他公鏈

| 項目 | Cardano | Ethereum | Solana |
|------|---------|----------|--------|
| 方法論 | 學術研究 | 快速迭代 | 性能優先 |
| 智能合約語言 | Plutus/Haskell | Solidity | Rust |
| TPS | 250+ | 15-30 | 數千 |
| 生態規模 | 小 | 最大 | 中等 |
| 開發者數量 | 少 | 最多 | 中等 |

### 投資 ADA 的考量

#### 潛在優勢

- 技術基礎紮實
- 創辦人影響力大
- 社群忠誠度高
- 長期願景明確

#### 風險

| 風險 | 說明 |
|------|------|
| 開發緩慢 | 競爭對手可能搶先 |
| 生態弱 | 缺乏殺手級應用 |
| 開發者少 | Haskell 門檻高 |
| 執行風險 | 願景好但能否實現？ |

### Cardano 的未來

| 方向 | 說明 |
|------|------|
| Hydra | Layer 2 擴容方案 |
| Voltaire | 完全去中心化治理 |
| 非洲市場 | 身分、教育、金融服務 |
| 跨鏈 | 與其他區塊鏈互通 |

{% hint style="success" %}
Cardano 是加密貨幣圈的「學院派」——它相信慢就是快，正確比快速更重要。這種理念有其價值，但在快速變化的市場中也可能是劣勢。投資 Cardano，是在投資「學術方法論能否贏得區塊鏈競賽」這個命題。到目前為止，市場給出的答案並不明確。
{% endhint %}
