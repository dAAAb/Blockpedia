# LOOMX

### Loom Network 是早期的以太坊擴容方案，讓開發者可以建立高效能的 DApp，而不用擔心以太坊主網的壅塞問題。它也是著名的區塊鏈教學平台「CryptoZombies」的創作者。

---

## 什麼是 Loom Network？

| 項目 | 說明 |
|------|------|
| 英文名 | Loom Network |
| 代號 | LOOM |
| 創立 | 2017 年 |
| 定位 | 以太坊 Layer 2 |
| 特色 | DPoS 側鏈 |

---

## 核心技術

### DAppChains

```
Loom 的設計理念：

問題：
以太坊太慢、太貴
→ 不適合遊戲和大量交易

解決方案：DAppChains
- 每個 DApp 可以有自己的側鏈
- 側鏈用 DPoS，速度快
- 資產可以橋接回以太坊

架構：
以太坊主網（安全、慢）
      ↕ 橋接
Loom 側鏈（快速、便宜）
      ↕
各種 DApp
```

### 技術架構

| 組件 | 說明 |
|------|------|
| PlasmaChain | Loom 的主要側鏈 |
| Transfer Gateway | 跨鏈資產橋接 |
| DPoS | 委託權益證明共識 |
| SDK | 開發工具包 |

{% hint style="info" %}
**Layer 2 先驅**

在 Polygon、Arbitrum 等方案出現之前，Loom 是最早的以太坊擴容解決方案之一。它的很多設計理念後來被其他項目採用。
{% endhint %}

## 發展歷程

### 重要事件

| 時間 | 事件 |
|------|------|
| 2017 | Loom Network 創立 |
| 2018 | CryptoZombies 爆紅 |
| 2018 | PlasmaChain 上線 |
| 2019 | 轉向企業區塊鏈 |
| 2020 | 團隊大幅縮減 |
| 2021+ | 專案活躍度下降 |

### 衰退原因

| 因素 | 說明 |
|------|------|
| 競爭激烈 | Polygon 等方案崛起 |
| 市場變化 | DeFi 熱潮帶動新選擇 |
| 定位模糊 | 企業 vs 消費者市場 |
| 團隊變動 | 核心成員離開 |

{% hint style="warning" %}
**專案現況**

Loom Network 目前的開發活動和社群活躍度都大幅下降。雖然網路仍在運作，但已不是主流的 Layer 2 選擇。
{% endhint %}

## 投資考量
