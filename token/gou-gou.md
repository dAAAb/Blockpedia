# 狗狗幣（Dogecoin）

## 狗狗幣是 2013 年作為「玩笑」創造的加密貨幣，以柴犬迷因為標誌。它沒有技術創新，沒有實用價值，卻成為市值前十的加密貨幣——因為 Elon Musk 喜歡它。

2013 年，兩個工程師花了大約 3 小時，創造了一種新的加密貨幣。

他們的目的？

**嘲笑加密貨幣圈的荒謬。**

他們用了當時最紅的網路迷因——一隻柴犬——作為標誌。

他們取了一個故意很蠢的名字：Dogecoin。

他們完全沒想到，十年後，這個「玩笑」會價值數百億美元。

### 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | Dogecoin |
| 代號 | DOGE |
| 創立 | 2013 年 12 月 |
| 創辦人 | Billy Markus、Jackson Palmer |
| 共識機制 | 工作量證明（PoW）|
| 總供應量 | 無上限（每年增發約 50 億枚）|
| 最高市值 | 超過 800 億美元（2021）|

### 起源故事

#### 玩笑的誕生

2013 年，澳洲人 Jackson Palmer 在 Twitter 上發了一條推文：

> 「投資 Dogecoin。我確定這會是下一個大事。」

這是諷刺——他在嘲笑當時加密貨幣圈的瘋狂投機。

但有人認真對待了。

美國工程師 Billy Markus 看到這條推文，決定把它變成現實。他花了幾個小時，把萊特幣的程式碼修改了一下，換上柴犬的圖案。

**Dogecoin 就這樣誕生了。**

#### 迷因的力量

「Doge」是什麼？

它是一個 2013 年爆紅的網路迷因：一隻柴犬的圖片，配上用 Comic Sans 字體寫的破英文。

```
        wow
              such coin
    very crypto
                    much value
        wow
```

這種「故意很蠢」的幽默，讓 Dogecoin 從一開始就不一樣。

{% hint style="info" %}
Dogecoin 的社群文化是「好玩、不認真」。早期它被用來在 Reddit 上打賞好內容，或贊助有趣的活動，而不是「投資」。
{% endhint %}

### 早期的瘋狂事蹟

Dogecoin 社群做過很多有趣的事：

| 年份 | 事件 |
|------|------|
| 2014 | 募款 $50,000 送牙買加雪橇隊去冬奧 |
| 2014 | 贊助 NASCAR 賽車手 |
| 2014 | 為肯亞打水井募款 |
| 各時期 | 在 Reddit 上打賞內容創作者 |

**這些活動的共同點**：好玩、有趣、不太認真。

沒有人真的把 Dogecoin 當投資。

直到 Elon Musk 出現。

### Elon Musk 效應

2020-2021 年，一切改變了。

#### Musk 的推文

全球首富 Elon Musk 開始頻繁發推提到 Dogecoin：

| 時間 | 推文 | 影響 |
|------|------|------|
| 2020.12 | 「One word: Doge」 | 價格上漲 |
| 2021.2 | 「Doge」配狗圖 | 暴漲 |
| 2021.4 | 「Doge Barking at the Moon」 | 再次暴漲 |
| 2021.5 | 稱 DOGE 是「人民的加密貨幣」| 歷史新高 |

**結果**：

- 2021 年初：$0.005
- 2021 年 5 月：$0.73
- 漲幅：**14,500%**

#### 為什麼是 Musk？

沒有人確定 Musk 為什麼喜歡 Dogecoin。

**可能的原因**：

- 他喜歡迷因和網路文化
- 他享受影響市場的權力
- 他真的覺得好玩
- 一些陰謀論者說他持有大量 DOGE

{% hint style="warning" %}
Musk 的推文可以在幾分鐘內讓 DOGE 漲跌 20%+。這不是健康的市場——這是賭場。
{% endhint %}

### SNL 的高點與轉折

2021 年 5 月 8 日，Musk 主持《Saturday Night Live》。

**社群的期待**：Musk 會大力宣傳 DOGE，價格會「To the moon」。

**實際發生**：

- 節目開始前，DOGE 漲到 $0.73（歷史新高）
- Musk 在節目中說 DOGE 是「a hustle」（騙局）
- 價格立刻暴跌

這一天成為 Dogecoin 的歷史高點。之後再也沒回到那個價格。

### Dogecoin 的「技術」

嚴格來說，Dogecoin 沒什麼技術創新：

| 項目 | 內容 |
|------|------|
| 基礎 | 萊特幣的分叉（萊特幣是比特幣的分叉）|
| 區塊時間 | 1 分鐘 |
| 共識機制 | PoW（與萊特幣合併挖礦）|
| 智能合約 | 無 |
| 特殊功能 | 無 |

**唯一的「創新」**：無限供應。

每年增發約 50 億枚 DOGE，這與比特幣的稀缺性完全相反。

### 投資 DOGE 的風險

{% hint style="danger" %}
投資 Dogecoin 本質上是在賭博，不是投資。以下是主要風險：
{% endhint %}

#### 1. 沒有基本面

| 問題 | 說明 |
|------|------|
| 沒有實用性 | 幾乎沒有 DApp、DeFi、NFT |
| 沒有開發 | 核心開發者很少 |
| 通膨設計 | 每年增發 50 億枚 |

#### 2. 完全依賴情緒

DOGE 的價格取決於：

- Musk 的推文
- 網路迷因的流行度
- 散戶的情緒

**不取決於**：技術、採用率、實際使用。

#### 3. 巨鯨集中

據分析，少數幾個地址持有大量 DOGE。

如果這些大戶賣出，價格可能崩盤。

#### 4. 歷史教訓

從歷史高點 $0.73，DOGE 已經跌了超過 85%。

### 那些靠 DOGE 致富的人

確實有人靠 DOGE 賺了大錢：

| 類型 | 情況 |
|------|------|
| 早期持有者 | 2013-2014 年買入，持有到 2021 年 |
| 精準交易者 | 在 Musk 推文前買入，高點賣出 |
| 運氣好的人 | 隨便買了一些，忘記了，後來發現暴漲 |

**但更多的人**：

- 在高點 FOMO 買入
- 看著價格下跌 80%+
- 至今還在「等解套」

### DOGE 的文化意義

不管你怎麼看 DOGE，它確實改變了一些事情：

#### 1. 迷因幣的先驅

DOGE 證明了：加密貨幣不需要「有用」也能有價值。

這直接催生了：SHIB、PEPE、FLOKI⋯⋯無數迷因幣。

#### 2. 加密貨幣大眾化

很多人第一次接觸加密貨幣，是因為 DOGE。

它讓加密貨幣變得「可愛」、「好玩」、「不那麼嚇人」。

#### 3. 社群力量的展示

DOGE 證明了：在加密貨幣世界，社群和敘事可以比技術更重要。

### 現在的 DOGE

截至目前，DOGE 仍然是市值前 10 的加密貨幣。

**近期發展**：

| 項目 | 說明 |
|------|------|
| X（Twitter）整合 | Musk 曾暗示可能在 X 上使用 DOGE |
| Tesla 商品 | 曾接受 DOGE 支付部分商品 |
| 開發更新 | 有一些小更新，但不多 |

### 如果你想買 DOGE

{% hint style="warning" %}
這不是投資建議。但如果你決定要買，請記住：
{% endhint %}

1. **只用你能承受損失的錢**：可能歸零
2. **不要 FOMO**：Musk 推文後才買通常已經晚了
3. **設好停損**：有紀律地交易
4. **了解你在做什麼**：這是投機，不是投資

### DOGE 的未來？

**樂觀的情境**：

- Musk 真的在 X 上整合 DOGE 支付
- 成為網路小費/打賞的標準貨幣
- 更多商家接受

**悲觀的情境**：

- Musk 興趣轉移
- 被新的迷因幣取代
- 慢慢被遺忘

**最可能的情境**：

繼續存在，價格隨 Musk 的推文和整體市場情緒波動。

{% hint style="success" %}
Dogecoin 是加密貨幣世界最荒謬的成功故事。一個「玩笑」變成了數百億美元的資產。它沒有技術創新，沒有實用價值，但它有最強大的力量——迷因和社群。DOGE 提醒我們：在加密貨幣世界，理性不是唯一的驅動力。人類是非理性的動物，而市場有時候就是反映這種非理性。
{% endhint %}
