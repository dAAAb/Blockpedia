# NEM
> **最後更新：2026 年 1 月**

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## NEM（New Economy Movement）是 2015 年推出的區塊鏈平台，以創新的「重要性證明」（PoI）共識機制聞名。它曾是市值前十的區塊鏈項目，後來演化為 Symbol。NEM 的故事是區塊鏈技術演進的縮影。

2015 年，當大家還在討論比特幣的工作量證明（PoW）時——

NEM 提出了一個問題：**為什麼只獎勵有錢的人？**

PoW：礦機越多，獎勵越多。

PoS：幣越多，獎勵越多。

NEM 說：**我們應該獎勵「重要」的人，不是「有錢」的人。**

這就是「重要性證明」（Proof of Importance）的由來。

### 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | NEM（New Economy Movement）|
| 代幣 | XEM |
| 上線 | 2015 年 |
| 共識機制 | PoI（重要性證明）|
| 創辦 | 社群發起（匿名團隊）|
| 總供應量 | 約 90 億枚 |
| 後繼者 | Symbol（XYM）|

### 重要性證明（PoI）

NEM 最大的創新是 Proof of Importance。

#### 什麼是「重要性」？

| 因素 | 說明 |
|------|------|
| **持幣量** | 你擁有多少 XEM |
| **交易活躍度** | 你發送了多少交易 |
| **交易對象** | 你和誰交易（網路分析）|

**用白話說**：如果你只是囤幣不動，你的「重要性」不高。如果你積極使用網路、和其他人互動，你的「重要性」就高。

#### PoI vs PoS

| 機制 | 獎勵誰 |
|------|--------|
| **PoS** | 持幣最多的人 |
| **PoI** | 對網路貢獻最大的人 |

{% hint style="info" %}
PoI 的設計理念是鼓勵「使用」而不是「囤積」。這與很多人「Hold 就對了」的投資策略相反——你必須使用才能獲得更高的獎勵。
{% endhint %}

### 「收穫」機制

NEM 不叫「挖礦」或「質押」，它叫「收穫」（Harvesting）。

#### 如何收穫？

```
1. 持有至少 10,000 XEM
2. 將帳戶連接到超級節點
3. 根據「重要性分數」獲得區塊獎勵
```

**特點**：

- 不需要挖礦設備
- 不需要一直在線
- 可以「委託收穫」給其他節點

### NEM 的技術特色

#### 1. 多重簽名

NEM 原生支援多重簽名：

```
例如：公司帳戶需要 3/5 的人簽名才能動用
```

這在當時是創新——比特幣和以太坊都需要額外的智能合約。

#### 2. 馬賽克（Mosaic）

NEM 上可以創建自定義資產，稱為「馬賽克」：

| 功能 | 說明 |
|------|------|
| 創建代幣 | 不需要寫程式 |
| 可轉讓性 | 可以設定能否轉讓 |
| 供應量 | 可以設定固定或可變 |
| 有效期 | 可以設定過期時間 |

**這是早期的「代幣化」工具**，比 ERC-20 更早出現。

#### 3. 命名空間

類似網域名稱，可以為地址取名：

```
例如：yourcompany:asset1
```

### NEM 的歷史

#### 2015-2016：起步

- 2015 年 3 月：主網上線
- 社群發起，沒有 ICO
- 專注亞洲市場（特別是日本）

#### 2017：輝煌時刻

| 時間 | 價格 | 事件 |
|------|------|------|
| 2017.1 | $0.004 | 年初 |
| 2017.5 | $0.20 | 開始上漲 |
| 2018.1 | $1.90 | 歷史高點 |

**漲幅**：從 2017 年初到高點，約 **400 倍**。

#### 2018：Coincheck 駭客事件

2018 年 1 月，日本交易所 Coincheck 被駭：

| 項目 | 數據 |
|------|------|
| 損失金額 | 約 5.23 億美元 |
| 被盜 XEM | 約 5 億枚 |
| 佔總供應量 | 約 5.5% |

{% hint style="danger" %}
這是當時歷史上最大的加密貨幣盜竊案。雖然是交易所的問題，不是 NEM 本身的漏洞，但對 NEM 的聲譽造成嚴重打擊。
{% endhint %}

#### 2021：Symbol 上線

NEM 基金會推出下一代區塊鏈 Symbol：

| 項目 | NEM (XEM) | Symbol (XYM) |
|------|-----------|--------------|
| 定位 | 原始鏈 | 企業級升級版 |
| 代幣 | XEM | XYM |
| 技術 | 較舊 | 更先進 |

**Symbol 的特點**：

- 更強的性能
- 更好的智能合約
- 混合區塊鏈（公有/私有）

### 價格歷史

| 時期 | 價格 |
|------|------|
| 2017.1 | $0.004 |
| 2018.1 | $1.90（歷史高點）|
| 2018.12 | $0.04 |
| 2021.2 | $0.70（反彈高點）|
| 2022-2024 | $0.02-0.05 |

{% hint style="warning" %}
NEM 從最高點 $1.90 跌到 $0.03，跌幅超過 98%。即使是曾經的前十項目，也無法避免市場的殘酷。
{% endhint %}

### NEM 在日本的影響

NEM 在日本特別受歡迎：

- 活躍的日本社群
- Coincheck 是日本最大交易所之一
- 很多日本企業探索 NEM 應用

**有趣的事**：Coincheck 被駭後，日本監管加強，推動了日本的加密貨幣法規發展。

### 投資考量

#### 潛在優勢

- PoI 概念仍有價值
- Symbol 是技術升級
- 日本社群基礎
- 價格已經很低

#### 風險

| 風險 | 說明 |
|------|------|
| 關注度低 | 市場熱度早已消退 |
| Symbol 採用慢 | 新鏈未獲得預期關注 |
| 競爭激烈 | 新一代區塊鏈太多 |
| 生態薄弱 | DApp 和開發者不足 |

### NEM 的歷史意義

不管現在的處境如何，NEM 有其歷史貢獻：

1. **共識創新**：PoI 是對 PoW/PoS 的重要思考
2. **早期代幣化**：馬賽克系統早於 ERC-20
3. **多簽先驅**：原生多重簽名設計
4. **市場教育**：讓很多人（特別是日本）認識區塊鏈

{% hint style="success" %}
NEM 是 2015-2018 年區塊鏈創新浪潮的代表之一。它的「重要性證明」概念提出了一個好問題：區塊鏈應該獎勵什麼行為？這個問題至今仍有意義。雖然 NEM 本身已經邊緣化，但它的思想影響了後來的許多項目。
{% endhint %}
