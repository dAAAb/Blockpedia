# 瑞波幣（XRP）
> **最後更新：2026 年 1 月**


## XRP 是 Ripple 公司開發的加密貨幣，專為跨境支付設計。它不用挖礦，交易只需 3-5 秒，手續費幾乎為零。但它也是最具爭議的加密貨幣之一——因為與 SEC 打了多年的官司。

銀行跨境匯款要多久？

3-5 個工作天。

手續費多少？

$25-50，外加匯率差價。

XRP 說：我可以在 3 秒內完成，手續費不到 $0.01。

這就是 Ripple 的願景：用區塊鏈重塑全球支付系統。

但這條路走得並不平順。

### 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | XRP（常被稱為 Ripple）|
| 公司 | Ripple Labs |
| 創立 | 2012 年 |
| 共識機制 | XRP Ledger 共識協議 |
| 總供應量 | 1,000 億枚（已全部創建）|
| 交易速度 | 3-5 秒 |
| 手續費 | 約 $0.0002 |

### XRP ≠ Ripple

這是常見的混淆：

| 名稱 | 是什麼 |
|------|--------|
| **Ripple** | 一家美國金融科技公司 |
| **XRP** | 一種加密貨幣 |
| **XRP Ledger** | 運行 XRP 的區塊鏈 |
| **RippleNet** | Ripple 的支付網路產品 |

**簡單說**：Ripple 是公司，XRP 是代幣。公司可以倒，代幣理論上還會存在。

{% hint style="info" %}
Ripple 公司持有大量 XRP（約 400-500 億枚），這是 XRP 被批評「中心化」的主要原因。
{% endhint %}

### Ripple 的商業模式

Ripple 不是做給散戶用的——它的客戶是銀行和金融機構。

#### 傳統跨境支付的問題

```
台灣 → 美國 匯款流程：

台灣銀行 → 台灣央行 → SWIFT 網路 → 美國央行 → 美國銀行

時間：3-5 天
費用：$25-50
問題：中間環節多、效率低、成本高
```

#### Ripple 的解決方案

```
使用 RippleNet：

台灣銀行 ←→ RippleNet ←→ 美國銀行

時間：幾秒鐘
費用：大幅降低
好處：即時結算、透明追蹤
```

#### 產品線

| 產品 | 功能 | 是否用 XRP |
|------|------|------------|
| xCurrent | 即時跨境支付訊息 | 不一定 |
| xRapid（現 ODL）| 流動性解決方案 | 是 |
| xVia | API 接入服務 | 不一定 |

{% hint style="warning" %}
重點：很多銀行用 Ripple 的技術，但不一定用 XRP 代幣。這是投資者需要注意的——Ripple 成功不等於 XRP 成功。
{% endhint %}

### XRP 的技術特點

#### 不是區塊鏈？

嚴格來說，XRP Ledger 不是傳統意義的「區塊鏈」。

| 項目 | 比特幣 | XRP Ledger |
|------|--------|------------|
| 共識機制 | PoW（挖礦）| 聯邦拜占庭協議 |
| 驗證者 | 任何人 | 受信任節點清單 |
| 區塊時間 | 10 分鐘 | 3-5 秒 |
| 能源消耗 | 極高 | 極低 |

#### 預先鑄造

XRP 的 1000 億枚代幣在 2012 年就全部創建完成。

- 沒有挖礦
- 沒有通膨
- Ripple 公司持有大部分

**代幣分配**：

| 持有者 | 數量 |
|--------|------|
| Ripple 公司 | ~400-500 億 |
| 創辦人 | ~數十億 |
| 市場流通 | ~500 億+ |

### SEC 訴訟案

這是 XRP 歷史上最重要的事件。

#### 事件經過

| 時間 | 事件 |
|------|------|
| 2020.12 | SEC 起訴 Ripple，指控 XRP 是未註冊證券 |
| 2021-2023 | 漫長的法律戰 |
| 2023.7 | 法官裁定：程式化銷售不構成證券 |
| 2024 | 持續的後續程序 |

#### 核心問題

SEC 認為：XRP 是「投資合約」，屬於證券，應該註冊。

Ripple 認為：XRP 是貨幣/商品，不是證券。

#### 2023 年判決的意義

法官 Torres 的裁決：

| 銷售類型 | 裁定 |
|----------|------|
| 機構銷售 | 構成證券（違法）|
| 程式化銷售（交易所）| 不構成證券 |
| 其他銷售 | 不構成證券 |

**這意味著**：散戶在交易所買 XRP，不算購買證券。

{% hint style="success" %}
這個判決對整個加密貨幣產業影響深遠——它為「代幣可能不是證券」提供了法律先例。
{% endhint %}

### XRP 的爭議

#### 1. 中心化

- Ripple 公司控制大量 XRP
- 驗證節點數量有限
- 與比特幣的去中心化精神相違背

#### 2. 預挖與拋售

- 創辦人和公司持有大量代幣
- 每月定期釋放代幣
- 被批評是「印鈔機」

#### 3. 實際採用

- 很多「合作夥伴」只是試點
- 真正使用 XRP（而非只用技術）的銀行有限
- 業務進展不如宣傳

### XRP 的支持者觀點

支持 XRP 的人認為：

1. **實際用途**：真的有銀行在用
2. **速度優勢**：比特幣太慢、以太坊太貴
3. **合規路線**：與監管合作而非對抗
4. **官司勝利**：SEC 案件的部分勝訴證明合法性

### 投資 XRP 的考量

#### 潛在優勢

- 如果被更多銀行採用，需求會增加
- 官司勝利後，可能重新上架美國交易所
- 跨境支付市場巨大

#### 風險

| 風險 | 說明 |
|------|------|
| 中心化 | Ripple 公司影響力過大 |
| 代幣釋放 | 持續的賣壓 |
| 競爭 | SWIFT 升級、CBDC、其他解決方案 |
| 法律不確定 | SEC 案件仍有後續 |
| 採用緩慢 | 銀行採用速度不如預期 |

### XRP 的價格歷史

| 時期 | 價格 | 事件 |
|------|------|------|
| 2017.12 | $0.006 | 牛市前 |
| 2018.1 | $3.84 | 歷史高點 |
| 2018.12 | $0.25 | 熊市低點 |
| 2020.12 | $0.20 | SEC 起訴前 |
| 2020.12 | $0.17 | SEC 起訴後暴跌 |
| 2023.7 | $0.80 | 部分勝訴後 |

{% hint style="warning" %}
XRP 從歷史高點 $3.84 跌了超過 80%，至今未能收復。這提醒我們：即使是「實用」的代幣，價格也可能長期低迷。
{% endhint %}

### XRP 與其他支付幣比較

| 幣種 | 定位 | 特點 |
|------|------|------|
| **XRP** | 銀行跨境支付 | 快速、便宜、但中心化 |
| **XLM（Stellar）** | 個人跨境支付 | XRP 創辦人另立門戶 |
| **SWIFT** | 傳統銀行網路 | 慢但可靠、正在升級 |
| **USDC/USDT** | 穩定幣 | 也可用於跨境支付 |

### Ripple 的未來

| 方向 | 說明 |
|------|------|
| IPO | 官司結束後可能上市 |
| CBDC | 與央行合作開發數位貨幣 |
| NFT | XRP Ledger 支援 NFT |
| 擴展 | 持續拓展亞太、中東市場 |

{% hint style="success" %}
XRP 是加密貨幣世界裡最「傳統金融友好」的項目之一。它不追求去中心化的極致，而是試圖在現有金融體系中找到位置。這是優勢也是劣勢——它更容易被機構接受，但也更依賴中心化實體。投資 XRP，本質上是在投資 Ripple 公司能否成功改變跨境支付產業。
{% endhint %}
