# 門羅幣

### 比特幣是「假名制」，門羅幣是「真匿名」。它讓每一筆交易的發送者、接收者、金額都隱藏起來——即使是區塊鏈分析公司也無法追蹤。這是把隱私當作基本權利的加密貨幣。

## 隱私技術

### 三大隱私機制

```
門羅幣如何實現隱私？

1. 環簽名（Ring Signatures）
   作用：隱藏發送者
   原理：你的簽名混在一群誘餌中
   類比：10 個人簽名，不知道是誰

2. 隱匿地址（Stealth Addresses）
   作用：隱藏接收者
   原理：每次生成一次性地址
   類比：每次收款用不同信箱

3. 環形機密交易（RingCT）
   作用：隱藏金額
   原理：加密交易金額
   類比：只知道有交易，不知道多少錢

結果：
- 無法追蹤發送者
- 無法追蹤接收者
- 無法知道金額
- 真正的金融隱私
```

### 與其他隱私幣比較

| 特性 | 門羅幣 | 零幣 | Dash |
|------|--------|------|------|
| 隱私方式 | 環簽名 | zk-SNARKs | CoinJoin |
| 預設隱私 | 是（強制）| 否（選擇）| 否（選擇）|
| 可追蹤性 | 完全不可追蹤 | z-addr 不可追蹤 | 可能被追蹤 |
| 區塊大小 | 動態 | 固定 | 固定 |

{% hint style="info" %}
**強制 vs 選擇性隱私**

門羅幣的所有交易都是隱私的，沒有「透明」選項。這是設計哲學的選擇——如果只有部分人使用隱私功能，那些使用的人反而更顯眼。
{% endhint %}

## 技術特色

### 抗 ASIC 挖礦

```
門羅幣的挖礦政策：

問題：
ASIC 礦機導致挖礦中心化
（少數人控制大部分算力）

解決方案：
定期更換挖礦算法
→ RandomX（目前）
→ 對 CPU 友好，ASIC 不友好

結果：
- 普通電腦也能挖
- 算力更分散
- 抗審查性更強
```

### 動態區塊大小

| 特性 | 說明 |
|------|------|
| 區塊大小 | 隨需求自動調整 |
| 無上限 | 理論上可無限擴展 |
| 懲罰機制 | 過大區塊會減少獎勵 |
| 優勢 | 不會像比特幣那樣塞車 |

## 使用場景

### 合法用途

| 場景 | 說明 |
|------|------|
| 金融隱私 | 不想被追蹤的正常需求 |
| 薪資支付 | 保護薪資資訊 |
| 捐款 | 匿名慈善捐款 |
| 記者/異議人士 | 保護資金來源 |

### 爭議用途

| 場景 | 說明 |
|------|------|
| 勒索軟體 | 部分勒索要求 XMR |
| 暗網市場 | 取代比特幣 |
| 洗錢 | 被認為有洗錢風險 |

{% hint style="warning" %}
**隱私權的兩面刃**

隱私是基本人權，但也可能被濫用。門羅幣的設計讓執法機關無法追蹤交易，這引發了持續的監管爭議。
{% endhint %}

## 投資考量

### 優勢與風險

| 優勢 | 風險 |
|------|------|
| 最強的隱私保護 | 監管打壓嚴重 |
| 活躍的開發社群 | 交易所下架風險 |
| 真正的去中心化 | 負面形象 |
| 抗審查能力 | 流動性可能降低 |

{% hint style="danger" %}
**監管風險警告**

門羅幣面臨全球監管壓力。多個主要交易所已下架 XMR。持有門羅幣可能面臨法律風險，請了解當地法規。
{% endhint %}
