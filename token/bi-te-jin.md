# 比特幣現金（Bitcoin Cash）
> **最後更新：2026 年 1 月**

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## 比特幣現金是 2017 年從比特幣分叉出來的加密貨幣。它主張「更大的區塊」，希望成為真正用於日常支付的電子現金——而不只是「數位黃金」。

2017 年，比特幣社群爆發了一場「內戰」。

核心問題只有一個：**區塊要多大？**

一派說：「1MB 夠了，擴容用 Layer 2。」

另一派說：「不行，要直接增大區塊到 8MB、32MB、甚至更大。」

雙方吵了好幾年，最後沒有共識。

2017 年 8 月 1 日，支持大區塊的人分叉出去。

那就是比特幣現金（Bitcoin Cash，BCH）。

### 基本資訊

| 項目 | 內容 |
|------|------|
| 名稱 | Bitcoin Cash |
| 代號 | BCH |
| 創立 | 2017 年 8 月 1 日（分叉）|
| 區塊大小 | 32 MB（BTC 為 ~1-4 MB）|
| 共識機制 | 工作量證明（PoW）|
| 總供應量 | 2,100 萬枚（與 BTC 相同）|

### 為什麼要分叉？

要理解 BCH，必須先理解比特幣的「擴容之爭」。

#### 比特幣的瓶頸

比特幣每 10 分鐘產生一個區塊，每個區塊約 1MB。

**這意味著**：

- 每秒約 7 筆交易（TPS）
- 相比之下，Visa 可以處理數千筆

當比特幣越來越熱門，問題就來了：

| 年份 | 情況 |
|------|------|
| 2016 | 交易開始擁堵 |
| 2017 | 手續費飆升到 $50+ |
| 用戶 | 小額支付變得不可行 |

**中本聰的願景**：比特幣是「點對點電子現金系統」。

**現實**：買杯咖啡要付比咖啡還貴的手續費。

#### 兩派的解決方案

| 派別 | 方案 | 代表人物 |
|------|------|----------|
| 小區塊派 | 維持 1MB，用 Layer 2（如閃電網路）擴容 | Bitcoin Core 開發者 |
| 大區塊派 | 直接增大區塊大小 | Roger Ver、Jihan Wu |

**小區塊派的觀點**：

- 大區塊會增加節點運營成本
- 導致中心化（只有大公司能運營節點）
- 違背比特幣去中心化的精神

**大區塊派的觀點**：

- 鏈上擴容最直接、最簡單
- Layer 2 太複雜，一般人不會用
- 比特幣應該是現金，不是「數位黃金」

{% hint style="info" %}
這場辯論的本質是：比特幣應該優先保證「去中心化」，還是優先保證「可用性」？兩派都有道理，這就是為什麼最終分道揚鑣。
{% endhint %}

### 分叉後的發展

2017 年 8 月 1 日，BCH 正式分叉。

**初期**：

- 區塊大小提升到 8MB
- 所有持有 BTC 的人都獲得等量 BCH
- 價格一度達到 BTC 的 1/10

**後來**：

- 2018 年再次分叉出 BSV（Bitcoin SV）
- BCH 區塊大小增加到 32MB
- 市值持續下滑，目前約為 BTC 的 1%

### BCH vs BTC 比較

| 項目 | BTC | BCH |
|------|-----|-----|
| 區塊大小 | ~1-4 MB | 32 MB |
| 每秒交易 | ~7 TPS | ~100+ TPS |
| 平均手續費 | $1-20 | $0.01 以下 |
| 市值排名 | #1 | #15-20 |
| 定位 | 數位黃金、價值儲存 | 電子現金、支付 |
| 機構採用 | 高 | 低 |
| 去中心化程度 | 較高 | 較低 |

### BCH 的核心主張

#### 1. 鏈上擴容

BCH 堅持在主鏈上擴容，而不是依賴 Layer 2。

**優點**：

- 用戶體驗簡單
- 不需要學習新技術
- 交易直接在鏈上完成

**缺點**：

- 區塊大需要更多儲存空間
- 節點運營成本增加
- 可能導致中心化

#### 2. 低手續費支付

BCH 希望成為真正的「電子現金」。

| 場景 | BTC | BCH |
|------|-----|-----|
| 買咖啡 | 手續費可能比咖啡貴 | 手續費不到 1 分錢 |
| 小額打賞 | 不實際 | 可行 |
| 日常支付 | 困難 | 可行 |

#### 3. 「真正的比特幣」

BCH 支持者認為他們才是繼承中本聰願景的人。

> 「中本聰的白皮書標題是『點對點電子現金系統』，不是『數位黃金』。」

{% hint style="warning" %}
這個爭論沒有標準答案。BTC 支持者認為「價值儲存」更重要，BCH 支持者認為「支付功能」更重要。市場目前明顯偏好 BTC。
{% endhint %}

### BCH 的困境

#### 1. 市場認可度低

- BTC 有 ETF、機構投資
- BCH 幾乎沒有機構關注
- 價格持續跑輸 BTC

#### 2. 閃電網路的崛起

BTC 的 Layer 2 解決方案——閃電網路——逐漸成熟。

**這削弱了 BCH 的核心論述**：

- 閃電網路也能做到低手續費、快速支付
- 而且保持了 BTC 的去中心化

#### 3. 社群分裂

2018 年，BCH 自己也分裂了：

| 幣種 | 支持者 | 理念 |
|------|--------|------|
| BCH | Roger Ver 等 | 32MB 區塊 |
| BSV | Craig Wright | 無上限區塊，聲稱是「真正的中本聰願景」|

分裂進一步削弱了大區塊派的力量。

#### 4. 網路效應

比特幣的網路效應太強大：

- 交易所流動性
- 商家接受度
- 品牌認知度

BCH 很難突破。

### 誰還在用 BCH？

雖然市值不如 BTC，BCH 仍有一些使用場景：

| 場景 | 說明 |
|------|------|
| 小額支付 | 手續費低，適合小額轉帳 |
| 某些商家 | 一些商家同時接受 BTC 和 BCH |
| 特定地區 | 一些發展中國家的用戶 |
| 理念支持者 | 相信「電子現金」願景的人 |

### 投資 BCH 的考量

#### 潛在優勢

- 如果閃電網路不成功，BCH 的論述可能重新被重視
- 手續費低，作為支付工具有實際用途
- 價格相對 BTC 便宜，波動可能更大

#### 風險

- 市場份額持續流失
- 機構興趣幾乎為零
- 社群分裂（BCH vs BSV）
- BTC 的網路效應難以撼動

{% hint style="danger" %}
BCH 從歷史高點已經跌了 95%+。它可能繼續存在，但要挑戰 BTC 的地位非常困難。投資 BCH 是在押注「支付優先」的願景會贏過「價值儲存」的願景——目前市場不這麼認為。
{% endhint %}

### BCH 的歷史教訓

BCH 的故事告訴我們幾件事：

#### 1. 網路效應很重要

即使技術有改進，新鏈也很難挑戰既有的龍頭。

#### 2. 社群共識很難達成

比特幣社群的分裂，證明了去中心化治理的困難。

#### 3. 市場會做出選擇

不管哪派的論述更「正確」，最終是市場決定價值。

{% hint style="success" %}
BCH 代表了區塊鏈世界裡「務實派」的聲音——他們認為加密貨幣應該首先是「可用的」，而不只是「稀缺的」。這個願景沒有錯，但市場目前選擇了 BTC 的「數位黃金」敘事。BCH 的故事提醒我們：在加密貨幣世界，不只是技術，敘事和共識同樣重要。
{% endhint %}
