# 萊特幣

### 如果比特幣是數位黃金，萊特幣就是數位白銀。它是比特幣的「輕量版」——同樣的配方，但出塊更快、手續費更低。作為最老牌的山寨幣，它見證了整個加密貨幣的發展史。

---

| 創立 | 2011 年 10 月 |

| 定位 | 比特幣的輕量替代品 |

## 與比特幣的比較

### 技術差異

```
萊特幣 vs 比特幣：

比喻：
比特幣 = 黃金（價值儲存）
萊特幣 = 白銀（日常交易）

參數比較：
                比特幣      萊特幣
出塊時間        10 分鐘     2.5 分鐘
總量            2,100 萬    8,400 萬
挖礦算法        SHA-256     Scrypt
首次減半        2012        2015

關係：
- 程式碼基礎相同
- 萊特幣常用於測試新功能
- 「比特幣的測試網」
```

### 關鍵參數

| 參數 | 比特幣 | 萊特幣 |
|------|--------|--------|
| 區塊時間 | 10 分鐘 | 2.5 分鐘 |
| 總發行量 | 2,100 萬 | 8,400 萬 |
| 挖礦算法 | SHA-256 | Scrypt |
| 確認速度 | 約 60 分鐘 | 約 15 分鐘 |

{% hint style="info" %}
**為什麼是 4 倍？**

萊特幣的很多參數都是比特幣的 4 倍——總量 4 倍、出塊速度 4 倍。這讓萊特幣可以處理更多交易，同時保持與比特幣相似的經濟模型。
{% endhint %}

## 發展歷程

### 重要里程碑

| 時間 | 事件 |
|------|------|
| 2011/10 | 萊特幣誕生 |
| 2013 | 首次價格飆漲 |
| 2017/5 | 啟用隔離見證（SegWit）|
| 2017/12 | 價格歷史高點 |
| 2019 | 第二次減半 |
| 2022 | MWEB 隱私升級 |
| 2023 | 第三次減半 |

### 創辦人故事

```
Charlie Lee 的故事：

背景：
- 前 Google 工程師
- MIT 畢業
- 2011 年創立萊特幣

2017 年爭議事件：
- 萊特幣價格歷史高點時
- Charlie Lee 宣布賣出所有 LTC
- 稱為「避免利益衝突」
- 社群反應兩極

現況：
- 仍是萊特幣基金會成員
- 持續參與開發
- 但不再持有 LTC
```

{% hint style="warning" %}
**創辦人賣幣事件**

2017 年牛市高點，Charlie Lee 賣出所有萊特幣。雖然他說是為了避免利益衝突，但這個舉動讓許多投資者感到被背叛。
{% endhint %}

## 技術創新

### 萊特幣的貢獻

| 創新 | 說明 |
|------|------|
| SegWit 先行者 | 比比特幣更早啟用 |
| 閃電網路測試 | 跨鏈原子交換 |
| MWEB | MimbleWimble 隱私協議 |

### MWEB 升級

```
2022 年 MWEB 升級：

什麼是 MWEB？
- MimbleWimble Extension Blocks
- 選擇性隱私功能
- 隱藏交易金額

工作原理：
主鏈 ←→ MWEB 側鏈
（透明）    （隱私）

用戶可選擇：
- 一般交易（公開）
- MWEB 交易（隱私）

影響：
- 部分交易所下架
- 合規性疑慮
- 但增加了使用場景
```

## 投資考量

### 優勢與風險

| 優勢 | 風險 |
|------|------|
| 歷史悠久、穩定 | 創新不足 |
| 比特幣的測試場 | 競爭者眾多 |
| 廣泛被支援 | 不再是「最快」的選擇 |
| 品牌認知度高 | 敘事缺乏新意 |

{% hint style="info" %}
**投資定位**

萊特幣是「穩健的老牌幣」。不會有爆發性增長，但也不太可能歸零。它就像加密貨幣世界的 IBM——不性感，但持續運作。
{% endhint %}
