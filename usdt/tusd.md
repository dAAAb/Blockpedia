# TUSD

### TUSD 曾經是「最透明的穩定幣」——它是第一個提供實時儲備證明的穩定幣。但在快速發展的穩定幣市場中，TUSD 的地位經歷了起伏。

---

## 什麼是 TUSD？

| 項目 | 說明 |
|------|------|
| 全名 | TrueUSD |
| 發行方 | TrustToken（後更名 Archblock）|
| 創立 | 2018 年 |
| 錨定 | 1 TUSD = 1 USD |
| 特色 | 實時儲備證明 |

---

## 核心特色

### 透明度機制

```
TUSD 的透明度特色：

1. 實時儲備證明
   - 鏈上可驗證
   - 透過 Chainlink Proof of Reserve
   - 任何人可查詢

2. 第三方託管
   - 資金存於受監管銀行
   - 多個託管銀行分散風險
   - 不由 TrustToken 直接持有

3. 定期審計
   - 獨立會計師事務所
   - 定期發布報告
```

### 與其他穩定幣比較

| 穩定幣 | 即時證明 | 審計頻率 | 監管 |
|--------|----------|----------|------|
| TUSD | 有 | 月度 | 部分州 |
| USDC | 無 | 月度 | 較完整 |
| USDT | 無 | 季度 | 有限 |

{% hint style="info" %}
**Chainlink Proof of Reserve**

TUSD 使用 Chainlink 的預言機網路來提供實時儲備證明。這讓任何人都可以在鏈上驗證儲備是否充足。
{% endhint %}

## 發展歷程

### 重要事件

| 時間 | 事件 |
|------|------|
| 2018 | TUSD 發行 |
| 2019 | 引入實時儲備證明 |
| 2020 | 更名為 Archblock |
| 2022 | 交易量增長 |
| 2023 | 幣安推動使用，後又暫停 |

### 2023 年爭議

```
TUSD 在 2023 年的起伏：

上升期：
- 幣安大力推廣
- 交易量大幅增加
- 一度成為主要交易對

爭議期：
- 審計公司更換
- 儲備透明度質疑
- 暫時脫離錨定

後續：
- 恢復錨定
- 但市場信心受影響
- 交易量回落
```

## 風險考量

### 主要風險

| 風險 | 說明 |
|------|------|
| 市場份額小 | 流動性不如 USDT/USDC |
| 審計變動 | 2023 年審計爭議 |
| 中心化 | 仍是中心化穩定幣 |
| 依賴交易所 | 交易所支持影響大 |

### 與大型穩定幣比較

```
選擇 TUSD 還是 USDT/USDC？

TUSD 優勢：
- 實時儲備證明
- 理論上更透明

TUSD 劣勢：
- 流動性較差
- 市場份額小
- 交易對較少
- 2023 年信心受損

建議：
- 若需大量交易，USDT/USDC 更方便
- 若重視透明度，可考慮 TUSD
- 但注意流動性風險
```

{% hint style="warning" %}
**流動性風險**

TUSD 的市值和交易量遠小於 USDT 和 USDC。這意味著在極端市場情況下，可能較難快速變現或有較大滑點。
{% endhint %}
