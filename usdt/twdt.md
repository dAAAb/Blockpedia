# TWDT

### 全球有 USDT，台灣有 TWDT——這是與新台幣錨定的穩定幣，讓台灣用戶可以用熟悉的貨幣單位進入加密世界。但本土穩定幣面臨的挑戰也不少。

---

## 什麼是 TWDT？

| 項目 | 說明 |
|------|------|
| 全名 | Taiwan Dollar Token |
| 錨定 | 1 TWDT = 1 TWD |
| 類型 | 法幣錨定穩定幣 |
| 定位 | 台灣本土穩定幣 |

---

## 為什麼需要 TWDT？

### 本土穩定幣的優勢

```
TWDT 的使用場景：

1. 避免匯率風險
   - 直接用台幣計價
   - 不需轉換美元
   - 熟悉的貨幣單位

2. 本地支付場景
   - 台灣商家收款
   - 本地對本地轉帳
   - 更低的轉換成本

3. 法規合規
   - 更符合台灣法規
   - 本地銀行合作
   - 在地化服務
```

### 與 USDT 比較

| 項目 | TWDT | USDT |
|------|------|------|
| 錨定貨幣 | 新台幣 | 美元 |
| 市場規模 | 小 | 極大 |
| 流動性 | 有限 | 充足 |
| 交易對 | 少 | 多 |
| 本地優勢 | 有 | 無 |

{% hint style="info" %}
**規模差異巨大**

TWDT 的市值和流動性遠小於 USDT。這意味著大額交易可能有滑點，可用的交易對也較少。
{% endhint %}

## 運作機制

### 發行與贖回

| 流程 | 說明 |
|------|------|
| 發行 | 存入台幣 → 獲得 TWDT |
| 贖回 | 交還 TWDT → 取回台幣 |
| 費用 | 可能有手續費 |
| KYC | 需要身份驗證 |

### 儲備機制

```
TWDT 的托底：

理論機制：
- 用戶存入 1 TWD
- 發行方銀行帳戶存 1 TWD
- 發行 1 TWDT 給用戶

關鍵問題：
- 儲備是否真的 100%？
- 有沒有獨立審計？
- 銀行帳戶在哪？
```

## 挑戰與風險

### 面臨的挑戰

| 挑戰 | 說明 |
|------|------|
| 流動性 | 交易量小，滑點高 |
| 採用率 | 使用場景有限 |
| 競爭 | USDT 已深入人心 |
| 法規 | 台灣穩定幣法規不明確 |

### 風險考量

| 風險 | 說明 |
|------|------|
| 發行方風險 | 小型發行方的信用風險 |
| 透明度 | 審計和儲備的透明度 |
| 流動性風險 | 可能無法快速變現 |
| 法規風險 | 法規變動可能影響運作 |

{% hint style="warning" %}
**小型穩定幣風險較高**

相較於 USDT、USDC 等大型穩定幣，小型穩定幣的風險通常較高。規模小意味著：更容易被擠兌、更少的審計、更不透明。
{% endhint %}

台灣穩定幣的發展方向：
