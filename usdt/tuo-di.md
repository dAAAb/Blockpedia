# 托底

### 穩定幣為什麼「穩定」？因為背後有東西「托底」——每發行一塊錢的穩定幣，就有一塊錢的資產支撐。但托底的資產是什麼、夠不夠，一直是市場關注的焦點。

---

## 什麼是托底？

| 概念 | 說明 |
|------|------|
| 定義 | 穩定幣背後的儲備資產 |
| 目的 | 確保可以兌換回價值 |
| 重要性 | 穩定幣信用的基礎 |
| 爭議 | 儲備是否足夠、透明 |

---

## 托底資產類型

### 不同形式

```
穩定幣的托底資產：

1. 現金及等價物
   - 銀行存款
   - 貨幣市場基金
   - 最高品質、最具流動性

2. 政府債券
   - 美國國債
   - 短期票券
   - 幾乎等同現金

3. 商業票據
   - 企業短期債務
   - 品質參差不齊
   - USDT 曾持有大量

4. 加密資產
   - 部分用於抵押
   - 風險較高
   - 價值波動大

5. 其他
   - 貸款
   - 投資
   - 透明度通常較差
```

### 各穩定幣托底比較

| 穩定幣 | 主要托底 |
|--------|----------|
| USDT | 現金、國債、商業票據 |
| USDC | 現金、短期國債 |
| BUSD | 現金、國債（已停止發行）|
| DAI | 加密資產超額抵押 |

{% hint style="info" %}
**托底品質很重要**

同樣是「100% 托底」，現金和商業票據的品質差很多。在極端情況下，低品質的托底資產可能無法快速變現。
{% endhint %}

## 透明度問題

### USDT 的爭議

```
USDT 托底爭議歷史：

2019 年：
- 紐約總檢察長調查
- 發現儲備曾借給 Bitfinex
- 和解，罰款 1850 萬美元

2021 年：
- 首次公布儲備組成
- 商業票據佔比高達 50%+
- 市場擔憂儲備品質

2022 年後：
- 逐步減少商業票據
- 增加美國國債
- 但仍不如 USDC 透明
```

### 審計 vs 證明

| 類型 | 說明 |
|------|------|
| 完整審計 | 獨立會計師深入檢查 |
| 證明 | 特定時點的快照 |
| 自我聲明 | 公司自己說的 |
| USDC | 有月度證明 |
| USDT | 季度證明（非完整審計）|

{% hint style="warning" %}
**證明不等於審計**

很多穩定幣只有「證明」（Attestation），不是「審計」（Audit）。證明只確認某個時點的餘額，不檢查整個會計流程。
{% endhint %}

## 風險分析

### 托底不足的風險

| 風險 | 說明 |
|------|------|
| 擠兌 | 大家同時要贖回 |
| 流動性危機 | 資產無法快速變現 |
| 信用損失 | 托底資產違約 |
| 脫鉤 | 價格跌破 $1 |

### 如何降低風險

| 方法 | 說明 |
|------|------|
| 分散持有 | 不要只持有一種穩定幣 |
| 選擇透明 | 選擇審計更透明的 |
| 了解風險 | 知道托底是什麼 |
| 保持流動 | 不要全部鎖定 |

## 監管趨勢

### 各地規定

| 地區 | 趨勢 |
|------|------|
| 美國 | 要求更嚴格的儲備 |
| 歐盟 | MiCA 規定儲備要求 |
| 日本 | 要求全額法幣儲備 |
| 香港 | 建立發牌制度 |

### 未來展望

```
穩定幣儲備的發展方向：

監管要求：
- 更高比例的現金/國債
- 更頻繁的審計
- 更透明的披露

市場壓力：
- 用戶更重視透明度
- 競爭促進改善
- UST 崩盤的教訓
```
