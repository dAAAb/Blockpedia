# USDT（Tether）
> **最後更新：2026 年 1 月**

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## USDT 是全球最大的穩定幣，由 Tether 公司發行，聲稱每一枚 USDT 背後都有等值的美元資產支撐。它是加密貨幣交易的基石——但也是最具爭議的代幣之一。

在加密貨幣圈，有一句話：

> 「如果 USDT 崩盤，整個市場都會跟著崩。」

這不是危言聳聽。

USDT 的市值超過 1000 億美元，幾乎每一個交易所都以它作為主要交易對。

它是加密貨幣的「血液」。

但它真的安全嗎？

### USDT 是什麼？

**全名**：Tether USD

**發行方**：Tether Limited（與交易所 Bitfinex 關係密切）

**機制**：聲稱 1 USDT = 1 USD，由等值資產儲備支撐

**創立**：2014 年（最早叫 Realcoin）

### 為什麼 USDT 這麼重要？

| 數據 | 說明 |
|------|------|
| 市值 | 超過 $1000 億（穩定幣 #1）|
| 日交易量 | 經常超過比特幣 |
| 支援鏈 | 以太坊、Tron、Solana、BSC 等 |
| 交易對數量 | 幾乎所有主流幣都有 USDT 交易對 |

**為什麼大家用 USDT？**

1. **流動性最高**：買賣最容易
2. **歷史最長**：2014 年就存在
3. **交易所支援**：幾乎所有交易所都用
4. **轉帳成本低**：尤其在 Tron 鏈上

### USDT 的爭議

USDT 從誕生以來就充滿爭議。

#### 爭議一：儲備透明度

Tether 長期宣稱「每一枚 USDT 都有 1 美元支撐」。

但實際上：

| 年份 | 事件 |
|------|------|
| 2017 | 與審計公司 Friedman LLP 終止合作，沒有完成審計 |
| 2019 | 紐約總檢察長調查，發現 Tether 曾動用儲備填補 Bitfinex 虧損 |
| 2021 | 與 NYAG 和解，承認儲備並非 100% 現金，支付 1850 萬美元罰款 |

**目前的儲備組成**（根據 Tether 公布的報告）：

| 資產類型 | 比例 |
|----------|------|
| 美國國庫券 | ~80% |
| 現金和銀行存款 | ~5% |
| 其他投資 | ~15% |

{% hint style="warning" %}
Tether 只提供「證明」(attestation)，不是完整審計 (audit)。證明只說「某一時刻」資產足夠，不保證「每一時刻」都足夠。批評者說這不夠透明。
{% endhint %}

#### 爭議二：操縱市場？

有研究論文指出 USDT 的發行可能與比特幣價格操縱有關：

- 2017 年牛市期間，大量 USDT 被鑄造
- 這些 USDT 流入市場買入比特幣
- 但 Tether 否認這些指控

#### 爭議三：可以凍結資產

Tether 有能力凍結任何地址的 USDT。

已知被凍結的案例：

- 與駭客攻擊相關的地址
- 美國制裁名單上的地址
- 執法機構要求凍結的地址

**這意味著**：USDT 不是「抗審查」的。如果你需要抗審查的穩定幣，USDT 不是好選擇。

### USDT 在不同區塊鏈上

USDT 存在於多條鏈上：

| 區塊鏈 | 優點 | 缺點 |
|--------|------|------|
| 以太坊 | 最多 DeFi 整合 | Gas 費高 |
| Tron | 手續費極低（幾乎免費）| 較中心化 |
| Solana | 快速、便宜 | 曾有網路問題 |
| BNB Chain | 便宜 | 幣安生態限定 |
| Polygon | 便宜、以太坊相容 | 較新 |

{% hint style="info" %}
不同鏈上的 USDT 不能直接互轉！以太坊上的 USDT 和 Tron 上的 USDT 是不同的代幣。要跨鏈需要用橋或交易所。
{% endhint %}

### USDT vs USDC

| 比較項目 | USDT | USDC |
|----------|------|------|
| 發行方 | Tether Limited | Circle |
| 市值 | #1（約 $1000 億）| #2（約 $300 億）|
| 透明度 | 較低 | 較高（月度審計）|
| 監管合規 | 爭議較多 | 較合規 |
| 流動性 | 最高 | 第二 |
| 可凍結 | 是 | 是 |
| 主要市場 | 亞洲、交易所 | 美國、DeFi |

### 如果 USDT 崩盤會怎樣？

這是加密貨幣圈最大的「黑天鵝」風險之一。

**可能的場景**：

1. Tether 被揭露儲備嚴重不足
2. 大量用戶嘗試贖回
3. Tether 無法滿足贖回需求
4. USDT 脫鉤（跌破 $1）
5. 恐慌蔓延到整個市場
6. 以 USDT 計價的資產暴跌

**但這也可能不會發生**：

- Tether 已經營運近 10 年
- 經歷過多次 FUD 都挺過來了
- 儲備中大部分是美國國庫券
- 市場對 USDT 的信心仍然存在

### 使用 USDT 的建議

#### ✅ 適合的情況

- 短期交易（買幣賣幣）
- 在 Tron 上低成本轉帳
- 需要最高流動性時

#### ❌ 不適合的情況

- 長期大額持有
- 需要最高透明度
- 需要抗審查

#### 🔒 安全建議

1. 不要把大部分資產放在 USDT
2. 分散持有不同穩定幣（USDT + USDC + DAI）
3. 交易完畢後換成其他資產
4. 關注 Tether 的新聞動態

### USDT 的未來

| 挑戰 | 說明 |
|------|------|
| 監管壓力 | 全球監管趨嚴 |
| USDC 競爭 | 合規市場份額可能流失 |
| CBDC | 央行數位貨幣可能取代部分用途 |
| 透明度要求 | 市場要求更高透明度 |

{% hint style="success" %}
USDT 是加密貨幣圈最成功也最具爭議的項目之一。它解決了加密貨幣價格波動的問題，成為整個生態系統的基石。但它的中心化、不透明和歷史爭議也是真實的風險。使用 USDT 時，要清楚知道你在承擔什麼風險——它不是「無風險的美元」，而是「Tether 公司的欠條」。
{% endhint %}
