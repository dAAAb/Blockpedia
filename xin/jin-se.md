# 金色財經

### 金色財經是中國幣圈的「主流媒體」。在中國禁止加密貨幣交易的背景下，它仍然在報導區塊鏈新聞，只是角度會比較「技術」而非「投資」。對於想了解中國市場動態的讀者來說，金色財經是重要的資訊來源。

### 報導風格

```
金色財經的報導特點：

1. 強調「區塊鏈技術」而非「加密貨幣投機」
2. 配合中國政策論述
3. 報導國際項目但避免敏感內容
4. 有大量付費軟文
```

{% hint style="warning" %}
**中國監管背景**

中國禁止加密貨幣交易，金色財經的報導需要在合規範圍內進行。這意味著：
- 不會直接教你如何交易
- 會強調區塊鏈技術應用
- 會配合政府政策論述
{% endhint %}
