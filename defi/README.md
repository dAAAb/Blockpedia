# DeFi 去中心化金融
> **最後更新：2026 年 1 月**

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## DeFi 是建立在區塊鏈上的金融服務，不需要銀行、不需要中介，任何人只要有錢包就能使用。

銀行開戶要身分證。
貸款要信用審查。
轉帳要工作日。
投資要達到門檻。

傳統金融充滿了門檻和限制。

**DeFi**（Decentralized Finance，去中心化金融）打破這一切：

- 沒有開戶流程
- 沒有信用審查
- 24/7 運作
- 任何人都能參與

這就是「金融民主化」。

### 什麼是 DeFi？

DeFi 是運行在區塊鏈上的金融應用總稱。

**傳統金融 vs DeFi**

| 面向 | 傳統金融 | DeFi |
|------|----------|------|
| 運營方 | 銀行、券商 | 智能合約 |
| 準入門檻 | 需要身分證、審核 | 只需要錢包 |
| 營業時間 | 工作日 | 24/7/365 |
| 透明度 | 不透明 | 完全透明 |
| 速度 | 數天 | 幾分鐘 |
| 地域限制 | 有 | 無 |
| 最低門檻 | 通常有 | 幾乎沒有 |

### DeFi 的核心組件

**1. 穩定幣**

DeFi 的「美元」：
- [USDT](../usdt/usdt.md)、[USDC](../stablecoin/usdc.md)、[DAI](../stablecoin/dai.md)
- 價值穩定，方便計價和結算

**2. 去中心化交易所（DEX）**

不需要註冊的交易所：
- [Uniswap](dex/uniswap.md)
- [Curve](dex/curve.md)
- 任何人都能交易任何代幣

**3. 借貸協議**

不需要信用審查的借貸：
- [Aave](lending/aave.md)
- [Compound](lending/compound.md)
- 存款賺利息，抵押借款

**4. 流動性挖礦**

提供流動性賺取獎勵：
- [Yield Farming](yield-farming.md)
- 把資產放入協議，賺取手續費和代幣

**5. 衍生品**

鏈上的期貨、選擇權：
- Synthetix
- dYdX
- GMX

### DeFi 如何運作？

以借貸為例：

**傳統銀行借款**

1. 填寫申請表
2. 提供收入證明
3. 銀行審核信用
4. 等待數天批准
5. 簽署合約
6. 獲得貸款

**DeFi 借款**

1. 連接錢包
2. 存入抵押品
3. 借出資產
4. 完成（幾分鐘）

為什麼這麼快？因為**智能合約**自動處理一切。

### 智能合約 = DeFi 的引擎

DeFi 的每個操作都由智能合約執行：

- 交易：自動撮合
- 借貸：自動計算利率
- 清算：自動執行
- 分配：自動發放獎勵

沒有人工，沒有延遲，沒有歧視。

代碼對所有人一視同仁。

### DeFi 的歷史

**2017-2018：萌芽期**
- MakerDAO 推出 DAI
- 第一代 DEX（EtherDelta）

**2020：DeFi 夏天**
- Compound 推出流動性挖礦
- 收益率狂飆
- 資金大量湧入
- TVL（鎖定總價值）從幾億飆升到幾百億

**2021：多鏈擴張**
- Polygon、BSC、Avalanche 等鏈的 DeFi 興起
- TVL 突破千億美元

**2022：調整期**
- Terra/Luna 崩盤
- 市場冷卻
- 更注重可持續性

**2023-2024：成熟期**
- RWA 整合
- 機構參與增加
- Layer 2 上的 DeFi 成長

### DeFi 的風險

**1. 智能合約風險**

代碼可能有漏洞。歷史上有很多被駭的案例。

**2. 無常損失**

提供流動性可能比單純持有還虧。

**3. 清算風險**

抵押品價值下跌可能被清算。

**4. 詐騙風險**

假項目、Rug Pull 屢見不鮮。

**5. 監管風險**

法規不明確，可能面臨限制。

### 如何開始？

**1. 準備錢包**

MetaMask 或其他 Web3 錢包。

**2. 獲得資產**

從交易所購買 ETH 或穩定幣，轉入錢包。

**3. 連接協議**

訪問 DeFi 協議網站，連接錢包。

**4. 開始使用**

交易、借貸、提供流動性...

**5. 謹慎操作**

- 從小金額開始
- 理解每個操作
- 注意風險

### TVL（鎖定總價值）

**什麼是 TVL？**

Total Value Locked，存放在 DeFi 協議中的資產總值。

**為什麼重要？**

- 衡量 DeFi 規模的指標
- 反映用戶對協議的信任
- 影響流動性和價格

**查看 TVL**

- [DefiLlama](https://defillama.com/)：最全面的 TVL 追蹤網站

### DeFi vs CeFi

| 面向 | DeFi | CeFi（中心化交易所）|
|------|------|---------------------|
| 託管 | 自己掌控 | 交易所託管 |
| 隱私 | 較高 | 需要 KYC |
| 風險 | 智能合約風險 | 交易所風險 |
| 費用 | Gas 費 | 交易手續費 |
| 速度 | 取決於區塊鏈 | 即時 |
| 產品豐富度 | 高 | 高 |

兩者各有優劣，很多人同時使用。

### DeFi 的未來

**1. 機構化**

更多傳統金融機構進入 DeFi。

**2. RWA 整合**

真實世界資產上鏈，擴大 DeFi 的範圍。

**3. Layer 2 普及**

更低成本、更高速度的 DeFi。

**4. 監管明確化**

合規 DeFi 可能成為新趨勢。

**5. 用戶體驗改善**

更簡單、更安全的操作介面。
