# SushiSwap

### SushiSwap 是從 Uniswap 分叉而來的去中心化交易所，以「吸血鬼攻擊」聞名。它的故事是 DeFi 歷史上最戲劇性的篇章之一——匿名創辦人、社群暴動、神秘轉讓、最終逆轉。

---

## SushiSwap 的誕生

2020 年 8 月，一個匿名開發者 **Chef Nomi** 做了一件當時看來很瘋狂的事：

1. 把 [Uniswap](uniswap.md) 的程式碼完整複製
2. 加入一個代幣獎勵機制（SUSHI）
3. 用更高的報酬吸引 Uniswap 的流動性提供者

這就是著名的「**吸血鬼攻擊**」（Vampire Attack）。

{% hint style="info" %}
**為什麼叫吸血鬼攻擊？**

因為 SushiSwap 像吸血鬼一樣「吸取」Uniswap 的流動性。LP 可以把在 Uniswap 的資金轉到 SushiSwap，同時獲得 SUSHI 代幣獎勵。
{% endhint %}

---

## 戲劇性的開局

### 瘋狂的成長

SushiSwap 上線後：
- 48 小時內，超過 10 億美元的流動性被「遷移」
- SUSHI 代幣從零漲到 $15
- DeFi 社群為之瘋狂

### Chef Nomi 跑路事件

就在所有人以為這是下一個傳奇時，災難發生了。

2020 年 9 月 5 日，Chef Nomi 把開發基金中的 SUSHI 代幣全部賣掉，套現約 1,400 萬美元。

SUSHI 價格暴跌，社群震怒。這看起來就像是一場精心設計的騙局。

### 神轉折

但故事沒有就此結束。

在社群的巨大壓力下，Chef Nomi 做了一件出乎意料的事：

1. **道歉** — 在 Twitter 上公開道歉
2. **歸還** — 把所有資金歸還給社群財庫
3. **退出** — 把控制權交給 FTX 創辦人 SBF

{% hint style="success" %}
**意外的結局**

Chef Nomi 歸還的資金，以當時價格計算約 1,400 萬美元。這在加密貨幣歷史上幾乎是前所未見的。
{% endhint %}

## SushiSwap 的特點

### 與 Uniswap 的差異

| 特點 | Uniswap V2 | SushiSwap |
|------|-----------|-----------|
| 代幣獎勵 | 無（當時） | SUSHI |
| 社群治理 | 較弱 | 較強 |
| 產品多元性 | 專注交換 | 多元化 |
| 跨鏈部署 | 較晚 | 較早 |

### SUSHI 代幣

SUSHI 是 SushiSwap 的治理和獎勵代幣：

- **流動性挖礦獎勵**：LP 可以獲得 SUSHI
- **治理投票**：持有者可以參與協議決策
- **手續費分成**：質押 SUSHI（xSUSHI）可以獲得協議收入分成

### 產品線

SushiSwap 不只是 DEX，它發展出完整的 DeFi 產品線：

| 產品 | 功能 |
|------|------|
| SushiSwap | AMM 交易所 |
| BentoBox | 資金庫（提高資金效率） |
| Kashi | 借貸市場 |
| MISO | 代幣發行平台 |
| Shoyu | NFT 市場（後來關閉） |

## 社群治理

SushiSwap 強調社群治理，但這也帶來了一些問題：

### 治理的挑戰

- **領導層動盪**：多次更換核心團隊
- **方向爭議**：社群對發展方向意見分歧
- **資金管理**：財庫使用引發爭議

### 2021-2022 年的風波

SushiSwap 經歷了多次內部動盪：
- 核心開發者離職
- 財務透明度問題
- 代幣激勵機制爭議

這些問題影響了協議的發展和 SUSHI 代幣的價格。

---

## SushiSwap 的貢獻

儘管經歷風波，SushiSwap 對 DeFi 生態有重要貢獻：

### 1. 證明了分叉的力量

開源協議可以被分叉和改進，這促使原協議加速創新。Uniswap 在 SushiSwap 出現後加速推出了 V3 和 UNI 代幣。

### 2. 代幣激勵模式

SushiSwap 普及了「代幣激勵流動性」的模式，這成為之後 DeFi 項目的標準做法。

### 3. 多鏈策略

SushiSwap 的跨鏈部署策略影響了後來許多 DEX 的發展方向。

---

## 現況與展望

截至 2024 年，SushiSwap：
- 仍是主要的 DEX 之一
- 在多條鏈上有部署
- 持續開發新功能
- 面臨來自 Uniswap、Curve 等的競爭

{% hint style="info" %}
**市場地位**

SushiSwap 的交易量和 TVL 已經被 Uniswap 大幅超越，但它仍然是 DeFi 生態中的重要參與者，尤其在某些 L2 和側鏈上保有一定市場份額。
{% endhint %}

## 給用戶的提醒

### 使用 SushiSwap 時注意

1. **確認正確網址**：只使用官方連結
2. **檢查流動性**：某些交易對流動性可能不足
3. **注意滑點**：設定合理的滑點容忍度
4. **了解風險**：流動性挖礦有無常損失風險
