# 挖礦
> **最後更新：2026 年 1 月**


## 挖礦是驗證區塊鏈交易並獲得獎勵的過程。礦工們用算力競爭，誰先解出數學難題，誰就能把新區塊加到鏈上，並拿到新發行的比特幣作為獎勵。

2009 年，你可以用筆電挖比特幣。

2013 年，你需要專業顯卡。

2017 年，你需要專業礦機。

現在？你需要一個礦場——成千上萬台機器，放在電費便宜的地方，24 小時運轉。

這就是挖礦的演變。

### 挖礦是什麼？

**用一句話解釋**：挖礦就是用電腦幫區塊鏈「記帳」，記對了就有獎勵。

**更技術一點**：礦工們競爭解決一個密碼學難題。誰先解出來，誰就獲得：

1. **區塊獎勵**：新發行的比特幣
2. **交易手續費**：該區塊內所有交易的手續費

```
問題：找到一個數字（nonce），使得：
SHA256(區塊資料 + nonce) < 目標難度

這基本上是在猜數字，沒有捷徑。
誰的電腦算得快，誰的機會大。
```

### 為什麼叫「挖礦」？

這是一個比喻：

| 黃金挖礦 | 比特幣挖礦 |
|----------|------------|
| 消耗體力 | 消耗算力 |
| 挖出黃金 | 產出比特幣 |
| 礦藏有限 | 總量 2100 萬枚 |
| 越挖越難 | 難度自動調整 |
| 需要礦機 | 需要礦機 |

**相似之處**：都是投入資源（勞力/電力）來獲得稀缺資產。

### 挖礦的演變

#### 第一階段：CPU 挖礦（2009-2010）

最初，用一般電腦的 CPU 就能挖礦。

- 中本聰本人就是用 CPU 挖的
- 一台筆電一天可能挖到好幾枚比特幣
- 那時候比特幣幾乎不值錢

#### 第二階段：GPU 挖礦（2010-2013）

有人發現顯卡（GPU）算得更快。

| CPU | GPU |
|-----|-----|
| 擅長複雜計算 | 擅長大量簡單計算 |
| 核心少（4-16）| 核心多（數千）|
| 挖礦效率低 | 挖礦效率高 100 倍+ |

**結果**：CPU 挖礦變得不划算，退出市場。

#### 第三階段：ASIC 挖礦（2013-至今）

專門為挖礦設計的晶片：ASIC（Application-Specific Integrated Circuit）

| 設備 | 算力 | 效率 |
|------|------|------|
| CPU | 低 | 最低 |
| GPU | 中 | 中 |
| ASIC | 極高 | 最高 |

**ASIC 特點**：

- 只能做一件事：挖礦
- 但做這件事比任何其他設備都好
- 一台 ASIC 礦機抵得上數百張顯卡

{% hint style="info" %}
這就是為什麼現在你不可能用家用電腦挖比特幣了——你在和全球的專業礦場競爭，他們有成千上萬台 ASIC 礦機。
{% endhint %}

### 主要的挖礦類型

#### 1. 工作量證明（PoW）挖礦

**代表幣種**：比特幣、萊特幣、狗狗幣

**機制**：比拼算力

```
算力大 → 解題機會高 → 獲得獎勵機會高
```

**優點**：
- 安全性經過 15 年驗證
- 去中心化程度高
- 難以攻擊

**缺點**：
- 極度耗電
- 設備不斷淘汰
- 算力集中化趨勢

#### 2. 權益證明（PoS）「挖礦」

**代表幣種**：以太坊（2022 年後）、Cardano、Solana

嚴格來說，PoS 不叫「挖礦」，而是「質押」或「驗證」。

```
持有代幣多 → 被選為驗證者機會高 → 獲得獎勵
```

**優點**：
- 不需要昂貴硬體
- 幾乎不耗電
- 進入門檻低

**缺點**：
- 「富者越富」疑慮
- 質押資金有鎖定期
- 安全性較新，未經長期驗證

{% hint style="warning" %}
以太坊在 2022 年從 PoW 轉為 PoS，這意味著以太坊礦機一夕之間變成廢鐵。投資挖礦設備前，要考慮這種系統性風險。
{% endhint %}

### 挖礦的經濟學

挖礦是一門生意，要計算：

| 項目 | 說明 |
|------|------|
| **收入** | 區塊獎勵 + 交易手續費 |
| **成本** | 電費 + 設備 + 場地 + 維護 |
| **利潤** | 收入 - 成本 |

#### 關鍵變數

**1. 比特幣價格**

價格漲，挖礦更賺；價格跌，可能虧本。

**2. 挖礦難度**

比特幣每 2016 個區塊（約兩週）自動調整難度：

- 礦工多 → 難度上升 → 每人分到的變少
- 礦工少 → 難度下降 → 每人分到的變多

**3. 電費**

挖礦最大的持續成本。

| 地區 | 電費（每度）| 競爭力 |
|------|------------|--------|
| 台灣 | ~$3-4 | 低 |
| 美國 | ~$2-4 | 中 |
| 中國（以前）| ~$1-2 | 高 |
| 冰島 | ~$1 | 高 |
| 薩爾瓦多 | 地熱免費 | 最高 |

**4. 設備成本與效率**

新機器更有效率，但也更貴。

### 比特幣減半

每 210,000 個區塊（約 4 年），區塊獎勵減半：

| 時間 | 區塊獎勵 |
|------|----------|
| 2009 | 50 BTC |
| 2012 | 25 BTC |
| 2016 | 12.5 BTC |
| 2020 | 6.25 BTC |
| 2024 | 3.125 BTC |
| 2028（預估）| 1.5625 BTC |

**減半的影響**：

- 礦工收入直接減半
- 效率不夠的礦工會被淘汰
- 歷史上，減半後價格都有上漲（供給減少）

{% hint style="info" %}
到 2140 年左右，比特幣將全部挖完。屆時礦工只能依靠交易手續費作為收入。
{% endhint %}

### 礦池

一個人挖礦，可能好幾年都挖不到一個區塊。

**解決方案**：礦池（Mining Pool）

```
單獨挖礦：
→ 中獎機率極低
→ 收入極度不穩定

加入礦池：
→ 大家算力合併
→ 中獎機率提高
→ 獎勵按算力分配
→ 收入穩定
```

**主要礦池**：

| 礦池 | 市占率（約）| 特點 |
|------|------------|------|
| Foundry USA | 30%+ | 美國最大 |
| AntPool | 15%+ | 比特大陸旗下 |
| F2Pool | 12%+ | 老牌礦池 |
| ViaBTC | 10%+ | 多幣種 |

{% hint style="warning" %}
如果一個礦池控制超過 50% 的算力，理論上可以發動 51% 攻擊。這是為什麼算力分散很重要。
{% endhint %}

雖然比特幣用 ASIC，但有些幣種刻意設計成「抗 ASIC」：

| 幣種 | 算法 | 適合設備 |
|------|------|----------|
| Ravencoin | KAWPOW | GPU |
| Ergo | Autolykos | GPU |
| Flux | ZelHash | GPU |

### 環境爭議

比特幣挖礦的能源消耗是最大的爭議點：

| 數據 | 說明 |
|------|------|
| 年耗電量 | 約 150 TWh（超過某些國家）|
| 碳排放 | 視電力來源而定 |
| 電子廢棄物 | 舊礦機的處理問題 |

**支持者的觀點**：

- 越來越多使用再生能源
- 可以利用閒置能源（如油田燃燒天然氣）
- 相比傳統金融系統，效率不一定更差

**批評者的觀點**：

- 能源消耗仍然巨大
- PoS 可以達成相同安全性，耗電量低 99.9%
- 環境成本被外部化

{% hint style="info" %}
這個辯論沒有標準答案。但如果你在意環境議題，可能需要考慮投資 PoS 幣種或綠色能源挖礦。
{% endhint %}

### 雲端挖礦

不想買設備？可以「租用」算力。

**雲端挖礦的模式**：

1. 你付錢給雲挖礦公司
2. 他們用你的錢運行礦機
3. 你按算力比例獲得收益

**風險**：

- 很多雲挖礦是詐騙
- 即使合法，扣除各種費用後往往不賺錢
- 你無法控制實際運作

{% hint style="danger" %}
雲端挖礦詐騙非常常見。很多「公司」收了錢就消失。如果報酬率聽起來太好，那幾乎肯定是騙局。
{% endhint %}

### 台灣挖礦的現實

在台灣挖礦面臨挑戰：

| 挑戰 | 說明 |
|------|------|
| 電費高 | 比中國、美國貴 2-3 倍 |
| 氣候熱 | 需要額外散熱成本 |
| 場地 | 住宅區不適合（噪音、用電）|
| 規模 | 難以與大型礦場競爭 |

**如果真的想參與**：

1. 加入礦池，獲得穩定小額收入
2. 選擇能效比最好的設備
3. 找到便宜電源（如離峰電、工業電）
4. 考慮其他 PoW 幣種（非比特幣）

### 挖礦還值得嗎？

**2024-2025 年的現實**：

| 情況 | 建議 |
|------|------|
| 有便宜電 | 可能值得，但要精算 |
| 一般電價 | 通常不划算 |
| 小規模 | 當興趣可以，別期望賺錢 |
| 大規模 | 需要專業規劃和資金 |

**替代方案**：

- **質押（Staking）**：參與 PoS 網路
- **流動性挖礦**：在 DeFi 協議提供流動性
- **直接買幣**：省去設備和電費的麻煩

{% hint style="success" %}
挖礦曾經是進入加密貨幣的主要方式，現在已經變成專業化、資本密集的產業。對一般人來說，直接購買比特幣或參與 PoS 質押，可能是更實際的選擇。但了解挖礦的原理，仍然是理解區塊鏈運作的重要基礎。
{% endhint %}
