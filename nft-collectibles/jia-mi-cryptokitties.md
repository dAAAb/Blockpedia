# 加密貓

### 2017 年底，一群虛擬貓咪讓以太坊差點癱瘓。這些貓不會喵喵叫、不會抓沙發，但最貴的一隻賣了 17 萬美元。CryptoKitties——史上第一個爆紅的 NFT 專案，開啟了數位收藏品的新紀元。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是 CryptoKitties？

| 項目 | 說明 |
|------|------|
| 中文名 | 加密貓、謎戀貓 |
| 推出時間 | 2017 年 11 月 |
| 開發團隊 | Dapper Labs |
| 技術標準 | ERC721（因此而誕生）|
| 類型 | 收藏 + 繁殖遊戲 |

## 遊戲機制

### 如何運作

```
CryptoKitties 的玩法：

1. 購買/領養貓咪
   - 每隻貓都是獨特的 NFT
   - 有不同外觀特徵
   - 從市場購買或繁殖獲得

2. 繁殖系統
   - 兩隻貓可以繁殖
   - 產生新的小貓
   - 基因組合決定外觀

3. 基因與稀有度
   - 每隻貓有 256 位基因
   - 基因決定「cattributes」
   - 稀有特徵價值更高

4. 世代系統
   - Gen 0：官方發行
   - Gen 1+：繁殖產生
   - 世代越低越稀有
```

## 歷史意義

### 里程碑事件

```
CryptoKitties 的歷史定位：

2017 年 11 月：
- 正式上線
- 迅速爆紅
- 交易量暴增

網路擁塞：
- 佔據以太坊 15%+ 交易量
- Gas 費飆升
- 其他 DApp 受影響
- 暴露以太坊擴容問題

市場高峰：
- 最貴的貓「Dragon」賣 600 ETH
- 當時約 17 萬美元
- 單週交易額超過 1200 萬美元

啟發後續：
- ERC721 標準因此制定
- NFT 概念普及
- Dapper Labs 後來創造 NBA Top Shot
```

{% hint style="info" %}
**ERC721 的起源**

CryptoKitties 的需求直接促成了 ERC721 標準的誕生。在此之前，以太坊沒有專門的 NFT 標準。
{% endhint %}

## Dapper Labs

### 創造者的後續發展

| 項目 | 說明 |
|------|------|
| 團隊 | Dapper Labs |
| 後續作品 | NBA Top Shot |
| 自建區塊鏈 | Flow |
| 成就 | NFT 領域領導者 |

```
Dapper Labs 的發展：

CryptoKitties（2017）
→ 發現以太坊的限制
→ 決定自建區塊鏈

Flow 區塊鏈（2020）
→ 專為 NFT 和遊戲設計
→ 更高效、更低費用

NBA Top Shot（2020）
→ 運動卡牌 NFT
→ 獲得 NBA 官方授權
→ 銷售額超過 10 億美元
```

## 現況與反思

### 市場表現

| 時期 | 狀況 |
|------|------|
| 2017-2018 | 熱潮高峰 |
| 2019-2020 | 熱度下降 |
| 2021 | NFT 復興但未回巔峰 |
| 現在 | 主要是歷史收藏價值 |

### 經驗教訓

```
CryptoKitties 的啟示：

成功之處：
- 證明 NFT 有市場
- 讓加密貨幣「可愛化」
- 吸引非技術用戶

侷限之處：
- 遊戲性不足
- 經濟模型問題（無限繁殖）
- 以太坊無法承載

對產業的影響：
- 催生 ERC721 標準
- 啟發無數 NFT 專案
- 暴露擴容問題
- Dapper Labs 成為重要玩家
```

{% hint style="warning" %}
**大多數加密貓已無流動性**

雖然 CryptoKitties 有歷史意義，但現在大多數貓咪交易量極低，除非是非常稀有的早期或特殊貓咪，否則難以出售。
{% endhint %}
