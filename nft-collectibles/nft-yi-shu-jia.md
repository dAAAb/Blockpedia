# NFT 藝術家

### 在傳統藝術市場，藝術家賣掉作品後，再怎麼升值都與他無關。但在 NFT 世界，每次轉手藝術家都能分到版稅。這個改變，讓數位藝術家第一次能靠作品持續獲益。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## 什麼是 NFT 藝術家？

| 概念 | 說明 |
|------|------|
| 定義 | 創作 NFT 數位藝術的創作者 |
| 作品形式 | 圖像、動畫、3D、生成式 |
| 發行方式 | 透過 NFT 平台鑄造銷售 |
| 收入模式 | 首次銷售 + 二次版稅 |

---

## NFT 如何改變藝術家

### 傳統 vs NFT 模式

| 面向 | 傳統藝術 | NFT 藝術 |
|------|----------|----------|
| 中間商 | 畫廊、拍賣行抽成高 | 平台抽成低（2.5-5%）|
| 版稅 | 首次銷售後無關 | 每次轉手可收版稅 |
| 市場 | 地區限制 | 全球化 |
| 證明 | 紙本鑑定 | 區塊鏈驗證 |

### 創作者經濟

```
NFT 對藝術家的改變：

收入結構：
- 首次銷售收入（通常扣除平台費）
- 二次銷售版稅（5-10% 不等）
- 持續被動收入可能

直接連結：
- 跳過中間商
- 直接面對收藏家
- 建立個人品牌

新可能性：
- 可編程藝術
- 互動式作品
- 分潤機制靈活
```

## 如何成為 NFT 藝術家

### 入門步驟

```
成為 NFT 藝術家的流程：

1. 創作作品
   - 確保是原創
   - 數位格式（圖片、影片、音樂等）
   - 建立個人風格

2. 設置錢包
   - 下載 MetaMask 等錢包
   - 準備少量 ETH（Gas 費）

3. 選擇平台
   - OpenSea：最大，門檻低
   - Foundation：邀請制，品質高
   - SuperRare：策展型，頂級藝術家
   - Zora：創作者友好

4. 鑄造作品
   - 上傳作品
   - 設定名稱、描述
   - 設定版稅比例

5. 行銷推廣
   - Twitter/X 是關鍵
   - 建立社群
   - 與收藏家互動
```

### 平台比較

| 平台 | 特色 | 適合 |
|------|------|------|
| OpenSea | 最大市場 | 初學者 |
| Foundation | 邀請制 | 成熟藝術家 |
| SuperRare | 高端策展 | 頂級藝術家 |
| Objkt | Tezos 鏈 | 環保考量 |

## 版稅機制

### 運作方式

| 環節 | 說明 |
|------|------|
| 設定版稅 | 鑄造時設定（通常 5-10%）|
| 觸發時機 | 每次二次市場交易 |
| 執行方式 | 智能合約自動分配 |
| 變化 | 近年部分平台取消強制 |

{% hint style="warning" %}
**版稅不是強制的**

2023 年後，部分平台（如 Blur）允許買家選擇是否支付版稅。這對藝術家收入造成衝擊，選擇支持版稅的平台變得重要。
{% endhint %}

## 挑戰與風險

### 常見問題

| 挑戰 | 說明 |
|------|------|
| 市場波動 | NFT 價格波動劇烈 |
| 版權盜用 | 作品被盜鑄 |
| 平台風險 | 平台可能關閉 |
| 稅務問題 | 收入需申報 |

```
NFT 藝術家需注意：

版權保護：
- 保留原始創作檔案
- 加入數位浮水印
- 監控盜用行為

市場現實：
- 99% 藝術家收入有限
- 成功需要持續曝光
- 社群經營很重要

長期策略：
- 不要 all-in NFT
- 多元收入來源
- 建立真正的藝術品牌
```
