# 同質 vs 非同質代幣

### 一張百元鈔票和另一張百元鈔票，可以互換——這就是同質。但蒙娜麗莎和星夜，能互換嗎？當然不行。這就是同質代幣和非同質代幣的核心差異。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是同質性？

| 概念 | 說明 |
|------|------|
| 同質（Fungible）| 可互換、等價 |
| 非同質（Non-Fungible）| 不可互換、獨特 |

### 生活中的例子

```
同質 vs 非同質的例子：

同質的東西：
- 鈔票：一張百元 = 另一張百元
- 黃金：一盎司 = 另一盎司
- 汽油：一公升 = 另一公升
- 比特幣：1 BTC = 1 BTC

非同質的東西：
- 房子：每棟都不同
- 藝術品：每件都獨特
- 演唱會門票：座位不同
- 你的寵物：每隻都獨一無二
```

## 代幣類型比較

### FT vs NFT

| 特性 | 同質代幣（FT）| 非同質代幣（NFT）|
|------|-------------|-----------------|
| 可互換性 | ✅ 完全可互換 | ❌ 每個獨特 |
| 可分割性 | ✅ 可分成小單位 | ❌ 通常不可分割 |
| 代表 | 價值、數量 | 身份、所有權 |
| 標準 | ERC20 | ERC721、ERC1155 |
| 例子 | ETH、USDT | CryptoPunks、BAYC |

### 技術標準

```
以太坊代幣標準：

同質代幣（FT）：
- ERC20：基本代幣標準
  - 每個代幣相同
  - 可分割到 18 位小數
  - 用於：貨幣、治理代幣

非同質代幣（NFT）：
- ERC721：NFT 標準
  - 每個代幣有唯一 ID
  - 不可分割
  - 用於：收藏品、藝術品

- ERC1155：多代幣標準
  - 同時支援 FT 和 NFT
  - 批量操作更高效
  - 用於：遊戲物品
```

## 應用場景

### 同質代幣用途

| 用途 | 說明 | 例子 |
|------|------|------|
| 支付貨幣 | 交換媒介 | ETH、BTC |

## 半同質代幣

### ERC1155 的彈性

```
ERC1155 的特殊設計：

可以同時處理：

同質部分：
- 遊戲金幣（每個相同）
- 材料道具（可堆疊）

非同質部分：
- 獨特武器（唯一編號）
- 限量角色（稀有度）

優點：
- 一個合約管理所有物品
- 批量轉移更省 Gas
- 適合遊戲經濟系統
```

## 分數化 NFT

### NFT 的新可能

| 概念 | 說明 |
|------|------|
| 分數化（Fractionalization）| 將 NFT 拆分成多份 |
| 運作方式 | 將 NFT 鎖定，發行 ERC20 代幣 |
| 目的 | 降低購買門檻 |

```
分數化 NFT 的例子：

假設：
- 一個 CryptoPunk 價值 100 ETH
- 一般人買不起

分數化後：
- 將它拆成 10,000 份
- 每份代表 0.01% 所有權
- 每份只要 0.01 ETH
- 人人都能擁有一小部分

問題：
- 誰控制原始 NFT？
- 如何決定出售？
- 治理複雜
```

{% hint style="info" %}
**FT 和 NFT 可以互通**

透過分數化，NFT 可以變成 FT。透過打包，多個 FT 也可以組成獨特的 NFT（如特定投資組合）。
{% endhint %}
