# NFT 藝術/蒐藏品

### 想像一下，你花了一百萬買了一幅畫——但你不能把它掛在牆上。這就是 NFT 藝術蒐藏品。數位藝術品、猿猴頭像、像素龐克⋯⋯這些「看不見摸不著」的東西，怎麼就值了天價？

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是 NFT 蒐藏品？

| 概念 | 說明 |
|------|------|
| 英文 | NFT Collectibles |
| 定義 | 以 NFT 形式存在的數位蒐藏品 |
| 特性 | 獨一無二、可驗證、可交易 |
| 載體 | 圖片、影片、音樂、3D 模型 |

---

## NFT 蒐藏品類型

### 主要類別

```
NFT 蒐藏品的類型：

1. PFP（Profile Picture）頭像類
   - 用於社群頭像
   - 代表身份認同
   - 例：BAYC、CryptoPunks

2. 數位藝術
   - 藝術家原創作品
   - 純藝術收藏
   - 例：Beeple、pplpleasr

3. 生成式藝術
   - 演算法生成
   - 每件獨特
   - 例：Art Blocks

4. 遊戲物品
   - 遊戲內資產
   - 可跨平台
   - 例：Axie、Loot

5. 音樂/影片
   - 數位音樂所有權
   - 影片收藏
   - 例：Sound.xyz

6. 域名
   - 區塊鏈域名
   - 數位身份
   - 例：ENS
```

### 經典專案比較

| 專案 | 類型 | 數量 | 地板價高峰 |
|------|------|------|------------|
| CryptoPunks | PFP | 10,000 | 100+ ETH |
| BAYC | PFP | 10,000 | 150+ ETH |
| Art Blocks | 生成式 | 多系列 | 因系列而異 |
| Azuki | PFP | 10,000 | 30+ ETH |

- 期望增值

## NFT 技術基礎

### 代幣標準

| 標準 | 特性 | 適用 |
|------|------|------|
| [ERC721](erc721.md) | 每個代幣獨特 | 單一收藏品 |
| [ERC1155](erc1155.md) | 可批量操作 | 遊戲物品 |

```
NFT 的技術結構：

鏈上數據：
- Token ID（唯一編號）
- 所有權地址
- 合約地址

鏈下數據（Metadata）：
- 圖片/媒體文件
- 屬性特徵
- 描述資訊
- 存儲在 IPFS 或中心化伺服器
```

{% hint style="warning" %}
**圖片可能不在鏈上**

大多數 NFT 的圖片存儲在鏈下（IPFS 或伺服器），只有所有權證明在鏈上。如果存儲服務商倒閉，圖片可能消失。
{% endhint %}

## 市場發展

### 歷史脈絡

```
NFT 蒐藏品發展史：

2017：CryptoKitties 誕生
- 第一個爆紅的 NFT 遊戲
- 導致以太坊擁塞

2020：DeFi Summer 帶動
- 開始受到關注
- 藝術品拍賣興起

2021：NFT 熱潮
- Beeple 作品拍出 6900 萬美元
- BAYC 等 PFP 專案爆發
- OpenSea 交易量暴增

2022-2023：市場回調
- 地板價大幅下跌
- 泡沫破裂
- 回歸理性

2024+：應用深化
- 更多實際用途
- 與實體結合
- RWA 概念興起
```

## 風險警示

### 主要風險

| 風險 | 說明 |
|------|------|
| 流動性風險 | 可能賣不掉 |
| 價值歸零 | 專案可能失敗 |
| 詐騙風險 | 假專案、Rug Pull |
| 版權爭議 | 原創性問題 |
| 技術風險 | 合約漏洞 |

{% hint style="danger" %}
**NFT 不等於投資**

大多數 NFT 最終價值歸零。請將其視為收藏品而非投資，只花你願意損失的錢。
{% endhint %}
