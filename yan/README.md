# 區塊鏈研討會

### 想深入了解區塊鏈技術？參加研討會是最好的方式。從學術研討會到開發者 Meetup，每個活動都是學習和交流的機會。

> ⚠️ **免責聲明**：參加活動請注意安全。

---

## 研討會類型

### 主要類型

| 類型 | 說明 |
|------|------|
| 學術研討會 | 論文發表、學術討論 |
| 技術研討會 | 開發者深度分享 |
| 產業研討會 | 商業應用討論 |
| Meetup | 定期社群聚會 |
