# Base
> **最後更新：2026 年 1 月**


## Base 是 Coinbase 推出的 Layer 2，基於 OP Stack 構建，目標是讓十億用戶進入區塊鏈世界。

Coinbase，全球最大的加密貨幣交易所之一。

1 億驗證用戶。
在納斯達克上市。
美國監管下的合規企業。

當這樣的公司推出 Layer 2，意義非凡。

這就是 **Base**：連接中心化世界與去中心化世界的橋樑。

### Base 是什麼？

Base 是 Coinbase 推出的 Optimistic Rollup Layer 2。

- **推出**：2023 年 8 月
- **基礎**：OP Stack
- **特色**：Coinbase 生態整合
- **代幣**：目前沒有原生代幣

### 為什麼 Coinbase 做 L2？

**1. 用戶入口**

Coinbase 有 1 億用戶，可以直接引導到 Base。

**2. 降低成本**

在 L1 上交易太貴，Base 讓小額交易變得可行。

**3. 控制命運**

不需要依賴其他 L2，掌握自己的技術棧。

**4. 新商業模式**

排序器收入、生態發展等新收入來源。

### 基於 OP Stack

Base 不是從頭開發，而是使用 [Optimism](optimism.md) 的 **OP Stack**。

**好處**

- 成熟的技術
- 與 Optimism 互操作
- 成為 Superchain 的一部分
- 共享安全性

**關係**

Base 向 Optimism 貢獻收入的一部分，支持 Superchain 發展。

### 沒有代幣

截至目前，Base **沒有發行原生代幣**。

**官方聲明**

Coinbase 表示沒有發行代幣的計劃。

**社區猜測**

很多人仍在「farming」Base，希望未來有空投。

**使用 ETH**

Base 使用 ETH 作為 Gas 代幣。

### Coinbase 生態整合

Base 的獨特優勢是 Coinbase 整合：

**1. 簡單入金**

從 Coinbase 交易所可以直接提款到 Base，幾乎即時。

**2. Coinbase Wallet**

Coinbase 錢包原生支援 Base。

**3. 法幣通道**

透過 Coinbase 可以法幣入金。

**4. 機構信任**

Coinbase 的品牌為 Base 背書。

### 生態系統

Base 的生態快速成長：

**DeFi**

| 協議 | 類型 |
|------|------|
| Aerodrome | 原生 DEX（Velodrome 分叉）|
| Uniswap | DEX |
| Aave | 借貸 |
| Moonwell | 原生借貸 |

**社交和 NFT**

- friend.tech（社交代幣）
- Farcaster 生態
- 各種 NFT 項目

**Meme 幣**

Base 成為 meme 幣的熱門地點：
- 低手續費
- 快速交易
- 社區活躍

### friend.tech 現象

2023 年，friend.tech 在 Base 上爆發。

**什麼是 friend.tech？**

社交代幣平台：
- 購買「Key」代表對某人的支持
- 價格隨需求上漲
- 持有 Key 可以進入私人聊天室

**影響**

- 帶動大量用戶到 Base
- 證明 Base 可以處理高流量
- 引發社交金融（SocialFi）討論

### Farcaster 生態

Farcaster 是去中心化社交協議，Base 是其重要基地。

**Warpcast**

Farcaster 的主要客戶端，很多功能與 Base 整合。

**Frame**

在 Farcaster 帖子中嵌入互動功能：
- NFT 鑄造
- 投票
- 小遊戲
- 交易

很多 Frame 在 Base 上運行。

### 費用

Base 是最便宜的 L2 之一：

**EIP-4844 後**

2024 年以太坊升級後，Base 費用大幅下降。

簡單交易可能只需要 $0.001。

**為什麼這麼便宜？**

- EIP-4844 降低數據成本
- 高交易量分攤固定成本
- 技術優化

### 使用 Base

**1. 添加網路**

在 MetaMask 添加：
- 網路名稱：Base
- RPC URL：https://mainnet.base.org
- Chain ID：8453

**2. 橋接**

官方橋：[bridge.base.org](https://bridge.base.org)

或從 Coinbase 直接提款（推薦，更快）。

**3. 使用 DApp**

選擇 Base 網路，開始探索。

### 中心化問題

Base 面臨的質疑：

**Coinbase 控制**

- 排序器由 Coinbase 運營
- 升級權限在 Coinbase 手中
- 如果 Coinbase 出問題？

**回應**

- 這是早期階段
- 正在走向去中心化
- Superchain 提供額外保障

### 與其他 L2 比較

| 面向 | Base | Arbitrum | Optimism |
|------|------|----------|----------|
| 背景 | Coinbase | Offchain Labs | OP Labs |
| 代幣 | 無 | ARB | OP |
| 費用 | 最低之一 | 低 | 低 |
| 入金便利 | Coinbase 直接 | 橋接 | 橋接 |
| 生態 | 成長中 | 最豐富 | 豐富 |

### 風險

**1. 中心化**

Coinbase 的控制是雙刃劍。

**2. 監管風險**

Coinbase 是美國公司，受美國監管：
- 可能被要求審查交易
- 政策變化可能影響運營

**3. 無代幣**

沒有代幣意味著：
- 沒有去中心化治理
- 沒有激勵對齊
- 用戶沒有「所有權」

### Onchain Summer

2023 年夏天，Base 推出「Onchain Summer」活動。

**內容**

- 免費 NFT 鑄造
- 各種合作項目
- 吸引新用戶

**效果**

- 大量新用戶體驗區塊鏈
- 降低入門門檻
- 建立 Base 品牌

### 未來發展

**1. 十億用戶目標**

Base 的願景是讓十億用戶上鏈。

**2. 更多 Coinbase 整合**

與 Coinbase 產品更深整合。

**3. Superchain 協作**

與其他 OP Stack 鏈互操作。

**4. 去中心化**

逐步降低 Coinbase 的控制。

{% hint style="info" %}
Base 是傳統金融擁抱區塊鏈的代表。有 Coinbase 的用戶基礎和品牌，它可能成為很多人進入區塊鏈的入口。但中心化問題仍需關注。
{% endhint %}
