# Arbitrum
> **最後更新：2026 年 1 月**


## Arbitrum 是 TVL 最大的以太坊 Layer 2，擁有最豐富的 DeFi 生態，是 Optimistic Rollup 的領導者。

如果說以太坊是區塊鏈的「世界電腦」，

那 Arbitrum 就是讓這台電腦**真正好用**的升級。

它有最大的 TVL，最活躍的 DeFi 生態，是很多人進入 L2 世界的第一站。

### Arbitrum 是什麼？

Arbitrum 是由 Offchain Labs 開發的 Optimistic Rollup。

- **開發者**：Offchain Labs
- **創立**：2021 年主網上線
- **代幣**：ARB
- **TVL**：L2 中最大

### 為什麼 Arbitrum 領先？

**1. 最早成熟**

Arbitrum One 是最早達到大規模使用的 L2 之一。

**2. DeFi 生態**

幾乎所有主要 DeFi 協議都在 Arbitrum 上：
- Uniswap
- Aave
- GMX（原生）
- Curve
- ...

**3. 技術優勢**

多輪欺詐證明提供更高的安全性。

**4. 開發者友好**

高度 EVM 兼容，開發者容易遷移。

### 技術特點

**多輪欺詐證明**

與 Optimism 的單輪證明不同：

- Optimism：一次提交完整證明
- Arbitrum：多輪互動，逐步縮小爭議範圍

**好處**
- 更高效（只驗證爭議部分）
- 鏈上成本更低

**Nitro 升級**

2022 年的重大升級：
- 用 WASM 替代自定義虛擬機
- 大幅提升性能
- 降低費用

### Arbitrum 家族

Arbitrum 不只一條鏈：

**Arbitrum One**

- 主力 L2
- 最大 TVL
- DeFi 中心

**Arbitrum Nova**

- 為遊戲和社交優化
- 更便宜
- 使用 AnyTrust（部分數據在鏈下）
- Reddit 的社區積分曾在此

**Arbitrum Orbit**

- 讓任何人建立自己的 L2/L3
- 類似 OP Stack
- 可自定義

### ARB 代幣

2023 年 3 月，Arbitrum 進行了大規模空投。

**空投**

- 對早期用戶空投
- 根據使用程度分級
- 很多人獲得數千美元

**代幣功能**
- 治理投票
- 決定協議參數和發展方向

**Arbitrum DAO**

ARB 持有者組成 DAO：
- 管理協議升級
- 資金使用
- 生態發展

### 生態系統

Arbitrum 有最豐富的生態：

**DeFi**

| 協議 | 類型 |
|------|------|
| GMX | 永續合約 DEX（原生）|
| Uniswap | DEX |
| Aave | 借貸 |
| Curve | 穩定幣 DEX |
| Camelot | 原生 DEX |
| Radiant | 借貸 |
| Pendle | 收益代幣化 |

**GMX 的成功**

GMX 是 Arbitrum 上最成功的原生協議：
- 去中心化永續合約交易
- 真實收益模式
- 帶動大量用戶和 TVL

**NFT 和遊戲**

- Treasure（遊戲生態）
- 各種 NFT 項目

### 費用

Arbitrum 的費用結構：

**組成**
- L2 計算費用
- L1 數據提交費用

**EIP-4844 後**

2024 年以太坊 Dencun 升級後，Arbitrum 費用大幅下降。

很多交易只需要幾美分。

### 使用 Arbitrum

**1. 添加網路**

在 MetaMask 添加：
- 網路名稱：Arbitrum One
- RPC URL：https://arb1.arbitrum.io/rpc
- Chain ID：42161

**2. 橋接**

官方橋：[bridge.arbitrum.io](https://bridge.arbitrum.io)

第三方橋：
- Hop Protocol
- Stargate
- Across

**3. 使用 DApp**

選擇 Arbitrum 網路，開始使用。

### 與 Optimism 比較

| 面向 | Arbitrum | Optimism |
|------|----------|----------|
| TVL | 更大 | 大 |
| 欺詐證明 | 多輪 | 單輪 |
| 開源框架 | Orbit | OP Stack |
| 生態豐富度 | 更豐富 | 豐富 |
| 原生協議 | GMX 等 | Velodrome 等 |
| 願景 | 技術領先 | Superchain |

**選擇建議**
- 想要最多 DeFi 選擇：Arbitrum
- 想參與 Superchain 生態：Optimism
- 兩者都可以用，根據具體 DApp 選擇

### Stylus

Arbitrum 的重要創新。

**什麼是 Stylus？**

允許用 Rust、C、C++ 等語言編寫智能合約。

**為什麼重要？**

- 不只限於 Solidity
- 更多開發者可以參與
- 某些場景性能更好

**影響**

可能吸引更多非 Web3 背景的開發者。

### 安全性

**繼承以太坊安全**

- 交易數據提交到以太坊
- 可以通過 L1 恢復資產

**風險因素**

- 排序器中心化（正在改善）
- 升級權限（有時間鎖）
- 欺詐證明系統（白名單限制中）

**L2Beat 評級**

L2Beat 提供詳細的風險評估，Arbitrum 在主要 L2 中評價較高。

### 未來發展

**1. 去中心化排序器**

讓排序器去中心化。

**2. BOLD 協議**

新的欺詐證明協議，提高安全性。

**3. Orbit 生態成長**

更多基於 Orbit 的 L2/L3。

**4. Stylus 採用**

更多用 Rust 等語言的智能合約。

### L3 和 Orbit

**什麼是 L3？**

建立在 L2 之上的網路。

**Arbitrum Orbit**

允許任何人創建：
- 基於 Arbitrum 的 L2
- 或基於 Arbitrum L2 的 L3

**應用場景**
- 遊戲專用鏈
- 企業專用鏈
- 特定應用鏈
