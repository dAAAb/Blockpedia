# Optimism
> **最後更新：2026 年 1 月**


## Optimism 是以太坊最重要的 Layer 2 之一，推出了 OP Stack 讓任何人都能建立自己的 L2，引領了「超級鏈」願景。

2021 年，以太坊手續費讓人痛苦。

一群理想主義者決定解決這個問題。

他們的名字就是他們的態度：**Optimism**（樂觀）。

樂觀地相信區塊鏈可以改變世界，樂觀地假設交易是有效的。

### Optimism 是什麼？

Optimism 是 Optimistic Rollup 類型的 Layer 2。

- **創立**：2019 年
- **代幣**：OP
- **類型**：Optimistic Rollup
- **TVL**：數十億美元

### 技術原理

**Optimistic Rollup**

1. 用戶在 L2 發送交易
2. 排序器打包交易
3. 交易數據提交到以太坊
4. 「樂觀地」假設交易有效
5. 7 天挑戰期內可以提出異議

**為什麼叫「Optimistic」？**

樂觀地假設所有交易都是有效的。只有在有人提出「欺詐證明」時才驗證。

### OP Stack

Optimism 最重要的貢獻是 **OP Stack**。

**什麼是 OP Stack？**

一套開源的軟體，讓任何人都能建立自己的 L2。

**為什麼重要？**

- 不需要從頭開發
- 繼承 Optimism 的技術和安全性
- 可以自定義

**使用 OP Stack 的 L2**

- [Base](base.md)（Coinbase）
- Zora
- Mode
- 還有很多...

這些 L2 一起形成「**超級鏈（Superchain）**」。

### Superchain 願景

Optimism 的長期願景是建立**超級鏈**。

**概念**

多個基於 OP Stack 的 L2 互相連接：
- 共享安全性
- 無縫互操作
- 統一的標準

**好處**

- 跨 L2 資產轉移更快更安全
- 共享排序器
- 網路效應

### OP 代幣

**功能**

- **治理**：投票決定協議發展
- **RetroPGF**：資助公共財（詳見下方）

**分配**

- 空投給早期用戶
- 生態基金
- 團隊和投資者

**空投**

2022 年 5 月，Optimism 對早期用戶空投 OP：
- 使用過 Optimism 的用戶
- 以太坊上的活躍用戶
- 多次空投（不只一次）

### RetroPGF（追溯性公共財資助）

Optimism 的獨特機制。

**什麼是 RetroPGF？**

Retroactive Public Goods Funding。
追溯性地資助對生態有貢獻的項目。

**怎麼運作？**

1. 項目先為生態做出貢獻
2. 貢獻被認可後，獲得 OP 資助
3. 由社區投票決定

**為什麼重要？**

- 獎勵公共財
- 不需要事先申請
- 社區決定價值

### 治理結構

Optimism 有獨特的雙院制治理：

**Token House**

- OP 代幣持有者
- 投票決定技術和經濟參數

**Citizens' House**

- 持有「靈魂綁定」NFT 的公民
- 負責 RetroPGF 資助決策
- 代表更廣泛的社區利益

這種設計平衡了代幣持有者和社區利益。

### 費用

Optimism 的費用比以太坊便宜很多：

**費用組成**

- **L2 執行費**：在 L2 上執行的成本
- **L1 數據費**：提交到以太坊的成本

**EIP-4844 影響**

以太坊的 blob 空間（2024 年上線）大幅降低了 L1 數據費。

現在很多交易只需要幾美分。

### 使用 Optimism

**1. 添加網路**

在 MetaMask 添加 Optimism 網路：
- 網路名稱：Optimism
- RPC URL：https://mainnet.optimism.io
- Chain ID：10

**2. 橋接資產**

使用官方橋或第三方橋把資產轉入。

**3. 使用 DApp**

連接錢包，在 Optimism 上使用 DApp。

**DeFi**

**NFT**

### 與 Arbitrum 比較

| 面向 | Optimism | Arbitrum |
|------|----------|----------|
| 技術 | 單輪欺詐證明 | 多輪欺詐證明 |
| 開源 | OP Stack | Arbitrum Orbit |
| 治理 | 雙院制 | ARB 代幣 |
| 生態 | 大 | 更大 |
| 願景 | Superchain | Arbitrum One + 生態 |

兩者都很優秀，選擇看需求和偏好。

### 風險

**1. 中心化排序器**

目前排序器由 Optimism 團隊運營：
- 正在走向去中心化
- 如果排序器停止，可以通過 L1 取回資產

**2. 升級風險**

合約可以被升級：
- 有安全理事會控制
- 需要信任這些實體

**3. 欺詐證明**

需要有人監控並提交欺詐證明：
- 目前有白名單限制
- 正在開放

### 未來發展

**1. 去中心化排序器**

讓排序器去中心化。

**2. Superchain 成長**

更多 L2 加入 Superchain。

**3. 欺詐證明開放**

讓任何人都能提交欺詐證明。

**4. 與其他 L2 互操作**

跨 L2 的無縫體驗。

{% hint style="info" %}
Optimism 不只是一個 L2，更是一個願景。OP Stack 和 Superchain 代表了 L2 未來的一種可能：多個互聯的 L2 共享安全性和互操作性。
{% endhint %}
