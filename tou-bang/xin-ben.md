# 恆心資本

### 恆心資本是亞洲區塊鏈投資版圖的一員。相較於西方的 a16z、Paradigm，亞洲的區塊鏈 VC 有自己的特色和生態。了解這些機構，有助於理解亞洲區塊鏈市場的資金流向。

### 亞洲 vs 西方 VC

| 比較 | 亞洲 VC | 西方 VC |
|------|---------|---------|
| 規模 | 相對較小 | 大型基金 |
| 風格 | 快進快出 | 長期持有 |
| 重點 | 交易、遊戲 | 基礎設施 |
| 監管 | 更複雜 | 相對明確 |

## 亞洲區塊鏈投資特點

### 投資偏好

```
亞洲 VC 的投資偏好：

熱門領域：
- 交易所和交易基礎設施
- GameFi 和遊戲
- 支付和匯款
- NFT 和數位收藏品

較少關注：
- 純技術基礎設施
- 需要長期研發的項目
```

### 地區差異

| 地區 | 特色 |
|------|------|
| 新加坡 | 監管友善，全球化 |
| 香港 | 傳統金融背景，合規導向 |
| 日韓 | 成熟市場，本地化強 |
| 東南亞 | 新興市場，高成長 |

{% hint style="info" %}
**亞洲的機會**

亞洲有大量的加密貨幣用戶，特別是在東南亞、韓國、日本。這為亞洲 VC 提供了獨特的投資機會——更了解本地市場。
{% endhint %}
