# 區塊鏈風投榜

### 區塊鏈風投是幣圈的「造王者」。他們投資的項目往往能獲得大量關注，他們的觀點影響市場走向，他們的投資組合就是一份「趨勢清單」。了解這些 VC，就是了解加密貨幣產業的未來方向。

---

## 為什麼要關注風投？

```
VC 的影響力：

投資決策 → 資金流向 → 項目獲得資源 → 產品上線 → 幣價表現

跟著 VC 投資不保證賺錢，但了解他們的思路有助於理解市場。
```

{% hint style="warning" %}
**VC 不是神**

VC 投資的項目也會失敗。FTX、Terra、Celsius 背後都有頂級 VC 的背書。「有 VC 投資」不等於「項目一定好」。
{% endhint %}

### 分析方法

```
分析 VC 投資的方法：

1. 追蹤投資組合
   - 他們投了哪些項目？
   - 最近投資趨勢？

2. 閱讀研究報告
   - a16z、Paradigm 都有公開研究
   - 了解他們的投資邏輯

3. 追蹤合夥人
   - Twitter 發言
   - 播客訪談
   - 會議演講
```

## VC 投資的啟示

### 正面參考

| 參考 | 說明 |
|------|------|
| 趨勢方向 | 多家 VC 同時投資的領域可能是趨勢 |
| 團隊背書 | VC 做過盡職調查 |
| 資源支持 | 有 VC 支持的項目通常有更多資源 |

### 警惕事項

| 警惕 | 說明 |
|------|------|
| 估值泡沫 | VC 投資的估值散戶買不到 |
| 利益不一致 | VC 的成本和你不同 |
| 解鎖壓力 | VC 代幣解鎖可能造成拋壓 |

{% hint style="danger" %}
**VC 代幣解鎖**

很多 VC 投資的代幣有鎖倉期。當鎖倉期結束，VC 可能會賣出，造成價格下跌。投資前要查看代幣經濟學。
{% endhint %}
