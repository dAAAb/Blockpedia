# a16z

### a16z 是加密貨幣產業的「造王者」。作為矽谷最具影響力的創投之一，它的 a16z crypto 部門管理超過 70 億美元的加密資產。它投資的項目往往能獲得巨大關注，它發布的研究報告影響整個產業的敘事。想要了解加密貨幣的未來？先看看 a16z 在投什麼。

## a16z 的投資哲學

### 核心理念

```
a16z 的投資邏輯：

1. 「軟體正在吞噬世界」
   - Marc Andreessen 的名言
   - 區塊鏈是新一代軟體基礎設施

2. 長期主義
   - 穿越牛熊週期投資
   - 不追逐短期熱點

3. 基礎設施優先
   - 投資「鏟子」而非「淘金者」
   - 開發者工具、基礎設施為主
```

### Chris Dixon 的觀點

Chris Dixon 是 a16z crypto 的領導者，他的觀點廣受關注：

| 觀點 | 說明 |
|------|------|
| Read-Write-Own | Web3 的核心價值是「擁有」|
| Token incentives | 代幣激勵是殺手級應用 |
| Builder culture | 熊市是建設者的好時機 |
