# Helium
> **最後更新：2026 年 1 月**


## Helium 是最知名的 DePIN 項目，建立了全球最大的去中心化無線網路，從 IoT 到 5G，挑戰傳統電信巨頭。

如果你問：「DePIN 最成功的案例是什麼？」

很多人會回答：**Helium**。

它證明了去中心化基礎設施不只是理論，而是可以實際運作的。

### Helium 是什麼？

Helium 是一個去中心化的無線網路。

傳統上，無線網路由電信公司建設：
- 買下頻譜執照
- 建設基地台
- 鋪設光纖
- 銷售給用戶

Helium 的方式不同：
- 普通人購買 Hotspot 設備
- 在家裡或辦公室架設
- 為附近的 IoT 設備提供網路
- 獲得 HNT 代幣獎勵

這讓 Helium 在短短幾年內，建立了覆蓋全球的無線網路。

### 發展歷程

**2013 年**：Helium 成立，最初專注於 P2P 無線網路硬體

**2019 年**：推出區塊鏈和 HNT 代幣，開始 DePIN 模式

**2020-2021 年**：爆發性成長
- Hotspot 數量從數千增長到數十萬
- HNT 價格飆升
- 成為加密圈熱門話題

**2022 年**：推出 5G 網路（Helium Mobile）

**2023 年**：遷移到 Solana 區塊鏈

**2024 年**：Helium Mobile 用戶持續增長，證明實用性

### 技術架構

Helium 使用的主要技術：

**LoRaWAN**

最初的 Helium 網路使用 LoRaWAN（Long Range Wide Area Network）技術：
- 低功耗
- 長距離（數公里）
- 低頻寬（適合 IoT 設備）

LoRaWAN 適合：
- 智能電表
- 資產追蹤器
- 環境感測器
- 寵物追蹤器

**5G/CBRS**

2022 年，Helium 推出 5G 網路：
- 使用 CBRS 頻段（美國的共享頻譜）
- 提供真正的手機網路
- 與 T-Mobile 合作漫遊

這讓 Helium 從 IoT 擴展到消費級手機網路。

**Solana 區塊鏈**

2023 年，Helium 從自己的區塊鏈遷移到 Solana：
- 更高的交易處理能力
- 更成熟的生態系統
- 降低維護成本

### 代幣經濟

Helium 有三種主要代幣：

**HNT（Helium Network Token）**

- 主要代幣
- 節點營運者獲得的獎勵
- 可以轉換成其他代幣

**MOBILE**

- 5G 網路專用代幣
- 5G Hotspot 營運者獲得
- 可以用 HNT 兌換

**IOT**

- IoT 網路專用代幣
- LoRaWAN Hotspot 營運者獲得
- 可以用 HNT 兌換

**經濟模型**

```
1. Hotspot 提供網路覆蓋 → 獲得 IOT/MOBILE
2. 用戶使用網路 → 支付費用（部分銷毀）
3. IOT/MOBILE 可以兌換 HNT
4. HNT 可以在交易所交易
```

### 如何參與？

**成為 Hotspot 營運者**

1. 購買 Helium 認證的 Hotspot 設備
   - IoT Hotspot：約 $200-500
   - 5G Hotspot：約 $2,000-5,000

2. 在家裡或辦公室安裝
   - 需要網路連接
   - 最好放在窗戶附近或戶外

3. 註冊並上線
   - 使用 Helium Wallet APP
   - 設備會自動開始工作

4. 獲得獎勵
   - 根據覆蓋範圍、網路使用量等獲得代幣

**收益預期**

收益取決於很多因素：
- 位置（附近有沒有其他 Hotspot）
- 覆蓋範圍（天線品質、高度）
- 網路使用量

早期（2021 年）收益很高，現在已經顯著下降。需要仔細計算投資回報率。

### Helium Mobile

Helium Mobile 是 Helium 的消費級產品：

**服務內容**
- 真正的手機網路服務（不是熱點分享）
- 月費 $20（基本方案）
- 與 T-Mobile 合作提供漫遊

**運作方式**
1. 用戶購買 Helium Mobile 方案
2. 優先使用 Helium 5G 網路
3. 沒有 Helium 覆蓋時，自動漫遊到 T-Mobile

**特色**
- 比傳統電信商便宜
- 使用 Helium 網路時獲得 MOBILE 獎勵
- 去中心化網路的實際應用

截至 2025 年，Helium Mobile 已有數十萬用戶，證明了 DePIN 模式可以服務真實消費者。

### 成功因素

Helium 成功的原因：

**1. 解決真實問題**

IoT 連接確實是個問題。蜂巢網路太貴，WiFi 範圍太小。LoRaWAN 填補了這個缺口。

**2. 激勵設計**

早期的高額獎勵吸引了大量參與者，快速建立網路覆蓋。

**3. 硬體標準化**

統一的 Hotspot 規格確保網路品質一致。

**4. 時機**

2020-2021 年加密牛市帶來大量關注和資金。

**5. 持續迭代**

從 IoT 擴展到 5G，從自有鏈遷移到 Solana，展示了適應能力。

### 爭議與挑戰

Helium 也面臨一些爭議：

**1. 實際使用量低**

早期很多 Hotspot 營運者發現，雖然獲得了代幣，但實際網路使用量很低。「挖礦」多於「服務」。

**2. 收益大幅下降**

隨著 Hotspot 數量增加，每個節點的收益大幅下降。很多早期參與者虧損。

**3. 硬體問題**

部分 Hotspot 設備有品質問題，導致用戶不滿。

**4. 詐騙問題**

有報導指出部分 Hotspot 營運者偽造位置或其他數據來獲得更多獎勵。

**5. 經濟可持續性**

如果網路使用量不能顯著增加，純靠代幣通膨不可持續。

### 與傳統電信的比較

| 特性 | 傳統電信 | Helium |
|------|----------|--------|
| 建設者 | 電信公司 | 個人參與者 |
| 資本投入 | 數十億美元 | 眾包 |
| 覆蓋擴展 | 緩慢 | 快速 |
| 服務品質 | 穩定 | 變化較大 |
| 價格 | 較高 | 較低 |
| 收益分配 | 公司股東 | 節點營運者 |

### 未來展望

Helium 可能的發展：

**1. Helium Mobile 擴張**

繼續增加 5G 覆蓋，吸引更多消費者。

**2. 企業 IoT**

吸引更多企業採用 Helium 網路進行 IoT 連接。

**3. 國際擴張**

目前主要在美國。可能擴展到其他國家（需要處理頻譜法規）。

**4. 新的垂直領域**

可能擴展到其他無線技術或基礎設施領域。

### 經驗教訓

Helium 給 DePIN 領域的啟示：

**1. 激勵很重要**

代幣激勵可以快速引導網路建設，但需要平衡。

**2. 真實需求是關鍵**

沒有真實使用需求，DePIN 就只是「挖礦」。

**3. 持續迭代**

市場和技術變化很快，項目需要持續適應。

**4. 管理預期**

過高的收益預期可能導致參與者失望。

Helium 是 DePIN 的先驅和試驗場。無論成敗，它都為這個領域提供了寶貴的經驗。
