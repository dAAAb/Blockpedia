# Filecoin

## Filecoin 是 IPFS 的激勵層，建立了全球最大的去中心化儲存網路，讓任何人都可以出租或使用分散式儲存空間。

你的硬碟有 2TB 空間，但只用了 500GB。

那 1.5TB 就這樣閒置著。

Filecoin 讓你可以把這些空間出租出去，賺取加密貨幣。

同時，需要儲存的人可以用比 AWS 更便宜的價格，獲得去中心化、抗審查的儲存服務。

### Filecoin 是什麼？

Filecoin 是一個去中心化的儲存網路。

傳統雲端儲存（AWS S3、Google Cloud Storage）：
- 中心化公司營運
- 資料存在他們的資料中心
- 他們可以審查、刪除你的資料
- 價格由他們決定

Filecoin 的方式：
- 全球的儲存提供者（礦工）參與
- 資料分散存儲在多個節點
- 沒有單一實體可以審查
- 價格由市場決定

### IPFS 與 Filecoin 的關係

要理解 Filecoin，先要了解 **IPFS**（InterPlanetary File System）。

**IPFS 是什麼？**

IPFS 是一個點對點的檔案系統：
- 檔案用內容哈希（CID）定址，而非 URL
- 檔案可以從任何有該檔案的節點獲取
- 去中心化、抗審查

**IPFS 的問題**

IPFS 本身沒有激勵機制。節點可以隨時刪除檔案，沒有人保證你的檔案會被持續儲存。

**Filecoin 解決這個問題**

Filecoin 是 IPFS 的「激勵層」：
- 用代幣激勵儲存提供者保存檔案
- 用加密經濟學確保檔案不被刪除
- 讓 IPFS 變得可靠

```
IPFS = 如何儲存和定址檔案
Filecoin = 為什麼有人願意幫你存檔案
```

### 運作機制

Filecoin 的核心機制：

**儲存礦工**

1. 購買硬碟設備
2. 加入 Filecoin 網路
3. 接受客戶的儲存訂單
4. 證明確實在儲存資料
5. 獲得 FIL 代幣獎勵

**儲存客戶**

1. 準備要儲存的資料
2. 選擇儲存礦工
3. 支付 FIL 代幣
4. 資料被儲存和複製
5. 需要時可以取回

**證明機制**

Filecoin 用獨特的證明機制確保礦工真的在儲存資料：

- **複製證明（PoRep）**：證明礦工真的複製了客戶的資料
- **時空證明（PoSt）**：證明礦工持續在儲存資料

如果礦工無法提供證明，會被罰沒質押的代幣。

### 代幣經濟

**FIL 代幣**

- 用途：支付儲存費用、質押、獎勵礦工
- 總供應量：約 20 億枚
- 流通方式：區塊獎勵 + 交易

**經濟模型**

1. **儲存訂單**：客戶付 FIL 給礦工
2. **區塊獎勵**：礦工獲得新鑄造的 FIL
3. **質押**：礦工需要質押 FIL 作為保證金
4. **懲罰**：違規礦工會被罰沒質押

### 儲存市場數據

截至 2025 年，Filecoin 網路：

- **總儲存容量**：超過 20 EiB（Exbibytes）
- **活躍礦工**：數千個
- **存儲的資料**：約 2-3 EiB 真實資料
- **主要地區**：中國、美國、歐洲

需要注意的是，大部分容量是「承諾容量」（礦工承諾可以儲存的空間），真實儲存的客戶資料比例較低。

### 如何使用？

**作為客戶（儲存資料）**

直接使用 Filecoin 比較複雜。建議使用：

- **web3.storage**：免費的 IPFS + Filecoin 儲存服務
- **NFT.Storage**：專門為 NFT 設計的儲存
- **Chainsafe Storage**：企業級解決方案

這些服務把複雜的 Filecoin 操作抽象掉，讓你像用雲端硬碟一樣簡單。

**作為礦工（提供儲存）**

1. 購買硬體（主要是大容量硬碟和一定的運算能力）
2. 設置 Lotus 節點（Filecoin 的主要實現）
3. 質押 FIL 代幣
4. 開始接單

門檻比較高，需要：
- 技術能力
- 大量硬碟
- 穩定的網路和電力
- FIL 作為質押

### 用例

Filecoin 的主要用例：

**NFT 資料儲存**

NFT 的實際內容（圖片、影片）需要存在某處。很多 NFT 平台用 Filecoin/IPFS：
- OpenSea 部分 NFT
- Foundation
- 很多 NFT 項目

**網頁備份**

Internet Archive 等組織用 Filecoin 備份網頁內容。

**科學資料**

大型科學資料集的存儲和分享。

**區塊鏈資料**

其他區塊鏈的歷史資料存儲。

**企業備份**

對去中心化、抗審查儲存有需求的企業。

### 與傳統雲端的比較

| 特性 | AWS S3 | Filecoin |
|------|--------|----------|
| 價格 | 約 $23/TB/月 | 約 $0.0001/GB/月（變化大）|
| 可靠性 | 99.999999999% | 取決於礦工 |
| 延遲 | 毫秒級 | 較高 |
| 審查 | 可能被審查 | 抗審查 |
| 去中心化 | 否 | 是 |
| 使用難度 | 簡單 | 較複雜 |

Filecoin 適合：
- 對成本非常敏感
- 需要抗審查
- 長期冷儲存
- 不需要即時存取

不太適合：
- 需要低延遲
- 需要高可靠性保證
- 頻繁存取的熱資料

### 挑戰

Filecoin 面臨的挑戰：

**1. 使用門檻高**

直接使用 Filecoin 很複雜。需要依賴中間層服務。

**2. 真實需求有限**

大部分容量是空閒的，真實付費客戶比例不高。

**3. 中心化風險**

大型礦工（特別是中國的）佔據大部分算力，有集中化風險。

**4. 代幣經濟**

如果 FIL 價格下跌，礦工可能虧損退出，影響網路穩定。

**5. 技術複雜**

對普通人來說，IPFS/Filecoin 的概念不容易理解。

### Filecoin Virtual Machine（FVM）

2023 年，Filecoin 推出 **FVM**（Filecoin Virtual Machine）。

這讓 Filecoin 可以運行智能合約，不只是儲存。

可能的應用：
- 資料 DAO（共同管理資料集）
- 自動化儲存交易
- DeFi 與儲存結合
- 資料代幣化

FVM 把 Filecoin 從單純的「儲存網路」變成「可程式化的資料層」。

### 未來展望

Filecoin 可能的發展：

**1. 使用體驗改善**

更多像 web3.storage 這樣的服務，降低使用門檻。

**2. 企業採用**

吸引更多企業用於備份和歸檔。

**3. 與 AI 整合**

AI 需要大量資料集，Filecoin 可以成為資料存儲層。

**4. 跨鏈整合**

與更多區塊鏈整合，成為 Web3 的資料層。

**5. FVM 生態發展**

基於 FVM 的應用生態成長。

{% hint style="info" %}
Filecoin 是去中心化儲存領域最大的項目，但還在證明其長期可行性的階段。
{% endhint %}

### 與其他儲存項目的比較

| 項目 | 特色 | 代幣 |
|------|------|------|
| Filecoin | 最大規模、有 IPFS 生態 | FIL |
| Arweave | 永久儲存、一次付費 | AR |
| Storj | 更接近傳統雲端體驗 | STORJ |
| Sia | 較早的去中心化儲存 | SC |

Filecoin 的優勢在於規模和 IPFS 整合，但每個項目都有其定位。
