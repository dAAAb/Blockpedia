# 糖果

### 在幣圈，「糖果」是免費代幣的俗稱。就像萬聖節發糖果一樣，項目方會把代幣免費「發」給用戶。這些糖果有時候一文不值，有時候卻能價值不菲。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是糖果？

| 概念 | 說明 |
|------|------|
| 定義 | 免費獲得的代幣 |
| 來源 | 空投、獎勵活動 |
| 俗稱 | 糖果 = 空投代幣 |
| 價值 | 從歸零到數千美元不等 |

---

## 糖果的來源

### 獲取方式

```
如何獲得糖果：

1. 空投（Airdrop）
   - 使用協議後獲得
   - 持有特定代幣獲得
   - 最常見的方式

2. 測試網獎勵
   - 參與測試網測試
   - 主網上線後獲得

3. 社群活動
   - 參與任務
   - 社群貢獻
   - 活動獎勵

4. 分叉幣
   - 區塊鏈分叉
   - 原幣持有者獲得新幣
   - 如 BCH、ETC
```

## 糖果的價值

### 價值分佈

| 類型 | 價值範圍 |
|------|----------|
| 垃圾糖果 | 歸零或近乎零 |
| 一般空投 | 幾美元到幾十美元 |
| 優質空投 | 數百到數千美元 |
| 大獎空投 | 數萬美元 |

### 經典糖果案例

```
歷史上的大糖果：

Uniswap（UNI）- 2020 年：
- 最低 400 UNI
- 高峰價值 $10,000+

ENS - 2021 年：
- 依註冊年限
- 最高 $30,000+

Arbitrum（ARB）- 2023 年：
- 依活躍度
- 最高 $10,000+

這些都是「追溯空投」
- 使用者在空投前就使用
- 沒有刻意去「擼」
- 真正獎勵早期用戶
```

{% hint style="info" %}
**最甜的糖果往往是意外**

最有價值的空投通常是你正常使用產品後意外獲得的，而不是刻意去「擼」來的。
{% endhint %}

### 什麼是擼糖果

擼糖果不是真的免費：

## 糖果安全

### 注意事項

| 風險 | 防範 |
|------|------|
| 假糖果 | 不點不明連結 |
| 釣魚網站 | 用官方網址 |
| 惡意授權 | 小心批准合約 |
| 私鑰洩漏 | 絕不透露種子詞 |

{% hint style="warning" %}
**很多「糖果」是詐騙**

如果有人私訊你說可以領糖果，99% 是詐騙。真正的空投不需要你額外操作，會直接發到你的錢包。
{% endhint %}
