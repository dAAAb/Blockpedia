# 糖果盒

### 空投太多，哪些值得參加？糖果盒平台幫你整理。這些平台聚合各種空投資訊，有的還提供任務系統讓你一站式完成。但要小心，不是所有糖果盒都是安全的。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## 糖果盒類型

### 平台類型

```
糖果盒的類型：

1. 資訊聚合型
   - 整理空投資訊
   - 不執行操作
   - 只是告訴你有什麼

2. 任務平台型
   - 提供任務系統
   - 完成任務獲得積分
   - 可能兌換獎勵

3. 錢包整合型
   - 整合在錢包中
   - 自動追蹤資格
   - 提醒可領取的空投

4. 社群型
   - Discord/Telegram 群組
   - 人工分享資訊
   - 互相交流
```

## 使用糖果盒

### 使用流程


## 風險與注意

### 使用風險

| 風險 | 說明 |
|------|------|
| 釣魚風險 | 假平台騙授權 |
| 資訊過時 | 空投可能已結束 |
| 時間浪費 | 任務可能白做 |
| 隱私洩漏 | 過多個資收集 |

### 安全使用

```
安全使用糖果盒：

1. 選擇知名平台
   - 用大家都在用的
   - 查看 Twitter 粉絲數

2. 分開錢包
   - 用專門的小錢包
   - 不放大額資產

3. 小心授權
   - 看清楚批准什麼
   - 定期撤銷授權

4. 不要貪心
   - 太好的可能是假的
   - 不給私鑰/種子詞
```

{% hint style="warning" %}
**使用專門錢包**

參與空投任務時，建議使用專門的小錢包，不要用存有大量資產的主錢包。萬一連接到惡意網站，損失可以控制。
{% endhint %}
