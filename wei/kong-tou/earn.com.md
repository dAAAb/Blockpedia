# Earn.com

### 在 2018 年，有一個網站讓你回覆郵件就能賺比特幣。這就是 Earn.com——透過完成任務、回覆調查來賺取加密貨幣。它後來被 Coinbase 收購，創辦人成為了 Coinbase CTO。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## Earn.com 的模式

### 運作方式

```
Earn.com 的商業模式：

對用戶（賺錢方）：
- 完成任務
- 回覆調查
- 回答問題
- 賺取 BTC

對企業（付費方）：
- 付費讓用戶看郵件
- 獲得高品質回覆
- 因為用戶有動機認真回答

核心理念：
- 你的時間和注意力有價值
- 應該被付費
- 用加密貨幣支付
```

### 任務類型

| 任務 | 說明 |
|------|------|
| 回覆郵件 | 閱讀並回覆 |
| 填寫調查 | 完成問卷 |
| 社群任務 | 關注、分享 |
| 推薦用戶 | 邀請朋友 |

## 歷史與發展

### 重要事件

```
Earn.com 的歷程：

2013：成立 21 Inc
- 開始做比特幣硬體
- 募資超過 $120M

2017：轉型為 21.co / Earn.com
- 從硬體轉向軟體
- 推出任務賺幣模式

2018：被 Coinbase 收購
- Balaji 成為 Coinbase CTO
- 整合進 Coinbase 生態

後續：
- Coinbase Earn 誕生
- 學習賺幣模式延續
```
