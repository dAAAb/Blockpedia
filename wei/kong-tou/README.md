# 空投

### 免費的代幣？天上掉下來的餡餅？空投確實可能讓你不花一分錢就獲得價值數千美元的代幣。但空投也有門檻、風險和陷阱。搞懂規則，才能安全地「白嫖」。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## 空投類型

### 常見類型

```
空投的類型：

1. 追溯空投（Retroactive Airdrop）
   - 獎勵早期用戶
   - 已經使用過協議
   - 最有價值的類型
   - 例：Uniswap、Arbitrum

2. 任務空投
   - 完成特定任務
   - 關注社群、轉推
   - 價值通常較低

3. 持有空投（Holder Airdrop）
   - 持有特定代幣
   - 快照時持有即可
   - 例：持有 ETH 獲得空投

4. 抽獎空投
   - 隨機抽選
   - 參與即有機會
```

### 追溯空投案例

| 項目 | 空投價值 | 條件 |
|------|----------|------|
| Uniswap | 最高 $10,000+ | 使用過 DEX |
| ENS | 最高 $30,000+ | 註冊過域名 |
| Arbitrum | 最高 $10,000+ | 橋接或使用 |
| Optimism | $1,000-$10,000+ | 多次使用 |

{% hint style="info" %}
**追溯空投最有價值**

追溯空投是獎勵「已經」使用過協議的用戶。這意味著你需要在空投宣布之前就使用協議，而不是宣布後才來。
{% endhint %}

## 如何參與空投

### 基本要求

```
空投獵人必備：

1. 自己的錢包
   - 非託管錢包
   - 如 MetaMask
   - 絕對不能用交易所錢包

2. 鏈上活動
   - 使用 DeFi 協議
   - 跨鏈橋接
   - 交易 NFT

3. 社群參與
   - Discord
   - Twitter/X
   - 論壇討論

4. 耐心
   - 空投可能很久才發
   - 不一定每個都成功
```

### 空投獵人策略

| 策略 | 說明 |
|------|------|
| 使用新協議 | 測試網、主網早期 |
| 多鏈活動 | L2、新公鏈 |
| 保持活躍 | 定期使用 |
| 分散風險 | 不要只盯一個 |

## 空投風險

### 常見陷阱

```
空投詐騙手法：

1. 假空投網站
   - 冒充官方網站
   - 要求連接錢包
   - 騙取授權後盜幣

2. 假代幣
   - 發送假代幣到錢包
   - 誘導你去「領取」
   - 實際是釣魚

3. 社群詐騙
   - 假官方帳號
   - 私訊說你中獎
   - 要求轉帳手續費

4. 惡意合約
   - 需要批准不明合約
   - 授權後被盜幣
```

### 安全原則

| 原則 | 說明 |
|------|------|
| 不要點不明連結 | 用官方網址 |
| 不要授權不明合約 | 看清楚再批准 |
| 不要相信私訊 | 官方不會私訊 |
| 使用小錢包 | 分離主要資產 |

{% hint style="danger" %}
**空投詐騙橫行**

大多數聲稱的「空投」都是詐騙。真正的空投不會要求你轉帳、不會私訊你、也不會在你不知情時出現。
{% endhint %}
