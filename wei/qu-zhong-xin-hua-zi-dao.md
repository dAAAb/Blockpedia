# 去中心化自組織（DAO）

### DAO 是一種沒有老闆、沒有董事會的組織形式。規則寫在智能合約裡，決策由代幣持有者投票。聽起來很理想，但實踐中也遇到了不少挑戰。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是 DAO？

| 概念 | 說明 |
|------|------|
| 全名 | Decentralized Autonomous Organization |
| 中文 | 去中心化自治組織 |
| 核心 | 智能合約 + 代幣治理 |
| 特色 | 無中心化管理層 |

---

## DAO 的運作

### 基本機制

```
DAO 的運作方式：

1. 治理代幣
   - 持有代幣 = 投票權
   - 代幣越多，投票權越大
   - 類似股東大會

2. 提案流程
   - 成員提出提案
   - 社群討論
   - 代幣持有者投票
   - 達到門檻則執行

3. 智能合約執行
   - 投票結果自動執行
   - 減少人為干預
   - 規則公開透明
```

### 投票機制

| 機制 | 說明 |
|------|------|
| 1 幣 1 票 | 最常見 |
| 二次方投票 | 減少大戶影響 |
| 時間加權 | 持有越久權重越高 |
| 委託投票 | 可委託他人投票 |

## DAO 的挑戰

### 常見問題

```
DAO 面臨的挑戰：

1. 投票率低
   - 大多數人不投票
   - 決策被少數人主導

2. 大戶控制
   - 持幣多的人說了算
   - 與「去中心化」矛盾

3. 效率問題
   - 每件事都要投票
   - 決策速度慢

4. 法律地位不明
   - DAO 不是公司
   - 法律責任誰承擔？

5. 安全風險
   - 智能合約漏洞
   - The DAO 事件教訓
```

### The DAO 事件

| 時間 | 2016 年 6 月 |
|------|--------------|
| 事件 | 駭客利用漏洞盜取 360 萬 ETH |
| 結果 | 以太坊硬分叉恢復資金 |
| 影響 | 催生 ETC、重新思考 DAO 安全 |

{% hint style="warning" %}
**DAO 不等於完美**

DAO 是一種實驗性的組織形式。它有去中心化的優點，但也有效率、安全、治理等方面的挑戰。
{% endhint %}
