# 上幣

### 「幣安上幣」曾經是幣圈最大的利好消息之一。上幣意味著項目從私募進入公開市場，散戶終於可以買到。但上幣也是一把雙刃劍——有時候上幣日就是最高點。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是上幣？

| 概念 | 說明 |
|------|------|
| 定義 | 代幣在交易所上架 |
| 英文 | Listing |
| 意義 | 進入公開交易市場 |
| 影響 | 流動性大幅增加 |

---

## 上幣流程

### 一般流程

```
項目上幣流程：

1. 申請
   - 向交易所提交申請
   - 提供項目資料
   - 可能需要付費

2. 審核
   - 交易所盡職調查
   - 評估技術、團隊、合規
   - 可能需要改進

3. 談判
   - 上幣費用
   - 做市商安排
   - 行銷配合

4. 上線
   - 公告上幣時間
   - 開放充值
   - 開始交易
```

### 上幣費用

| 交易所等級 | 費用範圍 |
|------------|----------|
| 一線（幣安等）| $100K-$1M+ |
| 二線 | $10K-$100K |
| 小型 | $1K-$10K |
| DEX | Gas 費用 |

{% hint style="info" %}
**上幣費用差異大**

不同交易所的上幣費用差異極大。大交易所可能要數十萬美元，但帶來的曝光和流動性也遠超小所。
{% endhint %}

## 不同類型的上幣

### 交易所類型

| 類型 | 說明 |
|------|------|
| CEX 上幣 | 中心化交易所上架 |
| DEX 上幣 | 去中心化交易所（無門檻）|
| Launchpad | 交易所新幣發行平台 |

### CEX vs DEX

```
CEX vs DEX 上幣比較：

CEX 上幣：
- 需要審核和費用
- 流動性通常較好
- 有做市商支持
- 帶來更多用戶
- 代表一定背書

DEX 上幣：
- 無需審核
- 任何人都可上
- 流動性可能差
- 垃圾幣也能上
- 不代表任何背書
```

## 上幣的影響

### 價格影響

```
上幣對價格的影響：

上幣前：
- 預期炒作
- 價格可能上漲
- "Buy the rumor"

上幣時：
- 流動性突然增加
- 早期投資者可能賣出
- 波動極大

上幣後：
- "Sell the news"
- 價格可能回落
- 真實價值逐漸顯現
```

### 常見現象

| 現象 | 說明 |
|------|------|
| 上幣即高點 | 早期投資者獲利了結 |
| 開盤暴漲後回落 | FOMO 買入後修正 |
| 持續下跌 | 沒有真實需求支撐 |
| 穩定發展 | 優質項目的表現 |

{% hint style="warning" %}
**上幣不等於成功**

很多代幣上幣後就是最高點，然後一路下跌。上幣只是起點，不是終點。不要 FOMO 在上幣時追高。
{% endhint %}

## 如何評估上幣消息

### 評估要點

| 要點 | 說明 |
|------|------|
| 交易所等級 | 大所上幣影響更大 |
| 代幣經濟 | 解鎖時程、流通量 |
| 早期估值 | VC 成本多少 |
| 項目基本面 | 是否有真實價值 |

### 投資建議

```
面對上幣消息的建議：

不要：
✗ FOMO 追高
✗ 用槓桿博開盤
✗ All-in 單一幣

應該：
✓ 研究項目基本面
✓ 了解代幣經濟
✓ 等價格穩定後再決定
✓ 分批進場
```
