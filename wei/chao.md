# 炒幣

### 「炒幣」是幣圈的日常——買低賣高，賺取差價。聽起來簡單，做起來難。統計顯示，大多數短線交易者最終都是虧損的。在進入這個戰場之前，你需要了解遊戲規則。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是炒幣？

| 概念 | 說明 |
|------|------|
| 定義 | 短線買賣加密貨幣 |
| 目的 | 賺取價差 |
| 時間框架 | 秒到週不等 |
| 風險 | 極高 |

## 交易類型

### 按時間分類

```
不同時間框架的交易：

1. 搶帽客（Scalping）
   - 秒到分鐘
   - 頻繁交易
   - 賺取微小價差
   - 需要極低延遲

2. 當沖（Day Trading）
   - 當天開平倉
   - 不過夜
   - 高強度操作

3. 波段（Swing Trading）
   - 數天到數週
   - 捕捉中期波動
   - 較不密集

4. 長線投機
   - 數週到數月
   - 但非長期持有
   - 仍是投機而非投資
```

### 按工具分類

| 工具 | 說明 | 風險 |
|------|------|------|
| 現貨 | 直接買賣幣 | 中 |
| 槓桿現貨 | 借錢買幣 | 高 |
| 永續合約 | 無到期日的合約 | 極高 |
| 期貨 | 有到期日的合約 | 極高 |
| 選擇權 | 買賣權利 | 高 |

{% hint style="danger" %}
**合約風險**

使用槓桿和合約可能讓你在短時間內損失所有資金，甚至負債。超過 90% 的散戶在合約交易中虧損。
{% endhint %}

## 風險管理

### 基本原則

```
炒幣風險管理：

1. 設定停損
   - 每筆交易都要停損
   - 不要移動停損點
   - 虧損是成本的一部分

2. 資金管理
   - 單筆不超過總資金 1-2%
   - 不要 All-in
   - 保留現金

3. 心理管理
   - 不要報復性交易
   - 虧損後休息
   - 不要情緒化

4. 記錄檢討
   - 記錄每筆交易
   - 定期檢討
   - 持續學習
```

### 常見錯誤

| 錯誤 | 後果 |
|------|------|
| 不設停損 | 小虧變大虧 |
| 過度交易 | 手續費吃掉利潤 |
| 報復交易 | 虧更多 |
| 高槓桿 | 爆倉 |
| FOMO 追高 | 買在頂部 |

## 炒幣 vs HODL

### 比較

| 比較 | 炒幣 | HODL |
|------|------|------|
| 時間投入 | 大量 | 很少 |
| 技術要求 | 高 | 低 |
| 壓力 | 極大 | 較小 |
| 手續費 | 高 | 極低 |
| 稅務 | 複雜 | 簡單 |
| 成功率 | 低 | 因標的而異 |

{% hint style="info" %}
**大多數人不適合炒幣**

研究顯示，超過 70-90% 的散戶交易者最終虧損。如果你沒有專業知識、充足時間和良好心理素質，長期持有優質資產可能是更好的選擇。
{% endhint %}
