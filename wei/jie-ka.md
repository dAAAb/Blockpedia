# 借幣

### 不只可以借錢，還可以借幣。在加密貨幣市場，你可以借入 BTC、ETH 等代幣。為什麼要借幣？可能是為了做空、可能是為了套利、也可能是為了避免賣出現有持倉。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是借幣？

| 概念 | 說明 |
|------|------|
| 定義 | 借入加密貨幣代幣 |
| 形式 | CEX 融券、DeFi 借貸 |
| 用途 | 做空、套利、質押 |
| 成本 | 借幣利息 |

---

## 借幣的用途

### 主要用途

```
為什麼要借幣：

1. 做空
   - 借入 BTC
   - 賣出換成 USDT
   - 等價格下跌
   - 買回 BTC 還款
   - 賺取差價

2. 套利
   - 借幣參與空投
   - 跨平台套利
   - 利率套利

3. 避免賣出
   - 不想賣掉長期持有的幣
   - 借穩定幣來用
   - 保持敞口

4. 做市
   - 提供流動性
   - 需要兩種代幣
   - 借其中一種
```

## 借幣方式

### 交易所借幣

| 形式 | 說明 |
|------|------|
| 現貨槓桿 | 借幣買/賣現貨 |
| 融券 | 專門借幣做空 |
| 合約 | 隱含借幣（保證金交易）|

### DeFi 借幣

```
DeFi 借幣流程：

1. 在 Aave/Compound 存入抵押品
   - 如存入 ETH

2. 借出其他代幣
   - 如借出 USDC 或其他代幣

3. 使用借來的代幣
   - 做空
   - 套利
   - 其他用途

4. 歸還
   - 還款 + 利息
   - 取回抵押品
```

## 做空機制

### 借幣做空

```
借幣做空範例：

步驟：
1. 借入 1 BTC（價值 $50,000）
2. 賣出換成 $50,000
3. BTC 跌到 $40,000
4. 用 $40,000 買回 1 BTC
5. 還給借方 1 BTC
6. 獲利 $10,000（扣除利息）

風險：
- 如果 BTC 漲到 $60,000
- 你需要 $60,000 才能買回
- 虧損 $10,000 + 利息
- 理論上虧損無上限
```

{% hint style="danger" %}
**做空風險無上限**

做多最多虧 100%（歸零），但做空理論上虧損無上限。如果做空的幣暴漲，你可能虧損數倍本金。
{% endhint %}

## 借幣成本

### 利息費用

| 因素 | 影響 |
|------|------|
| 幣種 | 熱門幣利率較低 |
| 平台 | 不同平台利率不同 |
| 市況 | 波動大時利率上升 |
| 時間 | 借越久成本越高 |

### 利率範例

```
借幣利率參考（浮動）：

穩定幣（USDT/USDC）：
- 年化 5-20%
- 視市場需求

主流幣（BTC/ETH）：
- 年化 1-10%
- 視做空需求

小幣種：
- 可能很高
- 流動性差
```

## 風險管理

### 借幣風險

| 風險 | 說明 |
|------|------|
| 清算風險 | 抵押品價值下跌 |
| 利息成本 | 時間越長成本越高 |
| 強制還款 | 可能被要求提前還 |
| 價格風險 | 做空時價格上漲 |

### 使用建議

```
借幣注意事項：

1. 控制規模
   - 不要借太多
   - 保持抵押率安全

2. 注意利率
   - 利率會浮動
   - 計算成本

3. 設定停損
   - 特別是做空
   - 虧損無上限

4. 短期為主
   - 利息會累積
   - 不適合長期
```
