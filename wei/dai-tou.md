# 代投

### ICO 熱潮時期，很多人找不到參與管道，於是「代投」應運而生。有人聲稱能幫你參加熱門項目的私募，只要給他錢就好。結果呢？詐騙橫行，很多人血本無歸。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是代投？

| 概念 | 說明 |
|------|------|
| 定義 | 代替他人參與項目投資 |
| 背景 | ICO 時期私募門檻高 |
| 形式 | 收集資金、統一參投 |
| 風險 | 極高，詐騙橫行 |

---

## 代投的起源

### ICO 時期背景

```
為什麼會有代投：

背景（2017-2018）：
- ICO 熱潮
- 熱門項目供不應求
- 私募有最低門檻（如 100 ETH）
- 有白名單限制

散戶困境：
- 金額不夠門檻
- 沒有管道參與
- 想買買不到

代投出現：
- 有人聲稱有私募管道
- 收集散戶資金
- 統一參與投資
- 代幣發放後分配
```

## 代投的風險

### 主要風險

| 風險類型 | 說明 |
|----------|------|
| 詐騙跑路 | 收了錢就消失 |
| 無法驗證 | 不知道是否真的投了 |
| 分配不公 | 實際拿到的少於承諾 |
| 法律風險 | 可能涉及非法集資 |
| 代幣歸屬 | 代幣在他人錢包中 |

### 常見詐騙手法

```
代投詐騙套路：

1. 建立信任
   - 在社群活躍
   - 分享「成功案例」
   - 展示「管道證據」

2. 收集資金
   - 聲稱有限額度
   - 製造稀缺感
   - 催促快速決定

3. 詐騙方式
   - 直接跑路（最常見）
   - 謊稱投資失敗
   - 給你假代幣
   - 剋扣大部分額度

4. 消失
   - 刪除帳號
   - 解散群組
   - 無法聯繫
```

{% hint style="danger" %}
**代投風險極高**

代投沒有任何法律保障。即使對方真的參與了投資，你也無法確保拿到應得的份額。詐騙案例數不勝數。
{% endhint %}

## 為什麼不要參與代投

### 問題分析

```
代投的根本問題：

1. 無法驗證
   - 你不知道錢去哪了
   - 無法確認是否投資
   - 全憑對方一張嘴

2. 無法追討
   - 沒有合約
   - 跨境難追
   - 金額通常不值得訴訟

3. 代幣控制
   - 代幣在對方錢包
   - 分配由對方決定
   - 你沒有任何控制權

4. 法律灰色地帶
   - 代投者可能違法
   - 但你也難以維權
   - 報警也不一定有用
```

## 如何保護自己

### 安全原則

| 原則 | 說明 |
|------|------|
| 不參與代投 | 最安全的方式 |
| 自己參與 | 直接參加公募 |
| 驗證管道 | 確認是否為官方 |
| 控制私鑰 | 代幣要到自己錢包 |

### 如果一定要參與

```
如果堅持要代投（極不建議）：

最低要求：
- 認識對方真實身份
- 有合約或書面協議
- 分批付款
- 要求即時分配

但即使這樣也不安全：
- 合約可能無法執行
- 對方仍可能跑路
- 損失難以追回
```

## 現況

### 代投的衰落

| 時期 | 狀況 |
|------|------|
| 2017-2018 | ICO 熱潮，代投盛行 |
| 2019-2020 | ICO 退燒，代投減少 |
| 2021 至今 | IDO/Launchpad 興起，代投式微 |

### 現代替代方案

```
不需要代投的參與方式：

1. Launchpad
   - 交易所的項目平台
   - 直接參與公募
   - 相對透明

2. IDO 平台
   - 去中心化發行
   - 自己錢包參與
   - 更公平

3. 公開市場
   - 等上交易所後購買
   - 雖然價格可能較高
   - 但更安全
```
