# 領投

### 當你看到某個項目宣布「獲得 a16z 領投」，這代表什麼？領投是投資輪次的主角，通常是最大的投資者，也是對項目最有信心的人。他們的參與往往是項目品質的背書。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

---

## 什麼是領投？

| 概念 | 說明 |
|------|------|
| 英文 | Lead Investor |
| 定義 | 投資輪次的主要投資者 |
| 角色 | 決定條款、帶頭投資 |
| 通常是 | VC、基金、知名機構 |

---

## 領投的角色

### 主要職責

```
領投的職責：

1. 盡職調查
   - 深入研究項目
   - 評估團隊能力
   - 審查技術和商業模式

2. 決定估值
   - 與項目方談判
   - 確定投資價格
   - 設定投資條款

3. 投資最大額度
   - 通常投最多錢
   - 承擔最大風險
   - 獲得最多分配

4. 背書效應
   - 吸引其他投資者
   - 增加項目可信度
   - 幫助後續募資
```

### 領投 vs 跟投

| 角色 | 領投 | 跟投 |
|------|------|------|
| 投資額 | 最大 | 較小 |
| 談判權 | 主導 | 跟隨 |
| 盡調深度 | 深入 | 依賴領投 |
| 風險 | 最大 | 分散 |
| 董事會席位 | 通常有 | 通常無 |

## 知名領投機構

### 頂級加密 VC

| 機構 | 說明 |
|------|------|
| a16z (Andreessen Horowitz) | 最知名的加密 VC |
| Paradigm | 頂級加密基金 |

| Coinbase Ventures | Coinbase 投資部門 |
| Binance Labs | 幣安投資部門 |

{% hint style="info" %}
**領投的背書效應**

當 a16z 或 Paradigm 領投一個項目，往往會帶來一波關注。但這不代表項目一定成功——頂級 VC 也有投錯的時候。
{% endhint %}

## 投資輪次

### 常見輪次

```
項目融資輪次：

種子輪（Seed）：
- 最早期
- 金額：$1-5M
- 估值最低
- 風險最高

A 輪：
- 有了一定進展
- 金額：$5-20M
- 產品初步成形

B 輪及之後：
- 更成熟階段
- 金額更大
- 風險相對較低

每輪可能有不同領投：
- 種子輪：天使投資人
- A 輪：早期 VC
- B 輪：成長型基金
```

## 對散戶的意義

### 如何參考

| 訊號 | 解讀 |
|------|------|
| 頂級 VC 領投 | 項目有一定水準 |
| 多家知名機構 | 獲得廣泛認可 |
| 無知名領投 | 需要更謹慎 |
| 估值過高 | 散戶進場空間有限 |

### 注意事項

```
散戶應注意：

1. VC 的優勢
   - 早期低價進場
   - 有解鎖期保護
   - 獲得更多資訊

2. 散戶的劣勢
   - 公開後才能買
   - 價格通常更高
   - 資訊不對稱

3. 解鎖壓力
   - VC 代幣會解鎖
   - 可能造成賣壓
   - 要注意解鎖時程

4. 不要盲目跟隨
   - VC 也會虧錢
   - 他們的成本比你低
   - 做自己的研究
```

{% hint style="warning" %}
**VC 投資 ≠ 必然成功**

即使是最頂級的 VC 領投，項目也可能失敗。FTX 有眾多頂級 VC 投資，最後仍然崩盤。永遠要做自己的研究。
{% endhint %}
