# 空氣幣

### 「空氣幣」這個詞在幣圈是最嚴重的指控之一——它代表這個代幣沒有任何實際價值，只是包裝精美的空氣。了解空氣幣的特徵，是保護自己的第一步。

---

## 什麼是空氣幣？

| 概念 | 說明 |
|------|------|
| 定義 | 沒有實際價值支撐的代幣 |
| 特徵 | 只有概念，沒有產品 |
| 目的 | 通常是為了圈錢 |
| 結局 | 大多數歸零 |

---

## 如何識別空氣幣

### 常見特徵

```
空氣幣的警訊：

技術面：
- 白皮書抄襲或空洞
- 沒有 GitHub 或代碼活動
- 沒有可驗證的產品
- 技術描述含糊不清

團隊面：
- 團隊成員匿名或造假
- 無法驗證的背景
- 沒有真實社群互動
- KOL 付費背書

經濟面：
- 團隊持有大量代幣
- 鎖倉期極短或沒有
- 代幣經濟不合理
- 只談價格不談應用
```

### 檢查清單

| 檢查項目 | 空氣幣特徵 |
|----------|------------|
| 白皮書 | 空洞、抄襲、只談願景 |
| 團隊 | 匿名、無法驗證、假背景 |
| 產品 | 沒有、PPT 階段、永遠開發中 |
| 代碼 | 沒有、不開源、零活動 |
| 社群 | 假的、只談價格、機器人 |
| 經濟模型 | 不合理、團隊佔比過高 |

{% hint style="warning" %}
**白皮書不等於產品**

很多人看到厚厚的白皮書就覺得是正經項目。但白皮書只是計畫書，不是產品。很多空氣幣的白皮書寫得很漂亮，但從來沒有實現。
{% endhint %}

## 空氣幣的包裝手法

### 常見敘事

| 敘事類型 | 套路 |
|----------|------|
| 技術創新 | 「革命性技術」但說不清楚 |
| 名人背書 | 付費代言或造假 |
| 合作夥伴 | 假合作或誇大 |
| 即將上線 | 永遠的「即將」|
| 錯過就沒了 | 製造 FOMO |

### 價格操控

```
空氣幣的價格模式：

發行階段：
1. 私募低價賣給內部人
2. 公募價格稍高
3. 上交易所拉盤

套利階段：
4. 製造交易量假象
5. 配合利好消息
6. 內部人分批出貨

崩盤階段：
7. 價格暴跌
8. 團隊消失
9. 投資者血本無歸
```

## 與其他幣種的區別

### 分類比較

| 類型 | 說明 | 例子 |
|------|------|------|
| 主流幣 | 有價值、有生態 | BTC、ETH |
| 山寨幣 | 模仿但有嘗試 | 各種 L1 |
| 迷因幣 | 沒有應用但誠實 | DOGE、SHIB |
| 空氣幣 | 沒有價值且欺騙 | 太多了... |

{% hint style="info" %}
**迷因幣 vs 空氣幣**

迷因幣和空氣幣都沒有實際應用，但有一個關鍵區別：迷因幣誠實地說自己只是 meme，而空氣幣假裝自己是正經項目。
{% endhint %}

## 如何保護自己

### 研究方法

| 方法 | 說明 |
|------|------|
| 查團隊 | LinkedIn、過往經歷 |
| 查代碼 | GitHub 活動 |
| 查社群 | 真實互動還是機器人 |
| 查合作 | 官方確認還是單方面宣稱 |
| 查鏈上 | 代幣分布、大戶持倉 |

### 基本原則

| 原則 | 說明 |
|------|------|
| 聽起來太好 | 就是太好，別信 |
| 做自己研究 | DYOR，不要只聽別人說 |
| 不懂不投 | 搞不懂就不要投 |
| 只投能虧 | 只投能承受歸零的錢 |

{% hint style="danger" %}
**沒有穩賺的投資**

任何承諾「穩定高回報」的加密貨幣項目，幾乎都是騙局。真正有價值的項目不需要用回報率來吸引人。
{% endhint %}
