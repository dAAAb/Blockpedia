# 垃圾幣

### 「垃圾幣」是幣圈最直白的用語——它指的是那些沒有技術、沒有應用、只靠炒作的代幣。這個詞帶有貶義，但了解什麼是垃圾幣，能幫你避開投資陷阱。

---

## 什麼是垃圾幣？

| 概念 | 說明 |
|------|------|
| 英文 | Shitcoin |
| 定義 | 低品質、無價值的加密貨幣 |
| 特徵 | 缺乏技術、應用或團隊 |
| 結局 | 大多歸零 |

{% hint style="warning" %}
**這是主觀判斷**

「垃圾幣」是主觀用語，不同人對此有不同定義。有些人把所有比特幣以外的幣都叫垃圾幣（比特幣極端主義），有些人只指明顯的騙局。
{% endhint %}

---

## 垃圾幣的特徵

### 常見特徵

```
垃圾幣的警訊：

技術面：
- 沒有原創技術
- 只是複製其他項目
- 沒有實際產品
- 代碼沒有活動

團隊面：
- 匿名或無法驗證
- 沒有相關經驗
- 只會做營銷

經濟面：
- 團隊持幣過多
- 流動性極低
- 只有少數交易所上架

社群面：
- 只討論價格
- 大量機器人
- 質疑者被封禁
```

### 與其他幣種比較

| 類型 | 說明 |
|------|------|
| 主流幣 | BTC、ETH 等有價值共識 |
| 山寨幣 | 模仿但有嘗試做事 |
| 迷因幣 | 承認是 meme，社群驅動 |
| 垃圾幣 | 沒有價值，只是圈錢 |

---

## 垃圾幣的危害

### 常見手法

```
垃圾幣如何傷害投資者：

1. Pump and Dump
   - 莊家低價囤幣
   - 炒作拉高價格
   - 吸引散戶進場
   - 莊家高位出貨
   - 價格崩盤

2. Rug Pull
   - 建立流動性池
   - 吸引投資者買入
   - 撤走流動性
   - 投資者無法賣出

3. 蜜罐（Honeypot）
   - 只能買不能賣
   - 合約有惡意代碼
   - 發現時已經太遲
```

### 損失案例

| 手法 | 特徵 | 結果 |
|------|------|------|
| Pump & Dump | 價格暴漲後暴跌 | 散戶接盤 |
| Rug Pull | 流動性突然消失 | 幣無法賣出 |
| Honeypot | 只進不出 | 全額損失 |

## 如何識別

### 檢查清單

| 檢查項目 | 警訊 |
|----------|------|
| 合約安全 | 未審計、有可疑功能 |
| 團隊背景 | 匿名、無法驗證 |
| 代幣分布 | 少數地址持有大量 |
| 流動性 | 極低、未鎖定 |
| 社群 | 只談價格、機器人多 |

### 工具使用

| 工具 | 用途 |
|------|------|
| Token Sniffer | 檢測合約安全 |
| DEXTools | 查看代幣數據 |
| RugDoc | 風險評估 |
| Etherscan | 查看代幣分布 |

{% hint style="info" %}
**做自己的研究（DYOR）**

沒有工具能 100% 識別垃圾幣。最重要的是培養自己的判斷力：問「這個幣解決什麼問題？」如果答不出來，可能就是垃圾幣。
{% endhint %}

## 投資原則

### 避免垃圾幣

| 原則 | 說明 |
|------|------|
| 不懂不投 | 搞不懂就不要投 |
| 只投能虧 | 只投能承受歸零的錢 |
| 做研究 | 不要只聽別人說 |
| 問問題 | 價值在哪？團隊是誰？|
| 小心 FOMO | 不要因為「怕錯過」而買 |

### 心態調整

```
面對垃圾幣的正確心態：

錯誤心態：
- 「別人賺了，我也要」
- 「跌了 90%，應該會反彈」
- 「這次不一樣」

正確心態：
- 大部分都會歸零
- 賺錢的是少數
- 倖存者偏差
- 把它當樂透，不是投資
```

{% hint style="danger" %}
**垃圾幣歸零是常態**

根據統計，大約 95% 以上的代幣最終會歸零或接近歸零。買垃圾幣前，請確保你能承受全額損失。
{% endhint %}
