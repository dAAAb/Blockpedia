# 分叉幣

### 區塊鏈可以「分裂」——當社群對發展方向有分歧，一條鏈可能變成兩條。分叉出來的新幣，就叫分叉幣。比特幣、以太坊都有過分叉，產生了新的幣種。

---

## 什麼是分叉？

| 概念 | 說明 |
|------|------|
| 分叉 | 區塊鏈的分裂 |
| 原因 | 社群分歧、升級需求 |
| 結果 | 產生新的區塊鏈和代幣 |
| 歷史 | 分叉前的紀錄共享 |

---

## 分叉的類型

### 硬分叉 vs 軟分叉

```
兩種分叉的區別：

軟分叉（Soft Fork）：
- 向後相容
- 舊節點仍可運作
- 不會產生新幣
- 例如：隔離見證（SegWit）

硬分叉（Hard Fork）：
- 不向後相容
- 舊節點必須升級
- 可能產生新幣
- 例如：比特幣現金（BCH）

圖示：
軟分叉：  ─────────────────→ 同一條鏈
         升級

硬分叉：  ─────────┬────────→ 原鏈
                  │
                  └────────→ 新鏈（分叉幣）
```

### 分叉原因

| 原因 | 說明 | 例子 |
|------|------|------|
| 技術分歧 | 對技術方向不同意 | BCH vs BTC |
| 重大事件 | 回應駭客或漏洞 | ETH vs ETC |
| 升級需求 | 新功能需要硬分叉 | 正常升級 |
| 社群分裂 | 治理和願景衝突 | BSV vs BCH |

## 重要分叉案例

### 比特幣分叉

```
比特幣的分叉歷史：

2017 年區塊大小之爭：

原因：
- 比特幣交易擁堵
- 擴容方案分歧
- 大區塊 vs 閃電網路

結果：
比特幣（BTC）────┬────→ BTC（維持 1MB）
2017/8         │
               └────→ BCH（8MB 區塊）

後續分叉：
BCH ────┬────→ BCH
2018/11 │
        └────→ BSV（Craig Wright）

其他分叉：
- Bitcoin Gold（BTG）
- Bitcoin Diamond（BCD）
- 數十種分叉幣...
```

### 以太坊分叉

| 事件 | 說明 |
|------|------|
| 背景 | 2016 年 The DAO 駭客事件 |
| 損失 | 約 360 萬 ETH 被盜 |
| 決定 | 硬分叉回滾駭客交易 |
| 結果 | ETH（回滾）vs ETC（不回滾）|

{% hint style="info" %}
**以太經典的堅持**

以太經典（ETC）社群認為「代碼即法律」，即使被駭也不應該回滾。這是區塊鏈不可變性的極端立場。
{% endhint %}

## 分叉幣的價值

### 價值來源

| 因素 | 說明 |
|------|------|
| 社群支持 | 有多少人認可 |
| 算力/質押 | 網路安全性 |
| 開發活動 | 持續改進 |
| 使用場景 | 實際應用 |
| 交易所支持 | 流動性 |

### 分叉幣的困境

```
大多數分叉幣面臨的問題：

認同危機：
- 不被視為「正統」
- 品牌價值較低
- 社群規模較小

技術困境：
- 開發資源不足
- 算力/質押較少
- 安全性較低

市場困境：
- 交易所支持有限
- 流動性不足
- 價格持續下跌
```

## 持有者須知

### 分叉時會發生什麼

| 情況 | 說明 |
|------|------|
| 自動獲得 | 持有原幣自動獲得分叉幣 |
| 快照時間 | 以特定區塊高度為準 |
| 領取方式 | 依錢包和交易所而定 |
| 風險 | 可能有詐騙分叉 |

### 注意事項

| 注意 | 說明 |
|------|------|
| 私鑰安全 | 不要為了領幣洩露私鑰 |
| 詐騙分叉 | 很多分叉是騙局 |
| 稅務問題 | 分叉幣可能有稅務影響 |
| 價值評估 | 大多數分叉幣會歸零 |

{% hint style="warning" %}
**分叉幣詐騙**

2017-2018 年，大量假分叉幣出現。有些要求你輸入私鑰來「領取」分叉幣，這是典型的詐騙手法。
{% endhint %}

## 分叉幣現況

### 主要分叉幣

| 分叉幣 | 原鏈 | 現況 |
|--------|------|------|
| BCH | BTC | 仍有市值 |
| ETC | ETH | 持續運作 |
| BSV | BCH | 爭議極大 |
| BTG | BTC | 大幅下跌 |

### 投資建議

{% hint style="danger" %}
**高風險警告**

絕大多數分叉幣都大幅跑輸原鏈。除非你有特殊理由支持某個分叉的願景，否則持有原鏈通常是更好的選擇。
{% endhint %}
