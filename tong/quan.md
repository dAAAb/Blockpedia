# 證券幣

### 證券幣是「合規版」的代幣——它代表真實資產的權益，像股票一樣受到監管。這意味著更多保護，但也更多限制。它是區塊鏈與傳統金融的交會點。

---

## 什麼是證券幣？

| 概念 | 說明 |
|------|------|
| 英文 | Security Token |
| 定義 | 代表證券權益的代幣 |
| 監管 | 受證券法規約束 |
| 權益 | 股權、債權、資產份額 |

## 證券幣 vs 應用幣

### 關鍵區別

```
證券幣 vs 應用幣：

證券幣（Security Token）：
- 目的：投資獲利
- 價值：來自底層資產
- 監管：嚴格，需合規
- 交易：限於合格投資者
- 例子：代幣化股票、房地產代幣

應用幣（Utility Token）：
- 目的：使用平台服務
- 價值：來自使用需求
- 監管：較寬鬆
- 交易：較自由
- 例子：ETH、BNB、LINK
```

### Howey 測試

| 條件 | 證券幣 | 應用幣 |
|------|--------|--------|
| 金錢投資 | ✓ | ✓ |
| 共同事業 | ✓ | 可能 |
| 期望獲利 | ✓ 主要目的 | 次要目的 |
| 依賴他人努力 | ✓ | 可能 |

{% hint style="info" %}
**Howey 測試**

美國 SEC 用 Howey 測試判斷什麼是證券。如果一個代幣符合這四個條件，就可能被認定為證券，需要合規發行。
{% endhint %}

## 證券幣的類型

### 常見類型

```
證券幣的分類：

1. 股權代幣
   - 代表公司股份
   - 享有股東權益
   - 例：代幣化股票

2. 債權代幣
   - 代表債務權益
   - 享有利息收入
   - 例：代幣化債券

3. 資產代幣
   - 代表實物資產
   - 例：房地產、藝術品

4. 收益分享代幣
   - 分享項目收益
   - 不一定有股權
```

### 代表案例

| 類型 | 案例 | 說明 |
|------|------|------|
| 房地產 | RealT | 代幣化房產 |
| 股權 | tZero | 證券交易平台 |
| 藝術品 | Masterworks | 藝術品份額 |

### 合規要求

```
證券幣的合規要求：

發行端：
- KYC/AML 驗證
- 投資者資格審查
- 合格投資者限制
- 資訊揭露要求

交易端：
- 在合規交易所交易
- 轉讓限制
- 報告義務

好處：
- 投資者保護
- 法律明確性
- 機構投資者可參與

壞處：
- 門檻高
- 流動性受限
- 成本高
```

## STO vs ICO

### 比較分析

| 項目 | ICO | STO |
|------|-----|-----|
| 監管 | 大多不合規 | 合規發行 |
| 投資者 | 任何人 | 合格投資者 |
| 門檻 | 低 | 高 |
| 保護 | 幾乎沒有 | 法律保護 |
| 成本 | 低 | 高 |
| 流動性 | 高 | 較低 |

{% hint style="warning" %}
**STO 不一定更安全**

雖然 STO 有監管，但不代表項目一定會成功或代幣一定會增值。合規只是基本門檻，投資風險仍然存在。
{% endhint %}

## 優缺點分析

### 優點

| 優點 | 說明 |
|------|------|
| 資產代幣化 | 讓不流動資產可交易 |
| 24/7 交易 | 區塊鏈不休市 |
| 全球流通 | 跨境交易更方便 |
| 透明度 | 鏈上可追蹤 |
| 合規 | 機構可參與 |

### 缺點

| 缺點 | 說明 |
|------|------|
| 流動性低 | 市場還小 |
| 門檻高 | 合格投資者限制 |
| 成本高 | 合規成本 |
| 複雜性 | 法規複雜 |
