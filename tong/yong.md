# 應用幣

### 應用幣就像遊樂園的代幣——你買它不是為了投資，而是為了使用園區內的設施。在區塊鏈世界，應用幣讓你使用平台服務、支付手續費、參與治理。

---

## 什麼是應用幣？

| 概念 | 說明 |
|------|------|
| 英文 | Utility Token |
| 定義 | 用於特定平台功能的代幣 |
| 用途 | 支付服務、獲取功能 |
| 特性 | 非投資性質（理論上）|

---

## 應用幣 vs 證券幣

### 關鍵區別

```
應用幣 vs 證券幣：

應用幣（Utility Token）：
- 用途：使用平台服務
- 價值來源：實際使用需求
- 監管：較寬鬆（理論上）
- 例子：ETH、BNB、LINK

證券幣（Security Token）：
- 用途：投資獲利
- 價值來源：項目收益分配
- 監管：嚴格，需合規
- 例子：代幣化股票

灰色地帶：
很多「應用幣」實際上被當作投資品
監管機構可能認定為證券
```

### Howey 測試

| 條件 | 說明 |
|------|------|
| 金錢投資 | 用錢購買 |
| 共同事業 | 投入共同項目 |
| 期望獲利 | 期待價值上漲 |
| 依賴他人 | 獲利來自他人努力 |

{% hint style="info" %}
**SEC 的立場**

美國 SEC 認為大多數 ICO 代幣都是證券。即使項目方說是「應用幣」，如果投資者主要是為了獲利而非使用，就可能被認定為證券。
{% endhint %}

## 常見用途

### 應用幣的功能

```
應用幣的典型用途：

1. 支付功能
   - 支付平台手續費
   - 購買服務
   - 例：ETH 支付 Gas 費

2. 訪問權限
   - 解鎖特定功能
   - 獲得會員服務
   - 例：某些 DeFi 協議

3. 治理投票
   - 參與項目決策
   - 提案和投票
   - 例：UNI、AAVE

4. 質押獎勵
   - 質押獲得收益
   - 維護網路安全
   - 例：ETH 質押

5. 折扣優惠
   - 持有代幣享折扣
   - 例：BNB 交易折扣
```

### 代表案例

| 代幣 | 平台 | 用途 |
|------|------|------|
| ETH | Ethereum | Gas 費、質押 |
| BNB | Binance | 手續費折扣 |
| LINK | Chainlink | 支付預言機服務 |
| UNI | Uniswap | 治理投票 |
| FIL | Filecoin | 支付存儲費用 |

## 代幣經濟學

### 價值驅動因素

| 因素 | 說明 |
|------|------|
| 使用需求 | 平台越多人用，需求越大 |
| 供應機制 | 通膨、通縮、燒毀 |
| 質押需求 | 需要質押才能參與 |
| 投機需求 | 價格預期 |

### 健康的代幣經濟

```
好的應用幣設計：

需求端：
- 使用平台必須持有代幣
- 功能有真實價值
- 用戶願意付費使用

供應端：
- 合理的發行量
- 明確的釋放時程
- 可能有銷毀機制

平衡：
- 使用需求 > 投機需求
- 長期價值 > 短期炒作
```

## 投資考量

### 評估應用幣

| 評估項目 | 問題 |
|----------|------|
| 實際用途 | 代幣有真實使用場景嗎？|
| 使用必要性 | 必須用這個代幣嗎？|
| 平台採用 | 有多少人真的在用？|
| 競爭優勢 | 為什麼選這個平台？|
| 代幣經濟 | 供需設計合理嗎？|

### 風險提示

| 風險 | 說明 |
|------|------|
| 使用量不足 | 沒人用，代幣沒價值 |
| 競爭淘汰 | 平台被取代 |
| 監管風險 | 被認定為證券 |
| 代幣經濟失敗 | 設計不當導致崩盤 |

{% hint style="warning" %}
**應用不等於價值**

代幣有應用場景，不代表它會增值。很多應用幣的使用量很大，但價格仍然下跌。代幣價值和平台價值不一定同步。
{% endhint %}
