# 傳銷幣

### 「拉一個人頭賺 10%」、「下線越多賺越多」——這種話術你聽過嗎？當區塊鏈遇上傳銷，就產生了傳銷幣。它們用加密貨幣包裝老鼠會，讓受害者變成加害者。

---

## 什麼是傳銷幣？

| 概念 | 說明 |
|------|------|
| 定義 | 以傳銷模式推廣的加密貨幣 |
| 特徵 | 獲利來自拉人頭而非投資 |
| 本質 | 加密貨幣包裝的龐氏騙局 |
| 結局 | 必然崩盤 |

---

## 傳銷幣的特徵

### 識別要點

```
傳銷幣的典型特徵：

收益結構：
- 推薦獎金（拉人頭）
- 多層級分潤
- 團隊業績獎勵
- 固定高回報承諾

推廣方式：
- 強調「被動收入」
- 「越早加入越賺」
- 「錯過就沒機會」
- 要求分享給親友

運作模式：
- 入場需要購買代幣
- 收益來自新人投入
- 沒有實際產品或服務
- 團隊抽成比例極高
```

### 檢查清單

| 檢查項目 | 傳銷幣特徵 |
|----------|------------|
| 收益來源 | 來自新會員，不是實際營運 |
| 獎勵制度 | 複雜的多層級結構 |
| 推廣重點 | 強調拉人，不是產品 |
| 入場門檻 | 必須先買幣才能參與 |
| 提領限制 | 複雜的提領規則 |

{% hint style="danger" %}
**傳銷的本質**

不管包裝多漂亮，傳銷的數學本質不會改變：它需要不斷有新人加入才能維持。當新人不夠，就會崩盤。底層的人永遠是輸家。
{% endhint %}

## 常見話術

### 推銷套路

| 話術 | 真相 |
|------|------|
| 「穩定高收益」| 沒有穩定高收益的投資 |
| 「區塊鏈技術」| 通常只是包裝 |
| 「錯過就沒了」| 製造緊迫感 |
| 「朋友介紹的」| 朋友也是受害者 |
| 「提領很方便」| 早期可能是，後期不會 |

### 心理操控

```
傳銷幣的心理操控術：

1. 社交壓力
   - 朋友、家人介紹
   - 「不支持就是不信任」
   - 群體認同感

2. 貪婪誘惑
   - 展示「成功案例」
   - 豪車、名錶、旅遊
   - 「你也可以」

3. 沉沒成本
   - 已經投入就不想承認錯誤
   - 「再拉幾個人就回本」
   - 越陷越深

4. 資訊隔離
   - 只聽正面消息
   - 質疑者被踢出群組
   - 「負面消息是競爭對手散布的」
```

## 知名案例

### 經典傳銷幣

| 案例 | 手法 | 結果 |
|------|------|------|
| OneCoin | 假的加密貨幣，根本沒有區塊鏈 | 損失超過 40 億美元 |
| BitConnect | 借貸 + 傳銷 | 2018 年崩盤 |
| PlusToken | 錢包 + 傳銷 | 損失超過 20 億美元 |

### OneCoin 案例

```
OneCoin 騙局始末：

2014-2017：
- 創辦人：Ruja Ignatova
- 宣稱：「下一個比特幣」
- 真相：根本沒有區塊鏈

運作方式：
- 購買「教育套餐」
- 獲得「代幣」
- 推薦他人獲得獎金
- 代幣只能在內部交易

結果：
- 全球數百萬受害者
- 損失超過 40 億美元
- 創辦人 2017 年消失，至今在逃
- FBI 最想追捕的嫌犯之一
```

## 法律風險

### 傳銷的法律問題

| 地區 | 規定 |
|------|------|
| 台灣 | 多層次傳銷管理法 |
| 中國 | 傳銷為刑事犯罪 |
| 美國 | 金字塔騙局違法 |

### 參與者的風險

| 風險 | 說明 |
|------|------|
| 財務損失 | 大多數參與者虧損 |
| 法律責任 | 可能被視為共犯 |
| 人際關係 | 傷害親友關係 |
| 心理創傷 | 發現被騙的打擊 |

{% hint style="warning" %}
**參與者也可能有法律責任**

在某些情況下，即使你也是受害者，如果你積極拉人頭，可能也要承擔法律責任。
{% endhint %}

## 如何保護自己

### 識別方法

| 方法 | 說明 |
|------|------|
| 問收益來源 | 錢從哪來？|
| 看獎勵結構 | 是否強調拉人？|
| 查詢項目 | 搜尋「XXX + 騙局」|
| 諮詢專業 | 問懂的人 |

### 如果已經參與

| 建議 | 說明 |
|------|------|
| 停止投入 | 不要再投錢 |
| 不要拉人 | 不要害別人 |
| 收集證據 | 保留交易紀錄 |
| 尋求幫助 | 報警或諮詢律師 |
