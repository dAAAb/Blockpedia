# 破發

### 破發是指加密貨幣的市場價格跌破首次發行價格（ICO/IEO/IDO 價格）。當你在公開發售時買入，結果上市後價格比你買的還低，這就是破發。對早期投資者來說，這是最不想看到的結果。

---

## 什麼是破發？

```
ICO 發行價：$1.00
上市開盤價：$2.00  ← 還好
幾天後：$0.50     ← 破發！

你的虧損：50%
```

{% hint style="info" %}
**為什麼叫「破發」？**

「發」是發行價的意思。「破發」就是「跌破發行價」的簡稱。這個詞源自股票市場，後來在幣圈也普遍使用。
{% endhint %}

---

## 破發的原因

| 原因 | 說明 |
|------|------|
| 估值過高 | 發行價就是泡沫價 |
| 市場環境差 | 熊市來了誰都跑不掉 |
| 項目不行 | 沒有實際產品或用戶 |
| 解鎖拋售 | 早期投資者和團隊賣幣 |
| 缺乏流動性 | 交易所上市後沒人接盤 |

---

## 2018 年：破發年

2018 年是幣圈「破發」最慘烈的一年：

| 統計 | 數據 |
|------|------|
| ICO 項目數 | 數千個 |
| 破發比例 | 超過 80% |
| 歸零比例 | 超過 50% |

很多人在 2017 年牛市尾聲參與 ICO，結果 2018 年全部破發，血本無歸。

{% hint style="danger" %}
**ICO 的教訓**

2017-2018 年的 ICO 狂熱告訴我們：發行價不等於價值。很多項目從一開始就是為了圈錢，根本沒打算做事。
{% endhint %}

---

## 如何避免被破發

| 策略 | 說明 |
|------|------|
| 研究項目 | 團隊背景、產品進度、代幣經濟模型 |
| 注意估值 | FDV（完全稀釋估值）是否合理 |
| 解鎖計畫 | 團隊和投資者何時能賣幣 |
| 市場時機 | 牛市尾聲要特別小心 |
| 分散風險 | 不要把全部資金投入一個項目 |

---

## 破發後怎麼辦？

| 選項 | 考量 |
|------|------|
| 割肉 | 及時止損，把資金用在更好的地方 |
| 持有 | 如果相信項目長期價值 |
| 加倉 | 如果認為是好的買入機會（風險極高）|

{% hint style="warning" %}
**沉沒成本謬誤**

很多人因為「已經虧了」就不願意賣出，結果越虧越多。記住：你買入的價格不影響這個幣未來的價值。
{% endhint %}
