# 接盤

### 接盤是加密貨幣圈的俚語，指在價格高點買入別人正在賣出的幣。就像打籃球時「接盤」——你接到的球可能是燙手山芋。在幣圈，「接盤俠」通常是買在最高點、然後眼睜睜看著價格下跌的人。

---

## 什麼是接盤？

```
莊家/早期持有者
       │
       │ 在高點賣出
       ↓
  ────────────
       ↑
       │ 在高點買入
       │
    接盤俠（你）
```

{% hint style="info" %}
**誰是接盤俠？**

通常是：
- FOMO 追高的散戶
- 聽信「喊單」的跟風者
- 相信「這次不一樣」的新手
{% endhint %}

---

## 接盤的形成過程

### 典型劇本

1. **早期**：聰明錢低價買入
2. **上漲**：消息開始傳播，價格上漲
3. **高潮**：媒體報導、KOL 喊單，FOMO 情緒蔓延
4. **出貨**：聰明錢開始賣出
5. **接盤**：散戶蜂擁而入，接下聰明錢的籌碼
6. **崩盤**：價格暴跌，接盤俠套牢

---

## 如何避免成為接盤俠

| 警訊 | 說明 |
|------|------|
| 價格已經漲很多 | 好的買點通常不會讓你 FOMO |
| 到處都在討論 | 當計程車司機都在談，可能已經太晚 |
| KOL 瘋狂喊單 | 他們可能正在出貨 |
| 只有利好消息 | 沒人提風險是危險信號 |
| 成交量暴增 | 可能是大戶在出貨 |

{% hint style="warning" %}
**經典警句**

「當你覺得穩賺不賠的時候，你可能就是那個接盤的人。」
{% endhint %}

---

## 接盤俠的心理

### 1. FOMO（錯失恐懼）

「漲這麼多了，不買就錯過了！」

### 2. 從眾心理

「大家都在買，應該沒問題吧？」

### 3. 過度自信

「我會是第一個跑的，不會被套。」

### 4. 選擇性偏誤

「我只看到別人賺錢，看不到虧錢的。」

---

## 接盤後的選擇

| 選項 | 分析 |
|------|------|
| 割肉止損 | 如果基本面不好，趁早離場 |
| 死守等待 | 如果相信長期價值，但可能等很久 |
| 加倉攤平 | 風險最高，可能越陷越深 |

{% hint style="danger" %}
**不要越接越多**

接盤後最危險的做法是「攤平成本」——在下跌過程中不斷買入。這可能讓你虧損更大。
{% endhint %}

---

## 其實每個人都可能是接盤俠

```
你買入 → 你覺得是投資
別人買入（在你之後）→ 你覺得他是接盤
你賣出 → 你覺得是獲利了結
別人賣給你 → 他覺得你是接盤俠
```

市場就是這樣運作的。重要的是：確保你買入的理由是基於研究，而不是 FOMO。

{% hint style="success" %}
**最好的心態**

接受自己可能是接盤俠的可能性，做好風險管理。控制投入金額，設定止損，不要把全部身家押在一個幣上。
{% endhint %}
