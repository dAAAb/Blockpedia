# 詐神

### 詐神是加密貨幣圈對專業騙子的戲稱。這些人善於包裝自己，用假的成功故事、誇大的履歷、和精心設計的話術來騙取投資者的信任和金錢。他們不是普通的詐騙犯，而是「神」級的騙術高手。

---

## 詐神的特徵

| 特徵 | 表現 |
|------|------|
| 豪華人設 | 名校背景、大公司經歷（多為假的）|
| 炫富展示 | 名車、豪宅、奢侈品照片 |
| 成功故事 | 「從零到百萬」的勵志故事 |
| 專業術語 | 說一堆聽不懂的技術名詞 |
| 名人背書 | 和「大佬」的合照（可能是P的）|
| 緊迫感 | 「機會難得」「限時優惠」|

{% hint style="warning" %}
**真正的成功人士**

真正的成功人士通常不需要到處炫富、不需要你的錢。如果有人特別想證明自己多成功，反而要小心。
{% endhint %}

## 詐神的常見騙術

### 1. 帶單騙局

```
詐神：「跟著我買這個幣，穩賺！」
你買入
詐神出貨給你
價格暴跌
詐神：「市場風險，自負盈虧」
```

### 2. 私募騙局

```
詐神：「我有內部額度，原價 $1，市價會到 $10」
你付錢「投資」
代幣從未上市 / 上市即破發
詐神消失
```

### 3. 課程騙局

```
詐神：「我的交易課程價值百萬」
你花高價購買
課程內容空洞無物
詐神：「學費嘛，都是這樣」
```

### 4. 社群騙局

```
詐神建立付費社群
承諾「alpha」資訊
實際上沒有任何價值
或者根本是詐騙推薦
```

## 如何識別詐神

| 警訊 | 說明 |
|------|------|
| 無法查證的背景 | Google 找不到任何證據 |
| 只有成功沒有失敗 | 正常人都有失敗經歷 |
| 過度活躍於社交媒體 | 花大量時間「建立影響力」|
| 收費才能加入 | 正常的資訊不需要付費 |
| 承諾保證收益 | 沒有人能保證收益 |

---

## 保護自己的方法

| 方法 | 說明 |
|------|------|
| 查證背景 | Google、LinkedIn、區塊鏈瀏覽器 |
| 問專業問題 | 真正懂的人能回答深入問題 |
| 不要 FOMO | 真正的好機會不會催你 |
| 不投你不懂的 | 聽不懂就不要投 |
| 小額試水 | 先用小錢測試，不要一次投入大量資金 |

{% hint style="success" %}
**最好的防禦**

記住這句話：「如果聽起來好得不像是真的，那它可能就不是真的。」

在幣圈，懷疑精神是最好的保護。
{% endhint %}
