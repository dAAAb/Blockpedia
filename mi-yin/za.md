# 砸盤

### 砸盤是加密貨幣圈形容大量拋售導致價格暴跌的詞。就像有人把一塊大石頭「砸」進水裡，濺起巨大的水花。在幣圈，砸盤通常是鯨魚、項目方或大戶一次性賣出大量籌碼，導致市場崩盤。

---

## 什麼是砸盤？

```
正常交易：
價格 ──────────

砸盤：
價格 ──────╲
            ╲
             ╲
              ╲ ↓ 暴跌
               ╲
```

{% hint style="info" %}
**砸盤 vs 自然下跌**

- **砸盤**：短時間內突然暴跌，伴隨巨大成交量
- **自然下跌**：緩慢下跌，成交量正常
{% endhint %}

---

## 誰在砸盤？

| 砸盤者 | 動機 |
|--------|------|
| 鯨魚 | 獲利了結、換倉 |
| 項目方 | 套現跑路、資金需求 |
| 機構 | 調整倉位、止損 |
| 駭客 | 盜幣後快速出貨 |
| 清算 | 槓桿倉位被強制平倉 |

---

## 砸盤的連鎖反應

```
大戶砸盤
   ↓
價格暴跌
   ↓
觸發槓桿清算
   ↓
更多強制賣單
   ↓
價格繼續跌
   ↓
散戶恐慌賣出
   ↓
價格崩盤
```

{% hint style="danger" %}
**連環清算**

2021 年 5 月 19 日，比特幣一天內跌超 30%。原因之一就是高槓桿倉位的連環清算——每次清算都產生新的賣壓，形成惡性循環。
{% endhint %}

---

## 歷史上的著名砸盤

### Mt. Gox 託管人拋售（2018）

Mt. Gox 破產後，託管人多次在公開市場賣出比特幣清償債務，造成市場大跌。

### Luna 崩盤（2022）

UST 脫鉤後，Luna 被大量鑄造並砸盤，價格從 $80+ 跌到幾乎歸零。

### FTX 破產（2022）

FTX 破產導致關聯代幣（FTT、SOL 等）被大量拋售。

---

## 如何識別砸盤

| 跡象 | 說明 |
|------|------|
| 異常大單 | 出現遠超正常規模的賣單 |
| 成交量暴增 | 比平常高出數倍 |
| 價格快速下跌 | 幾分鐘內跌幅數個百分點 |
| 鏈上轉帳 | 大額代幣轉入交易所 |

---

## 砸盤時該怎麼辦？

### 如果你持有

| 選項 | 考量 |
|------|------|
| 持有不動 | 如果相信長期價值，波動只是噪音 |
| 割肉離場 | 如果認為是趨勢性下跌 |
| 加倉 | 如果認為是超跌機會（風險高）|

### 如果你場外

| 選項 | 考量 |
|------|------|
| 抄底 | 可能是好機會，也可能是陷阱 |
| 觀望 | 等情況明朗再行動 |
| 做空 | 如果你有這個能力和風險承受度 |

{% hint style="warning" %}
**不要試圖接飛刀**

砸盤時抄底很危險——你不知道底在哪裡。很多人在「抄底」時抄在半山腰。
{% endhint %}

---

## 砸盤與監管

某些類型的砸盤可能涉及市場操縱：

- **拉高出貨**：先拉高價格，再砸盤出貨
- **內幕交易**：提前獲知利空消息後砸盤
- **協調砸盤**：多方合作打壓價格

在傳統金融市場，這些行為是違法的。但加密貨幣市場監管較少，這類行為更常見。
