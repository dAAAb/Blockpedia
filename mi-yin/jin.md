# 資金盤

### 資金盤是一種金融騙局，本質上是龐氏騙局（Ponzi Scheme）的變體。它用後來投資者的錢支付早期投資者的「收益」，製造出「穩定獲利」的假象。當新資金進入速度跟不上支出時，整個系統就會崩潰。

---

## 資金盤的運作方式

```
投資者 A 投入 $1000
         ↓
「回報」$200 給 A（來自 B 的錢）
         ↑
投資者 B 投入 $1000
         ↓
「回報」$200 給 B（來自 C 的錢）
         ↑
投資者 C 投入 $1000
         ↓
        ...

當沒有新的 D、E、F 進來時：
        崩盤！
```

{% hint style="danger" %}
**資金盤的數學必然**

資金盤需要不斷增長才能維持。當增長停止，它就會崩潰。這不是「可能」崩潰，而是「必然」崩潰。
{% endhint %}

---

## 資金盤的特徵

| 特徵 | 說明 |
|------|------|
| 超高回報 | 承諾 10%、20% 甚至更高的月回報 |
| 穩定收益 | 聲稱「穩賺不賠」 |
| 拉人頭 | 鼓勵你邀請更多人加入 |
| 模糊商業模式 | 說不清錢從哪來 |
| 限制提款 | 設置各種提款門檻 |
| 豪華包裝 | 炫富、名車、豪宅照片 |

---

## 幣圈常見的資金盤類型

### 1. 「挖礦」類

聲稱有高效的挖礦設備，實際上根本沒有。

### 2. 「量化交易」類

聲稱有神奇的交易策略，實際上只是拿新錢付舊帳。

### 3. 「質押」類

假借 DeFi 名義，承諾超高年化收益。

### 4. 「遊戲」類

如 Fomo3D 類遊戲，最後一個入場者拿走大獎。

## 著名的加密貨幣資金盤

| 案例 | 規模 | 結果 |
|------|------|------|
| BitConnect | $25 億+ | 2018 年崩盤 |
| PlusToken | $20 億+ | 創辦人被捕 |
| OneCoin | $40 億+ | 創辦人被捕 |

{% hint style="warning" %}
**「這次不一樣」**

每個資金盤都會說自己「不一樣」，有「真正的商業模式」。但如果說不清楚錢從哪來，它就是資金盤。
{% endhint %}

## 如何識別資金盤

| 問題 | 正常項目 | 資金盤 |
|------|----------|--------|
| 收益來源？ | 清楚說明 | 模糊不清 |
| 年化多少？ | 合理（10-30%） | 離譜（100%+） |
| 能否提款？ | 隨時可以 | 各種限制 |
| 商業模式？ | 透明公開 | 神秘兮兮 |
| 拉人獎勵？ | 沒有或很少 | 超高獎勵 |

---

## 為什麼人們會被騙？

### 1. 貪婪

「這麼高的回報，我要賺！」

### 2. 僥倖心理

「我不是最後一個，我能跑掉。」

### 3. 社交證明

「我朋友賺錢了，應該沒問題。」

### 4. 資訊不對稱

真的不知道這是騙局。

{% hint style="success" %}
**最簡單的判斷方法**

如果一個投資機會：
1. 承諾超高回報
2. 說不清錢從哪來
3. 鼓勵你拉人

那它 99% 是資金盤。
{% endhint %}
