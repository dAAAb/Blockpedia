# DYOR

## DYOR 是 "Do Your Own Research" 的縮寫，意思是「自己做功課」。這是加密貨幣圈最重要的建議，也常被濫用來推卸責任——「我告訴你這幣會漲 100 倍，但記得 DYOR 喔！」

有人在推特上喊：「這個幣要起飛了！早買早享受！」

底下有人問：「這幣是幹嘛的？」

回覆：「DYOR。」

這就是加密貨幣圈的日常。

### DYOR 的起源

DYOR 不是加密貨幣圈發明的，但在這裡發揚光大。

原因很簡單：這個市場**太野了**。

- 沒有監管（或監管很弱）
- 詐騙橫行
- 項目可以一夜之間消失
- 「投資建議」隨處可見，但沒人負責

在這種環境下，「自己做研究」從好心的建議，變成了**生存法則**。

### DYOR 真正的意思

**本意**：在投資之前，你應該親自了解這個項目，而不是只聽別人說。

**具體包括**：

| 研究項目 | 要看什麼 |
|----------|----------|
| 白皮書 | 項目要解決什麼問題？技術方案是什麼？ |
| 團隊 | 是誰在做？有沒有真實身分和過往經歷？ |
| 代幣經濟 | 代幣怎麼分配？團隊持有多少？有沒有鎖倉？ |
| 社群 | Discord/Telegram 活躍嗎？有沒有真實用戶？ |
| 程式碼 | GitHub 有沒有活動？有沒有審計？ |
| 競爭對手 | 市場上有沒有類似項目？這個有什麼優勢？ |

### DYOR 的正確用法

**好的 DYOR**：

> 「我覺得這個項目不錯，但投資有風險，建議你自己做研究再決定。」

這是在說：「我分享我的看法，但你應該有自己的判斷。」

**壞的 DYOR**：

> 「這個幣下個月要漲 10 倍！趕快買！（不是投資建議，DYOR）」

這是在說：「我要喊單，但加個 DYOR 就不用負責了吧？」

{% hint style="warning" %}
很多詐騙項目和 KOL 會用「DYOR」當免責聲明。就像香菸盒上的警語一樣——它在那裡，但不代表這個東西對你好。
{% endhint %}

### 如何真正做 DYOR

#### 第一步：基本面檢查

問自己這些問題：

- 這個項目解決什麼問題？
- 這個問題真的需要區塊鏈嗎？
- 為什麼這個項目能成功？
- 代幣有什麼用途？不持有代幣能用這個產品嗎？

#### 第二步：團隊背景

- 創辦人是誰？有沒有 LinkedIn？
- 他們以前做過什麼？
- 是不是匿名團隊？（匿名不一定是壞事，但風險更高）

#### 第三步：代幣經濟學

```
危險訊號 🚩：
- 團隊持有 > 50% 代幣
- 沒有鎖倉期
- 代幣分配不透明
- 「買幣就能賺錢」是唯一用途
```

#### 第四步：社群和媒體

- Discord/Telegram 有沒有真實討論？
- 還是全是「🚀🚀🚀 月球見」？
- 有沒有獨立媒體報導？

#### 第五步：智能合約

- 有沒有審計報告？
- 合約是不是開源的？
- 有沒有可疑的函數（如 mint 無上限、owner 可以凍結資產）？

### DYOR 的限制

說實話，大多數人沒有能力真正「做研究」。

- **讀白皮書**？大部分人讀不懂技術細節
- **看程式碼**？需要專業知識
- **分析代幣經濟**？需要金融背景

這就是為什麼「DYOR」有時候是一個**不公平的要求**。

它把所有責任推給投資者，忽略了：

- 項目方可能在說謊
- 資訊可能是假的
- 詐騙可能設計得很精密

{% hint style="info" %}
DYOR 不是萬能的。即使你做了研究，也可能被精心設計的詐騙騙過。保持謙虛，不要投入無法承受損失的金額。
{% endhint %}

### DYOR vs NFA

你常會看到兩個縮寫一起出現：

- **DYOR**：Do Your Own Research（自己做研究）
- **NFA**：Not Financial Advice（這不是投資建議）

當有人說「買這個幣！DYOR, NFA」時，他的意思是：

> 「我在喊單，但如果你虧錢，不關我的事。」

### 為什麼 DYOR 很重要？

即使 DYOR 常被濫用，它的核心概念仍然重要：

1. **這是你的錢**：別人不會為你的損失負責
2. **資訊不對稱**：內幕人士知道的永遠比你多
3. **激勵不一致**：推薦你買幣的人可能想把幣賣給你
4. **市場不成熟**：監管薄弱，詐騙容易

### DYOR 檢查清單

在投資任何加密貨幣前：

- [ ] 我了解這個項目要解決什麼問題
- [ ] 我知道團隊是誰（或理解匿名的風險）
- [ ] 我看過代幣分配，知道誰持有多少
- [ ] 我查過這個項目有沒有被報導過詐騙
- [ ] 我用的是「虧光也不心疼」的錢
- [ ] 我沒有因為 FOMO 而急著買入
- [ ] 我有離場計畫

### DYOR 的反面教材

**Luna/UST 崩盤（2022）**：

很多人「做了研究」，看到：
- 知名創辦人 Do Kwon
- 大型機構投資
- 「穩定幣」的安全感
- 20% APY 的收益

但他們沒看到（或選擇忽略）：
- 演算法穩定幣的致命缺陷
- 死亡螺旋的可能性
- 過度膨脹的承諾

結果：數百億美元蒸發。

**教訓**：DYOR 不只是「看好消息」，更要主動尋找**壞消息**和**風險**。

{% hint style="success" %}
真正的 DYOR 不是為了說服自己買入，而是為了找到不買的理由。如果你找遍了還是想買——那才是經過深思熟慮的決定。
{% endhint %}
