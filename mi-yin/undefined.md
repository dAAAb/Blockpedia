# 開車

### 「開車」是加密貨幣圈的俚語，意思是帶領別人投資。當有人說「老司機要開車了」，意思是他要分享投資標的或操作策略，邀請別人跟著一起進場。這個詞源自中國網路文化，後來在幣圈廣泛使用。

---

## 什麼是「開車」？

```
老司機（KOL/群主）：「兄弟們，上車！買 XXX 幣！」

散戶 A：「好，跟！」
散戶 B：「老司機帶帶我！」
散戶 C：「開往月球！」

然後...
可能漲 🚀
可能崩 💥
```

{% hint style="info" %}
**相關用語**

- **老司機**：經驗豐富的投資者，帶頭開車的人
- **上車**：跟著買入
- **下車**：賣出離場
- **翻車**：投資失敗虧錢
- **錯過這班車**：錯過買入時機
{% endhint %}

## 開車的風險

### 1. 老司機可能是莊家

有些「開車」其實是：

```
莊家低價買入 → 喊單「開車」→ 散戶跟進推高價格 → 莊家高價賣出 → 散戶套牢
```

### 2. 老司機可能收錢喊單

很多 KOL 的「推薦」是項目方付費的廣告。

### 3. 老司機可能也不知道

就算是真心推薦，老司機也可能看錯。市場面前，人人平等。

{% hint style="danger" %}
**翻車是常態**

幣圈有句話：「開車不一定到月球，可能直接進溝裡。」跟車要謹慎，虧錢只能自己負責。
{% endhint %}

## 如何判斷是否該上車

| 考量 | 問自己 |
|------|--------|
| 了解這個幣嗎？ | 是投資還是賭博？ |
| 老司機可信嗎？ | 過往紀錄如何？ |
| 有利益衝突嗎？ | 他是不是收了錢？ |
| 虧得起嗎？ | 最壞情況能接受嗎？ |

{% hint style="success" %}
**最安全的方式**

DYOR（Do Your Own Research）——自己做功課，自己做決定。把別人的「開車」當作參考，不是指令。
{% endhint %}
