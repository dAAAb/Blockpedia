# Fomo3D

### Fomo3D 是區塊鏈史上最「瘋狂」的遊戲之一。它公開宣稱自己是「公平的龐氏騙局」，卻吸引了大量玩家投入資金。最後的結局更是戲劇性——贏家透過堵塞整個以太坊網路來確保勝利。

## 遊戲機制

### 規則說明

```
Fomo3D 的遊戲規則：

1. 玩家用 ETH 購買「鑰匙」
2. 每次有人買鑰匙，倒計時重置
3. 倒計時結束時，最後買鑰匙的人贏得獎池
4. 鑰匙價格隨時間上漲

資金分配：
- 部分給之前的鑰匙持有者（分紅）
- 部分進入最終獎池
- 部分給團隊

本質：
- 先進場的人賺後進場的錢
- 最後一個人贏走獎池
- 公開的龐氏+博弈
```

### 為什麼會火？

| 因素 | 說明 |
|------|------|
| FOMO 心理 | 害怕錯過 |
| 分紅機制 | 早期參與有收益 |
| 巨額獎池 | 千萬美元誘惑 |
| 公開透明 | 規則寫在合約裡 |
| 博弈心理 | 賭最後一個人是自己 |

{% hint style="info" %}
**「公平的龐氏」**

Fomo3D 的開發者公開說這是龐氏騙局，但因為規則透明（寫在智能合約裡），所以是「公平的」。這種黑色幽默反而吸引了更多人。
{% endhint %}

## 戲劇性結局

### 最後贏家

```
Fomo3D 的結局：

2018 年 8 月 22 日：

1. 贏家在倒計時快結束時買入
2. 同時發送大量高 Gas 交易
3. 堵塞整個以太坊網路
4. 其他人無法買入新鑰匙
5. 倒計時結束
6. 贏家獲得約 10,469 ETH（約 300 萬美元）

這暴露了什麼問題？
- 以太坊的擴展性限制
- 「公平」遊戲可以被博弈
- 礦工/區塊順序可以被利用
```

### 教訓

| 教訓 | 說明 |
|------|------|
| 網路可以被攻擊 | 高 Gas 可以堵塞網路 |
| 時間依賴不可靠 | 區塊時間可以被操控 |
| 鏈上遊戲的局限 | 有些遊戲不適合鏈上 |

{% hint style="warning" %}
**這不是投資建議**

Fomo3D 和類似的遊戲本質上是博弈和投機。大多數參與者會輸錢。不要參與這類項目。
{% endhint %}
