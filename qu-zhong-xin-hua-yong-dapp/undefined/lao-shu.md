# 區塊鏈老鼠會

### 老鼠會（龐氏騙局）不是新東西，但區塊鏈給了它新包裝。打著「去中心化」、「智能合約」的旗號，很多項目本質上還是用後來者的錢付給先來者。了解這些手法，才能保護自己。

---

## 什麼是區塊鏈老鼠會？

| 概念 | 說明 |
|------|------|
| 定義 | 用新投資者的錢支付舊投資者回報 |
| 特徵 | 承諾高額固定回報 |
| 結局 | 必然崩盤，後期進入者損失 |

---

## 常見手法

### 包裝方式

```
區塊鏈龐氏的包裝：

常見敘事：
- 「挖礦」收益
- 「質押」高報酬
- 「套利」機器人
- 「DeFi」協議
- 「遊戲」邊玩邊賺

共同點：
- 承諾固定高回報（每日 1%、每月 30%）
- 鼓勵拉人頭
- 資金來源不透明
- 聽起來太好以至於不真實
```

### 識別特徵

| 特徵 | 說明 |
|------|------|
| 高額固定回報 | 每日/每月固定收益 |
| 推薦獎勵 | 拉人有分潤 |
| 資金池不透明 | 不知道錢去哪 |
| 團隊匿名 | 創辦人不公開 |
| 急迫感 | 「限時」、「快沒額度」|
| 複雜結構 | 讓你搞不懂的層級 |

{% hint style="danger" %}
**如果太好以至於不真實，它就不是真的**

沒有任何合法投資能夠持續提供每月 20%+ 的回報。如果有人這樣承諾你，那幾乎肯定是騙局。
{% endhint %}

## 為什麼人會上當？

### 心理因素

| 因素 | 說明 |
|------|------|
| 貪婪 | 想要高回報 |
| FOMO | 看到別人賺錢 |
| 信任 | 朋友介紹的 |
| 無知 | 不懂區塊鏈 |
| 僥倖 | 覺得自己不會是最後一個 |

### 騙局的設計

```
龐氏騙局的心理操控：

早期：
- 真的付錢（用新錢付）
- 建立信任
- 讓你帶朋友來

中期：
- 鼓勵「複投」
- 減少提領金額
- 製造提領門檻

後期：
- 找藉口延遲付款
- 「升級」、「維護」
- 關站跑路
```

{% hint style="warning" %}
**朋友介紹不代表安全**

很多人因為是朋友介紹才投資。但你的朋友可能也是受害者，只是還沒意識到。
{% endhint %}

## 如何保護自己

### 檢查清單

| 問題 | 警訊 |
|------|------|
| 收益來源？ | 說不清楚 |
| 團隊是誰？ | 匿名或假身份 |
| 有審計嗎？ | 沒有或假審計 |
| 合約開源？ | 不公開或有後門 |
| 提領限制？ | 複雜的提領規則 |

### 基本原則

| 原則 | 說明 |
|------|------|
| 做功課 | 研究項目背景 |
| 問問題 | 問到懂為止 |
| 不貪心 | 高回報=高風險 |
| 分散 | 不要全押 |
| 可承受 | 只投能虧的錢 |
