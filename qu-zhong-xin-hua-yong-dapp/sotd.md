# SOTD

### State of the DApps 是 DApp 世界的「黃頁」——在 DappRadar 等工具出現之前，它是人們發現和探索 DApp 的重要入口。

---

## 什麼是 SOTD？

| 項目 | 說明 |
|------|------|
| 全名 | State of the DApps |
| 類型 | DApp 目錄和資訊平台 |
| 創立 | 早期 DApp 生態 |
| 功能 | DApp 清單、數據 |

{% embed url="https://www.stateofthedapps.com/" %}

---

## 主要功能

### 提供的資訊

| 功能 | 說明 |
|------|------|
| DApp 目錄 | 瀏覽各類 DApp |
| 統計數據 | 活躍用戶、交易量 |
| 分類瀏覽 | 依類別探索 |
| 搜尋功能 | 尋找特定 DApp |

### 覆蓋範圍

```
SOTD 追蹤的內容：

區塊鏈：
├── Ethereum
├── EOS
├── TRON
└── 其他鏈

類別：
├── Games
├── Finance
├── Social
├── Exchanges
└── Gambling
```

## 如何使用

| 用途 | 說明 |
|------|------|
| 發現新 DApp | 瀏覽目錄 |
| 研究項目 | 查看數據 |
| 了解趨勢 | 觀察熱門類別 |

{% hint style="info" %}
**歷史意義**

SOTD 是早期 DApp 生態的重要工具。雖然現在有更多選擇（如 DappRadar），它仍是了解 DApp 生態的參考資源。
{% endhint %}
