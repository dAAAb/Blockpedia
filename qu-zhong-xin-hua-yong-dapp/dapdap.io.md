# DapDap.io

### 每次想用不同的 DApp，都要跑到不同的網站，連接錢包，很麻煩對吧？DapDap.io 想解決這個問題——把所有 DApp 整合到一個地方，讓你一站式體驗 Web3。

---

## 什麼是 DapDap.io？

| 項目 | 說明 |
|------|------|
| 類型 | Web3 整合平台 |
| 功能 | 一站式 DApp 探索 |
| 特色 | 聚合多個 DApp |

---

## 核心功能

### 主要特色

| 功能 | 說明 |
|------|------|
| DApp 聚合 | 在一個介面使用多個 DApp |
| 跨鏈支援 | 支援多條區塊鏈 |
| 任務系統 | 完成任務獲得獎勵 |
| 社交功能 | 追蹤其他用戶 |

### 使用場景

```
DapDap.io 的使用場景：

探索新 DApp：
- 瀏覽熱門 DApp
- 發現新項目

一站式使用：
- 不用跳轉多個網站
- 統一的介面體驗

賺取獎勵：
- 完成任務
- 累積積分
```

---

## 如何使用

| 步驟 | 說明 |
|------|------|
| 1. 連接錢包 | 支援 MetaMask 等錢包 |
| 2. 探索 DApp | 瀏覽不同類別 |
| 3. 使用功能 | 直接在平台上操作 |
| 4. 完成任務 | 獲得獎勵 |

{% hint style="info" %}
**聚合器的價值**

類似 DapDap.io 的聚合平台，讓新手更容易探索 Web3，不需要自己尋找和驗證各個 DApp 的網址。
{% endhint %}

## 優缺點

### 優點

| 優點 | 說明 |
|------|------|
| 方便 | 一站式體驗 |
| 發現 | 容易找到新 DApp |
| 任務獎勵 | 額外收益機會 |

### 缺點

| 缺點 | 說明 |
|------|------|
| 中間層風險 | 增加一個信任層 |
| 可能有限制 | 功能可能不如原生完整 |

{% hint style="warning" %}
**安全提醒**

使用任何聚合平台時，都要確認：
- 平台的安全性和信譽
- 授權的範圍
- 定期檢查錢包授權
{% endhint %}
