# 區塊鏈遊戲

### 玩遊戲還能賺錢？這就是區塊鏈遊戲的承諾。透過 NFT 和代幣經濟，玩家在遊戲中獲得的物品和角色可以真正「擁有」並交易。但這個美好的願景，也經歷了泡沫和現實的洗禮。

---

## 什麼是區塊鏈遊戲？

| 概念 | 說明 |
|------|------|
| 別名 | GameFi、P2E（Play-to-Earn）|
| 核心 | 遊戲資產上鏈 |
| 特色 | 玩家擁有遊戲資產 |
| 經濟 | 可以交易、賺取代幣 |

---

## 主要模式

### Play-to-Earn（P2E）

```
Play-to-Earn 模式：

傳統遊戲：
玩家花錢 → 遊戲公司獲利
（玩家只能消費）

P2E 遊戲：
玩家投入時間 → 獲得代幣/NFT → 可以出售
（玩家可以獲利）

關鍵問題：
錢從哪裡來？
- 新玩家的投入
- 遊戲內經濟
- 外部投資
```

### 遊戲類型

| 類型 | 說明 | 代表 |
|------|------|------|
| RPG | 角色扮演 | Axie Infinity |
| 移動賺 | Move-to-Earn | StepN |
| 卡牌 | 收集卡牌 | Gods Unchained |
| 虛擬世界 | 元宇宙土地 | Decentraland、The Sandbox |
| 賽車 | 競速遊戲 | Nitro League |

{% hint style="info" %}
**NFT 在遊戲中的角色**

在區塊鏈遊戲中，你的角色、裝備、土地都是 NFT。這意味著：
- 你真正「擁有」這些資產
- 可以在遊戲外交易
- 遊戲公司無法隨意沒收
{% endhint %}

## 經濟模型挑戰

### 核心問題

```
GameFi 的經濟困境：

問題：
- 早期玩家賺錢 = 後期玩家付錢
- 如果沒有新玩家 → 經濟崩潰
- 本質上類似龐氏結構

Axie 的例子：
- 2021：菲律賓玩家月入數百美元
- 2022：代幣價值暴跌 90%+
- 很多玩家虧損嚴重
```

### 永續性挑戰

| 挑戰 | 說明 |
|------|------|
| 通膨 | 代幣不斷發行 |
| 依賴新玩家 | 需要持續增長 |
| 遊戲性不足 | 很多遊戲不好玩 |
| 投機為主 | 玩家為賺錢而非娛樂 |

{% hint style="warning" %}
**投資警告**

很多 GameFi 項目的經濟模型不永續。不要投入無法承受損失的金額。很多「Play-to-Earn」最終變成「Pay-to-Lose」。
{% endhint %}
