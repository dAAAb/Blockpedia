## 如何評估 NFT 藝術家

### 考量因素

```
評估 NFT 藝術家的指標：

藝術層面：
- 風格獨特性
- 作品一致性
- 藝術史脈絡

市場層面：
- 歷史銷售
- 二級市場活躍度
- 收藏家質量

社群層面：
- 追蹤者數量
- 互動程度
- 同業認可

長期價值：
- 持續創作能力
- 跨界合作
- 藝術發展性
```

{% hint style="warning" %}
**銷售額不等於藝術價值**

排行榜主要反映市場表現，不完全代表藝術價值。有些優秀藝術家可能尚未被市場發現。
{% endhint %}
