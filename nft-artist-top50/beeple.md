# Beeple

### 2021 年 3 月 11 日，一幅「看不見、摸不著」的數位藝術品在佳士得拍出 6934 萬美元。創作者 Beeple 一夜之間成為在世最貴藝術家第三名。這一刻，NFT 正式進入主流藝術世界。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。加密貨幣投資有風險，請自行研究（DYOR）後再做決定。

## Everydays 計畫

### 5000 天不間斷創作

```
Everydays 的故事：

2007 年 5 月 1 日：
- Beeple 開始一個挑戰
- 每天創作一件數位藝術
- 不管發生什麼，一天不落

持續 13 年：
- 歷經結婚、生子
- 從未中斷一天
- 每件作品都發布在網上

2021 年：
- 將 5000 張作品合成一件
- 「Everydays: The First 5000 Days」
- 佳士得拍賣，$69,346,250 成交
```

### 作品風格

| 特色 | 說明 |
|------|------|
| 社會諷刺 | 反映時事、政治 |
| 流行文化 | 結合名人、迷因 |
| 科幻元素 | 未來感視覺 |
| 超現實 | 荒誕但有深意 |

### 佳士得拍賣細節

| 項目 | 說明 |
|------|------|
| 作品 | Everydays: The First 5000 Days |
| 拍賣行 | 佳士得（Christie's）|
| 成交價 | $69,346,250 |
| 買家 | Metakovan（Vignesh Sundaresan）|
| 意義 | 首件主流拍賣行 NFT |

```
這場拍賣的意義：

對藝術界：
- 數位藝術獲得認可
- NFT 成為新的藝術形式
- 傳統拍賣行擁抱區塊鏈

對 Beeple：
- 成為在世最貴藝術家之一
- 證明每日創作的價值
- 改變了他的人生

對 NFT 產業：
- 引發全球關注
- 帶動 NFT 熱潮
- 吸引無數人進場
```

{% hint style="info" %}
**為什麼是佳士得？**

佳士得是全球最頂級的拍賣行之一。當它願意拍賣 NFT，代表傳統藝術界正式認可這個新形式。這對 NFT 的正當性意義重大。
{% endhint %}

## 創作歷程

### 從無名到巨星

```
Beeple 的成長路：

早期（2007-2019）：
- 每日創作，默默耕耘
- 商業案例：Nike、Apple、SpaceX
- Instagram 累積追蹤者
- 免費分享作品

轉折（2020）：
- 開始接觸 NFT
- 首次 NFT 銷售成功
- 發現數位藝術的新可能

爆發（2021）：
- Nifty Gateway 拍賣 $350 萬
- 佳士得拍賣 $6900 萬
- 成為全球焦點
```

### 後續發展

|------|------|
| 2021 | 多件作品高價成交 |
| 2022 | 實體雕塑作品 |
| 持續 | 每日創作不中斷 |

```
關於 Beeple 的爭議：

藝術性質疑：
- 有人認為只是商業插畫
- 傳統藝術家不認同
- 但市場給了答案

每日創作的意義：
- 支持者：紀律與累積
- 批評者：量大於質
- Beeple：這是我的修行

NFT 泡沫？：
- 2022 年市場大跌
- 但 Beeple 作品仍有流動性
- 長期價值仍待觀察
```
