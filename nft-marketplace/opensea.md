# OpenSea
> **最後更新：2026 年 1 月**


## OpenSea 是最老牌的 NFT 市場，曾經統治整個 NFT 交易，見證了 NFT 從小眾到爆發再到冷卻的完整週期。

2021 年，NFT 爆發。

每個人都在談論 Bored Apes、CryptoPunks、Art Blocks。

而幾乎所有交易都發生在同一個地方：**OpenSea**。

它是 NFT 世界的「淘寶」，是通往數位收藏品的入口。

### OpenSea 是什麼？

OpenSea 是最大的 NFT 交易市場。

- **創立**：2017 年
- **創辦人**：Devin Finzer、Alex Atallah
- **總部**：美國紐約
- **估值**：2022 年曾達 133 億美元

### 歷史

**2017-2019：早期**

- 2017 年 CryptoKitties 熱潮後創立
- 早期 NFT 市場
- 用戶較少但穩定成長

**2020：DeFi 夏天**

- NFT 開始受關注
- 藝術家開始進入
- Beeple 等藝術家成名

**2021：爆發年**

- NFT 交易量飆升
- Bored Ape Yacht Club 爆紅
- OpenSea 成為獨角獸
- 月交易量達數十億美元

**2022：調整**

- 估值達 133 億美元
- 但市場開始冷卻
- 競爭者（如 Blur）出現

**2023-2024：挑戰**

- 市場大幅萎縮
- 裁員
- 面臨 Blur 的激烈競爭
- 尋找新方向

### 如何運作？

**對賣家**

1. 連接錢包
2. 選擇要出售的 NFT
3. 設定價格或拍賣
4. 列表（可能需要初次設定費）
5. 買家購買後，收到付款

**對買家**

1. 連接錢包
2. 瀏覽或搜尋 NFT
3. 購買或出價
4. 確認交易
5. NFT 轉入你的錢包

**手續費**

OpenSea 收取交易額的 2.5%（已調整為 0% 部分情況）。

### 支援的區塊鏈

OpenSea 支援多條區塊鏈：

- **Ethereum**：主要市場
- **Polygon**：低手續費選項
- **Solana**：獨立生態
- **Arbitrum**
- **Optimism**
- **Base**
- **Avalanche**
- 還有更多...

### 主要功能

**1. 列表和交易**

基本的買賣功能。

**2. 拍賣**

英式拍賣（價高者得）和荷蘭式拍賣（價格逐漸下降）。

**3. 出價系統**

可以對未上架的 NFT 出價。

**4. 收藏夾和追蹤**

追蹤喜歡的項目和藝術家。

**5. 懶鑄造（Lazy Minting）**

創作者可以免費創建 NFT，只有在售出時才支付 Gas。

**6. 多鏈支援**

在不同區塊鏈上交易。

### OpenSea Pro

2023 年，OpenSea 推出專業版 OpenSea Pro（前身是 Gem）：

**功能**
- 批量購買
- 跨市場聚合
- 專業分析工具
- 適合活躍交易者

這是對 [Blur](blur.md) 的回應。

### 與 Blur 的競爭

2022 年底，Blur 出現並迅速搶佔市場：

**Blur 的優勢**
- 零手續費
- 代幣激勵
- 專業交易介面
- 更快的交易體驗

**OpenSea 的回應**
- 降低手續費（部分情況為 0%）
- 推出 OpenSea Pro
- 強調創作者版稅

**結果**

Blur 在交易量上超越 OpenSea，但 OpenSea 仍有大量用戶基礎。

### 版稅爭議

NFT 版稅是重要議題：

**什麼是版稅？**

NFT 轉手時，原創作者獲得一定比例（如 5%）。

**問題**

版稅不是鏈上強制的，市場可以選擇不執行。

**Blur 的做法**

最初不強制版稅，後來有所調整。

**OpenSea 的立場**

曾經強制版稅，後來放鬆以應對競爭。

這引發了創作者 vs 交易者的爭論。

### 安全問題

OpenSea 曾有一些安全事件：

**釣魚攻擊**

假 OpenSea 網站騙取用戶授權。

**惡意列表**

利用舊列表低價購買 NFT。

**建議**
- 確認官方網址
- 仔細檢查交易內容
- 定期撤銷不需要的授權

### 使用 OpenSea

**1. 準備錢包**

MetaMask 或其他 Web3 錢包。

**2. 連接**

訪問 [opensea.io](https://opensea.io)，連接錢包。

**3. 瀏覽**

探索不同的收藏品和藝術家。

**4. 交易**

購買或出售 NFT。

**5. 管理**

查看你擁有的 NFT，管理列表。

### 費用結構

| 操作 | 費用 |
|------|------|
| 創建收藏 | 免費 |
| 列表 | 首次需要 Gas |
| 購買 | 交易額 + Gas |
| 平台費 | 0-2.5%（視情況）|
| 創作者版稅 | 視收藏品設定 |

### 未來挑戰

**1. 市場萎縮**

NFT 交易量從高峰大幅下降。

**2. 競爭**

Blur 和其他平台的競爭。

**3. 監管**

SEC 等監管機構對 NFT 的關注。

**4. 創新**

需要找到新的增長點。

### OpenSea 的地位

儘管面臨挑戰，OpenSea 仍有優勢：

**品牌認知**

「NFT 市場」幾乎等於 OpenSea。

**用戶基礎**

大量普通用戶熟悉 OpenSea。

**多鏈佈局**

在多個區塊鏈上有存在。

**創作者關係**

與藝術家和創作者的長期關係。

{% hint style="info" %}
OpenSea 是 NFT 歷史的重要一頁。它見證了 NFT 的爆發和冷卻。雖然面臨競爭和挑戰，它仍是最知名的 NFT 平台之一。
{% endhint %}
