# Blur
> **最後更新：2026 年 1 月**


## Blur 是新一代 NFT 市場，用零手續費和代幣激勵顛覆了 OpenSea 的統治，成為專業 NFT 交易者的首選。

2022 年底，NFT 市場冷卻。

OpenSea 看似不可動搖。

然後 **Blur** 來了。

它用零手續費和瘋狂的空投，在幾個月內改變了整個格局。

### Blur 是什麼？

Blur 是專為專業交易者設計的 NFT 市場。

- **創立**：2022 年
- **代幣**：BLUR
- **特色**：零手續費、代幣激勵、專業介面
- **定位**：NFT 的「彭博終端」

### 快速崛起

**2022 年 10 月**

Blur 正式上線。

**策略**
- 零平台手續費
- 大量空投預期
- 專業交易介面

**2023 年 2 月**

BLUR 代幣空投，吸引大量關注。

**結果**

交易量迅速超越 OpenSea。

### 核心特色

**1. 零手續費**

OpenSea 收 2.5%，Blur 不收。

對大量交易者來說，這是巨大差異。

**2. 專業介面**

- 快速列表
- 批量操作
- 深度圖表
- 即時數據

像交易終端，不像購物網站。

**3. 聚合器**

聚合多個市場的列表，找到最佳價格。

**4. Bid 池**

批量出價功能，可以對整個收藏品出價。

**5. 代幣激勵**

活躍交易和提供流動性可以獲得 BLUR 代幣。

### BLUR 代幣

**空投**

Blur 進行了多輪空投：
- 第一季：對早期用戶
- 後續季度：持續激勵

**獲取方式**
- 交易 NFT
- 出價提供流動性
- 持有特定 NFT

**代幣功能**
- 治理投票
- 未來可能有更多用途

### Blend 借貸

2023 年，Blur 推出 **Blend**：NFT 抵押借貸。

**什麼是 Blend？**

- 用 NFT 抵押借 ETH
- 沒有到期日
- 沒有預言機
- 點對點借貸

**怎麼運作？**

1. 借款人抵押 NFT
2. 貸款人提供 ETH
3. 借款人支付利息
4. 可隨時還款取回 NFT
5. 或貸款人可以要求還款

**意義**

讓 NFT 持有者可以獲得流動性而不用賣出 NFT。

### 版稅政策

Blur 的版稅政策引發爭議：

**初始政策**

不強制創作者版稅。

**爭議**

創作者認為這損害了他們的收入。

**後續調整**

提供了可選的版稅支持。

**影響**

引發了整個 NFT 市場對版稅的重新思考。

### 與 OpenSea 比較

| 面向 | Blur | OpenSea |
|------|------|---------|
| 手續費 | 0% | 0-2.5% |
| 目標用戶 | 專業交易者 | 一般用戶 |
| 介面 | 交易終端 | 購物網站 |
| 代幣激勵 | 有（BLUR）| 無 |
| 版稅 | 可選 | 有過調整 |
| 借貸 | Blend | 無 |

**選擇建議**
- 頻繁交易、注重效率：Blur
- 偶爾買賣、重視體驗：OpenSea

### 交易量對比

2023 年後，Blur 在交易量上超越 OpenSea：

**原因**
- 零手續費吸引大戶
- 代幣激勵促進交易
- 專業工具適合活躍交易

**但**

OpenSea 在用戶數和新手體驗上仍有優勢。

### 使用 Blur

**1. 連接錢包**

訪問 [blur.io](https://blur.io)，連接 MetaMask。

**2. 瀏覽市場**

探索不同收藏品。

**3. 交易**

- 購買：直接購買或出價
- 出售：一鍵列表

**4. 賺取獎勵**

- 交易賺積分
- 出價提供流動性
- 積分轉換為 BLUR

### 積分系統

Blur 用積分激勵活動：

**賺取方式**
- 交易 NFT
- 保持列表在地板價
- 提供競標流動性

**轉換**

積分定期轉換為 BLUR 代幣空投。

**策略**

很多人專門「刷」Blur 積分，希望獲得更多空投。

### 風險

**1. 代幣價格波動**

BLUR 價格不穩定。

**2. 激勵結束後**

如果激勵減少，交易量可能下降。

**3. 市場風險**

NFT 市場整體仍在調整。

**4. Blend 風險**

借貸涉及清算風險。

### 對 NFT 市場的影響

Blur 改變了 NFT 市場：

**1. 費用競爭**

迫使 OpenSea 降低費用。

**2. 專業化**

NFT 交易變得更專業化。

**3. 版稅討論**

引發對版稅可持續性的討論。

**4. 代幣激勵模式**

證明代幣激勵可以快速獲取市場份額。

### 未來展望

**1. 持續創新**

Blur 持續推出新功能。

**2. 代幣經濟調整**

如何維持激勵的可持續性。

**3. NFT 市場復甦**

等待 NFT 市場整體回暖。

**4. 跨鏈擴展**

可能擴展到其他區塊鏈。

{% hint style="info" %}
Blur 用激進的策略顛覆了 NFT 市場格局。它代表了 Web3 的「閃電戰」打法：零手續費 + 代幣激勵 + 專業工具。雖然長期可持續性仍待觀察，但它已經永久改變了 NFT 市場的競爭態勢。
{% endhint %}
