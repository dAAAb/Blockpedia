# JBA

## 日本加密貨幣監管

### 監管特色

```
日本加密貨幣監管的特點：

先行者：
- 2017 年認定比特幣為合法支付方式
- 全球最早建立完整監管框架
- Mt.Gox 事件後加強監管

監管架構：
- 金融廳（FSA）主管
- 交易所需取得牌照
- 自律組織協助監管

行業自律：
- JVCEA（交易所自律）
- JBA（區塊鏈產業自律）
- 政府認可的自律規範
```

### Mt.Gox 的影響

| 事件 | 影響 |
|------|------|
| 2014 Mt.Gox 倒閉 | 85 萬 BTC 損失 |
| 監管反應 | 加強交易所監管 |
| 結果 | 發牌制度建立 |

## 日本市場特色

### 獨特之處

| 特色 | 說明 |
|------|------|
| 合法支付 | 比特幣可合法支付 |
| 嚴格監管 | 交易所需牌照 |
| 消費者保護 | 客戶資產分離 |
| 自律機制 | 行業自我規範 |

### 對業者要求

```
日本交易所的要求：

牌照申請：
- 向金融廳申請
- 審查標準嚴格
- 需要時間和資源

營運要求：
- 客戶資產分離保管
- 冷錢包存儲大部分資產
- 定期審計
- KYC/AML 合規

安全要求：
- 資訊安全標準
- 內控機制
- 員工培訓
```
