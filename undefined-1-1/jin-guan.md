# 金管會

### 在台灣，如果你想開交易所、發行代幣、或提供任何加密貨幣金融服務，金管會就是你要打交道的對象。它是台灣金融世界的守門人。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成法律或投資建議。台灣法規持續更新，請諮詢專業律師。

---

## 什麼是金管會？

| 項目 | 說明 |
|------|------|
| 全名 | 金融監督管理委員會 |
| 英文 | Financial Supervisory Commission (FSC) |
| 成立 | 2004 年 |
| 職責 | 金融監督、管理、檢查 |

## 金管會與加密貨幣

### 監管立場

```
台灣對加密貨幣的定位：

目前定位：
- 虛擬通貨 = 數位商品
- 不是貨幣
- 不是有價證券（目前）
- 不是電子支付工具

意義：
- 不受《銀行法》管轄
- 不受《證交法》管轄
- 但受《洗錢防制法》管轄
```

### 監管範圍

| 項目 | 是否監管 |
|------|----------|
| 交易所 | ✅ 洗錢防制申報 |
| ICO/STO | ✅ 需個案審查 |
| 穩定幣 | ⚠️ 研議中 |
| DeFi | ❌ 尚未明確 |

## 台灣加密貨幣監管

### 發展歷程

| 時間 | 事件 |
|------|------|
| 2018 | 虛擬通貨納入洗錢防制 |
| 2021 | 交易所強制申報 |
| 2023 | 專法研議 |
| 2024 | 虛擬資產專法草案 |

### 洗錢防制要求

```
交易所必須做的事：

登記申報：
- 向金管會申報
- 非許可制，但必須登記

客戶識別（KYC）：
- 實名認證
- 身份證件驗證
- 風險評估

交易監控：
- 可疑交易申報
- 大額交易通報
- 保留交易紀錄

違規處罰：
- 未申報：最高 500 萬罰鍰
- 洗錢協助：刑事責任
```

## STO 規範

### 證券型代幣

| 項目 | 說明 |
|------|------|
| 定義 | 具有證券性質的代幣 |
| 監管 | 比照有價證券 |
| 發行 | 需經金管會核准 |

```
台灣 STO 規範：

發行限制：
- 單一專案上限 3000 萬台幣
- 專業投資人限制
- 需透過持牌平台

投資人資格：
- 專業投資人：無限制
- 一般投資人：上限 10 萬
- 限年滿 20 歲

交易平台：
- 需取得許可
- 目前極少數
- 流動性有限
```

{% hint style="warning" %}
**STO 市場發展緩慢**

台灣 STO 規範較嚴格，實際發行案例有限。相比之下，一般虛擬貨幣交易更為活躍。
{% endhint %}

## 對業者的影響

### 合規要求

| 要求 | 說明 |
|------|------|
| 洗錢防制 | 必須申報並執行 |
| 客戶識別 | KYC 強制要求 |
| 資訊揭露 | 風險警語 |
| 爭議處理 | 建立機制 |

### 未合規的風險

| 風險 | 後果 |
|------|------|
| 行政罰鍰 | 最高 500 萬 |
| 刑事責任 | 洗錢罪 |
| 營業禁止 | 勒令停業 |

## 對投資人的保護

### 現有保護

```
台灣投資人的處境：

有什麼保護：
- 交易所需實名制
- 有爭議處理管道
- 政府有監督機制

沒有什麼保護：
- 沒有存款保險
- 沒有投資人保護基金
- 虧損自負
```

### 投資人須知

| 須知 | 說明 |
|------|------|
| 風險自負 | 政府不保證投資 |
| 選擇合規交易所 | 確認已申報登記 |
| 保留紀錄 | 交易紀錄要保存 |
| 稅務申報 | 獲利需報稅 |

{% hint style="info" %}
**查詢交易所合規狀態**

可至金管會網站或法務部洗錢防制處查詢交易所是否已完成申報登記。
{% endhint %}
