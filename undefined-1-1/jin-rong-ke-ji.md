# 金融科技協會

### 金融科技（FinTech）正在改變我們的金融生活。從行動支付到區塊鏈，金融科技協會作為產業與政府的橋樑，推動創新發展同時維護金融秩序。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。

## 監管沙盒

### 概念說明

```
什麼是監管沙盒：

定義：
- 受控的實驗環境
- 允許創新測試
- 暫時放寬部分法規

目的：
- 讓創新不受過多限制
- 但仍有監管保護
- 測試後決定正式規範

台灣的做法：
- 金管會設有監管沙盒
- FinTech 可申請實驗
- 區塊鏈項目亦可適用
```

{% hint style="info" %}
**監管沙盒是雙贏**

監管沙盒讓創新業者有空間實驗，也讓監管機構有機會了解新科技。對區塊鏈產業來說，是推動友善監管的重要機制。
{% endhint %}
