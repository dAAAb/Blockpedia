# ICORating

### 2017-2018 年的 ICO 熱潮中，每天都有幾十個新項目上線。哪個值得投資？哪個是詐騙？ICORating 等評級機構試圖提供答案，但它們自己的公正性也受到質疑。

> ⚠️ **免責聲明**：本文僅供教育目的，不構成投資建議。評級僅供參考，不保證準確。

---

## 什麼是 ICORating？

| 項目 | 說明 |
|------|------|
| 類型 | ICO/加密項目評級 |
| 成立 | 2017 年 |
| 服務 | 項目分析、風險評估 |
| 收入 | 評級費用、廣告 |

## 評級機構的功能

### 評估內容

```
ICO 評級通常評估：

團隊分析：
- 成員背景
- 過往經歷
- 顧問陣容

技術評估：
- 白皮書品質
- 技術可行性
- 代碼審計

商業模式：
- 市場需求
- 競爭分析
- 盈利模式

代幣經濟：
- 代幣分配
- 解鎖機制
- 激勵設計

風險評估：
- 法律風險
- 執行風險
- 市場風險
```

### 評級標準

| 評級 | 說明 |
|------|------|
| A+ / A | 高品質專案 |
| B+ / B | 良好專案 |
| C+ / C | 一般專案 |
| D / F | 高風險 / 詐騙 |

## 評級的爭議

### 利益衝突

```
評級機構的爭議：

收費評級問題：
- 項目方付費獲得評級
- 付費可能影響評分
- 利益衝突明顯

SEC 執法案例：
- 2019 年 SEC 起訴 ICORating
- 未揭露收費關係
- 被認定為隱藏廣告

結果：
- 評級公信力受損
- 投資者需謹慎參考
- 行業自律不足
```

### SEC 對 ICORating 的行動

| 時間 | 事件 |
|------|------|
| 2019 | SEC 提起訴訟 |
| 指控 | 未揭露付費關係 |
| 和解 | 支付罰款 |

{% hint style="warning" %}
**評級不等於背書**

即使是高評級的項目，也可能失敗或是詐騙。評級只是參考之一，不應該是投資的唯一依據。
{% endhint %}

## 評級的價值與限制

### 仍有價值

```
評級的正面價值：

節省時間：
- 初步篩選大量項目
- 快速了解基本面

風險警示：
- 標記明顯問題
- 避免最差的項目

資訊整理：
- 統一格式比較
- 關鍵資訊彙整

但要記住：
- 過去績效不保證未來
- 評級有時間延遲
- 無法預測黑天鵝事件
```
