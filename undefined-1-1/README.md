# 區塊鏈組織

### 加密貨幣是去中心化的，但它存在的世界不是。監管機構決定什麼是合法的、行業協會推動自律、評級機構提供參考。了解這些組織，才能在這個產業安全前行。

## 監管趨勢

### 全球動向

```
加密貨幣監管趨勢：

明確化：
- 從模糊到明確
- 定義什麼是證券
- 發牌制度建立

合規化：
- KYC/AML 要求
- 交易所發牌
- 穩定幣監管

國際協調：
- FATF 旅行規則
- 跨境監管合作
- 資訊交換機制
```

{% hint style="info" %}
**監管是雙面刃**

監管帶來合規成本，但也帶來正當性和機構資金。成熟的監管框架可能是加密貨幣被主流採納的關鍵。
{% endhint %}
