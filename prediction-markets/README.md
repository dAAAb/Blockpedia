# 預測市場

## 預測市場（Prediction Market）是讓人們用真金白銀對未來事件下注的市場，被稱為「群眾智慧的聚合器」。

你有沒有想過，誰能最準確地預測未來？

是專家？是媒體？還是統計模型？

答案可能讓你意外：**是市場**。

### 賭一把的智慧

讓我們做個思想實驗。

假設明天要選舉，有 A、B 兩位候選人。你問專家，專家說「根據我的分析，A 會贏」。你問媒體，媒體說「民調顯示 B 領先」。眾說紛紜，不知道該信誰。

現在換個方式：開一個賭盤，讓大家用自己的錢押注。如果 A 贏，押 A 的人賺錢；B 贏，押 B 的人賺錢。

會發生什麼事？

- 真的有內幕消息的人，會押下大注
- 隨便喊喊的人，不敢押太多（畢竟是真錢）
- 最後的賠率，反映的是「所有人用真金白銀投票」的結果

這就是預測市場的核心：**讓人們把錢放在嘴巴的位置（put your money where your mouth is）**。

### 為什麼預測市場準？

預測市場有幾個神奇的特性：

**1. 激勵真實**

專家可以隨便說，反正說錯也不用負責。但在預測市場，說錯會虧錢，這讓人們有強烈動機去找到真相。

**2. 資訊聚合**

每個人都有一點點資訊：你知道當地的氣氛、他知道內部的消息、她看到一些數據。預測市場把所有人的資訊透過價格聚合起來。

**3. 即時更新**

民調一週做一次，但預測市場每分每秒都在更新。新消息出來，價格立刻反應。

**4. 打敗偏見**

人類有很多認知偏見：確認偏誤、樂觀偏誤、從眾效應。但當你要押上真金白銀，這些偏見會被「虧錢」矯正。

{% hint style="info" %}
學術研究顯示，預測市場的準確度經常超越民調、專家預測，甚至政治分析師。
{% endhint %}

### 預測市場的歷史

預測市場不是新概念。

**1988 年**：愛荷華大學創建「愛荷華電子市場」（IEM），讓人們押注美國大選。結果比大多數民調更準確。

**2000 年代**：好萊塢股票交易所（HSX）讓人們押注電影票房，準確度驚人。

**2014 年**：區塊鏈預測市場 Augur 誕生，但因為用戶體驗差，沒有爆發。

**2020 年**：[Polymarket](polymarket.md) 上線，成為第一個真正好用的去中心化預測市場。

**2024 年**：Polymarket 在美國大選期間爆紅，交易量突破 90 億美元，成為主流媒體的參考指標。

### 區塊鏈帶來什麼？

傳統預測市場（如博彩公司）存在一些問題：

- **地域限制**：很多國家禁止線上賭博
- **中心化風險**：公司可以拒絕支付、操控賠率
- **隱私問題**：需要實名註冊
- **高手續費**：博彩公司要抽成

區塊鏈預測市場解決這些問題：

| 問題 | 區塊鏈解法 |
|------|-----------|
| 地域限制 | 去中心化，無法封鎖 |
| 中心化風險 | 智能合約自動執行，無法賴帳 |
| 隱私 | 只需要錢包地址，不用身分證 |
| 手續費 | 通常只有 1-2%，遠低於傳統博彩 |

### 預測市場 vs 賭博

你可能會問：這跟賭博有什麼不同？

從機制上，確實很像。但有幾個關鍵差異：

**1. 目的不同**
- 賭博：娛樂、刺激
- 預測市場：資訊發現、決策輔助

**2. 社會價值**
- 賭博：零和遊戲，創造不了價值
- 預測市場：產出有價值的預測資訊

**3. 設計意圖**
- 賭博：莊家設計來賺錢
- 預測市場：設計來聚合資訊

當然，這條界線不是絕對的。很多人也把預測市場當賭博玩，監管機構對此也有爭議。

### 寶博士觀點

寶博士常說，預測市場是「把市場當成真相機器」。

在這個假消息滿天飛的時代，我們需要更好的方法來辨別真假。預測市場提供了一個有趣的工具：不是問「誰說得對」，而是問「誰願意押錢」。

當然，預測市場也不是萬能的。它預測的是機率，不是確定性。低機率的事件還是可能發生。

但作為「眾人智慧的聚合器」，預測市場確實提供了一個獨特的視角。
