# Stripe + Bridge

## 2024 年 10 月，支付巨頭 Stripe 宣布以 11 億美元收購穩定幣基礎設施公司 Bridge，創下加密領域最大收購紀錄。

這不只是一筆併購，而是一個信號：**傳統支付巨頭認真了**。

### Stripe 是誰？

如果你不熟悉 Stripe，簡單介紹：

- **成立**：2010 年，由愛爾蘭兄弟 Patrick 和 John Collison 創辦
- **估值**：約 650 億美元
- **客戶**：Amazon、Google、Shopify、Uber 等
- **市場**：全球線上支付領導者

Stripe 讓開發者可以輕鬆在網站和 APP 中接入支付功能。如果你在網路上付過款，很可能經過 Stripe。

Stripe 有句名言：「我們讓網路商業的 GDP 增長」。

現在，他們想把這個故事延伸到加密貨幣。

### Bridge 是誰？

Bridge 是一家相對低調的公司，但在行內很有名。

**定位**：穩定幣基礎設施

Bridge 提供 API，讓企業可以：
- 接受穩定幣付款
- 發送穩定幣付款
- 在不同穩定幣之間轉換
- 在法幣和穩定幣之間轉換

**客戶**：

- SpaceX（是的，馬斯克的公司用 Bridge 處理部分資金）
- 加密交易所
- 跨境支付公司
- 金融科技公司

**團隊**：

創辦人來自 Square、Coinbase 等公司，對支付和加密都有深厚背景。

### 11 億美元的邏輯

為什麼 Stripe 願意花 11 億美元？

**1. 跨境支付的痛點**

Stripe 的客戶有大量跨境支付需求。傳統方式：
- 慢（3-5 天）
- 貴（手續費 3-5%）
- 複雜（涉及多個中介銀行）

穩定幣可以：
- 快（分鐘級）
- 便宜（不到 1%）
- 簡單（點對點）

**2. 新興市場機會**

很多新興市場的銀行基礎設施不完善，但手機普及率高。

穩定幣可以讓這些地區的商家和消費者參與全球經濟，不需要依賴傳統銀行。

**3. 競爭壓力**

PayPal 已經推出 PYUSD 穩定幣。Visa、Mastercard 都在佈局。Stripe 不能落後。

**4. Web3 商業**

NFT 平台、DeFi 協議、加密遊戲...這些新興商業模式需要原生的加密支付解決方案。

### 收購後的整合

2025 年 2 月，Stripe 完成對 Bridge 的收購。

整合方向：

**1. Stripe 接受穩定幣**

Stripe 的商戶將可以接受穩定幣付款，就像接受信用卡一樣簡單。

消費者選擇「用 USDC 付款」→ 掃描 QR Code → 完成付款 → 商戶收到美元（或穩定幣，自選）

**2. Stripe 發送穩定幣**

Stripe Connect（用於平台付款給賣家）可以用穩定幣發放款項。

對於跨境電商平台，付款給海外賣家會變得更快更便宜。

**3. 穩定幣出入金**

Bridge 的法幣-穩定幣轉換能力，讓用戶可以：
- 把銀行帳戶的錢轉成穩定幣
- 把穩定幣轉回銀行帳戶

這是連接傳統金融和加密世界的關鍵。

{% hint style="info" %}
Stripe CEO Patrick Collison 說：「穩定幣是金融服務的超導體——將讓貨幣在全球流動時幾乎沒有摩擦。」
{% endhint %}

### 技術架構

Stripe + Bridge 的技術優勢：

**多鏈支援**

Bridge 支援多條區塊鏈：
- Ethereum
- Solana
- Polygon
- Base
- 其他

這讓 Stripe 可以選擇最適合的鏈——比如 Base 或 Solana 的低手續費。

**企業級可靠性**

Stripe 的核心優勢是可靠性和合規。Bridge 需要達到同樣標準：
- 高可用性
- 嚴格的 KYC/AML
- 完善的監控和報告

**開發者友善**

Stripe 以「開發者體驗」聞名。整合後的產品會保持這個傳統：
- 簡潔的 API
- 詳細的文件
- 快速的整合

### 對商戶的意義

如果你是商戶，Stripe + Bridge 意味著：

**1. 新的支付選項**

可以接受加密貨幣支付，吸引新客群。

**2. 降低跨境成本**

如果你的客戶在海外，用穩定幣收款可能比信用卡便宜。

**3. 更快的結算**

傳統信用卡結算要 1-2 天。穩定幣可以即時。

**4. 無縫整合**

如果你已經用 Stripe，增加穩定幣支付可能只需要開啟一個選項。

### 對消費者的意義

**1. 更多支付選擇**

喜歡用加密貨幣的人，可以在更多地方使用。

**2. 跨境購物更便宜**

用穩定幣付款，可能省下信用卡的外幣手續費。

**3. 無銀行帳戶也能消費**

只要有加密錢包和穩定幣，就可以線上購物。

### 競爭格局

Stripe + Bridge 的競爭對手：

**PayPal / Venmo**

- 已經有 PYUSD 穩定幣
- 整合進 PayPal 的龐大用戶群

**Coinbase Commerce**

- 加密原生的商戶解決方案
- 跟 Coinbase 交易所整合

**Circle**

- USDC 發行方
- 有自己的支付 API

**MoonPay / Transak**

- 專注法幣-加密轉換
- 但商戶功能較弱

Stripe 的優勢是：**已經有百萬商戶基礎**。這是冷啟動最難解決的問題。

### 挑戰

Stripe + Bridge 面臨的挑戰：

**1. 監管不確定性**

穩定幣在各國的法律地位不同。Stripe 需要確保合規。

**2. 商戶教育**

很多商戶不了解加密貨幣，需要教育他們為什麼要接受。

**3. 消費者習慣**

大部分消費者習慣用信用卡。改變習慣需要時間和激勵。

**4. 波動風險**

雖然穩定幣理論上價值穩定，但有脫鉤風險（雖然低）。

**5. 私鑰管理**

消費者需要管理自己的錢包。丟失私鑰就丟失資金。

### 長期影響

這筆收購的長期影響可能很深遠：

**1. 加速主流採用**

當 Stripe 這樣的主流支付商支援穩定幣，更多商戶和消費者會接觸到。

**2. 推動監管明確**

Stripe 的規模和影響力，可能推動監管機構更快制定清晰框架。

**3. 壓縮傳統支付成本**

如果穩定幣支付更便宜，傳統支付商可能被迫降價。

**4. 催生新商業模式**

微支付、AI 支付、跨境小額交易...過去因為成本太高不可行的模式，可能變得可行。

### 你可以期待什麼

如果你是 Stripe 的用戶（商戶或開發者），可能很快會看到：

1. Dashboard 中出現「穩定幣」選項
2. API 中新增穩定幣相關端點
3. 新的定價結構（穩定幣可能比信用卡便宜）
4. 新的結算選項

關注 Stripe 的公告，這將是 Web3 支付的重要里程碑。
