# Block Bitcoin 支付
> **最後更新：2026 年 1 月**


## Block（前身 Square）是 Jack Dorsey 創辦的支付公司，正在大力推進比特幣支付，目標是讓每個商戶都能接受 BTC。

如果說 [Stripe + Bridge](stripe-bridge.md) 代表「穩定幣」路線，那 Block 代表的是「比特幣」路線。

這兩種路線的差異，反映了對加密支付未來的不同願景。

### Jack Dorsey 是誰？

Jack Dorsey 是科技圈的傳奇人物：

- **Twitter（現 X）創辦人**：社群媒體巨頭
- **Square（現 Block）創辦人**：支付科技公司

更重要的是，他是**比特幣的狂熱支持者**。

Dorsey 曾說：「比特幣可以成為網際網路的原生貨幣。」「如果我沒在經營 Square 和 Twitter，我會全職投入比特幣。」

他不只是說說而已。Block 公司持有約 8000 枚比特幣（價值數億美元）作為資產負債表的一部分。

### Block 是什麼？

Block 的業務版圖：

**Square**
- 商戶支付終端
- 讓小店也能接受信用卡
- 美國市場占有率極高

**Cash App**
- 點對點轉帳 APP
- 已經支援比特幣買賣
- 超過 5000 萬用戶

**Afterpay**
- 先買後付（BNPL）
- 2021 年以 290 億美元收購

**Spiral**
- 比特幣開源開發
- 專注 Bitcoin 和 Lightning Network

**TBD**
- 去中心化身分和數據
- 建立 Web5 願景

可以看到，Block 不只是支付公司，而是圍繞比特幣建立了完整生態系。

### Bitcoin for Business

2025 年，Block 推出重磅產品：**Bitcoin for Business**。

這是什麼？

讓使用 Square 的商戶可以**直接接受比特幣支付**。

運作方式：
1. 消費者用 Cash App 或任何 Lightning 錢包
2. 掃描商戶的 QR Code
3. 透過 Lightning Network 即時付款
4. 商戶可以選擇：
   - 收到比特幣
   - 自動轉換成美元

重點是：**2026 年底前，商戶手續費 0%**。

沒錯，免費。

### 為什麼選 Lightning Network？

比特幣主網有個問題：交易慢、手續費高。

Lightning Network 是比特幣的「第二層」，解決這個問題：

| 特性 | 比特幣主網 | Lightning Network |
|------|-----------|-------------------|
| 速度 | ~10 分鐘 | 秒級 |
| 手續費 | 幾美元到幾十美元 | 幾分錢 |
| 容量 | ~7 TPS | 百萬級 TPS |

Lightning 讓比特幣終於可以用來「買咖啡」了。

### 與穩定幣路線的比較

Stripe 選穩定幣，Block 選比特幣。為什麼？

| 特性 | 穩定幣（Stripe） | 比特幣（Block） |
|------|-----------------|----------------|
| 價格波動 | 無 | 有 |
| 發行方 | 有（Circle、Tether） | 無 |
| 哲學 | 實用主義 | 理想主義 |
| 監管風險 | 中等 | 較低 |
| 用戶基礎 | DeFi 用戶 | 比特幣持有者 |

Jack Dorsey 選擇比特幣有幾個原因：

**1. 去中心化**

穩定幣有發行方，可能被監管、被封鎖。比特幣沒有這個問題。

**2. 硬通貨**

比特幣總量有限（2100 萬枚），是「數位黃金」。穩定幣只是「數位美元」，會隨美元一起通膨。

**3. 全球性**

比特幣在全球通用，不依賴任何國家的貨幣政策。

當然，這也是信仰問題。Dorsey 相信比特幣是更好的貨幣，所以願意押注。

{% hint style="info" %}
有趣的是，Cash App 也支援 USDC。所以 Block 不是完全排斥穩定幣，只是在商戶支付上押注比特幣。
{% endhint %}

### Cash App 的比特幣業務

Cash App 是 Block 比特幣策略的核心。

**已有功能**：
- 買賣比特幣
- 發送/接收比特幣（包括 Lightning）
- 自動定期購買（DCA）

**用戶數據**：
- 超過 5000 萬月活用戶
- 顯著比例的用戶買過比特幣

Cash App 讓比特幣對普通人變得簡單。不需要懂錢包、私鑰，就像用 Venmo 一樣。

### 商戶激勵

為什麼商戶要接受比特幣？

**Block 的激勵**：

1. **0% 手續費**（至 2026 年底）
   - 信用卡手續費通常 2-3%
   - 比特幣免費，省下的就是賺的

2. **即時結算**
   - 信用卡要 1-2 天才能拿到錢
   - 比特幣即時到帳

3. **無退款風險**
   - 信用卡有「拒付」（chargeback）問題
   - 比特幣交易不可逆

4. **吸引新客群**
   - 比特幣持有者是一個獨特的消費族群
   - 他們願意在支持比特幣的商家消費

### 風險

Block 的比特幣策略有風險：

**1. 波動**

比特幣價格波動大。如果商戶收到比特幣後價格大跌，可能虧損。

Block 的解法：提供即時轉換美元選項。

**2. 用戶習慣**

大部分人習慣用信用卡，有回饋、有延遲付款。比特幣沒有這些。

**3. 監管**

比特幣的監管地位在各國不同。政策改變可能影響業務。

**4. 技術門檻**

Lightning Network 還在發展中，有時會有技術問題。

### 對比特幣生態的影響

Block 的動作對比特幣生態有重大影響：

**1. 提升實用性**

如果大量商戶接受比特幣，「比特幣能幹嘛？」就有了答案。

**2. 推動 Lightning 普及**

Block 的規模可以大幅增加 Lightning Network 的使用量。

**3. 帶動其他公司**

如果 Block 成功，其他支付公司可能跟進。

**4. 壓力測試**

大規模商用會暴露 Lightning 的問題，推動技術改進。

### 全球化野心

Block 的比特幣策略有全球化野心。

在新興市場：
- 很多人沒有銀行帳戶
- 但有手機
- 本國貨幣可能不穩定

比特幣可以提供：
- 價值儲存
- 跨境支付
- 不依賴本地銀行

Block 已經在非洲、拉丁美洲等地擴展。比特幣是打開這些市場的鑰匙。

### 與 Stripe 的競爭

Stripe 和 Block 的競爭變得更有趣：

| 路線 | Stripe | Block |
|------|--------|-------|
| 加密策略 | 穩定幣為主 | 比特幣為主 |
| 收購 | Bridge（11 億） | 無（自建） |
| 核心客戶 | 線上商戶 | 線下商戶（也有線上）|
| 哲學 | 實用 | 信仰 |

最終誰會贏？可能不是零和遊戲——兩種路線可能同時存在，服務不同需求。

### 你可以期待什麼

作為消費者：
- Cash App 買比特幣會更方便
- 更多商家接受比特幣
- 用比特幣消費的體驗會改善

作為商戶：
- 免費接受比特幣的機會
- 可能吸引新客群
- 需要學習比特幣基礎知識

作為比特幣持有者：
- 終於可以「花」比特幣了
- 不再只是「持有」和「投機」
