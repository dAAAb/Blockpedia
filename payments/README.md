# Web3 支付

## Web3 支付是指使用區塊鏈和加密貨幣進行的支付，正在從加密原生場景走向主流商業。

在「[幣](../currency.md)」的條目中，我們說貨幣的核心功能是「交換價值」。區塊鏈從誕生那天起，就被寄予「成為新支付方式」的期望。

十幾年過去了，這個期望正在成為現實。

### 從投機到支付

過去很長一段時間，加密貨幣主要是「投資標的」而非「支付工具」。

為什麼？

**1. 波動太大**

比特幣一天可以漲跌 10%。誰敢用它買咖啡？今天花的 5 美元比特幣，明天可能值 6 美元。

**2. 速度太慢**

比特幣確認一筆交易要 10 分鐘以上。買個三明治等 10 分鐘？

**3. 手續費太高**

以太坊繁忙時，一筆交易手續費可能超過 50 美元。

**4. 體驗太差**

錢包地址一長串、私鑰要保管、交易要簽名...普通人搞不懂。

這些問題讓加密貨幣停留在「投機」階段。

### 2024-2025：轉折點

但事情正在改變。

**穩定幣普及**

[USDT](../usdt/usdt.md)、[USDC](../stablecoin/usdc.md) 等穩定幣解決了波動問題。1 USDC 永遠約等於 1 美元。

**Layer 2 成熟**

[Arbitrum](../layer2/arbitrum.md)、[Base](../layer2/base.md)、[Optimism](../layer2/optimism.md) 等 Layer 2 讓交易變得又快又便宜。

**傳統金融入場**

Stripe、PayPal、Visa 等支付巨頭開始認真佈局加密支付。

**用戶體驗改善**

帳戶抽象、社交登入等技術讓錢包變得像普通 APP 一樣簡單。

這些因素匯集，讓 Web3 支付終於有了「可用」的基礎。

### 支付場景

Web3 支付正在滲透各種場景：

**線上購物**

越來越多電商接受穩定幣支付。結帳時選擇「用 USDC 支付」，掃碼付款，完成。

**跨境轉帳**

傳統跨境匯款要 3-5 天、手續費 5-10%。穩定幣轉帳幾分鐘到帳、手續費不到 1%。

**自由工作者**

遠端工作者可以用穩定幣收款，不用擔心銀行帳戶、匯率問題。

**B2B 支付**

企業之間的跨境付款，用穩定幣更快更便宜。

**AI Agent 支付**

如「[AI x Web3](../ai-web3/README.md)」章節所述，AI 需要自主付款能力。

### 主要玩家

Web3 支付領域的主要玩家：

**傳統金融**
- [Stripe + Bridge](stripe-bridge.md) — 支付巨頭收購穩定幣公司
- [Block（Square）](block-btc.md) — Jack Dorsey 的比特幣支付
- PayPal — 發行 PYUSD 穩定幣
- Visa — 穩定幣結算試點

**加密原生**
- Circle — USDC 發行方
- Coinbase Commerce — 商戶加密收款
- BitPay — 老牌加密支付
- MoonPay — 法幣入金通道

**區塊鏈**
- Base — Coinbase 的 L2，手續費極低
- Solana — 高速低費，適合支付
- Lightning Network — 比特幣的支付層

### 支付 vs 投資：不同的思維

這裡要強調一個觀念轉換。

當我們說「比特幣」，很多人想到的是投資、價格、牛市熊市。

但「支付」是不同的思維：

| 投資思維 | 支付思維 |
|----------|----------|
| 價格會漲嗎？ | 能不能順利付款？ |
| 長期持有 | 即時使用 |
| 波動是機會 | 波動是風險 |
| 關注市值 | 關注手續費 |

Web3 支付更多是「穩定幣」的天下，而非比特幣、以太幣。

{% hint style="info" %}
有人說，穩定幣是加密貨幣最成功的應用。它解決了波動問題，讓加密貨幣真正可以當「貨幣」用。
{% endhint %}

### 監管挑戰

Web3 支付面臨監管挑戰：

**KYC/AML**

支付服務需要知道「誰在付款」，但區塊鏈的假名制讓這變難。

**牌照要求**

很多國家要求支付服務持有特定牌照。加密支付公司需要合規。

**稅務問題**

用加密貨幣支付，可能觸發資本利得稅。這增加了複雜度。

**消費者保護**

傳統支付有退款、爭議處理機制。區塊鏈交易不可逆，出問題怎麼辦？

這些問題正在被逐步解決。監管框架在各國逐漸明確。

### 機會與風險

**機會**

- 比傳統支付更便宜（尤其跨境）
- 比傳統支付更快
- 可以觸及無銀行帳戶的人群
- 可程式化支付（智能合約）
- 24/7 全球運作

**風險**

- 監管不確定性
- 用戶教育成本
- 私鑰風險（丟失、被盜）
- 智能合約風險
- 穩定幣脫鉤風險

### 寶博士觀點

寶博士常說，比特幣的願景是「點對點的電子現金系統」。

這個願景說起來簡單，做起來很難。比特幣本身因為各種限制，沒有成為主流支付工具。

但穩定幣和 Layer 2 讓這個願景重新有了希望。

當你可以用手機掃碼、秒級確認、幾分錢手續費，把錢從台北發到紐約——這才是「電子現金」該有的樣子。

Web3 支付正在讓這成為現實。
