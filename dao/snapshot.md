# Snapshot
> **最後更新：2026 年 1 月**


## Snapshot 是最受歡迎的 DAO 投票平台，讓代幣持有者可以免 Gas 投票，是去中心化治理的基礎設施。

投票是民主的核心。

但在區塊鏈上投票要付 Gas？

**Snapshot** 說：不用。

它讓投票變得免費、簡單、靈活。

幾乎每個重要的 DAO 都用 Snapshot。

### Snapshot 是什麼？

Snapshot 是去中心化的投票平台。

- **類型**：鏈下投票（Off-chain voting）
- **費用**：免費（不需要 Gas）
- **用途**：DAO 治理投票

### 為什麼需要 Snapshot？

**鏈上投票的問題**

傳統的鏈上投票：
- 每次投票都要交易
- 每次交易都要 Gas
- Gas 費可能比投票更重要

**結果**

- 小持有者不願投票
- 投票參與率低
- 治理被大戶主導

**Snapshot 的解決方案**

- 投票不上鏈
- 用簽名代替交易
- 完全免費

### 運作原理

**1. 簽名而非交易**

投票時，你只需要用錢包簽名：
- 不需要發送交易
- 不需要付 Gas
- 只是用私鑰證明你的意願

**2. 快照機制**

在提案創建時，對區塊鏈拍「快照」：
- 記錄那個時間點的代幣持有情況
- 投票權基於快照時的餘額
- 防止在投票期間轉移代幣操縱結果

**3. IPFS 存儲**

投票數據存儲在 IPFS：
- 去中心化存儲
- 不可竄改
- 永久保存

### 投票策略

Snapshot 支援靈活的投票策略：

**基本策略**

- **ERC-20 餘額**：持有多少代幣 = 多少票
- **ERC-721（NFT）**：持有特定 NFT 才能投票
- **代幣鎖定**：考慮鎖定時間

**進階策略**

- **多代幣**：組合多種代幣
- **二次方投票**：票數 = 代幣數量的平方根
- **委託**：可以把投票權委託給他人

**自定義策略**

可以編寫自定義策略滿足特殊需求。

### 如何使用 Snapshot？

**投票**

1. 訪問 [snapshot.org](https://snapshot.org)
2. 連接錢包
3. 找到你想參與的 DAO
4. 查看活躍提案
5. 點擊投票
6. 簽名確認

**創建提案**

1. 進入 DAO 空間
2. 點擊「New proposal」
3. 填寫標題和內容
4. 設定投票選項
5. 設定投票時間
6. 發布

**創建 DAO 空間**

1. 點擊「Create space」
2. 連接擁有 ENS 域名的錢包
3. 設定空間參數
4. 設定投票策略
5. 邀請成員

### 投票類型

**單選投票**

從多個選項中選一個。

**贊成/反對**

二選一投票。

**加權投票**

可以把票數分配給多個選項。

**排名投票**

對選項排序。

### 鏈下 vs 鏈上

| 面向 | Snapshot（鏈下）| 鏈上投票 |
|------|-----------------|----------|
| 費用 | 免費 | 需要 Gas |
| 速度 | 即時 | 取決於區塊鏈 |
| 執行 | 需要額外步驟 | 可自動執行 |
| 安全 | 信任平台 | 區塊鏈安全 |
| 適用 | 大多數治理 | 正式決策 |

**結合使用**

很多 DAO 結合使用：
- Snapshot 用於討論和初步投票
- 重要決策再用鏈上投票

### 著名使用者

### Snapshot X

Snapshot 推出的新版本：

**什麼是 Snapshot X？**

可選的鏈上執行層：
- 投票仍在 Snapshot
- 結果可以自動在鏈上執行
- 結合鏈下投票的便利和鏈上執行的安全

**目前狀態**

正在開發和推廣中。

### 限制

**1. 信任問題**

需要信任 Snapshot 平台正確記錄投票。

**2. 執行問題**

投票結果不會自動執行，需要額外步驟。

**3. 可能的攻擊**

- 快照時間操縱
- 閃電貸攻擊（在快照前借代幣）

**4. 法律約束力**

Snapshot 投票沒有法律約束力。

### 安全建議

**對於投票者**

- 確認在官方 Snapshot 網站
- 仔細閱讀提案內容
- 了解投票影響

**對於 DAO 管理者**

- 設定合理的提案門檻
- 給予足夠的投票時間
- 有爭議的提案要充分討論

### 替代方案

**Tally**

鏈上投票平台：
- 支援 OpenZeppelin Governor
- 投票結果可自動執行

**Aragon**

完整 DAO 平台：
- 包含投票功能
- 更多 DAO 管理工具

**Commonwealth**

討論 + 投票平台：
- 整合論壇和投票
- 社區討論功能

### 未來發展

**1. Snapshot X 普及**

鏈上執行的廣泛採用。

**2. 更多策略**

支援更多投票策略。

**3. 跨鏈支援**

支援更多區塊鏈的代幣。

**4. 整合增強**

與更多工具的整合。

{% hint style="info" %}
Snapshot 是 DAO 治理的基礎設施。它解決了鏈上投票的成本問題，讓更多人能夠參與治理。雖然是「鏈下」投票，但它的透明性和可驗證性讓它成為業界標準。
