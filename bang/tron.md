# TRON

### TRON 是幣圈最有爭議的公鏈之一。不是因為技術，而是因為創辦人孫宇晨。但拋開爭議不談，TRON 確實有一個殺手級應用：它是 USDT 流通量最大的區塊鏈。

---

|------|------|

| 啟動 | 2018 年 |
| 代幣 | TRX |
| 共識機制 | DPoS |
| 特色 | USDT 大戶 |

## TRON 的定位

### 核心特點

| 特點 | 說明 |
|------|------|
| 低手續費 | 轉帳幾乎免費 |
| 高速 | TPS 可達數千 |
| USDT 支援 | 最大 USDT 流通鏈 |
| 娛樂導向 | 原定位是娛樂平台 |

### USDT 的成功

```
TRON 上的 USDT：

現況：
- USDT 在 TRON 上的流通量超過 Ethereum
- 特別受亞洲、新興市場歡迎
- 低手續費是主因

為什麼用 TRON？
- 轉 USDT 幾乎免費
- Ethereum 手續費太貴
- 交易所支援廣泛
```

{% hint style="info" %}
**為什麼 USDT 選擇 TRON？**

對於想轉穩定幣的人，在 Ethereum 上可能要付 $10-50 手續費，在 TRON 上幾乎免費。這個簡單的經濟邏輯讓 TRON 成為 USDT 大戶。
{% endhint %}

## 使用場景

### 適合

| 場景 | 原因 |
|------|------|
| 轉穩定幣 | 手續費低 |
| 交易所間轉帳 | 快速便宜 |
| 新興市場 | 當地用戶多 |

### 不適合

| 場景 | 原因 |
|------|------|

| NFT | 市場小 |
