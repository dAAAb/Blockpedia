# 公鏈榜

### 公鏈（Public Chain）就像是不同的國家，各有各的法律、貨幣和生態系統。Bitcoin 是最老的王國，Ethereum 是智能合約的發源地，EOS、TRON 則是想挑戰以太坊的競爭者。選擇在哪條鏈上活動，就像選擇在哪個國家定居。

---

## 什麼是公鏈？

| 概念 | 說明 |
|------|------|
| 定義 | 任何人都可以參與的區塊鏈 |
| 特點 | 開放、去中心化、抗審查 |
| 對比 | 私有鏈、聯盟鏈 |

## 公鏈比較

### 核心指標

| 公鏈 | TPS | 手續費 | TVL |
|------|-----|--------|-----|
| Ethereum | ~30 | 高（波動）| 最高 |
| Solana | ~5000 | 極低 | 高 |
| Bitcoin | ~7 | 中等 | N/A |
| TRON | ~2000 | 極低 | 高（USDT）|

### 生態系統

```
公鏈生態比較：

Ethereum：
├── DeFi（最完整）
├── NFT（主流平台）
├── Layer 2（多條）
└── 開發者（最多）

Solana：
├── DeFi（快速成長）
├── NFT（知名項目）
└── 開發者（Rust 為主）

Bitcoin：
├── 支付（原始用途）
├── Ordinals（新興 NFT）
└── Lightning（Layer 2）
```

{% hint style="info" %}
**選擇公鏈的考量**

選擇在哪條鏈上活動，要考慮：
- 手續費：你能負擔多少 Gas？
- 生態：你需要的應用在哪條鏈上？
- 安全性：這條鏈有多去中心化？
{% endhint %}

## 公鏈戰爭

### 以太坊殺手們

```
「以太坊殺手」的歷史：

2017-2018：
- EOS：「免手續費！」→ 中心化爭議
- Cardano：「學術研究！」→ 發展緩慢
- TRON：「娛樂平台！」→ 爭議不斷

2020-2021：
- Solana：「超快速！」→ 多次當機
- Avalanche：「子網路！」→ 穩定成長
- Polygon：「Layer 2！」→ ETH 生態一部分

結果：
- 沒有人「殺死」以太坊
- 多鏈並存成為常態
```

### 現況

| 時期 | 主題 |
|------|------|
| 2021 | Layer 1 大戰 |
| 2022 | Layer 2 崛起 |
| 2023-2024 | 模組化區塊鏈 |
| 2025+ | 跨鏈互操作 |
