# Ethereum

### 如果 Bitcoin 是數位黃金，Ethereum 就是數位世界電腦。它不只是貨幣，更是一個可以運行任何程式的去中心化平台。DeFi、NFT、DAO——這些革命性應用都誕生在 Ethereum 上。

---

## 基本資訊

| 項目 | 說明 |
|------|------|
| 創建者 | Vitalik Buterin 等人 |
| 啟動 | 2015 年 7 月 30 日 |
| 代幣 | ETH |
| 共識機制 | 權益證明（PoS）|
| 供應量 | 無上限（但有銷毀機制）|

---

## 為什麼重要？

### 智能合約革命

| 創新 | 說明 |
|------|------|
| 智能合約 | 自動執行的程式 |
| DApp 平台 | 去中心化應用的基礎 |
| 代幣標準 | ERC-20、ERC-721 |
| 生態系統 | 最豐富的開發者社群 |

### 與 Bitcoin 的差異

```
Bitcoin vs Ethereum：

Bitcoin：
- 目的：點對點電子現金
- 功能：轉帳、價值儲存
- 腳本：有限
- 定位：數位黃金

Ethereum：
- 目的：世界電腦
- 功能：任何可程式化的應用
- 語言：Solidity（圖靈完備）
- 定位：去中心化平台
```

{% hint style="info" %}
**為什麼叫「世界電腦」？**

Ethereum 就像一台全世界共享的電腦。任何人都可以在上面部署程式（智能合約），而且沒有人能關閉它。
{% endhint %}

### The Merge

```
The Merge（2022 年 9 月）：

發生了什麼？
- 從 PoW 轉向 PoS
- 能耗降低 99.95%
- 不再需要礦工

為什麼重要？
- 區塊鏈史上最大升級
- 證明可以無縫切換共識
- 為未來升級鋪路
```

## Layer 2 生態

### 主要 L2

| L2 | 類型 | 特色 |
|----|------|------|
| Arbitrum | Optimistic | 最大 TVL |
| Optimism | Optimistic | OP Stack |
| Base | Optimistic | Coinbase 推出 |
| zkSync | ZK | 零知識證明 |
| StarkNet | ZK | 獨特語言 Cairo |

```
Layer 2 的必要性：

主網問題：
- Gas 費太高（高峰期 $50-100+）
- TPS 太低（15-30）

L2 解決方案：
- 在 L2 處理交易
- 定期結算到主網
- 繼承主網安全性
- Gas 費降低 10-100 倍
```

{% hint style="info" %}
**該用哪個 L2？**

選擇 L2 要考慮：
- TVL 和生態豐富度
- 你想用的 DApp 在哪
- Gas 費和提款時間
{% endhint %}
