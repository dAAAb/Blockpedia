# EOS

### EOS 曾經是幣圈的「明日之星」。2018 年募資 40 億美元創下記錄，號稱要成為「以太坊殺手」。免手續費、高速度、DPoS 共識——一切聽起來很完美。但故事沒有按劇本走。

---

## 基本資訊

| 項目 | 說明 |
|------|------|
| 創建者 | Daniel Larimer、Block.one |
| 啟動 | 2018 年 6 月 |
| 代幣 | EOS |
| 共識機制 | DPoS（委託權益證明）|
| 募資金額 | 約 40 億美元 |

---

## EOS 的承諾

### 核心賣點

| 特點 | 說明 |
|------|------|
| 免手續費 | 不需要 Gas 費 |
| 高速度 | 號稱可達數千 TPS |
| 可擴展 | 設計用於大規模應用 |
| 易開發 | 對開發者友善 |

### 技術架構

```
EOS 的設計：

共識：DPoS
- 21 個超級節點（Block Producer）
- 代幣持有者投票選舉
- 節點輪流出塊

資源模型：
- RAM：存儲資源
- CPU：計算資源
- NET：網路資源
- 需要質押 EOS 獲取資源

「免費」的代價：
- 必須持有 EOS 才能使用
- 資源價格波動
- 實際上不是免費
```

{% hint style="info" %}
**為什麼「免手續費」？**

EOS 不收交易手續費，但用戶需要質押 EOS 獲取 CPU/NET 資源。這不是真正免費，只是計費方式不同。
{% endhint %}

## 興衰歷程

### 時間線

| 時期 | 事件 |
|------|------|
| 2017-2018 | 一年 ICO，募資 40 億美元 |
| 2018 年 6 月 | 主網上線 |
| 2018-2019 | DApp 熱潮、博弈類應用 |
| 2020 | 活躍度下降 |
| 2021 | 被 Solana 等新公鏈超越 |
| 2022 | Block.one 和社群分裂 |
| 2023+ | ENF 接管，嘗試復興 |

### 為什麼衰落？

```
EOS 衰落的原因：

1. 中心化問題
   - 21 個節點太少
   - 節點串通疑慮
   - 中國節點一度主導

2. Block.one 的問題
   - 募資後投入不足
   - 與社群關係破裂
   - 被 SEC 罰款 2400 萬美元

3. 生態問題
   - DApp 品質低（多為博弈）
   - 開發者流失
   - 被新公鏈搶走注意力

4. 資源模型複雜
   - RAM 價格炒作
   - 新手難以理解
   - 實際使用門檻高
```

{% hint style="warning" %}
**中心化的代價**

只有 21 個節點驗證，雖然速度快，但也容易被批評「不夠去中心化」。這成為 EOS 最大的軟肋。
{% endhint %}

## 教訓

### 對投資者

| 教訓 | 說明 |
|------|------|
| 募資金額不等於成功 | 40 億美元也可能失敗 |
| 技術不是一切 | 生態和治理同樣重要 |
| 中心化有代價 | 速度快但失去核心價值 |
| 注意團隊 | Block.one 的態度是警訊 |

### 對區塊鏈發展

```
EOS 的教訓：

技術層面：
- DPoS 的中心化問題是真實的
- 資源模型需要更好設計

治理層面：
- 基金會和社群關係很重要
- 募資後的執行力更重要

市場層面：
- 「殺手」很難真的殺死對手
- 先發優勢不一定持久
```
