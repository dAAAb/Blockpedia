# Bitcoin

### Bitcoin 是一切的起點。2008 年，中本聰發布白皮書；2009 年，創世區塊誕生。十五年後，它從極客玩具變成萬億美元資產。如果你只了解一條區塊鏈，那就是 Bitcoin。

## 為什麼重要？

### 歷史意義

| 創新 | 說明 |
|------|------|
| 第一個區塊鏈 | 證明去中心化貨幣可行 |
| 解決雙花問題 | 不需要中央機構 |
| 抗審查 | 沒有人能凍結你的 BTC |
| 稀缺性 | 固定供應量，類似黃金 |

### 數位黃金敘事

```
Bitcoin 為什麼被稱為「數位黃金」？

與黃金的相似處：
- 稀缺：總量有限
- 難以取得：需要「挖礦」
- 持久：不會腐爛或損壞
- 全球認可：跨國界價值儲存

優於黃金的地方：
- 易於攜帶：只需記住私鑰
- 易於驗證：任何人都可以驗證
- 可分割：可以只買 0.00000001 BTC
- 可程式化：可以設定條件
```

{% hint style="info" %}
**Bitcoin vs 黃金**

截至 2025 年，Bitcoin 市值約占黃金的 10-15%。有些人認為 Bitcoin 最終會達到或超過黃金的市值。
{% endhint %}

## 重大里程碑

### 歷史時間線

| 年份 | 事件 |

| 2010 | 首次用 BTC 買披薩 |

| 2021 | 突破 $60,000 |
| 2024 | Bitcoin ETF 獲批准 |

### 減半事件

```
Bitcoin 減半（Halving）：

每 210,000 區塊（約 4 年），挖礦獎勵減半

歷史：
- 2012：50 → 25 BTC
- 2016：25 → 12.5 BTC
- 2020：12.5 → 6.25 BTC
- 2024：6.25 → 3.125 BTC

影響：
- 新增供應減少
- 歷史上價格在減半後上漲
- 礦工收入減少
```

## 技術特點

### 基本參數

| 參數 | 數值 |
|------|------|
| 區塊時間 | 約 10 分鐘 |

## 爭議與挑戰

### 主要爭議

| 爭議 | 說明 |
|------|------|
| 能源消耗 | 挖礦耗電量大 |
| 速度慢 | TPS 遠低於 Visa |
| 波動性 | 價格劇烈波動 |
| 監管壓力 | 各國政策不一 |

### 能源爭議

```
Bitcoin 能源爭議：

批評：
- 年耗電量約等於一個小國
- 碳排放問題
- 資源浪費

回應：
- 越來越多使用可再生能源
- 能源成本是安全性的來源
- 許多傳統金融系統也耗能大
```

{% hint style="warning" %}
**投資風險**

Bitcoin 價格波動劇烈，曾多次跌幅超過 80%。只投資你願意歸零的金額。
{% endhint %}
