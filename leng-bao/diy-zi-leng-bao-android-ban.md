# MEW Connect（iOS, Android）

### MEW 是老牌的以太坊錢包，MEW Connect 則是它的手機版。雖然不是真正的「冷錢包」，但搭配正確的使用方式，可以大幅提升安全性。它是進入以太坊世界的便捷入口。

## 核心功能

### MEW wallet 特色

| 功能 | 說明 |
|------|------|
| 建立錢包 | 在手機上產生私鑰 |
| 掃碼連接 | 掃 QR Code 連接網頁版 |
| 簽署交易 | 在手機上確認 |
| NFT 支援 | 查看 NFT 收藏 |
| Swap | 代幣兌換 |

### 安全機制

```
MEW wallet 的安全設計：

私鑰管理：
- 私鑰存在手機本地
- 不上傳到伺服器
- 使用時需解鎖

連接方式：
- 掃描 QR Code 配對
- 交易需手機確認
- 類似硬體錢包的流程

與真正冷錢包的差異：
- 手機仍然連網
- 沒有獨立安全晶片
- 但比純軟體錢包更安全
```

{% hint style="info" %}
**定位說明**

MEW wallet 不是真正的冷錢包，但它提供了「二因素確認」的機制：電腦發起交易，手機確認簽署。這比單一裝置的軟體錢包更安全。
{% endhint %}

## 進階：離線使用

### 如何提升安全性

```
將 MEW wallet 當冷錢包用：

方法：
1. 在舊手機上安裝 MEW wallet
2. 建立錢包、備份種子詞
3. 開啟飛航模式
4. 之後只用來簽署交易

流程：
- 在電腦上準備交易
- 顯示 QR Code
- 離線手機掃描
- 手機簽署後顯示 QR Code
- 電腦掃描廣播

注意：
- 比正常使用更麻煩
- 但安全性接近硬體錢包
- 適合進階用戶
```

{% hint style="warning" %}
**這不是標準用法**

上述離線使用方式需要額外設備和步驟。對大多數人來說，直接使用 MEW wallet 已經提供足夠的安全性。如果需要更高安全性，建議直接購買硬體錢包。
{% endhint %}
