# 冷錢包

### 熱錢包像是放在口袋裡的現金，方便但危險。冷錢包則像銀行金庫——離線保存，駭客摸不到。如果你的加密資產值得保護，你需要一個冷錢包。

---

## 什麼是冷錢包？

| 概念 | 說明 |
|------|------|
| 定義 | 私鑰離線保存的錢包 |
| 別名 | 硬體錢包、離線錢包 |
| 特色 | 私鑰永不接觸網路 |
| 目的 | 最大化安全性 |

---

## 冷錢包 vs 熱錢包

### 關鍵差異

```
冷熱錢包比較：

熱錢包（Hot Wallet）：
- 私鑰存在連網裝置
- 例：MetaMask、交易所錢包
- 優：方便、快速
- 缺：有被駭風險

冷錢包（Cold Wallet）：
- 私鑰存在離線裝置
- 例：Ledger、Trezor
- 優：極度安全
- 缺：使用較不方便
```

### 使用場景

| 場景 | 建議 |
|------|------|
| 小額日常 | 熱錢包 |
| 大額持有 | 冷錢包 |
| 頻繁交易 | 熱錢包 |
| 長期儲存 | 冷錢包 |

{% hint style="info" %}
**80/20 法則**

許多人會把 80% 的資產放在冷錢包，20% 放在熱錢包用於日常操作。這樣兼顧安全與便利。
{% endhint %}

## 冷錢包如何運作

### 核心原理

```
冷錢包的安全機制：

1. 私鑰永不離開裝置
   - 私鑰在裝置內生成
   - 簽署交易在裝置內完成
   - 電腦/手機只看得到公鑰

2. 交易流程
   - 電腦組成未簽署交易
   - 傳送到冷錢包
   - 在冷錢包螢幕確認
   - 按實體按鈕簽署
   - 已簽署交易傳回電腦
   - 電腦廣播到區塊鏈

3. 為何安全
   - 即使電腦被駭
   - 駭客也拿不到私鑰
   - 因為私鑰從未離開冷錢包
```

### 安全晶片

| 術語 | 說明 |
|------|------|
| Secure Element | 專用安全晶片 |
| CC EAL5+ | 安全認證等級 |
| 防篡改 | 物理攻擊也無法提取私鑰 |

## 使用注意事項

### 購買安全

| 注意 | 說明 |
|------|------|
| 官方購買 | 只從官網或授權經銷商 |
| 檢查包裝 | 確認無拆封痕跡 |
| 勿買二手 | 可能被動過手腳 |
| 自己設種子 | 絕不使用預設的種子詞 |

### 種子詞保管

```
種子詞（Seed Phrase）保管原則：

絕對不要：
✗ 拍照存手機
✗ 存在電腦/雲端
✗ 截圖
✗ 用打字的
✗ 告訴任何人

應該這樣做：
✓ 用紙手寫
✓ 多份備份
✓ 存放不同地點
✓ 考慮防火防水
✓ 考慮金屬備份板
```

{% hint style="danger" %}
**種子詞 = 所有資產**

任何拿到你種子詞的人，都可以完全控制你的資產。即使冷錢包壞了、丟了，只要有種子詞就能恢復。但如果種子詞洩漏，資產就沒了。
{% endhint %}

## 常見迷思

### 迷思澄清

| 迷思 | 事實 |
|------|------|
| 冷錢包壞了錢就沒了 | 有種子詞就能恢復 |
| 冷錢包存著幣 | 幣在鏈上，錢包只存私鑰 |
| 冷錢包 100% 安全 | 還是要防物理攻擊和釣魚 |
| 一個錢包一個地址 | 一個錢包可產生無限地址 |
