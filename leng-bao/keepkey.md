# KeepKey

### KeepKey 的特色是那塊超大螢幕——在確認交易時，你可以清楚看到每個細節。它曾被 ShapeShift 收購，整合了便捷的幣幣交換功能，但這段歷史也讓它經歷了起伏。

---

## 產品概覽

| 項目 | 說明 |
|------|------|
| 創立 | 2015 年 |
| 總部 | 美國 |
| 收購 | 2017 年被 ShapeShift 收購 |
| 特色 | 大螢幕、整合交易 |

---

## 核心特色

### 大螢幕設計

```
KeepKey 的螢幕優勢：

螢幕規格：
- 3.12 吋 OLED
- 256 x 64 解析度
- 比 Ledger/Trezor 大很多

實際好處：
- 完整顯示收款地址
- 不需要捲動就能看全
- 確認交易更放心
- 老花眼友善
```

### 與 ShapeShift 整合

| 功能 | 說明 |
|------|------|
| 幣幣交換 | 直接在錢包內換幣 |
| 無需帳戶 | 去中心化交換 |
| 免 KYC | 不需身份驗證 |

{% hint style="info" %}
**ShapeShift 的轉變**

ShapeShift 原本是中心化交易所，後來轉型為去中心化平台。KeepKey 的整合也跟著演進。
{% endhint %}

## 使用體驗

### 設定流程

```
KeepKey 初始化：

1. 下載 ShapeShift 應用
   - 桌面版或網頁版

2. 連接 KeepKey
   - USB 連接電腦

3. 初始化裝置
   - 建立新錢包
   - 產生種子詞

4. 備份種子詞
   - 12 字或 24 字
   - 手寫記錄

5. 設定 PIN
   - 裝置螢幕顯示亂序數字
   - 在電腦上點選對應位置
```

### 日常使用

| 操作 | 體驗 |
|------|------|
| 查看餘額 | 透過 ShapeShift 介面 |
| 發送交易 | 大螢幕確認細節 |
| 接收 | 顯示完整地址 |
| 換幣 | 內建 DEX 功能 |

## 優缺點分析

### 優點

| 優點 | 說明 |
|------|------|
| 超大螢幕 | 確認交易最清楚 |
| 設計質感 | 鋁合金外殼 |
| 開源 | 可審計程式碼 |
| 整合 ShapeShift | 便捷換幣 |
| 價格合理 | 中等價位 |

### 缺點

| 缺點 | 說明 |
|------|------|
| 幣種較少 | 不如 Ledger 多 |
| 體積大 | 不好攜帶 |
| 更新較慢 | 新幣支援慢 |
| 市佔低 | 社群較小 |
| 無 Secure Element | 物理攻擊風險 |

{% hint style="warning" %}
**市場定位**

KeepKey 在 Ledger 和 Trezor 的夾擊下，市佔率較小。這意味著社群資源和第三方支援可能較少。
{% endhint %}
