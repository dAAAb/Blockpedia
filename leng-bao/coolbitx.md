# CoolBitX

### 在一片歐美品牌中，CoolBitX 是台灣之光。它的 CoolWallet 薄如信用卡，可以放進錢包隨身攜帶。這是全球第一款行動式硬體錢包，展現了台灣的創新能力。

## 核心特色

### 卡片式設計

```
CoolWallet 的獨特設計：

外形規格：
- 尺寸：85.6 x 54 x 0.8 mm
- 重量：6g
- 外觀：如同信用卡
- 材質：不鏽鋼、防水防塵

實際優勢：
- 放進實體錢包
- 隨身攜帶方便
- 不怕忘記帶
- 耐用度高

對比傳統冷錢包：
- Ledger/Trezor 像 USB 隨身碟
- CoolWallet 像信用卡
- 便攜性完全不同等級
```

### 藍牙連接

| 特色 | 說明 |
|------|------|
| 無線連接 | 透過藍牙與手機配對 |
| App 操作 | iOS/Android 專屬 App |
| 加密傳輸 | 藍牙通訊加密 |
| 行動優先 | 專為手機使用設計 |

{% hint style="info" %}
**行動錢包先驅**

CoolWallet 是最早專注「手機搭配」的硬體錢包。在 Ledger 還只能用 USB 連電腦時，CoolWallet 就已經能用手機操作了。
{% endhint %}

## 安全設計

### 硬體安全

```
CoolWallet 的安全架構：

硬體層面：
- CC EAL5+ 安全晶片
- 與銀行卡同級
- 私鑰永不離開晶片

軟體層面：
- 藍牙 AES-256 加密
- 生物辨識支援
- 多重驗證機制

實體安全：
- 防水防塵
- 耐彎折
- 不怕日常使用損耗
```

### 驗證機制

| 機制 | 說明 |
|------|------|
| 實體按鈕 | 卡片上有確認按鈕 |
| 2+1 驗證 | 手機 + 卡片 + 按鈕 |
| E-ink 顯示 | 電子墨水顯示地址 |

## 使用體驗

### 初始設定

```
CoolWallet 設定步驟：

1. 下載 CoolWallet App
   - iOS 或 Android

2. 藍牙配對
   - 按下卡片按鈕開機
   - App 自動搜尋配對

3. 建立錢包
   - 產生種子詞
   - 手寫記錄

4. 設定密碼
   - App 登入密碼
   - 可設生物辨識

5. 開始使用
   - 新增幣種
   - 接收/發送
```

### 支援幣種

| 類型 | 支援 |
|------|------|

| ETH | ✓ |
| ERC-20 代幣 | 多數支援 |
| BNB Chain | ✓ |

## 優缺點分析

### 優點

| 優點 | 說明 |
|------|------|
| 極致便攜 | 卡片式，放錢包 |
| 台灣品牌 | 本地支援、中文介面 |
| 行動優先 | 手機操作體驗好 |
| 防水防塵 | 耐用度高 |
| 安全晶片 | CC EAL5+ |

### 缺點

| 缺點 | 說明 |
|------|------|
| 幣種較少 | 不如 Ledger 多 |
| 依賴手機 | 沒有桌面版 |
| 小螢幕 | 顯示資訊有限 |
| 社群較小 | 國際知名度低 |

{% hint style="warning" %}
**藍牙安全考量**

藍牙連接理論上增加了攻擊面，但 CoolWallet 使用 AES-256 加密和多重驗證。只要正常使用，安全性是足夠的。
{% endhint %}
