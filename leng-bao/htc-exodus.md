# HTC Exodus

### 當手機大廠 HTC 宣布做「區塊鏈手機」時，很多人都驚訝了。Exodus 是一台真正的智慧型手機，但內建了硬體級的安全晶片，讓你的手機就是冷錢包。這是一個大膽的嘗試。

## 核心概念

### 什麼是區塊鏈手機

```
Exodus 的設計理念：

傳統方式：
- 手機裝 App 錢包（熱錢包）
- 私鑰存在手機系統中
- 系統被駭 = 私鑰洩漏

Exodus 方式：
- 內建獨立安全晶片（Zion）
- 私鑰存在安全晶片中
- 即使 Android 被駭，私鑰也安全

就像是：
- 傳統手機 + 熱錢包 App
- Exodus = 智慧手機 + 內建冷錢包
```

## Zion 安全晶片

### 硬體架構

| 特性 | 說明 |
|------|------|
| 獨立晶片 | 與主系統隔離 |
| ARM TrustZone | 安全執行環境 |
| 私鑰保護 | 永不離開晶片 |
| 生物辨識 | 指紋驗證 |

### 安全設計

```
Zion 的安全機制：

硬體隔離：
- Zion 晶片獨立運作
- 與 Android 系統完全隔離
- 即使系統被 root 也無法存取

交易簽署：
- 私鑰在晶片內
- 簽署在晶片內完成
- Android 只看到已簽署交易

備份機制：
- 社交密鑰恢復（Social Key Recovery）
- 將種子詞分給信任的朋友
- 類似 Shamir 備份概念
```

{% hint style="info" %}
**社交密鑰恢復**

Exodus 的創新功能：將種子詞拆分成多份，分給你信任的朋友保管。需要恢復時，從朋友那裡收集足夠的份數。這是區塊鏈手機獨有的功能。
{% endhint %}

## Exodus 1s 特色

### 比特幣全節點

| 功能 | 說明 |
|------|------|
| 全節點 | 可在手機上運行 BTC 全節點 |
| 驗證交易 | 自己驗證，不依賴第三方 |
| 儲存需求 | 需要 400GB+ SD 卡 |
| 同步時間 | 需要數天至數週 |

```
為什麼要跑全節點：

傳統方式：
- 手機錢包連接遠端節點
- 依賴第三方提供交易資料
- 隱私較差

全節點方式：
- 自己驗證所有交易
- 不需要信任第三方
- 最高等級的去中心化
- 但需要大量儲存空間
```

## 安全性討論

### 與傳統冷錢包比較

| 比較 | Exodus | 傳統冷錢包 |
|------|--------|------------|
| 連網 | 手機連網 | 完全離線 |
| 攻擊面 | 較大 | 最小 |
| 便利性 | 最高 | 較低 |
| 私鑰隔離 | 晶片隔離 | 完全隔離 |

```
Exodus 的安全定位：

比熱錢包安全：
- 私鑰在獨立晶片
- 即使 Android 被駭也安全

但比傳統冷錢包差：
- 手機始終連網
- 攻擊面較大
- 可能有未知漏洞

適合對象：
- 重視便利性
- 中等金額
- 不想帶多個裝置
```
