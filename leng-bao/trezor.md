# Trezor

### 2014 年，Trezor 開創了硬體錢包這個品類。它不只是「第一個」，更是「完全開源」的代表。對於相信「Don't trust, verify」的密碼龐克來說，Trezor 是信仰的選擇。

## 核心特色

### 完全開源

```
Trezor 的開源哲學：

開源範圍：
- 硬體設計圖 → 公開
- 韌體程式碼 → 公開
- 桌面軟體 → 公開
- Bootloader → 公開

為什麼重要：
- 任何人都可以審計程式碼
- 社群可以發現漏洞
- 沒有後門的可能
- 真正的「Don't trust, verify」

與 Ledger 的差異：
- Ledger 的韌體部分閉源
- Trezor 完全透明
- 這是很多人選擇 Trezor 的原因
```

### 安全設計

| 特性 | Model One/T | Safe 3/5 |
|------|-------------|----------|
| 開源 | 完全 | 完全 |
| Secure Element | 無 | 有 |
| 觸控螢幕 | T 有 | 5 有 |
| Shamir 備份 | 支援 | 支援 |

{% hint style="info" %}
**Secure Element 的取捨**

早期 Trezor 刻意不用 Secure Element，因為這類晶片通常不開源。Safe 3/5 採用了開源 Secure Element，兼顧安全與透明。
{% endhint %}

## 特色功能

### Shamir 備份（SLIP39）

```
Shamir 備份是什麼：

傳統備份：
- 一組 24 字種子詞
- 全有或全無
- 誰拿到都能用

Shamir 備份：
- 將種子詞分成多份
- 例：分成 5 份，需要 3 份才能恢復
- 單獨一份沒有用

使用場景：
- 分散存放在不同地點
- 給不同家人保管
- 即使一份被盜也沒事
- 但需要足夠份數才能恢復
```

### Passphrase（第 25 字）

| 功能 | 說明 |
|------|------|
| 作用 | 在種子詞外加上密碼 |
| 效果 | 產生完全不同的錢包 |
| 隱藏錢包 | 可以有多個錢包 |
| 偽裝 | 被迫時可交出空錢包 |

## 使用流程

### 初始設定

```
Trezor 設定步驟：

1. 下載 Trezor Suite
   - 官方管理軟體
   - 桌面版最安全

2. 連接裝置
   - USB 連接電腦
   - 按照畫面指示

3. 初始化
   - 選擇建立新錢包
   - 選擇備份方式
     - 標準 24 字
     - 或 Shamir 備份

4. 記錄種子詞
   - 手寫在紙上
   - 絕不拍照或存電子檔

5. 設定 PIN
   - 用於解鎖裝置
   - 錯誤多次會鎖定
```

### Trezor Suite

| 發送/接收 | 管理交易 |
| 兌換 | 內建交易功能 |
| Tor 支援 | 隱私保護 |

## 安全紀錄

### 歷史漏洞

```
Trezor 安全事件：

已修復漏洞：
- 2017：韌體漏洞（已修復）
- 2019：物理提取攻擊（需要物理接觸）
- 2020：側通道攻擊研究（需要專業設備）

重要觀點：
- 所有已知漏洞都已修復
- 大多數攻擊需要物理接觸裝置
- 開源讓漏洞更快被發現和修復
- 沒有「完美」的安全，只有更好的安全
```

### 如何保護

| 威脅 | 防護 |
|------|------|
| 遠端攻擊 | 私鑰離線，免疫 |
| 物理盜竊 | PIN 碼保護 |
| 暴力破解 | 錯誤次數限制 |
| $5 扳手攻擊 | Passphrase 隱藏錢包 |

{% hint style="warning" %}
**$5 扳手攻擊**

最難防的攻擊是有人拿著扳手威脅你交出密碼。Passphrase 功能讓你可以有一個「誘餌錢包」，被迫時交出，真正資產在另一個隱藏錢包。
{% endhint %}

## 與 Ledger 比較

| 比較 | Trezor | Ledger |
|------|--------|--------|
| 開源 | 完全開源 | 部分開源 |
| 歷史 | 2014（更早）| 2016 |
| 幣種數 | 較少 | 最多 |
| 藍牙 | 無 | Nano X 有 |
| 價格 | 略低 | 略高 |
| 信任模式 | 可驗證 | 需信任 |

### 如何選擇

```
Trezor vs Ledger：

選 Trezor 如果：
- 重視開源、可驗證
- 信奉「Don't trust, verify」
- 不需要藍牙
- 偏好開源社群

選 Ledger 如果：
- 需要支援冷門幣種
- 要藍牙行動使用
- 偏好主流品牌
- 不介意部分閉源
```
