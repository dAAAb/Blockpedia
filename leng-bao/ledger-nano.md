## 使用流程

### 初始設定

```
Ledger 設定步驟：

1. 開箱檢查
   - 確認包裝完整
   - 確認是全新未拆封
   - 絕不使用預設種子詞

2. 下載 Ledger Live
   - 官方管理軟體
   - 支援桌面/手機

3. 初始化裝置
   - 設定 PIN 碼
   - 產生 24 字種子詞
   - 手寫記錄種子詞

4. 驗證種子詞
   - 裝置會要求確認
   - 確保記錄正確

5. 安裝應用程式
   - 依需求安裝幣種 App
   - 例：Bitcoin、Ethereum
```

### 交易流程

| 步驟 | 說明 |
|------|------|
| 1. 在 Ledger Live 發起交易 | 輸入收款地址和金額 |
| 2. 裝置顯示交易詳情 | 確認地址和金額正確 |
| 3. 按下兩個按鈕確認 | 在裝置上簽署 |
| 4. 交易廣播 | Ledger Live 將已簽署交易發送 |

## 2023 年爭議

### Ledger Recover 事件

```
事件經過：

背景：
- 2023 年 5 月 Ledger 推出 Recover 服務
- 目的：幫用戶備份種子詞

機制：
- 將種子詞拆分成三份
- 分別存於三個託管商
- 用戶可透過身份驗證恢復

爭議：
- 社群強烈反彈
- 「私鑰怎麼可能離開裝置？」
- 質疑這違背冷錢包初衷
- 擔心政府或駭客可取得私鑰

Ledger 回應：
- Recover 是可選服務
- 不啟用就不會上傳任何資料
- 程式碼會開源供審計
```

### 影響與後續

| 影響 | 說明 |
|------|------|
| 信任危機 | 部分用戶轉投其他品牌 |
| 開源承諾 | Ledger 承諾開源更多程式碼 |
| 行業討論 | 引發對硬體錢包安全性的廣泛討論 |

{% hint style="warning" %}
**風險評估**

如果你不使用 Recover 服務，你的 Ledger 仍然是安全的。但這個事件確實顯示了即使是領先品牌，也可能做出有爭議的決定。
{% endhint %}
